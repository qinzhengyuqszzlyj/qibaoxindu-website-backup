保护幼儿体验和探究新食物的权利——记闵行区学前生活中心组研讨活动

发布时间 : 2023-11-03   作者（来源）: 张静   阅读 : 218

3/8
←
→

2023年10月26日，闵行区学前生活中心组在闵行区康城幼儿园开展了研讨活动。活动分为观摩现场活动和食育微课程分享两部分。

观摩现场活动  追随幼儿的经验

  首先，闵行区康城幼儿园的金佳老师执教了活动《水果变变变》，从幼儿问题出发，将幼儿的个体经验以分享交流的形式转化到集体中，帮助幼儿提升和梳理对水果的认知及生活经验。活动前，金老师准备充分，结合幼儿的已有经验，从幼儿的角度切入，寻找活动的关键点，并在组织和开展活动中，追随幼儿，发挥了幼儿的主体性，在“以幼儿发展优先”的理念下，不断激励幼儿自主探索、大胆表达。整个活动将幼儿认

知与饮食习惯等相融合，体现了食育课程的意义。

课程故事分享  发现孩子的力量

随后，金佳老师针对本次活动进行了现场的反思和课程故事的分享。课程故事的缘起是幼儿在生活点心环节中出现的种种“小问题”“小困惑”——水果的皮可以吃吗？金老师抓住这一契机点，开展了食育课程的实施，从分享中看到金老师前期对于个性化发展的幼儿年龄特点的把握和不同发展幼儿需求都有自己的思考和研究。在观察幼儿、了解幼儿的基础上，进一步加强了幼儿对于水果吃什么部位的认识。

在研讨中，老师们各抒已见，从幼儿自主选择水果的皮是不是要吃；从水果分类的角度出发，让幼儿进行大胆地探索及分类；延伸到果皮的价值和饮食安全性等等的观点，大家在研讨中对现场活动有了后续更深层次地剖析和实施方向的建议。

接着，来自闵行区第一幼儿园的李兰芬老师分享了食育课程故事《饭泛而谈》，从幼儿日常生活中熟悉的食物——米饭出发，从白米饭到各种不同的杂粮饭，观察孩子的喜好、口感，记录一周的米饭，从孩子的提问中帮助他们梳理经验，并拓展了不同国家的不同米饭制品，给孩子更多关于主食的认知和经验，激发他们爱上米饭。同时，食思到如何能更好的衔接到孩子对于饮食的重要性认识，如何激发孩子们的探知能力等，是十分接地气和贴近孩子的课程。

梅锦幼儿园的薛飞老师分享了两个课程故事《食与生命》和《暑相“莲”》,第一个课程故事从幼儿饲养小兔子出发，在饲养过程中思考小动物爱吃什么、能吃什么、不能吃什么，拓展了食物与生命的联系性，也进一步帮助孩子知道正确饮食的重要性。第二个课程故事以小暑节气为切入点，小暑吃什么，也成为孩子们讨论的话题，《暑相“莲”》课程，让孩子知道小暑节气有莲子、有莲藕等，这些与节气相关联的食物。拓展了幼儿对人与自然关系的认识，挖掘了幼儿的潜能，促进幼儿的个性发展，同时，也培养了幼儿求知精神和实践精神，锻炼了幼儿的独立意识。  

古美中心幼儿园的沈丽洁老师分享的课程故事《与你有“圆”》，从托班幼儿在日常生活中发现的圆圆的番茄为切入点，延伸到寻找生活中“圆圆的东西”，将课程融入日常活动、身边环境等，以和“园”的游戏贯穿一日活动，让幼儿寻找身边的“圆圆”，探索哪些东西是圆圆的，从而引发到圆圆的食物，充分让幼儿在实践中探索和感知体验。

梳理提升经验  食育故事在继续

最后，张静老师对整个活动进行了总结与梳理，提出了对后续课程研究的期望，帮助中心组成员进一步明确了研究重点和方向：多观察分析孩子的经验，课程要往深处想往浅处做，明确活动对于真正推进食育的价值；给幼儿更多机会，了解食物本身的样子，进一步挖掘开展食育课程的价值；要给幼儿更多的实践体验，动手操作的机会，在认知基础上不断提升经验；寻找符合幼儿的课程，尊重幼儿的自主选择，但教师也要把握好方向，给到幼儿积极、正面的引导。

虽然本次活动结束了，但是我们的食育故事还在继续，我们也将从新的思路出发，带着新的思考和不断地创新，去进一步观察幼儿、了解幼儿、剖析幼儿，在课程实施中支持他们，给他们更多“食”的体验，“食”的认知，积累“食”的经验，让“食”更健康、更开心。


撰稿 :  康城幼儿园 金佳
