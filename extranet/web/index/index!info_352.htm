躬身实践 示范引领 ——闵行区学前种子基地活动

发布时间 : 2023-11-17   作者（来源）: 顾莹   阅读 : 437

2023年11月14日上午，顾莹老师带领学习中心组（社会）和种子基地教师在闵行区梅陇金都幼儿园开展集体教学实践活动。顾莹老师亲自执教展示了大班科学活动《博物馆奇案——名画失窃》。

在顾老师的带领下，孩子们一步步寻找线索，观察墙面上遗留下的布料花色、监控时间段，不断剖析作案时间点和人物特征。最后通过辨认指纹的纹路，确定了博物馆中名画失窃案的主谋。

顾老师教学风格灵动，尊重和关注用各种方法助力幼儿发展。在活动中提供了幼儿独立思考的机会，在提问和追问中幼儿建立人物与逻辑间的链接。同时，顾老师倾听幼儿对于线索的理解和分析，观察与识别幼儿的能力发展水平，及时给予回应，有效的促进了幼儿逻辑思维能力的发展。

活动后，顾莹老师对活动设计与组织进行了分享，并呼吁种子老师们要关注从过去的“以科学知识为中心的科学”转变为今天的“以科学探究为中心的科学”的观念。只有老师从外在的、客观的科学知识体系转向幼儿生动、丰富的科学探究，才能让幼儿体验和发现科探活动的奥妙之处。结合基地目前研究的“童蒙养正 立德树人”主题，顾老师鼓励大家多关注幼儿的态度教育和德育教育，从态度入手培养幼儿的关键能力。

种子基地老师被顾老师生动和深度的教育教学风貌所感动。顾老师鼓励青年教师在饱含职业热忱的同时不断精进自己的教育教学水平，不忘初心，砥砺前行。
