基于新课标，探索新课堂----第17周小学体育区级教研活动

发布时间 : 2023-06-09   作者（来源）: 顾佳伟   阅读 : 719

3/8
←
→

6月7日下午，以“基于新课标 探索新课堂”为主题的闵行区体育教研活动在闵行区七宝明强第二小学举行。出席本次活动的有上海市体育教研员王立新老师，上海市特级教师、正高级教师冯敏老师，徐汇区体育教研员陈善军老师，青浦区体育教研员张衡老师，闵行区体育教研员顾佳伟老师，闵行区七宝明强第二小学郭芳校长、陈骁炯副校长等。青浦区小学体育骨干团队、闵行区体育学科中心组、闵行区小学体育名师工作室及中北片体育教师共300余人，闵行区体育教研员顾佳伟老师主持本次活动。

活动伊始，七宝明强第二小学校长郭芳致辞并介绍了学校“智慧蝴蝶”项目化课程、“我爱欢校”数字化校园平台、学校“蝶文化”融合体育教学工作三大特色内容以及学校在体育学科领域取得的一些荣誉和成绩，为本次活动拉开了序幕。

课堂展示环节中，首先明强二小体育组老师们通过组内设计、磨课、研讨等情景再现，向大家展示了微型教研活动《体育跨学科主题学习教学设计和实施》。其后，吴计风老师执教一堂二年级《立定跳远》跨学科体育课。吴老师设计了“小青蛙勇闯数学岛”主题情景，设置了跳过荷塘-跳跃河流-密宝争夺赛等闯关游戏，将数学学科中的直角、锐角、度量、测算、加减乘除融入立定跳远教学中，有效解决教学重难点，全面激发学时兴趣，充分挖掘学生潜能，彰显育人价值。随后，姜晓磊老师执教一堂二年级《模仿动物爬行》跨学科体育课。姜老师创设生动有趣的“动物与自然”主题情境，立足体育学科核心素养，融合自然学科中动物的习性、栖息地、声音、脚印等于课中，发挥学科融合功能，在新颖的教学场地和信息技术赋能中，让学生综合运用知识来解决现实情境中的真实问题，让身体练习与思维活动产生紧密联系。

专家点评环节中，徐汇区体育教研员陈善军老师充分肯定了吴计风老师的课，尤其是在跨学科主题学习时是基于学生实际基础，融入数学知识，丰富课堂学练，有效解决课的关键问题。青浦区体育教研员张衡老师对姜晓磊老师的课也赞不绝口，表示信息技术赋能教学，用贴近生活的教学设计认识动物与自然的关系，了解动物特性和习惯，打破学科内容的限制，彰显跨学科学习内涵。

同时，上海市特级教师，正高级教师冯敏老师指出，我们在跨学科主题学习设计时一定要站在体育学科立场上，设置五育并举的学习主题，学习目的要和学练手段相符合，利用学生已有的其他学科知识助力于体育课技能学练。

最后，上海市体育教研员王立新老师作领导讲话。他高度肯定了明强二小体育教学工作是富有成效的，并向大家提出三个“新”，即新课标：学习研究新课标，体育学科核心素养是关键；新课堂：打破学科界限，将不同学科内容有机融合，体现教学方式创新；新发展：五育并举，技术赋能课堂，努力提升学生体质健康。同时希望明强二小要不断努力，继续在体育教学中与时俱进、聚焦时代脉络，丰富时代新质。

本次区级教研活动成功举行，反映了区域和学校对体育教育工作的深入理解和扎实做法，凸显出体育教学不仅指向动作技能的形成，更指向育人、指向学生核心素养培育。
