探索高质量办学的创新路径与家校社协同育人机制建设——2024年闵行区教育系统优秀年轻干部第四次专题培训

发布时间 : 2024-09-27   作者（来源）: 朱凤   阅读 : 316

3/8
←
→

9月25日，2024年闵行区教育系统优秀年轻干部第四次专题培训在闵行区教育学院举行。闵行区教育学院党总支书记朱靖、副书记院长林唯、导师仇忠海校长等，近20位领导、专家出席活动。

朱靖作领导讲话。她肯定学员们在探索家校社协同育人机制建设中的实践智慧，呈现了作为一名后备干部在学校教育改革中“开路先锋、事业闯将”的作用。并提出了三点要求，第一，加强学习，把准方向，领会国家教育部署。正确领会党的二十届三中全会精神，深入学习贯彻习近平总书记关于教育的重要论述，弘扬教育家精神，为党育人，为国育才；第二，引领学校教育创新发展提升履职能力。敢于在教育改革发展的关键事、关键时中迎难而上，攻坚克难、闯关夺隘，在教育改革实践中提升教育领导力；第三，保持敬畏，做忠诚干净的好干部。以德立身，以德立学，以德施教，做一个一心为公、一身正气的好干部。

本次培训分为专题报告和学员汇报两部分。

上海市实验西校管理团队分享了学校高质量办学的创新之路。章志强校长、王静书记、王爱华副校长和林建刚副校长分别作了《“一体两翼”教学改革推动学校高质量发展》、《“五育融合”理念下大思政课程的创新实践》、《多元课程与教学创新，助力教学品质全面提升》、《创建开放多元互利的食堂治理体系》主题报告，通过价值引领，机制建设、实践探索，实现学校高质量发展，为学员们提供了一次学习转化实践的典范。

学员们聚焦“教育高质量发展背景下的家校社协同育人机制建设”这一主题，展开分组汇报。专家们根据学员的主题汇报内容和临场表现进行综合评估，经过推荐，七宝中心幼儿园田闻文、中福会浦江幼儿园陈劲草、交大附属闵实验冯志兰、罗阳小学彭琼、华坪小学陆洲、马桥复旦万科谈珺嫣、七宝中学金亚伦、上外高中唐晔、七宝实中蔡媛媛、文琦中学杨丽君10位学员代表各自小组进行集中汇报，上外高中唐晔主持集中汇报。学员们深刻解读了家校社之间的相互关系以及建设三者协同育人机制的价值意义。基于校情，通过课程、项目等载体，分享了家校社协同育人机制建设智慧。

专家们对学员们的育人理念和汇报表现给予高度评价并建议：构建家校社协同育人机制要体现专业性、针对性和时效性，学员需充分解读主题，把握重点，定位准确；汇报内容立足岗位，由点及面，展现先进教育思想与独到见解。同时鼓励学员领悟教育家精神，热爱教育，不畏困难，增强信心，为闵行教育高质量发展注入强大动力。
