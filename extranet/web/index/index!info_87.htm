自查互检，提升学校语言文字工作水平 ——闵行区语委办开展语言文字规范化达标建设“回头看”活动

发布时间 : 2023-10-07   作者（来源）: 倪宏   阅读 : 261

3/8
←
→

根据《上海市语言文字工作委员会办公室关于开展2023年学校语言文字工作达标建设“回头看”抽查工作的通知》的文件要求，为推进闵行区学校语言文字工作高质量开展，闵行区语委办组织专家开展2023年学校语言文字工作达标建设“回头看”区级抽查工作。

2023年9月22日，闵行区语委办倪宏老师带领市语委办、兄弟区语委办、区教育局、区教育学院、区兄弟学校相关领导、专家莅临闵行区君莲幼儿园，开展集中抽查工作。本次接受检查的学校涉及中小幼各学段，分别为：七宝中学、文来中学、莘松中学、华坪小学、七宝实小和君莲幼儿园。

首先，各校（园）长从语言文字工作“制度建设”“能力建设”“教育教学”“宣传普及”等四个板块进行了本校（园）语言文字工作汇报。各学校（园）结合校本特色，既呈现了语言文字工作的规范性，也凸显出各学段语言文字工作的不同特色。

汇报结束后，专家组查阅了各校（园）语言文字工作档案材料；对君莲幼儿园进行了校园语言文字环境巡查与随机访谈，实地查看和了解了幼儿园内的用字用语情况。

互动交流环节中，领导和专家们肯定了各校（园）对本次语言文字工作达标建设“回头看”工作的重视，并建议各校（园）以本次“回头看”抽查工作为契机，继续扎实、有序地开展语言文字规范化工作，将语言文字和办学特色进一步紧密结合。同时，建议各校（园）加强对上级最新文件、讲座的学习，领悟最新的工作精神，希望学校语言文字工作能与时俱进，开创学校（园）语言文字工作新局面。
