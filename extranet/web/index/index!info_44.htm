专家引领明方向 切思且行促提升 ——2023学年第一学期闵行区中学两类课程学期工作会议

发布时间 : 2023-09-13   作者（来源）: 马丽敏   阅读 : 410

2023年9月12日下午，闵行区中学两类课程学期工作会议在上海市七宝中学附属闵行金都实验中学召开，全区初高中两类课程负责人、STEM+课程负责人、学生研究性学习指导教师、拓展型课程常任教师代表等参与本次会议。

会议邀请了原虹口区教育学院柳栋老师为全体老师作了题为《综合实践活动的一些思考》的专题讲座。柳老师从认识综合实践活动、研究性学习的由来、研究性学习的透视和综合实践活动实施等四个方面，以理论结合实践的方式详细阐述了相关背景、具体操作方法。深入浅出又幽默风趣的娓娓讲述，精彩纷呈又彰显睿智的答疑解惑，让大家禁不住一次次地鼓掌、喝彩。

随后研训员马丽敏老师对2023年中学生研究性成果评选活动进行了总结：近两年里学生参赛数量不仅逐年增加，而且品质也维持在一定水准，但更希望看到背后学校课程的支持、教师的引导、过程的规范管理，以形成求真务实的学术氛围；随后，就本学期区初高中研究型课程中青年教师教学比赛、学校研究性学习课程负责人及常任教师细聊会、教研员短学期教学实践公开课、智能创造研修坊、追光影视育人教师沙龙、“智慧校园”主题项目学习展评活动等相关活动作了预告；最后还对编制/修订《学校综合实践活动课程实施方案》提出了具体要求，以便学校操作落实。

通过今天的会议，大家明确了本学期的常规工作和重点项目，对后续工作开展达成了共识，希望各校在这一轮课程转型中顺利做好传承与对接，彰显闵行风采，也为更多学生提供有品质的课程、有意义的学习经历！
