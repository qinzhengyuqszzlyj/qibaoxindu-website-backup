重规范 讲服务 党建引领塑造“四有”教师 ——2018年闵行区新任校（园）长带教培训赵瑛群导师组活动通讯（3月）

发布时间 : 2019-03-27   作者（来源）: 曹雅明   阅读 : 2182

为深入贯彻落实习近平新时代中国特色社会主义思想和党的19大精神，推动全面从严治党向纵深发展，进一步严格和规范党内政治生活，强化党员教育管理，充分发挥党支部战斗堡垒作用和党员先锋模范作用。党组织领导下的校长负责制作为一项基本制度已全面推进，闵行区“书记校长‘一肩挑’背景下强化党组织书记角色的实践研究”项目的实践研究也日益深入。

目前，校长、书记一肩挑对本人而言是一大挑战，在支部引领和行政管理的有效融合上，在行政团队和支部组织的建设提升方面尚缺乏经验。因此，新学期开学之后，我向师傅学习的主要任务就聚焦了党建工作如何规范有效的开展。

一、借参观学习，明晰党建规范要求

2月28日，重庆市璧山区25位学校书记来到田园外小考察学习，我也借此机会，在书记赵瑛群、副书记张瀛两位同志的带领下一起对党建工作进行了一次实物培训。 

赵书记开展以“引领 融合 发展”为主题的支部建设研讨活动，赵书记毫无保留地介绍了田园外小党建工作的方方面面，尤其着重介绍了学校在党员“双培养”、主题党日活动开展中的宝贵经验，立志要打造“被党员需要、被师生需要、被百姓需要”的家门口好学校。会上讨论气氛热烈，互动环节中重庆部分书记对今天的参观、研讨也提出了自己的疑问和想法，将两省市的党建交流推向高潮。 

会后，在赵校长的带领下，大家一起来到了田外的党建活动室，对学校党建档案资料的整理和归档工作进行了现场学习，浏览了文件柜中排列整齐的资料夹，查阅了学校日常工作的记录和整理，进一步感受到了田外党建工作的规范和扎实。

通过本次活动，我受到了很大的启发，对如何将党建工作融入学校的日常工作之中，有了更加全面的了解，将帮助我尽快明确书记的岗位职责，在学习借鉴的基础上，基于校情开展切实有效的支部活动，发挥示范引领的作用。

二、借专题培训，推进师德提升工程

2018年，闵行区推出了“师德师风三年提升工程”，2019年闵行区师德师风建设将紧紧围绕“大培训”、“大讨论”、“大研究”、“大检查”、“大选树”五大项工作开展，以全面提升闵行教师师德师风来促进闵行教育质量提升、内涵发展，办好人民满意的教育。但是在学校中究竟如何依托党组织的有力领导，推进学校的师德师风建设，仍需不断探索。

 3月5日下午，闵行区教育系统党支部片组长会议在我校举行，内容涉及教育系统党建的各项工作，我有幸旁听会议，机会弥足珍贵。

会上，朱雪平书记对“闵行区‘守初心 为人师’新时代‘四有’好教师选树活动”做了重点部署，强调了此次评选活动的目的，要进一步强化师德师风建设，营造提振师道尊严的良好氛围，使闵行“守初心 为人师”新时代“四有”好教师选树活动形成一定的社会影响力，全面展现闵行教育工作者争做“四有”好教师的良好形象，推动教师成为先进思想文化的传播者、党执政的坚定支持者、学生健康成长的指导者。朱书记建议各校要以此为契机，开展好校内的选树活动，挖掘教师中的典型，讲好师德典型人物的故事。

为此，我校结合上学期期末进行的“四叶草”教师评选活动，开展了一次“四叶草教师”故事分享会，大家一起畅谈两位四叶草教师的点滴故事，会上大家畅所欲言，讲述了许多发生在日常的小故事，传递了满满的正能量。最终，推选了朱挥副校长参加区级评选，他不仅是一位优秀的英语老师，还是一个爱读绘本故事、会讲故事的老师，无偿捐赠“个人图书馆”，推进学校“520阅读成长计划”的落地实施，成为了校内的师德典范。

虽然学校起步阶段规模较小，但是我们仍旧要高度重视政治建设，勇于创新突破，强化组织领导，把握机遇，坚定信心，谋好篇、开好局、鼓足劲、加满油，争做新时代的追梦人。
