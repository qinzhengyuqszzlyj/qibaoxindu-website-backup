提高实践能力 提升综合素养 ——2019年云南省保山市部分高中校长、教师来闵行学习交流

发布时间 : 2019-09-18   作者（来源）: 朱凤   阅读 : 3572


9月17日， 云南省保山市部分高中校长、物理教师来闵行学习交流开班仪式在区教育学院举行。教育学院副院长董学平以及相关教师、保山市教研所教研员雷积才以及部分高中校长和教师、跟岗学校具体负责人共计70人参加了开班典礼。

教育学院校长发展中心主任俞琳主持，并从培训形式、培训内容、组织纪律等方面解读了本次培训方案。雷积才感谢闵行区教育局和教育学院的精心组织，虽然保山还没有实施高考综合改革，但提前思考与谋划才能实现弯道超车。他希望学员们通过理论和实践学习，拓宽视野，提高实践能力，完善知识结构，发挥示范和引领的作用。

董学平代表教育学院欢迎保山教育同行来闵行学习交流，他分享了闵行教育这几年的改革举措以及取得的成效，尤其信息化建设在上海乃至全国有一定的影响力，同时也介绍了教育学院的功能定位。希望彼此珍惜每一次相互学习和交流的机会，相互借鉴，进一步推进区域教育教学改革。

开班仪式后，教育局普教二科科长马卫锋、金汇高中校长余长辉、教育学院副院长董学平分别做了《新高考改革背景下的闵行高中建设》、《探索有情怀有温度的责任教育》、《闵行区教师队伍建设经验分享》专题报告，围绕高中育人方式改革，从课程建设、队伍建设、生涯教育、特色高中建设等方面进行探讨与交流，揭示了在育人方式变革中，坚持立德树人，积极探索高中学生综合素质评价的新思路和新方法，培养学生的创新精神与实践能力，促进学生全面、可持续、个性化发展。

本次学习交流为期两周，第一周为集中培训，将聘请区内外知名校长、专家等15人，与保山校长及物理教师分享闵行经验和闵行智慧。第二周为跟岗培训，学员们将分别深入七宝中学、闵行中学、上闵外、莘庄中学和华理科高五所高中，进一步学习和体验。把握高中新课程改革和高考综合改革方向，创设适合学生全面、可持续、个性化发展的学校文化和环境，推动高中教育多样化、特色化发展。
