浸润田园教研文化，拓宽学科研究视野 ——2018年闵行区新任校（园）长带教培训赵瑛群导师组活动通讯（12月）

发布时间 : 2018-12-18   作者（来源）: 曹雅明   阅读 : 2384

百年大计，教育为本；教育大计，教师为本。如果说教育改革的蓬勃发展是一股浪潮奔涌向前，那么，学校就是一艘航船，一支主动成长的教师队伍就是这艘航船上奋勇拼搏的桨手。师傅赵瑛群校长在20年的办学历程中，始终坚持“教师优先发展”战略，践行“为了每一个学生终身发展”的核心理念，以教师专业发展工程为引领，走出了一条从外力驱动走向专业自觉的教师发展之路。作为一名校长，既要有自己的学科专业特长，还要有对全学科的整体把握能力，赵校长坚持参加各级各类学科现场展示或研讨活动，甚至在繁忙的事务管理之余，深入学科研究参加磨课活动。我也提醒自己，要让课堂研究成为自己和教师情感的纽带，让学校充满学术的气息。可以说，12月就是一个高频度的各领域、各学科学习月。

一、跨区联合教研，高水准的学科深度互动研修

11月的市教学调研刚结束，12月的“田园·大宁”第九次跨区联合教研便如期而至，共推出了语文、数学、英语、道法四个学科专场的研修活动，各学科市区教研员全程参与。我有幸参与了语文、英语、道法三个学科的活动，共听了5节精彩纷呈的课堂教学，聆听了三场具有深度的互动研讨，感受颇深。作为曾经的田园人，我亲历了之前的八场跨区联合教研，参与过活动的组织和学科的指导，与大宁学校建立了良好的姐妹情谊。但这一次以不同的身份参与活动，给了我不同于以往的心灵冲击，更清晰地看到了田外跨越式发展背后的密码。

与市区教研室长期而紧密的合作，与各位学科专家建立了亦师亦友的感情。田外的学科团队高度重视教师梯队教师和学科课堂研究，研究的步伐始终紧跟市教研室，用最高的标准严格要求自己，如8月底刚推出的“各学科单元设计指南”刚推出，但是田外的单元教学研究早已开始探索实践。学科教师对课堂改进的热情和对学习的渴望，给了田外源源不断的发展动力，田外的造血功能极强，一批批的新教师进入田外，不断搭建适合的历练平台，在很短的时间内就能成长起来。田外的精细化管理还体现在各方面工作的精益求精，哪怕是一次会务的安排，都把细节落实到位。

在田外诸多办学特色培育的过程中，基于课程、立足课堂、面向学生的校本研修是核心策略，从教师教学技能和新基本功锤炼入手，解决教师专业发展的归属感问题，形成职业认同；从专业素养和新研究力培养入手，提升教师主动成长的成就感，形成价值认同；从学术自觉和新生长力培育入手，提升教师专业自觉的幸福感，达到文化认同。

二、学生电影节，高规格的综合文化艺术盛宴

学生的成长与他的经历紧密相关，学校要培养怎样的人才和它的课程设置也紧密相关，学校丰富多彩的课程为学生个性化的发展提供了广阔的空间。学校的课程设计已经打破了单一学科的格局，越来越多的综合性项目化的课程在学校应运而生并茁壮成长，不仅丰富了学生的成长经历，也使学校的发展呈现了百花齐放的态势。

经过数年的摸索与实践，田外的校园电视台已成为了闵行区乃至上海市信息化建设的一张靓丽名片，首届“田园杯”学生电影节的隆重举行，更是为教育信息化赋予了更为鲜活和丰富的教育意义。学生电影节的各项活动设计，站在改革开放40周年的大背景下，以电影为核心，设计了丰富的易于学生参与和体验的项目，还融合了观影、绘画创作、视频设计、亲子配音等多种样态的表现形式，呈现了一场温情与梦想交织的视听盛宴。历时近4个月的学生电影节参与面广、受众面大，影响深远，给参与校的学生们提供了丰富的影视传媒体验。

透过“田园杯”学生电影节，思考学校特色打造和学生综合体验项目的设计，感受颇多。首先，综合活动的推出是要有学校日常扎实的实践为根基的，是需要有广大教师的价值认同为支撑的；其次，综合活动的设计是长程的，学生可以通过较长一段时间的参与体验，是突破时空向社会和家庭延神的；再次，综合活动是有重要的节点设计的，通过节点更好地展现活动成果，促进互动交流，提升认知和感受，甚至成为可以烙印在心底的重要成长印记；另外，综合活动也可以成为学校办学特色的集中式呈现，使内涵更加丰富。

田外是一块肥沃的土壤，有着一群不断进取超越的田园人，他们有着前瞻的教育理念，有着不倦的教育情怀，以发展的眼光、现代的手段、教育的初心，不断创造着一个又一个的教育里程碑。我想，年轻的二外小有着同样的田外教育基因，作为校长，要能尽快掌握治校的方法，把握教育发展的规律，因地制宜，合理谋划学校的发展，夯实日常的教育实践，为学生创造有意义的成长经历，为教师提供有品质的研修生活，让每一个田园人都能在这里得到成长和发展。
