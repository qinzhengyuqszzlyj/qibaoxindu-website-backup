在聚焦核心素养 促进文化理解的实践中寻新路、求发展 ——暨闵行区小学音乐2022学年第二学期第15周教研活动

发布时间 : 2023-05-26   作者（来源）: 刘庆江   阅读 : 1157

3/8
←
→

2023年5月25日下午，在闵行区七宝镇明强小学(东校区)二楼远程教室开展了闵行区小学音乐学科2022学年第二学期第15周研讨活动,全区小学共200多位音乐教师和松江区九亭五小音乐教师一起参与了研讨和学习活动。

本次的研讨议程分为两个活动版块，活动一为现场课堂教学研讨，活动二为合唱比赛评分原则与实施策略的专家专题培训。

活动版块一：音乐课堂教学研讨

音乐课堂教学指向落实新课标、新课堂的研究，围绕着 “聚焦核心素养内涵  促进音乐文化理解”为主题开展现场的教学研讨。由小学音乐学科中心组成员崔征老师主持并介绍本次研讨活动的教研主旨。明强小学音乐教研组一直以来遵循音乐新课程标准的指引，强调学科综合，在日常课堂教学中不断进行落实核心素养的探索与研究。在今天的研讨活动中，通过两节课堂教学展示与各学校的音乐教师共同分享交流。

第一节课是由吴思成老师执教的一年级第二学期《玩具兵进行曲》，吴老师把握大单元的学习设计方向，以童话故事的情境创设，关注学生音乐聆听能力的培养。通过不同的音乐实践活动帮助提升审美感知能力，在活动中引导学生们展开联想和想象、学生们自由且有序的在音乐中进行大胆创作，一年级同学在音乐中感受表现美，理解美，在有趣的活动中感悟、领会作品的人文内涵。课堂活动中各环节自然衔接，贴近孩子们的生活。从学唱到运用小乐器伴奏，再到列队行进，丰富了音乐课堂表现形式。吴老师可爱风趣的教学语言和自然亲切的教态，和孩子们玩成了一片。让整个音乐课堂生动有趣，将唱游课中的“玩中学，乐中学”落实到实处。

第二节课是由王晋妮老师执教的四年级第二学期《采茶舞曲》，王老师从情境导入让学生了解茶文化，通过实际的品茶环节活跃课堂，创造非常良好的课堂学习氛围。本着研究真问题、展示真课堂，本节课王晋妮老师采用了借班上课的方式，她从容淡定的教学风格，就如一道清茶温润学生们心灵，得到了各位学校老师们的一致好评。课堂教学的每个环节都自然过渡，结构顺畅，让观课的老师们时而赞叹，时而忍不住一起哼唱。王老师的说课环节中采用了思维导图的方式，贯穿茶文化的理解，向我们呈现了课堂中“跳茶舞、听茶曲、唱茶歌、奏茶曲”的课堂教学环节。整个说课环节逻辑清晰，有条有理，风趣幽默，引得各位老师阵阵掌声。

在评课环节中，老师们纷纷主动地发表自己的观课感受，对吴思成、王晋妮两位不同教龄老师执教的两堂课给予了较高的评价；对于明强小学教研组在课堂教学中遵循新课标的教学理念，追求研究真实的问题、呈现真实的音乐课堂教学的追求给予了积极的评价。

 

活动版块二：合唱比赛专题培训

第二版块为“中小学合唱比赛的评分原则与实施策略”的专题培训，是由闵行区教育学院教研中心主办，闵行区七宝镇明强小学承办。活动由闵行区教育学院小学音乐学科教研员刘庆江老师主持，特别邀请上海音乐学院音教系副主任，合唱指挥专家王海灵老师进行现场指导。

首先活动开场由明强小学王丽华老师、王琳老师分别对两首合唱《小白船》、《爱是我的眼睛》进行指挥排练。王海灵老师对闵行合唱学习与训练给予了极高的赞扬，肯定了各学校优质的合唱团队。王老师指出合唱最追求的就是能打动人心的声音，表达出孩子们最纯真的声音音色。但在此基础上王老师也提出，在一个合唱团队的比赛中，作为评委专家更关注的是音准、音色、声音声部的平衡、和声色彩的和谐以及对原谱的忠实程度。并对在合唱训练和比赛中的这些重要因素进行详细讲解，作为合唱老师，对于音准的把控要考虑到声部比例问题，声部人数的均衡。对于学生音色上一定要注意减少声带的使用。再进行全曲演唱中，音乐的表现力、音乐中乐句感、句子中的连贯、句子中的语气感、逻辑重音等都是表现合唱魅力细节要点。同时王老师也提出，作为合唱教师，要留给自己清晰的耳朵，客观的去聆听自己团队的声音。像评价其他团队一样去合理评价自己的团队，更有可能找到问题并给予解决。合唱追求的是多维立体的、有漂亮的音色、有和谐均衡的声部支撑，这样才能让我们的合唱团队更立体，更多彩。

在王琳老师指挥的《爱是我的眼睛》中，王海灵老师给予了建议和指挥动作的指导，什么时候吸气，吸气的动作准备。所有歌唱状态的面部表情准备，指挥习惯的调整，手部掌心相对的指挥状态，脸部表情的状态，在舞台上与孩子们的暗号这些等等的要素都是合唱训练中需要养成的默契。

在王丽华老师指挥的《小白船》中，王老师也呼应全体老师一起参与到指挥和演唱的过程中，大家学习得非常认真，虽到结束时间但还意犹未尽，依旧沉浸在音乐当中。

音乐教研员刘庆江老师对本场活动进行总结，既肯定了大家的努力，也激励各位合唱教师们不断前进，在追求最佳的音乐表现中实现学科育人。

本次区级研讨活动在研讨和学习的头脑风暴中圆满落幕。活动一的音乐课堂研讨不仅是一次对音乐新课标的落地实践，也是一次让青年教师在小学音乐课堂教学的实践活动中扎实好根基，树立好职业理想，往更好更高的平台勇往直前的学习机会。活动二的合唱培训活动是引导老师们以音乐相伴，探寻碰撞出绚烂火花的理念、方法和路径培训，时间虽短，但老师们收获颇多。让我们一起凝心聚力，扬帆远航，做高质量音乐教育的践行者！
