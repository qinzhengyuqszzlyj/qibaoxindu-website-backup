落实课标要求 构建育人蓝图 丰富文化体验 —— 闵行区小学英语四年级主题教研活动于七宝实小举行

发布时间 : 2023-05-17   作者（来源）: 沈雯晴   阅读 : 1212

3/8
←
→

立夏过半，小满将至。2023年5月16日下午，主题为“落实课标要求 构建育人蓝图 丰富文化体验”的闵行区小学英语四年级主题教研活动活动在七宝实验小学开展。本次活动由区小学英语中心组成员、上海市民办万源城协和双语学校的候明翠老师主持，区四年级英语教师近100人参与活动。

本次活动包括单元整体规划、单课教学实施、教学设计说明、教师教学反思、互动交流研讨、专家指导点评等六个环节。

活动伊始，李祺老师和顾敏凤老师从“课程内容、单元目标、学科育人”等方面进行《英语（牛津上海版）》4B Module4 More things to learn Unit2 Festivals in China单元整体规划。

其次，韩悦老师和王潇渊老师分别展示了4BM4U2 Festivals in China第一课时 Chinese festivals I know和第二课时The Spring Festival两堂研讨课。韩老师在学校文化周的语境中，以Festival book作为载体，引导学生开展指向主题意义探究的学习活动，充分体会节日所要表达的文化内涵，体验中国传统节日的多样性、趣味性以及重要性，从而培育文化意识。王老师在春节语境中，通过关联递进的学习探究活动，引导学生获取关于节日的相关信息，体会中国传统节日的重要性，理解文化意涵，增强文化自信。两位执教老师均基于学生实际生活经历，优化活动设计，丰富学生文化体验，最终践行学科育人。

课后，潘佳佳老师和施陆珺老师结合两堂课的教学实施，从“依托多元资源，丰富文化体验”和“优化活动设计，深化学科育人”两大方面对两节课进行教学设计说明。

在韩悦和王潇渊老师的现场教学反思之后，四位小青椒成员老师：莘松小学朱洁蕾老师，田园外小储逸琼老师，明强小学顾丹老师及北桥小学黄艳红老师分别从“课标落实”、“活动设计”、“育人价值”、“学习体验”、“跨学科学习” 等方面进行了互动评课。

接着，候明翠老师对此次活动进行指导和点评。候老师首先肯定了七宝实小教研团对于本次主题的充分理解，每一环节都围绕着课程标准的理念与要求实施，强化素养育人。随后，候老师从“落实课标要求，强化素养育人”、“整合多元资源，辅助自主探究，丰富文化体验”、“设计多种思辨问题，融于活动，达成素养提升”对本次活动进行了深入浅出的剖析，也为如何将新课标要求真正落实于课堂实践指明了方向。

七宝实小英语组教研团队在此次教研活动中落实课标要求，构建育人蓝图，丰富文化体验，学生核心素养和教师专业素养均得到提升。区教研员沈雯晴老师、区中心组候明翠老师以及小青椒各位老师多次的教研指导为七宝实小教研团队提供了宝贵建议，指明了研究方向。相信在在新课标的引领下，专家的指导下，闵小英通过不断的努力奋斗，会不断勇攀高峰。 


撰稿：七宝实验小学 潘佳佳

审稿: 闵行区教育学院 沈雯晴
