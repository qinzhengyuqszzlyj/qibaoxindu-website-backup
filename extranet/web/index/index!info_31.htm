亲近名家名园 创新办园理念 ——郁亚妹“名校长培养工作室”研修活动系列

发布时间 : 2023-10-27   作者（来源）: 贾建芬   阅读 : 358

为了进一步实现学校高质量发展的目标，提升园长对于学校建设的顶层设计与思考。2023年10月20日，闵行区郁亚妹“名校长培养工作室”成员在导师带领下，聆听了华东师范大学赵勇教授关于《知己知彼：美国教育新动向与研究新进展》、荷花池幼儿园宋青园长关于《建构育人导向的创造性艺术评价，追求回归儿童本真的艺术教育》的专题报告，以“亲近名家名园”为重点，引导园长们“从不一样的视角出发，跳出教育谈教育”，围绕目前学前教育发展大方向展开积极的思考与研讨。

赵勇教授从国际大环境、西方教育研究现状和教育的未来发展三个大方向讲述并引发大家对于目前存在的教育问题的思考，他的讲座提供了对当前教育领域的深刻洞察，以及对未来教育的展望，对幼儿园的教育工作具有积极的指导意义。荷花池的幼儿园的宋青园长以园为案，介绍了荷花池幼儿园在“共生共享”理念下的课程发展、实践与评价。尤其是他们在柔活师幼关系、优化学习方式、关注学习品质等方面的探究与突破，更是给了学员很好的借鉴与思考。

在互动研讨过程中，学员们这样说：“我们需要具备全球化视野和多元文化教育的理念，帮助孩子理解世界的多样性，促进幼儿对不同文化的尊重和理解。要积极鼓励教师探索创新的教育方法，以便在特殊情况下保持教育的连续性。”（浦燕群）

“从赵教授的报告中我捕捉到了两个未来我们需要去关注的教育走向，也正是我们在努力实践与突破的方向，即关注个性化教育与信息化素养教育。我们要承认每个幼儿都有自己的优势和弱势领域，推崇优势理论，注重发展每个人的特长，鼓励每个幼儿在自己擅长的领域得到更自信的发展。作为未来的教师应当思考如何引导幼儿有效地利用人工智能等工具，以及培养幼儿的判断力和社会素养。”（吴兰）

“两位专家帮助我们打开思路，寻找解惑的切入口。我在思考是否可以从分析孩子喜欢什么，适合什么，什么好玩，怎么玩，尝试调整为在一些区域提供材料，至于孩子们怎么玩？在哪里玩？想和谁一起玩，玩的内容是什么，玩的结果怎样，则希望是开放的、自主的，鼓励孩子主动参与，用他们的各种经验，用自己喜欢的方式去表征，去探索……教育就在细微之处着力，倾听孩子们的声音，因为转化、改变、成长就在那些瞬间。”（周莉）

“把握时代脉搏，建设与时俱进的幼儿园，是一名管理者的职责与使命。在高质量发展背景下，作为园长我们应当重新审视：学校、家庭、社会之间的关系，秉持开放办学的理念，跨越地域、时间、空间等界限获取教育资源，为教师和幼儿创设更开放、更自主、更有活力的成长环境。”（谢贤群）

“专家的精彩讲座荷花池幼儿园“共生共享”理念下的园所发展让我看到园所特色塑造的重要性。重视培养幼儿热爱艺术和生活的根基，强调艺术融入日常生活，能帮助孩子建立积极的艺术认知和情感基础，描绘他们的底色，实践五彩斑斓的艺术生活。在实施课程时，我们要学会更全面地了解每个孩子的兴趣、天赋和需求，以便为他们提供个性化的教育支持，确保课程内容与儿童的年龄和发展水平相适应。”（余菁）

最后，导师郁亚妹园长这样说：“教育无小事，事事皆教育。作为一名教育管理者，我们要认识到，教师队伍的发展直接关系到学前教育质量和教育改革的成败。要思考如何“求同存异”地促进教师群体和谐发展，因人而异地挖掘每一个教师身上的资源和特长，使得教师在合适的平台上找到专业自信，从而推动教师队伍有效执行与实施课程。我想我们可以经常性的跨出教育看教育，通过借鉴优秀企事业人力资源开发和管理案例，在把握园所队伍发展的现状和问题的基础上，尊重并诊视教师差异性所蕴含的丰富性和多样性价值，以‘管理即服务’的理念促进团队和谐发展，推动园所的高质量发展。”

现代科学管理之父德鲁克说：“管理要解决的问题，90%都是相通的。”通过本次学习，学员们深刻感受到：无论行业是否相同，管理的方法和智慧是可以借鉴和模仿的，解决问题的路径和方式是普适的、可以复制的。在“高质量发展”背景下，必须跳出教育看教育、立足全局办教育，从而有效提升管理效能，打造具有本园特色的高质量园所。
