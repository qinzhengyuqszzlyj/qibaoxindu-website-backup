“智慧纸笔”赋能 促劳动素养培育 ——闵行区劳动学科教研员教学实践暨公开课研讨活动

发布时间 : 2023-10-27   作者（来源）: 刘至宁   阅读 : 1294

10月26日下午，闵行区劳动学科教研员教学实践暨区公开课研讨活动在华理梅陇实验学校举办，全区初中劳动学科及部分小学劳动教师参加了本次活动。刘至宁老师在上课之前，首先就“智慧纸笔”赋能劳动课程教学的数字化背景情况做了介绍，讲解了智慧纸笔及闵智作业平台中对比讲评、作品点赞、创意思维呈现等功能如何使用；意在引导老师有目的、有方向地进行观课，重在了解“智慧纸笔”如何在教学中精准赋能，如何在互动、展示及评价等教学环节中如何发挥作用，如何促进学生劳动素养培育和提升教学实效的。

接下来，刘老师为大家展示了一堂别开生面的区公开课——《纸艺小品向日葵①》，首先刘老师通过向日葵的“向阳而生”、“温暖明亮”、“丰收秋日”等美好寓意引导学生讨论出作品的设计理念；再通过小组合作、任务分工、自主学习等方式完成作品设计和技能实践。在本节课的设计草图绘制和互动交流环节中，刘老师巧妙运用了“闵智作业”数字化平台的“全屏通览”、“对比讲评”、“思维呈现”等功能，通过精准指导帮助学生明确设计主题、表达设计理念；学生利用“智慧笔”快速完成草图设计并进行展示交流，学生在交流中不仅展示了草图的主题、构图设计，还体现了设计元素的个性化，更是在表达中体现了向日葵的寓意和内涵。

其中有一位同学在设计中还创作了一首小诗：“金秋季节(十月)麦粒香，向阳花开天天响（想）”，表达了丰收景象和对美好生活的思考，温暖的颜色还让人觉得不再孤独。学生的展示与交流，让在座的师生都感受到作品设计体现了温暖与力量，体现了劳动课程育人育美的课程理念。

展示课结束后，刘老师就《纸艺小品——向日葵》这一单元的整体设计进行了说课，刘老师强调了单元设计要详略得当，重点突出，单元重点关注单元目标设计和学习活动设计。刘老师还详细阐述了本单元的设计意义是体现“新课标”的教育导向，劳动课程从单纯的技能训练到包含着劳动情景创设、劳动技能习得、创新设计挑战等环节的整个劳动过程体验，学生在劳动项目中体认劳动价值，感受劳动带来的成果、快乐与成就感。另外，刘老师还强调教师要给到学生充分的实践活动时间，在小组合作中精准指导，并在过程中善于捕捉生成性资源，恰当点评，以帮助学生达到更高的作品完成度。

最后，活动以教研论坛的形式进行了评课研讨，来自华理梅陇实验的张仲林老师、七宝三中的谭珊珊老师、交大马桥实验的胡晓翠老师、田外初中的陈婷老师、七宝实验的刘琳琳老师和明星学校的孙守思老师对本节课做了精彩点评。

张仲林老师高度赞扬了刘老师以每时每刻的微笑鼓励学生，通过与学生心的交流把控住了课堂；特别是对学生创作“小诗”的呼应，张老师在发言中表达了，由学生原创想到了传统文化诗词《长歌行》——“青青园中葵，朝露待日晞。阳春布德泽，万物生光辉。常恐秋节至，焜黄华叶衰。百川东到海，何时复西归。少壮不努力，老大徒伤悲!”还可以把这一生成性资源加以利用，更好地发挥其育人功能。

谭珊珊老师谈到本节课智慧笔的恰当使用是劳动课中设计课程的实施是一种突破；胡晓翠老师结合自身实践经验，分享了“智慧笔”的“数字画像”功能对于课堂效率的提高。

陈婷老师作为新任学科教师，在听课中感受到对于跨学科教学与研究的启发；刘琳琳老师也表示本节课让自己感触最深的是课型反转和大胆放手让学生去设计和实践，再配合教师的精准指导，是有力助推了学生综合实践能力的培养和劳动素养的培育；孙守思老师强调刘老师的劳动课堂扎根学生内心，让学生更加热爱美好生活，一节启迪心灵的劳动课也许会影响学生一生的成长和生活状态，所以上好每一节劳动课，都是我们每一位劳技教师的使命和责任。

活动在热烈的教研气氛中圆满结束，会后老师们交流的热情还在继续，相信本次教研活动不仅在数字化促劳动素养培育方面给广大师生以启发，更是在学科育人、育美、育德、育心等方面都都为广大教师提供了启示和参考，为未来的劳动课程教学实践提供了动力和方向。
