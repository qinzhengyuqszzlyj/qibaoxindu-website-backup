大连市甘井子区中小幼学校教育管理者高级研修班上海市闵行区名校跟岗研修活动启动

发布时间 : 2023-11-13   作者（来源）: 曹雅明   阅读 : 794

为进一步加快大连市甘井子区中小学、幼儿园教育发展步伐，加强上海闵行、大连甘井子区两地学校管理者在教育理念、教育行为上的沟通与提升，2023年11月13日，大连市甘井子区中小幼学校教育管理者高级研修班上海市闵行区名校跟岗研修活动开班仪式在闵行区教育学院举行。教育学院工会主席刘建勋主持仪式。

研修班班主任曹雅明作研修方案解读。本次研修由甘井子区教育局和闵行区教育局共同领导，闵行区教育学院校长发展中心具体负责研修项目实施。本次跟岗研修为期两周，参加对象是大连市甘井子区中小学24名教导主任、6名正副幼儿园园长，将通过对我区名校跟踪观摩学习，深入学校各项业务管理工作，参与教育教学等，加强甘井子区与闵行区中小幼学校校际间、干部间、教师间的联系、沟通与交流。

甘井子区周水子小学教导主任刘韵代表全体学员发言。作为跟岗学员要“知其所来，识其所在，明其将往”。表示一定珍惜研修机会，履行研修计划，自觉遵守跟岗学校规章制度，尊重带教导师研修安排，确保为期两周的跟岗研修学习保质保量。

闵行区教育学院校长发展中心主任俞琳欢迎大连市甘井子区中小幼学校教育管理者高研班一行来到闵行，并介绍了闵行区近几年的教育改革成效，以及“四级六列”教师队伍建设、“六个层级”干部队伍建设情况；分享了闵行教育通过队伍建设促进学校发展、个人成长、区域学生受益的良好生态。建议学员基于问题开展跟岗研修，在深入学校各项业务管理工作、参与教育教学中寻求解决问题的路径和方法，提升自身管理水平。

大连市甘井子区教育局副局长谢田军、甘井子区教师进修学校培训部主任孙金波参加开班仪式并赠送礼物。谢田军副局长感谢闵行区教育局、教育学院，以及各跟岗学校的大力支持与辛勤付出，他指出：大连甘井子区与上海闵行区曾开展一系列深度合作，共享教育理念与改革成果，关系源远流长。希望本期研修班学员做到“五个一”：一要积极深入学校，学习先进的教学模式和育人理念，并与自身实际相结合，进一步提升专业素养。二要积极深入课堂，研讨探究教学问题，拓展教育教学新思路、新方法。三要积极结交高人，与闵行区优质学校管理者和老师成为朋友，学习规划学校发展、领导课堂教学、优化内容管理、引领教师成长的本领和能力。四要学以致用，努力提升自身能力，并把学到的经验和精神带回学校。

开班仪式后，30位学员将分成中学、小学、幼儿园三个学习组，分别进入闵行区莘松中学教育集团、实验小学教育集团、莘庄幼儿园教育集团的几所学校或校区，通过观摩课堂教学、参与研训活动、体验学校管理、感受校园文化，深层次了解闵行名校的教育教学与管理方法等。莘松中学党委书记张建国、实验小学党委副书记徐太生、莘庄幼儿园办公室主任黄晓蕾作为三个教育集团跟岗研修的联系人亲临开班仪式现场，确保研修活动顺利高效。
