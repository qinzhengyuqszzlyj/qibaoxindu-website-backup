共思共研 悦享运动 —— 华师紫幼课程基地研究共同体研讨活动

发布时间 : 2023-05-23   作者（来源）: 鲁鸣   阅读 : 633

3/8
←
→

2023年5月16日上午，华师紫幼闵行区第三轮幼儿园课程基地“幼儿发展优先”项目研讨活动在浦江宝邸幼儿园顺利开展。本次活动由华师紫幼主办，浦江宝邸幼儿园承办。项目研究共同体成员共计40余人聚集一堂，观摩学习、分享交流、互动研讨，促进教师专业化成长。


现场观摩，“慧”玩运动

活动伊始，项目研究共同体成员对浦江宝邸幼儿园中、大班幼儿的自主运动现场进行观摩。幼儿自由结伴、自主探索，将多种材料和器械创造性组合，玩出花样，乐在其中。我们看到了孩子们浓厚的运动兴趣，多元的动作发展，更看到了幼儿的合作能力、思维变通能力、问题解决能力、坚持力和自信心。


案例分享，精彩纷呈

研究共同体的3位学员分别从研究缘起、研究过程、研究思考等方面展开进行了案例分享。

浦江宝邸幼儿园李霞老师分享了大班案例《“垫”燃乐趣》，抓住孩子们偶发的事件，与孩子们进行了一场关于垫子的N+1种玩法。教师在过程中观察支持、复盘推进、分享交流、持续探索的过程，也是不断尝试、调整、改变的过程。垫子游戏还在继续，游戏玩法还在更新，创意无限制，运动无止境。

颛桥镇幼儿园顾晓菁老师分享了大班案例《玩转轮胎》，小小的轮胎，大大的乐趣，教师多元角度看到幼儿运动的发展，多途径给予幼儿个性支持。轮胎山路、轮胎翘翘板、轮胎跳山羊、趣味滚轮胎……孩子们敢于创新，不断探索运动可能。

华东师范大学闵行永德实验幼儿园刘杨老师分享了中班案例《“滑”进山洞》，孩子们探索着火车进山洞的游戏，在偶然发现、制定计划、寻找材料、自主制作、发现问题中创意玩法、不断优化。每个孩子都用自己的方式进行思考和参与。

三个案例从科学、专业的角度观察、识别与分析幼儿的动作发展和学习经历，同时也梳理了具体的支持策略，为共同体教师后续的研究提供了思路和方向。


互动研讨，共探发展

分享结束后，共同体成员基于运动现场、案例分享进行了充分的研讨。大家积极发言，肯定了现场活动和案例中教师们将场地规划权、材料投放权归还给孩子，幼儿在运动中学会交往，学会分享、合作……


专家点评，指明方向

活动最后，华东师范大学紫竹幼儿园徐园长以“燃”为主线，从材料、过程、幼儿日记进行点评。徐园长肯定了老师们的支持、追踪、记录、思考，同时也抛了三个问题：

1.对于材料，周边除了运动材料还有什么？这些材料仅仅只是看到吗？

2.当我们观察的过程变得长程的时候，我们要看什么？

3.一对一的倾听，哪些是有持续性的价值？

这些问题给予老师们后续工作的思考方向，让老师们受益匪浅。


唯有学思并进，方能行稳致远。教师们不断转变观念、改变行动、追求突破，关注幼儿的整体发展、主动发展、差异发展，从多种角度观察和识别幼儿，支持幼儿在运动中的一切可能。
