在聆听中汲养 在学习中提升 ——闵行区小学校长培训基地第五期培训班系列活动十七

发布时间 : 2018-10-22   作者（来源）: 曹雅明   阅读 : 1779

闵行区小学校长基地第五期培训班在经过前期一系列形式独特、内容新颖的培训课程后，基地主持人何学锋校长又一次精心策划，带领学员前往深圳这座中国改革开放最前沿，在教育发展与改革领域极具创新活力的城市，进行为期两天的考察学习。

10月17日上午，学员们来到了“深圳行”的第一站——南山文理实验学校。这所学校创办于2015年，在“跨界与融合”办学理念引领下，吴希福校长带领大家致力于培养具有国际视野与国际交流能力的未来创新型人才，走出了一条“以信息技术领先、引领学校全面发展”的办学之路，逐渐形成了“云教育”等八大核心特色。学员们在聆听中了解到：信息技术已经被学校广泛运用于课堂教学，信息化实验研究得到了有力推进，只通过几年实践就把学校打造成了区域办学的新品牌。

下午，学员们马不停蹄奔赴“深圳行”的第二站——龙华行知小学。满小螺校长秉持“造一个真善美爱乐群的儿童世界”的办学思想，明确“育人理念”、“为师形象”、“德育策略”、“课堂文化建设”等办学方略。在她的带领下，行知“1+1”学科课程体系完备，校园艺术氛围浓郁。满校长与学员们分享办学的成功与喜悦，当然也有困惑与阻碍，她认为“培养一个完整的人，才有真教育”。通过真情对话，学员们认识到校长作为一校之魂，必须有思想、有定力、有行动、有坚持。

10月18日，全体学员前往“深圳行”的第三站——光明新区玉律小学。这所学校加入“新基础教育”研究行列只有短短4年时间。在课堂开放中，学员们感受到了教师对生成资源的捕捉、回应和利用意识，而反思重建的研究生态反映了玉律小学对“新基础教育”理念引领学校整体转型变革的坚定决心。在与学校管理团队交流互动中，学员们针对学生工作、教师发展、学校管理等领域进行了经验分享，还聚焦队伍建设——年段管理模式进行了探讨。大家畅所欲言，互相启发与促进。最后，何学锋校长作为参加新基础教育研究二十年的领衔校校长和资深实践者，谈了自己对新基础教育的理解和认识。

本次“深圳行”学习活动，开放又融合，学员们既领略到了国家经济改革开放最前沿城市的经济发展风貌，也感受到了深圳基础教育的发展速度和创新活力。大家通过梳理与总结，进一步增强了使命感、责任感和紧迫感，同时集成他校办学智慧和经验，不断拓宽办学视野，提升办学水平，创生办学思路。

 

摄影、供稿：闵行区小学校长培训基地办公室
