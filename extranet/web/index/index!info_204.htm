依标施教  精准教学

——闵行区七年级道德与法治学科教学研讨活动

2023年5月25日，以“依标施教 精准教学 打造高效课堂”为主题，闵行区七年级道德与法治教学研讨活动在七宝文来学校百川剧场举行。参加本次研讨活动的有闵行区道德与法治教研员刘芳老师，金山区道德与法治教研员陆春娟、何花老师，松江区教师代表及闵行区兄弟学校百余名道德与法治一线教师。

课堂教学——《我们与法律同行》

活动伊始，七宝文来学校道德与法治学科陈秋坤老师执教《我们与法律同行》公开课。陈老师以《素不相识的欺凌》热点事件导入，带领学生剖析案情本质。结合学习任务单的设计，启发学生思考从不同当事人的角度出发，如何依法办事才能避免案件的发生？结合《“家校社协同 建法治校园”倡议书》以知促行，培养学生解决实际问题的能力。课程尾声，陈老师结合《习近平总书记在党的二十大上的报告》升华主旨，促使学生理解体会未成年人对于法治中国建设的应有的责任与担当，逐步成长为社会主义法治的忠实崇尚者、自觉遵守者、坚定捍卫者。整节课堂，教师语言行云流水、自然舒畅，学生互动积极，乐于表达。

主题研讨——教学评一致性

作为对课堂教学的呼应，来自浦航实验中学曹雪玮老师、吴嘉琦老师选择同一单元七年级第四单元《走进法治天地》，进行了单元教学设计的分享。两位老师立足单元整体知识架构，从单元要求、学情分析、课标依据来设计每节课的教学目标。通过情景创设、案例分析启发学生思考，生活与法律息息相关。在知识铺陈中，明晰人类探寻法治之路；在观点辨析中，理解人类如何走上依法治国的道路，人治到法治，助推中国梦；在实践探索中，学生感悟并思考如何更好促进依法治国的建设，落点“奉法者强则国强”。从教学设计的用心、精心，与会教师感受到曹老师、吴老师扎实的学科素养。

来自闵行区文来中学王洁老师肯定陈老师采用一例到底的课堂模式，问题设计层层推进，紧扣教材内容。文绮中学的贺思琦老师肯定陈老师结合学科特性，关注学生道法思维培养，捕捉二十大热点，并在板书设计上独具特色，让人耳目一新。

金山区资深教研员陆春娟老师，感动于陈秋坤老师对课堂教学每一处设计精益求精。更在教学中打通不同年级间的知识体系德育培养目标。金山区教研员何花老师，感谢闵行经验分享，促进更高层面的区级交流。课堂学习单形式设计契合当下的学科中考试题，立足基础年级，培养学科思维品质。板书在形象可爱的小人画板书设计中实现知行合一、知行统一，贴切教学评一致性研讨主题。

九亭二中的孙晨佳老师，用“有态度”、“有温度”、“有深度”、“有高度”评价陈老师的公开教学；用“精准”、“精细”、“精致”、“精进”表达了对闵行区道德与法治学科教研活动的钦佩学习。

闵行区道德与法治学科教研员刘芳老师，高瞻远瞩依托新课标的解读，针对实践活动设计的可操作性，视频资源信息提取的有效性，课堂学生表述的生动性等三个方面，提出了切实的优化建议，拓展一节优质公开课的知识宽度、思维高度、情感深度。

初三复习研讨——道德与法治学科中考中的“变”与“不变”

中考临近，刘芳老师以道德与法治学科中考中的“变”与“不变”助力一线教师的“黄金冲刺”期。刘老师建议教师要立足模考真题，关注题型、紧扣教材、落实课标，来应对多变的考试角度与考察点。应对教师群体复习迎考的紧张情绪，刘老师也以“行之力则知愈进，知之深则行愈达”给与老师心理层面的关怀与鼓励，激励老师们“心向往，行必至”。

结尾

本次教研活动是一次助力青年教师成长与学科中考冲刺的务实研讨活动，以课堂教学为尝试，以主题研讨促思考，以复习指导聚中考，扎实推进闵行区道德与法治学科的教研品质。
