上海市委党校卜新兵教授为闵行区优秀校园长等做专题培训

发布时间 : 2019-10-28   作者（来源）: 曹雅明   阅读 : 7179

2019年10月10日，中共上海市委党校教授卜新兵在闵行区教育学院作“不忘红色初心，牢记党员使命，为实现新时代伟大梦想努力奋斗”的专题报告。闵行区第二届“教育家培育工程”、第三届骨干园长培训班、“初中强校工程”培训班全体学员，以及教育学院全体党员和教师等100多人参加。教育学院党总支书记、院长朱靖出席，校长发展中心主任俞琳主持活动。

法学博士卜新兵是上海市委党校马克思主义学院精品课教师，上海市习近平新时代中国特色社会主义思想研究中心研究员，上海党校系统“双七步案例教学法”和“快乐党课”创始人。

卜教授从“不忘初心牢记使命是实现新时代历史使命的必然要求”、“认真学习领会讲话精神和中央部署是开展好主题教育的重要前提”、“在学懂弄通理论、历史、实践逻辑中牢记初心使命推进自我革命”三大方面，深入浅出地阐明了“不忘初心，牢记使命”的深远意义，指出当今世界正在经历百年未有之大变局，开展“不忘初心，牢记使命”主题教育活动是中国共产党在新时代继续成功应对各种艰难险阻与风险挑战的需要。

他从今年国庆大阅兵谈起，分析了当下国际国内形势，剖析了“中美贸易战”的深远影响，以及“香港乱局”的由来及走向等，指出每位共产党员要认真学习领会好习总书记两次讲话精神，再次明确中国共产党人的初心和使命：为中国人民谋幸福，为中华民族谋复兴。作为新时代共产党人，要坚持用党的科学理论武装头脑，坚持在党的光辉历程中汲取智慧营养，坚持在总结反思中坚定走向未来，坚持实事求是、解放思想、保持清醒、自我革命。既要明确主题教育的总要求：守初心、担使命、找差距、抓落实，还要明确主题教育的根本任务和具体目标，在今后的发展道路上要时刻牢记“稳、紧、改、闯”四字，立足本职，在“学、思、践、悟”中增强工作本领和人生智慧。

卜教授幽默风趣的话语体系，严谨缜密的逻辑思维，引经据典的情势分析，让整场报告高潮迭起，掌声阵阵。在场的各位领导和老师受益匪浅、意气奋发，决心面对新挑战，不忘初心，牢记使命，重整行装再出发，为闵行教育继续作贡献！

 

供稿：教育学院校长发展中心    摄影：教育学院技术中心
