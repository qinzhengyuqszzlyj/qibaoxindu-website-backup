岗位历练 助力优秀年轻干部快速成长 ——2021年闵行区教育系统优秀年轻干部挂职培训启动

发布时间 : 2021-03-19   作者（来源）: 曹雅明   阅读 : 2131

3/8
←
→

2021年3月18日，为进一步加强后备干部梯队建设，注重岗位实践锻炼，激励年轻干部拓展视野、砥砺品质，2021年闵行区教育系统优秀年轻干部挂职锻炼动员会在田园外小举行。闵行区中小幼学校、教育局机关的35名优秀年轻干部，以及26所挂职学校校长等参加活动。区教育党工委书记朱雪平、教育局局长恽敏霞出席，区教育党工委副书记李光华主持活动。

在开班动员仪式上，区教育党工委党群科科长林火生对本期优秀年轻干部挂职培训工作进行部署，明确了挂职培训的方式、时间、内容以及相关要求。田园外小副校长何敏、龙茗路幼儿园园长浦燕群分别代表优秀年轻干部及挂职接收学校进行发言。挂职干部与挂职学校导师签订了带教协议。

恽敏霞局长以自身经历告诉年轻干部，挂职是干部成长的必要途径，只有经过不同岗位的历练，视野才能更开阔，看问题更全面，对事情的理解更透彻。她希望每位学员好好把握这次培训锻炼机会，全面融入挂职学校，在挂职培训及日常工作中努力做到“和润于心、尽力于事”，严格要求、激发动力，真正使自己在思想上、能力上、境界上有新的提高。同时希望挂职单位把学员当作自己人，让他们放手去干，从本校与挂职学校的文化特质、管理模式、工作氛围的比较中去思考和审视问题，大胆质疑，敢于表达自己的见解和观点，真正做到彼此尊重、彼此融入。

朱雪平书记在讲话中指出，“十四五”期间是我区校园长新老更替以及优化教育资源布局的重要节点和关键期，校园长配备面临质和量的双重压力，而多岗位锻炼、多文化交流对优秀年轻干部的成长很有必要。他希望各位年轻干部好好抓住这个提升自我的机会，与挂职单位领导和老师多交流、多沟通，在挂职锻炼过程中开阔眼界，增长见识，主动跳出一校一地的局限；要勤于思考，善于总结，能看到、学到提升办学质量的着力点；在挂职锻炼过程中，精准把握政策导向和要求，注重自身修炼，提高领导艺术，争取快速成长。他也希望挂职学校大胆、严格管理年轻干部，用其所长、补其所短，让他们在急难险重中显身手，通过压担子，提升学员看问题、处理问题的能力，树立挂职干部良好形象。

启动仪式后，田园外小校长赵瑛群为学员们作《引领自觉 群体成长 团队发展》的首场专题报告。赵校长从“高标准培养干部梯队”、“高品质成就教育梦想”两大方面分享了学校在教师队伍建设中的一些做法与感受。她从习近平总书记“有信念、有梦想、有奋斗、有奉献的人生，才是有意义的人生。” 这一论述说起，谈到学校面对“年龄结构趋同、专业发展后置、锻炼岗位有限”等教师专业发展难题，如何打破定势、团队共长，做到岗位历练，机制激发内在动力；梯队建设，文化厚植团队情怀；群团凝聚，精神涵养时代气质。让学员们深切感悟国家对于青年及青年干部的殷切希望，明白挂职意味着要负起双重责任，承担双重压力，经受双重考验，把任职学校和挂职学校的工作都做好，这是机遇也是挑战，正如习近平总书记所说：干部成长无捷径可走，经风雨、见世面才能壮筋骨、长才干。在摸爬滚打中增长才干，在层层历练中积累经验。

区教育学院校长发展中心主任俞琳主持培训活动，并希望学员们真正做到从大局出发，树立责任担当意识，发扬勇于实践、艰苦奋斗的精神，通过一年的锻炼，拓宽视野，经受历练，积累经验，努力成长为具有全局视野的新时代教育管理者，为闵行教育现代化建设积蓄力量。
