新教师如何面对“双新”背景下的欣赏教学 ——闵行区小学音乐2023学年第一学期第十一周教学活动

发布时间 : 2023-11-11   作者（来源）: 刘庆江   阅读 : 1074

3/8
←
→

2023年11月9日下午，闵行区2023学年第一学期小学音乐第十一周音乐教研活动在闵行区田园外语实验小学（银都校区）举行。闵行区小学音乐5年内（含五年）教龄新教师，以及闵行区音乐教研员刘庆江老师共同参与了此次教研活动。

围绕本次活动主题《新教师如何面对“双新”背景下的欣赏教学》，田园外小的两位青年教师李珍和吴斌分别进行教学实践。

第一节课由李珍老师执教沪教版四年级第一学期第五单元“多彩的歌声”中的第一课时《赛马》。李老师充分利用自身的二胡演奏专业优势融入课堂教学。整节课教学过程流畅自然，通过小乐器演奏、哼唱主题旋律、身体律动等学习活动，引导学生层层深入体验乐曲的情绪、感受蒙古族音乐的无穷魅力。

第二节课由吴斌老师执教沪教版五年级第一学期第二单元“异国风情”中的作品《单簧管波尔卡》。吴老师把萨克斯演奏融入音乐课堂，引导学生了解“木管家族”各类乐器，并基于音乐新课标进行教学设计，为学生创设了“舞会”的音乐情境。通过节奏创编、身体律动、小乐器伴随等方法，引导学生表现乐曲的不同主题，感受捷克民间舞蹈--“波尔卡”的音乐风格。教师教学语言轻松幽默、教学风格自然大方，充分调动课堂氛围，师生共同营造富有活力的课堂。

之后，教研员刘庆江老师为与会教师，对2022年版《义务教育艺术课程标准》中关于第一、第二学段中的“聆听音乐”和“欣赏与评述”进行了深入浅出的解读。针对教学中出现的各种问题，刘老师分享了自己多年来学习及教学方法，给现场的新教师们提供了非常宝贵的意见与建议。最后，刘老师提出，希望新教师们在课堂教学中逐渐积累方法，为闵行的音乐教学出自己的一份力。

通过此次教研活动，教师们不仅深入研讨了教育教学方法，还收货了宝贵的教学经验和教育理念。相信这些必将成为我们未来教学工作的有力支撑，为我们音乐教学工作注入新的活力与动力！
