自主发展润童心，共话课程促成长 —— 闵行区鹤庆幼儿园课程基地活动

发布时间 : 2023-06-18   作者（来源）: 鲁鸣   阅读 : 547

3/8
←
→

2023年6月16日上午，闵行区鹤庆幼儿园举办了“欢乐一夏”的课程基地活动展示与研讨。来自基地共同体园所和共同体教师、江川学区化各园教师准时出席并参与了本次活动。

方案介绍

一楼门厅签到后，教师们集中到两楼多功能厅，一起聆听、了解“欢乐一夏”的活动的方案。鹤幼课程基地中心组从尊重孩子意愿出发设计了问卷星，对全园每个孩子进行“夏天宝贝最喜欢做的事情”的大调研，随后各班孩子再次投票决定自己班级的活动内容。经过将近一个月的活动开展，本周全园孩子一起欢度“欢乐一夏”。组员们在了解具体方案后，分组认领自己组重点观察的内容，拿着现场观察记录表，带着问题来到户外现场观摩。

现场观摩

户外操场上，孩子们人手一本护照，两两结伴，选择自己喜欢的内容去参与活动。基地教师们被孩子们的热情感动了，一边参与孩子们的活动，一边拍摄、记录孩子们的表现。

“冰爽一夏”：“观赏区”的小小讲解员正和客人老师介绍美丽的冰花雨；“体验区”的孩子们分成了冰中寻宝组、冰船组、滑冰组以及彩绘冰雕组，鲜花、树叶、小宝石等点缀在各种形状的冰块里。小小介绍员们热情地邀请客人老师和小朋友们看一看、摸一摸、玩一玩，体验“趣”玩冰的乐趣，感受夏日的清凉。

“炫酷水枪”：在“炫彩砰砰砰”区，孩子们举起水枪颜料与白布来一个亲密的碰撞组合，在洁白画布上“砰”发出美丽的色彩，体验色彩的魅力；在“水枪大战”区，孩子们自主选择武器——水枪、水蛋等，进行多人对抗。在躲避、奔跑、投掷的过程中充分感受水枪的好玩与刺激。

“夏日侏罗纪”：在“恐龙故事展示区”，几个小导游们热情地向客人老师和小伙伴介绍关于恐龙的秘密，如“恐龙的灭绝”“恐龙的食物”“恐龙化石的挖掘”等；在“沙池体验”区，小客人在沙坑里尽情地挖掘恐龙化石，并根据挖出来的恐龙材料，分别来到“水中孵化恐龙蛋”“化石试验室”“钻石恐龙DIY”三个区域，自主选择喜欢的材料进一步探究和感受“与恐龙亲密接触”的快乐。

“水宝宝大作战”：“欣赏区”的孩子们快乐地用各式各样的瓶子打捞晶莹剔透的水宝宝，大的，小的，方的，圆的，快乐地装进小瓶子里，并通过研磨灌装，得到一个闪亮的水宝宝许愿瓶；在“实验区”，孩子们通过在水中加上不同介质，改变水的密度来探索水宝宝浮起来的秘密；在“制作区”，宝贝们还能用果冻胶在磨具里制作自己喜欢的各种图案，边观察边操作，沉浸在水宝宝带来的快乐里。

……

案例分享

观摩活动结束后，教师们又集中在一起观摩案例分享。中一班丁老师围绕“你好，小萌鸡”，从自然角讨论饲养小动物到决定饲养芦花鸡，讲述了照蛋行动、鸡舍创想、照顾小鸡、小能手月展示，最后到小鸡葬礼的生动故事。中二班何老师向大家介绍了“春风有信、花开有期”的案例，重点围绕留住鲜花、制作干花、干花书签、其他书签、挂上流苏进行分享。两个案例无不渗透教师呵护孩子童心，不断支持与助推班本化活动，促进幼儿自主性发展的用心与用情。

互动研讨

随后，教师们重点围绕“孩子活动中自主性发展表现”“教师支持性策略”两个话题进行交流与研讨。研讨结束后，每一组教师代表进行互动与交流，大家结合《上海市评价指南》中的观察指引，全面分析幼儿现场中的“幼儿自主性发展”具体表现。随着研讨的进行，并结合现场提炼了活动中的教师支持策略。

结语

“欢乐一夏”活动展示了鹤幼孩子们的自主与创想。各班教师不断倾听幼儿的想法，观察幼儿的行为，整合各类教育资源，支持幼儿的学习与探索，真正从“教师视角”走向“儿童视角”，让班本课程更适合班级幼儿发展，助推幼儿获得更好的学习和发展。
