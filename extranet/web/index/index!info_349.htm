让学习成为工作常态，让教育保持生命温度 ——2018年闵行区新任校（园）长带教培训赵瑛群导师组活动通讯（11月）

发布时间 : 2018-12-02   作者（来源）: 曹雅明   阅读 : 3096

“学然后知不足，教然后知困。知不足，然后能自反也；知困，然后能自强也”。每一个人无论身处何时何地或担任何种岗位，学习都应是生活的重要组成部分。作为一名校长，更应率先垂范，积极营造浓厚的学习氛围，建设学习型团队，让学校充满向上的勃勃生机，让每一名教师都有不断创新、不断超越自我的勇气和决心。曾经看过师傅赵瑛群校长的读书和会议笔记，永远是密密麻麻的记录，还有自己的即时思考。即便已经是特级校长，却始终保有求知的饥渴和创新的热情，带领着田园外小创造一张又一张的教育名片。师傅的言传身教，让我耳濡目染，因此我坚持投入较多的时间和精力，通过学习汲取新的成长养分，力争能保持对教育本真的独立思考和扎实实践。

一、新基础读书分享，让教育理念转化为创造性的行动

自2018年6月开始，参加了“新基础教育”新校长研修班，第一轮的研修从《“新基础教育”论》的系统阅读开始。在我看来，如果缺乏了个人的静悄悄阅读，无论有多少外力助推你，也是缺乏发展后劲的。真正的发展，必定来于自身内部的变化。而改变自己的内部基质，除了阅读，别无他法。读书是一种内心的修炼，以坚定前行道路和教育情怀；是一种实践的反思，以梳理日常工作和教育认知；是一次提升，以重塑人生价值和教育意义。

近半年来，共开展了三个主题的阅读分享活动，分别是“学校管理变革”、“课堂教学改革”和“班级建设改革”。分别有叶澜教授、庞庆举博士和李家成教授参与了分享活动，并进行了精彩的点评与指导。第三次读书交流会在我校举行，特邀赵瑛群校长作专家指导，田园外小和田园二外小的部分行政参与学习。

本次读书会聚焦《“新基础教育”论》第十章“班级建设改革”，每位校长通过一个月的时间认真阅读本章节内容，对“新基础教育”的班级建设有了更加系统的理论性理解，并结合本校学生工作和班级建设的实际情况进行了充分的交流分享。在分享中，看到了校长们由于自己学校的发展状况不同，阅读的关注点也有所不同，航华一小叶萍校长的假日小队实践，浦江三小吴春晓校长的分层阅读，在校内广泛开展“新基础教育”系列研究成果的读书学习活动，帮助教师提升理解，颛桥中学李卫军校长对班级岗位价值的深刻诠释，纪王学校祝庆校长的“一班一品”建设等，都给了我极大的启发。

李家成教授的指导更进一步帮助我们认识到阅读与校本实践之间的关系。因为有书的支持，会帮助我们更加聚焦本校存在的问题，提升对问题分析的力度，对自我反思的深度，帮助寻找到新的学校和个人的提升空间，久而久之书会变成自己思想的一部分，最终作用于日常的学校管理和变革性实践中。

二、走出去参观学习，借他山之石助办学清思

学习的途径各式各样，不仅有请进来的指导，还有主动走出去的请教。我和管理团队始终怀揣着一颗学习求教之心，去提高管理的能力、拓宽管理的视野、寻求管理的良方。

当我将赴新梅小学参观学习的想法跟师傅赵校长沟通后，师傅非常支持，认为新梅小学的优质发展之路具有极高的学习借鉴价值，一所办学规模不大的年轻学校，在办学特色、学业质量、教师发展等方面位居全区领先，肯定有它成功的奥秘和规律，值得我们去学习。

我们的外出学习得到了新梅小学汤泳清校长及新梅管理团队的大力支持。在汤泳清校长的陪同下，参观了新梅小学的校园环境，尽管学校不大，但是小而精致，各种绿植错落有致；虽然只有20多年的办学历史，但是学校品质优异，办学特色凸显。校门口各种学生陶艺作品的展示，马上就能让人置身于“以陶育德，以陶育人”的校园文化氛围中；各楼层走道及楼梯空间，富有活力的“乐省、乐言、乐健、乐雅、乐探”的“五乐”生本课程独具匠心。汤校长还详细介绍了学校在“建设陶乐文化，创办优质学校”过程中的深入探索和积极实践，20多年来，新梅小学始终走在“陶文化”特色发展的道路上，以“以陶育德，以陶育人”办学理念为魂，以“陶品管理、陶人教师、五乐陶娃、陶韵课程、陶香校园”为发展体系，打造富有陶特质的校园文化。

作为一所新开办学校，我们要依托一体化办学的强有力支撑，不断做强做优自己。更要有不断汲取和吸纳的意识，走出去学习和借鉴他校的成功经验，在互动学习中逐步清晰办校治校的思想和策略。这一次新梅小学的拜访，不仅增进了两校间的了解和友谊，也更坚定了“办家门口好学校”的信心。

三、广泛参与体验，百姓剧社让我进一步认识教育戏剧

近年来，戏剧教育有“一夜燎原”之势，在上海北京乃至全国很多城市，涌现了大大小小有关“戏剧教育”的夏令营、体验营、工作坊和培训班，既有针对小朋友的，也有针对白领观众的。在学校内，戏剧教育也不断被重视和开设。在国外，戏剧教育是一个很重要的课程，尤其是英国这样有着深厚戏剧传统的国家，戏剧课从小就在艺术教育中占有重要的地位，它是一种全人教育。

我也一直有戏剧教育的情怀，有幸和本校的两位青年教师经过海选一起加入了颛桥百姓戏剧体验工作坊第二期学员班。这是一个以吕凉戏剧工作室为领衔的公益组织百姓剧社，由戏剧专业教师执教，每周六半天的学习至今已经坚持了两个多月的时间。戏剧课程以剧场游戏角色扮演为主要形式，通过实践让参与者由外而内的重新寻找和思考自己，课程内容涉及讲述自己的生命故事，学习倾听的艺术，提升说话的外在行动与内在节奏性等。

年过四十学戏剧，为时未晚。在我看来教育戏剧不是单纯的学习表演，是通过戏剧的表现手法实现对人事的洞察、人格的塑造、人性的诠释、人生的体悟。同时还涉及跨领域的融合，如：剧本的创作、情境的创设、舞台的设计、音效的配合、技法的运用、道具的运用等，它对人的综合素养的要求极高，而不同个性的人、不同才能的人，均能在其中找到自己的用武之地，并在合作中实现戏剧的最佳呈现。

问渠哪得清如许，为有源头活水来。学无止尽，学无边界，希望我们在繁忙的工作之余，始终能给学习留有时空，让学习成为工作的重要组成部分，启迪教育的创造，保持教育的温度。
