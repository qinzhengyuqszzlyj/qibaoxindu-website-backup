“开学第一餐”里的幼儿与教师 ——记闵行区学前生活中心组研讨活动

发布时间 : 2023-09-27   作者（来源）: 张静   阅读 : 939

3/8
←
→

2023年9月21日，闵行区学前生活中心组在闵行区龙柏第二幼儿园开展研讨活动，此次活动迎来了上海市教师教育学院（上海市教育委员会教学研究室）贺蓉老师出席指导。

一、专题分享：“幼儿发展优先理念下的食育课程构建”阶段成果

区学前教研员张静老师分享了“幼儿发展优先理念下的食育课程构建的研究”阶段成果，对于以儿童形象为标志的食育课程的目标设立、经历三次迭代的食育课程内容、柔活边界的课程实施方式进行了重点阐述，中心组老师们在回顾中学习着，也在学习中领悟着。

接着中心组的三位教师代表通过课程故事的方式，分享了对于食育课程实施的理解与思考：

华漕金色幼儿园的徐维娅老师分享了“菜谱的秘密”，从一次幼儿对于吃鱼的吐槽出发，开展了持续了两个月的观察、记录、分享，在“润物细无声”的过程中发现了菜谱的秘密。

爱博果果幼儿园的曹春燕老师带来的食育课程故事是“从1到N：一棵樱桃树开始”，幼儿们在采摘中品味不同果实的酸酸甜甜，也在分享中建立与身边的人的联系，而在不同果实的采摘中，果实的颜色、形状、口味、采摘时间等等经验也编织进幼儿的成长经历中。

七宝皇都幼儿园的王莉老师分享了《霜与柿》，用早中晚的温差比较；自制霜；通过亲身操作、实验、探索等方法认识柿子、柿饼等等儿童可感知可探究的方式，感知感受并过属于自己的霜降节气。

二、主题研讨：“开学第一餐”里的幼儿发展

依惯例，生活中心组的老师们会拍摄幼儿的开学第一餐的视频。这次在教研员张静老师的带领下，老师们从开学第一餐的拍摄视频中探讨孩子的学习与发展。贺老师询问大家拍摄第一餐时的出发点和关注点是什么？激发了大家的深度反思与研讨。

老师们关注“幼儿的习惯、情绪、礼仪、工具运用……”；老师们能联系地变化地看幼儿的发展，不同的幼儿在同以餐中，同一名幼儿在不同学期的开学第一餐中，幼儿在进餐中与其他活动中等等的相同与不同；老师们还从师幼互动、同伴、环境等角度思考影响幼儿进餐情况的因素；老师们还讨论如何进一步支持幼儿的发展。“开学第一餐”的日常里藏着幼儿发展的点点滴滴，也让我们看到了老师们的用心与智慧。

三、专家指导:回归食本身，探寻对儿童精神和身体的养育

 活动最后，贺蓉老师分享了对于食育的理解，它关乎幼儿现在与未来的成长、关乎幼儿对于生命和美好生活的渴望、关乎幼儿精神和身体的养育。她也带领我们思考：“如何提升每个幼儿进餐时的幸福感”“如何养成与食物自然可持续的连接？”“如何以自己对世界的认识、习惯来影响幼儿”“如何撬动幼儿食相关的活动中容易被忽视的问题的解决”……一系列的追问引发了中心组的老师们的深深思考！

  正如老师们说的，贺老师提醒我们要回到初心，回到食本身，回到日常生活中，去真正意义上影响幼儿的身心健康。
