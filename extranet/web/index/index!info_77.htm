研思并进 蓄力成长 ——“童蒙养正，立德树人”大班幼儿德育实践研讨活动（交大闵幼专场）

发布时间 : 2023-10-04   作者（来源）: 顾莹   阅读 : 599

3/8
←
→

2023年9月28日，上海交通大学闵行幼儿园开展了“童蒙养正，立德树人” 大班幼儿德育实践的片级研讨活动，以实践观摩、教研分享、互动研讨、剖析点评等四个环节展开。闵行区教育学院教研员顾莹老师、南片公办幼儿园的大班教研组长出席了此次活动。

实践观摩  “研”之有道

由孟凡力老师执教大班社会活动《桃园三结义》，以传统故事为背景，孩子们通过观察、评价、演绎，尝试站在他人的角度去思考。活动以有趣的互动方式支持幼儿主动表达；以“剥笋式提问”引导幼儿倾听同伴、接纳与自己不一样的想法；以“对话”、“对手戏”的方式，有效梳理解决问题的办法，进而促进了幼儿自我意识的提升。

现场教研  “研”之有理

现场观摩后，保教主任谢斌英老师带领老师们开展了以“基于社会领域核心价值，研讨大班社会活动过程和方法”为主题的现场教研。基于《桃园三结义》研讨活动实施的有效性，聚焦“过程与方法对幼儿发展的推动性价值”，从环节设计价值点、互动方式两方面展开积极讨论，并结合现场观摩分析，说亮点、提想法，探讨大班社会领域活动的设计。

互动研讨  “研”之有获

紧接着，顾莹老师组织姐妹园的老师们进行交流互动，在宽松的研讨氛围中，大家各抒己见、畅所欲言。老师们结合教学活动，从目标、内容、环节、方法等角度肯定了活动：新颖的素材、灵动的设计；同时也诚恳地提出了建议：可以将“互动”做得更亮，将“可‘示’”做得更大；并针对每一个细节进行剖析，为完善本次教学活动提出了很多宝贵建议。

剖析点评  “研”之有悟

最后，顾莹老师对社会领域活动的价值判断、核心素养进行解析，指向“关键能力”和“必备素养”对活动给予专业点评：肯定了活动的设计、孩子的表现和教师的从容；并结合孩子的年龄特点，提出了在活动实施过程中如何对逻辑和判断有更多的思考，引发大家探讨在幼儿教育中“如何晓之以理、动之以情？”“如何以浸润方式讲道理？”为我们后续更好地开展社会领域活动提供思路、指明方向。

摄影、撰稿： 上海交通大学闵行幼儿园 谢斌英
