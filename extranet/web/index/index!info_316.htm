伊甸新田园 幸福乐成长 ——2018年闵行区新任校（园）长带教培训赵瑛群导师组活动通讯（1月）

发布时间 : 2019-02-18   作者（来源）: 曹雅明   阅读 : 3546

作为闵行教育的新成员，田园二外小开办至今始终得到了区镇两级教育主管部门的鼎力支持和殷切关怀，尤其在田园外小一体化办学扶持和颛桥镇学区化办学的大背景下，学校顺利达成了期初制定的“立规范、树形象、稳开局”的工作目标，赢得地区百姓的广泛认可，迈出了坚定的开局之步。

1月是盘点之时，也是展望之际，如何将期末的各项工作进行统整规划，发挥学期节点特殊的功效，我与师傅赵校长做了较为深入的探讨，最终决定借鉴田外期末智慧分享活动的形式，注入二外小的独特性，与期初的一体化办学启动仪式呼应，同时亦是一次重要的节点回顾与梳理。

一、智慧分享的不仅仅是做过的事，还有共同的教育理想

智慧分享的方案几易其稿，主要纠结的是智慧分享的核心价值是什么？我们分享什么？传递什么？摈弃唱唱跳跳的联欢，需要彰显的开局之年的探索、实践与成长。和师傅多次商议，加上之前在田外多年的活动策划经验，最终和管理团队共同确定了三个分享的主要板块：“伊甸小当家  担当中成长”学生工作分享、“一体化课程  分享中探索”课程教学工作分享、“师德提升工程 互动中引领”师训工作分享。组建了核心团队，明确第一责任人，进行分板块落实。整个活动的筹备过程就是一次极佳的实战培训，大家在推进调整中不断反思半年来的工作，思考价值与意义。

1月16日下午，主题为“伊甸新田园，幸福乐成长”的期末智慧分享活动顺利举行，上海市教研室教研员、特级教师朱浦老师，区教育学院朱靖院长、李群主任，颛桥镇马霞清副镇长、镇教委陈建时主任，闵行区督导室吴晓华校长，颛桥镇文体中心周海婕副主任、学区化办学成员校校长、田园外小一体化领导班子、社区共建单位的领导，以及田园二外小一年级部分家长代表莅临现场，共同见证学校的成长。

活动伊始，一部短片带大家一同回顾了一学期来的实践与探索。倪冰老师和一4班的孩子们通过别开生面的微型班队活动课，再现了学生在“伊甸小当家”活动中的担当与成长。教导主任陈淑萍老师带领四位不同学科的教师，聚焦“课程、教学、评价”，以微论坛的形式从“多元课程有效衔接、多维教研有效提升、多样评价有效撬动”三方面分享了学校在课程建设与教师主题研修中实践与创新。颛桥百姓剧社的朱瑞老师和6位教师，呈现了一场微型的教育戏剧培训课程。通过教育戏剧体验的方式开展师德培训，凝聚人心，统一认知，激发教师的教育热情，体悟育人的初心。

分享会上，通过一个回顾视频，展现了一学期来各方面的探索与成效，朱靖院长揭晓了首届“四叶草教师”获奖名单，马霞清副镇长为新成立的的“精灵剧社”和“启跳篮球”项目负责人颁发了聘书，由全体学生和教职员工共同参演的迎新短片《这就是学校的样子》将活动推向了高潮，既送上了新年的诚挚祝福，又开启了崭新的教育新篇章。最后，上海市小学英语教研员、特级教师朱浦老师从“了解、走进、畅想”三方面做了精彩的发言：田园外小是家门口好学校的标志、是外语类公办学校的榜样、是教师成长的摇篮、是国家意志和国家课程得以实施的优质教育场所。优质办学经验的复制，最难的就是教研能量和课堂实施力，但二外小呈现的就是“同一个田园”，办学理念先进、学校环境优美、师生阳光自信，初步实现了高起点的办学。对一所学校来说，理念建校、学科立校、课程文化名校、教师强校，期望年轻的二外小能在国家课程优质化实施的基础上，建设学校特色课程，打造专业教师队伍，用教师队伍推动学科的强健，用学科的强健引领学校的前行。

一个半小时的活动浓缩了一学期的成长与感悟，我坐在底下欣赏着台上每一位教师的精彩亮相，无比激动。“TSFL”是我们共同的名字，我们与学校同生共长，组建仅半年的团队展现了责无旁贷的使命感和卓而不群的教育追求，让我看到了学校未来强大的发展源动力。市区镇各级领导和专家的鼎力支持与热情洋溢的讲话，给予了莫大的鼓舞，身处新时代，教育人的使命与担当，让我们内心炽热。

二、工会策划的不仅仅是教工活动，还有共同的精神家园

学校工会是全体教职工共同的组织，既维护每一位工会成员的合法权益，又可以通过形式多样的工会活动来凝聚人心。我校的工会组建时间尚短，平时的工会活动尚未能常态化开展。适逢学期末，在完成各项期末结束工作的同时，我们共同策划并组织了一场别有趣味的“qinqing伊甸园  浓浓关爱情”系列工会活动。

1月22日上午，全体教职工汇聚在三楼音乐教室，开展了一场别开生面的教育戏剧体验活动。今年我们学校有3位老师参加了颛桥镇“百姓剧社”的培训学习，他们成为了教育戏剧的“种子教师”，将培训资源进行校本转化，带领全体教职工参与到戏剧艺术的实践活动中。大家通过身体接龙游戏、一句话台词情境表演、故事串联表演、学科游戏活动创编等不同的活动形式，使每位教师都在愉悦互动中释放自我，在热情参与中分享快乐，在体验创造中生成智慧。尤其是最后的“我记得”分享环节，大家敞开心扉分享了戏剧活动带来的别样感受，也道出了与年轻的二外小共同成长的难忘瞬间与感人故事。

中午，老师们又来到了食堂，大家围在一起，有的包起了馄饨，有的包起了饺子，有的在耐心地教，有的在认真地学，一招一式有板有眼。不在意包得如何好，不在乎包得有多快，大家聚在一起，有说有笑，就像是家人一样快快乐乐地迎新年。等食堂的师傅端上来一碗碗热气腾腾的馄饨饺子，吃着自己亲手包的，觉得特别暖心好吃。

下午，学校行政分成几个小组，对全校教职工进行了慰问走访。送上温暖的祝福，感谢老师们一学期来的辛苦创业；送上温暖的关怀，关心老师们的生活，了解他们的所思所想。鼓励每一位教职工继续脚踏实地，为田园二外小美好的明天齐心协力、努力奋斗！

学校的工作千头万绪，在推进各项工作的同时，一定要心中有校，目中有人，摒弃私信杂念。校内的各岗位没有贵贱之分，只有分工不同，我们要成为相亲相爱的一家人才会把学校的发展看成己任，并为之付出自己的汗水和智慧。

送走了幸福启航的2018，迎来了充满希望的2019。全体田园二外小人将始终不忘教育初心和使命，砥砺前行，与田园共芳华，一起努力成为更好的自己。
