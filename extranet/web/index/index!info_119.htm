结 对 跟 岗     同 研 共 长

--记2022年云南保山市校（园）长、教师闵行跟岗研修结业仪式

11月18日上午，2022年云南省保山市校（园）长、教师闵行跟岗研修班结业仪式在闵行区实验小学举行。本次结业仪式由闵行区教育学院教师发展中心骨干培养管理办公室主任兰平主持。闵行区教育党工委副书记李光华、闵行区教育学院校长发展中心朱凤老师以及来自保山市跟岗研修的校（园）长、教师等共40余人参加了此次结业仪式。

结业仪式上，来自保山市各学段跟岗老师代表吴红翠、徐华昌、鲁建臣和朱茁萱分别发表了跟岗研修感言。他们分享了两周跟岗研修的学习收获，交流了在跟岗学校学习过程中所见、所学、所思、所感、所悟，同时也表达了对闵行区教育局、区教育学院和各跟岗实践学校的谢意，特别强调了结对导师手把手、心贴心的带教，让他们倍感温暖。他们表示，将把这次在闵行区教育学院和各跟岗学校学习中收获到的好经验好做法都带回去，返回后及时做好培训梳理与归纳，并与同事分享与交流，在工作实践中再做深入提炼与总结，争取把这次跟岗研修活动的“种子”带回去，在保山教育的土壤用心耕耘，根植出美丽的花朵。

保山市教育教学研究所李娟老师作了跟班研修总结发言。李娟老师表示在这次跟岗研修学习中，学员们学到了如何全方位看待和思考教育，如何在“双减”政策背景下，发挥教育工作者育人的使命，如何在高质量发展的愿景下，开展自我创新。

李光华书记给研修班学员代表颁发了结业证书并作领导讲话。李书记首先表扬学员们排除地域和生活习惯等困难，以始终如一的热情顺利完成学习任务的工作态度，祝贺老师们获得了感悟和收获。同时也给参训学员提出了希望，他希望老师们回滇返岗实践能深入思考，要做到学以致用，也希望跟岗研修的校（园）长和老师们后续能加强联系，常来常往，进一步深入交流合作，共促成长。

结业仪式在学员们热烈的掌声和依依不舍的道别气氛中结束。这种浸润式的教育帮扶工作，不仅让每一个学员收获了专业成长，还让沪滇两地的教育工作者收获了友谊和精神食粮，提升了两地教师的岗位效能。相信带着这份收获，后续会有更多的精彩！
