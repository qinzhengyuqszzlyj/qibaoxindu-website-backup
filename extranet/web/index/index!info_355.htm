支持幼儿经验连续性生长的集体学习活动 ——闵行区北片幼儿园主题教研活动（虹鹿幼儿园专场）

发布时间 : 2023-11-17   作者（来源）: 张静   阅读 : 261

3/8
←
→

2023年11月15日，沐浴在初冬的暖阳下，主题为“支持幼儿经验连续性生长的集体学习活动”闵行区幼儿园北片教研活动在虹鹿幼儿园举行。在闵行区教育学院学前教研员张静老师的带领下，北片的保教主任们参与了本次活动。

虹鹿幼儿园保教主任罗老师主持本场活动，在热烈欢迎大家的到来后，罗老师指出今天的片区开放活动意在关注幼儿的寻常时刻，践行“幼儿发展优先”的理念，聚焦大班幼儿的交往合作方面的已有经验和真实问题。   

陶老师执教的大班社会活动《有趣的“传送带”》，引导幼儿在“接力传送小球”的游戏情境下，尝试与同伴分工、合作、协商来解决问题，现场运用表征记录、战队PK、集体挑战等形式梳理、感知合作，并在游戏中获得与好友合作带来的愉快情感体验。

针对上述集体活动，虹鹿幼儿园大班组的老师们在教研组长黄老师的带领下现场展开教研，聚焦活动设计，深入探讨对幼儿合作行为与教师支持策略的观察，梳理教师促进幼儿经验生长的有效策略，提升教师的观察与支持能力。同时，紧扣本次北片教研主题关注幼儿合作经验的连续性生长，在观点的碰撞中促进大班幼儿有效合作交往策略的形成与梳理。

随后，张静老师组织大家探讨大班幼儿人际交往方面的经验连续性生长“话题，并对集体教学活动与现场教研分别做了精彩的点评和总结。她对执教老师与大班教研团队给予了高度的肯定和赞赏，同时，对于活动中教师如何更好地把握集体教学活动的节奏与优化内容选择、更好地根据幼儿的需求灵活生成，以及教研组织中如何引发更深层次的观点碰撞等方面，提出了可行性的建议，让在场参与活动的教师都受益匪浅。活动最后，张园长感谢张静老师给予展示的机会与精彩的点评，感谢各位北片同行一行的宝贵意见与骨干教师们的参与！

本次片级研讨活动，为虹鹿幼儿园的教师搭建了专业历练与成长的舞台，对前来交流学习的保教主任与骨干教师也是一次思维的激荡。在不断实践、反思、交流的过程中，教师专业能力得到了提升，共同为促进幼儿经验连续性生长而努力。
