基于新课标，探索新课堂----第15周小学体育区级教研活动

发布时间 : 2023-05-25   作者（来源）: 顾佳伟   阅读 : 405

3/8
←
→

2023年5月24日下午，以“基于新课标 探索新课题”为主题的闵行区小学体育教师研讨活动在闵行区浦江第三小学举办。本次活动由闵行区小学体育种子基地主持人、浦江小学教育联盟体育名师工作室主持人、闵行区中心组正高级教师吴玉莉老师主持，参加的老师们来自闵行区小学体育种子基地、浦江小学教育联盟体育名师工作室、区七校联合教研的全体老师们。

此次的研讨活动由浦江三小孙明老师和曹晓豪老师执教。他们带来了两节新兴体育课，给老师们都有眼前一亮的感觉。


                                             课堂观摩

先是由孙明教师执教的二年级《拳击—直拳》第三课时。根据低年级学生的年龄特点，孙老师创设了“熊熊森林保卫战”的故事情境。老师化身为熊老师，带着小熊熊们成为“森林小卫士”进行练本领、打害虫、森林灭火，让学生在情节中学习拳击，这不仅提高了学练氛围，还提高了学生的学习兴趣，真正体现了兴趣化教学。课中，孙老师还采用了“光电球”与多媒体信息技术相结合，不仅激发了学生学练兴趣，而且课中利用“光电球”技术，及时采集课中数据，及时反馈评价，学习目标的达成度使看课老师一目了然，真正提升了课堂教学的时效性。

第二节课是曹晓豪老师执教的五年级《极限飞盘—反手掷飞盘》第三课时。极限飞盘属于新兴体育运动，作为五年级《体育与健身》拓展教材之一，曹老师充分发挥了学生的学习自主性和对新兴项目的好奇心，让学生们通过探究式学习，从观察视频动作到学生自学，再是发现问题采取教学游戏，逐步改进动作，最后到教学模拟比赛。学生从观察、模仿、思考、学练、比赛这几个环节中进行学习，这不仅提高了学生的学练能力，真正给了学生学习的空间，发挥了高年级学生自主学练能力，而且把新课标中的“学、练、赛”一体化教学呈现的淋漓尽致。

                                       

                                            说课评课

课后，两位老师分别围绕自己的课堂阐述了教学设计意图，孙明老师利用学校已有拳击社团项目的开展，了解孩子们对拳击兴趣浓厚，所以他借助校本资源拓展了拳击课程，使新兴体育项目真正融入课堂；曹晓豪老师从自身出发，喜欢玩极限飞盘项目，因此把这项目带进课堂与学生一起分享，但他又站在小学生安全的角度考虑，飞盘换为了柔软的橡胶材质，这样使学生的练习更具有安全性，让学生也能放心大胆的区学习。同时两位老师也对自己课堂中存在的不足进行了反思，也虚心接受各位老师们的宝贵建议。

接着，蔷薇小学、上师三附小、吴泾实验小学、田园外语实验小学等老师们纷纷提出了自己看课后的想法，充分表扬和肯定了两位老师敢于创新、敢于追求的精神，称赞课堂给大家提供了众多的教学思路，值得大家带回家尝试。同时老师们也给到了更多的教学建议，让此次研讨活动达到了举一反三，触类旁通的。

吴玉莉老师对两位青年教师的第一次公开课表达了充分肯定，首先称赞老师们在前期花了很大功夫，不光是体育技能的钻研，还有教学方法、手段的研究，体育器材的设计、自制能力等多方面的研究，借此也鼓励青年教师们多学习、多钻研，尽可能的多方面展示自己的特长，用来辅助自己的课堂教学；其次引导老师们在进行新兴课程开发时，尤其是成人运动之类的拓展项目，需转化成孩子们能够接受的、具有童趣化的运动项目，要注意到学习的主体；最后吴老师也提醒各位青年教师们对于新课标的解读与实践，仍需加强专研，要多思考、多实践、多创新。
