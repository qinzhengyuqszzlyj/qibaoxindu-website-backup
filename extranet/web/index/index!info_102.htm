跨区交流探特色，交流互动促成长 ——闵行区小学校长（田园外小）基地在奉教院附小举行培训活动

发布时间 : 2023-04-21   作者（来源）: 曹雅明   阅读 : 618

3/8
←
→

为更好立足于办学实践问题的探索与改进，提升学员对教育的理解力、领导力、执行力。4月19日下午，闵行区小学校长（田园外小）基地导师赵瑛群校长带领基地学员与带教学员，一同赴上海市奉贤区教育学院附属实验小学参观学习。

参与本次活动的导师与专家有：特级校长、全国教育学会小学教育管理委员会理事，奉教院附属实验小学何哲慧校长；特级校长、正高级教师，田园外小赵瑛群校长。以及校长基地学员：明强二小郭芳校长、鹤北小学蒋红梅校长、昆阳路小学夏权书记、浦江二小赵美华书记等。

在何哲慧校长带领下，导师和学员们充分感受奉教院附小“自主生产劳动场域”的文化建设、良好的全员育人氛围、体建设学校的文化环境，木创教室、纺织空间、学生超市等都给大家留下深刻印象。余安勤副校长对学校如何开展劳动教育工作进行了详细解说，介绍学校因地制宜，开辟校园劳动基地，分年段、分季节、分品种，按兴趣、按科目分门别类进行多样化、灵动性的生产劳动，构建全学段一体化的劳动教育体系。

参观结束后，何哲慧校长与学员们进行了互动交流。何哲慧校长谈到，学校的办学历程，三校区建设情况，集团所带6所成员校的共研共育模式等。就如何培育学校管理者，支教辐射骨干教师流动，一同教研和一同融合督导，全程参与集团的学校的教育教学全过程，这也为教师的发展和基础性工作做出贡献，评价等各方面也要做细做实，引领管理的整体智慧等做了介绍。

学员们也一同探讨与交流了感受，分享了自己办学和区域建设中的所思所想。感叹学校大格局办学，岗位聘任与梯队建设对教师的促进，新机制激活了教师的主动性，循环式的流动为教师提供了新的机遇，骨干培养与输出为区域带来的活力。共同就综合落实各种政策制度、学校优质资源衔接、融于文化和新颖的教育设施设计等如何助推学校高质量发展等开展了互动交流。

最后，赵瑛群校长对本次交流活动做了总结，她表示奉教院附小营造的良好育人氛围令人交口称赞，劳动育美，让每一面墙都会说话，让每一块草地都会抒情，让每一个角落都彰显文化。

本次校长基地的跨区参观交流，不仅加强了学校间的深厚友谊，更让学员们对于奉教院附小的文化理念、课程教学、师资建设、评价体系等方面有了更深的了解。后续校长基地将以更加开放的姿态，与其他学校一起相互交流学习，携手共进，奋力前行！
