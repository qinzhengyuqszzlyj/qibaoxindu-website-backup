普特并举，其乐“融融” ——梅陇金都幼儿园区际联合教研活动

发布时间 : 2023-05-19   作者（来源）: 潘静   阅读 : 784

3/8
←
→

5月18日，梅陇金都幼儿园举办了以“普特并举，其乐‘融融’”为主题的区际联合教研活动。上海市教委教研室邱轶老师，闵行区特教教研员潘静老师，杨浦区鞍山幼稚园顾骁嫣园长，杨浦区特教教研员李静老师，杨浦、闵行两区特教班点的教师共同参与了此次活动。

虽然前日正逢甘霖，户外场地潮湿，但丝毫不影响孩子们的运动兴致。室内走廊上、教室里，活动室中，随处可见孩子们矫健的身影。他们在趣味性、情境化的路线中自由穿梭，在高大的攀爬墙上挑战自己，从室内滑梯上倾斜而下，尽情挥洒汗水，如此活力四射！其中有几个特别的身影，虽然动作不够灵活协调，但依旧不断地尝试，收获了同伴们的“加油！”，他们备受鼓舞。运动中，我们也会欣喜地看到孩子们抢着为特殊小伙伴们擦汗、倒水、整理垫背巾的有爱场景。

融合运动活动观摩结束后，老师们来到多功能厅，郁彩英园长向今天参与活动的人员表示了热烈的欢迎。郁园长简单介绍了我园特教班的办学情况、融合课程的开展状况等。同时，衷心感谢市、区级教研员在前期给与的指导和帮助。感谢他们能搭建平台，通过区际联合教研的形式，促进两区教师互相学习，共同发展。

接着沈靓靓老师带来了大班融合活动《笔先生》。结合《我要上小学》的主题，让幼儿在情景式、游戏化的活动中，发现笔的特征并进行借形想象，激发幼儿的艺术表现与创造能力。活动过程中，沈老师善用多媒体资源，引导孩子细致地观察，大胆地联想，自信地表达表现。并充分给予特殊幼儿机会，引导同伴互助。分享环节中，画作的呈现和同伴的赞美，让特殊幼儿在融合活动中收获了巨大的成就感。

短暂的茶歇后，杨茗老师进行了《我用时光，记录下你的融合之路》的个案分享。跟随杨老师的镜头，老师们对刚刚参与融合活动的特殊幼儿有了更深入的了解。看到他如何从一位社交退缩的幼儿，成长到现在的大方自信。杨老师在坚持融合的同时，运用同伴结对、个别化训练、问题行为矫正等多种支持手段，帮助特殊幼儿拉近与普通孩子的距离。使用视频、卡片、社会故事等支持手段，锻炼幼儿在各类场景中的社交技能。我们从孩子巨大的转变中，看到了老师的付出和大爱。

 接下来沈靓靓老师进行了执教反思，在座的各位特教班点老师对教学现场进行点评。大家认为沈老师的活动设计非常符合大班孩子的年龄特点，教师也非常善于启发，敢于放手让孩子们进行自主地表达表现。

专家点评环节，顾骁嫣园长充分肯定了我园融合环境的创设，丰富的自然环境和和谐的人文环境，共同造就了一个包容的氛围。让特殊幼儿能全程保持身心愉悦的状态，参与到融合班级的各类活动中去。杨老师的个案分享，让大家对特殊幼儿的融合历程有了更清晰的了解，也感动于特教教师为矫正特殊幼儿问题行为所做出的努力。

李静老师指出，幼儿园的教育是一个人终身教育的重要组成部分。沈老师能考虑到幼儿处于幼小衔接的阶段，运用常见的学习物品——笔，合理地进行主题式教学设计。孩子在老师的循循善诱下，发展了有意注意、细致观察、语言表达、创意想象等多种能力。助教杨老师的共同协同和个别辅导，充分考虑了特殊幼儿的需要，对其进行了有效的支持。融合是态度、是思维、是目标、是观念，我们要关注幼儿核心素养的发展，创设有意义、有挑战、有温度的融合教学体系。

孩子们的阳光和活力深深地感染了邱轶老师，他高度赞扬了我园室内运动场地开放、情景创设十分有趣，给与了孩子自由、自主的运动机会。他鼓励老师们要坚定幼儿优先的理念，让孩子敢于探索、乐于分享、勇于表达。并指出学前特教的团队现阶段还比较弱小，希望通过更多的区际教研活动，提升特教专职教师、融合教师的业务能力，期待一批批既有专业，又有爱的老师能够涌现出来。

最后潘静老师用“敬业、专业”四字来进行总结，充分肯定了幼儿园所有教职员工在前期的准备，为在坐的各位特教同仁们带来了一场别开生面的开放活动。在所有环节中，老师和孩子们都呈现了非常真实、自然的活动状态，从容淡定，张弛有度。同时，她提醒特教教师要做好资料积累工作，用多种形式跟踪和记录特殊幼儿的成长。

普特并举，其乐“融融”。融合活动在我园的实施，顺应了当前学前教育改革的需要。特殊幼儿获得了更多社会交往的机会，普通幼儿也在融合中学习爱和包容。金都幼儿园的所有教职员工，会用自己的爱心、耐心、责任心，努力做好的教育，做让家长们放心的教育。期待在融合的蓝天下，普特幼儿能够温暖携手，共同成长！
