托班微世界，邂逅微时光 ——闵行区南片幼儿园托班教研活动在马桥富国幼儿园举行

发布时间 : 2023-05-27   作者（来源）: 鲁鸣   阅读 : 635

3/8
←
→

2023年5月25日上午，上海市闵行区马桥富国幼儿园进行了一场《托班微世界，邂逅微时光》的托班主题教研活动。闵行教育学院教研中心学前教研室鲁鸣老师带领区南片各幼儿园托班教师、马桥街镇联合教研组教师齐聚马桥富国幼儿园，共同参与活动。

现场观摩·个性教学凸显园本特色

活动伊始，马桥富国幼儿园的老师们呈现了托班活动。其中由马桥富国幼儿园赵吉莉老师和俞蓓老师展示了圆圈活动《爆米花》和《可爱的小脚丫》，以游戏化的形式进行开展活动。幼儿在《爆米花》中尽情翻滚，“小爆米花”们，翻、爬、滚样样在行，并以蘸取果酱请客人老师品尝的方式，大胆表达自己的感受。《可爱的小脚丫》活动里，俞蓓老师邀请幼儿脱去鞋袜，让他们在小脚丫踩印的世界里尽情释放天性，展露童真。

马桥富国幼儿园吴婷老师和李守燕老师顺应托班幼儿的兴趣，适时开展了《沙沙，你好》和《多彩蛋壳》的小组游戏。《沙沙，你好》活动是吴老师在观察幼儿玩沙的过程中适时发起的活动。活动中，吴老师引导幼儿触摸沙、玩沙，并让幼儿利用多种材料进行沙画创作。《多彩的蛋壳》活动中，李守燕老师借幼儿在草地上发现的蛋壳秘密，顺势进入游戏。情景式的互动鼓励幼儿自由表达创意。活动中，教师借助户外自然环境的优势，满足孩子的“我要玩”，顺应孩子的“随意玩”，实现了孩子的“开心玩”，并用照片的方式及时捕捉了幼儿的精彩瞬间，一方面用了拍立得，即拍即现，另一方面用了移动多媒体“小天”，方便孩子们后续查看和欣赏，为孩子们的成长留下痕迹。

研讨交流·思维碰撞促进内生成长

马桥镇联合教研组组长吴婷老师携教研组教师共同开展了马桥镇联合教研活动，在教研初期，大家一致认同师幼互动是一切进行教养活动的基础，因此在当日观摩活动后，深度研讨了《幼儿发展优先理念下的托班师幼互动的研究》。

过程中，联合教研组组员各抒己见。马桥富卓幼儿园的缪绿群和马桥实验幼儿园的胡媛媛老师都从亲密关系展开了阐述，提出教师要给予幼儿温暖的情感支持。马桥富国幼儿园的几位老师从托班特有的活动组织形式——小组游戏和圆圈活动进行讨论，认为在活动支持方面，教师应追随孩子、顺应孩子，满足他们的需求。马桥元祥幼儿园的何嘉琦和马桥富杰幼儿园的曹国焕老师分别从环境的创设和自然、童趣的材料投放进行了分享，并一致认为师幼互动除了关注教师与幼儿之间发生的相互作用，也要重视幼儿与环境和材料发生的互动。

在教研中，老师一起贡献智慧、共享经验、积淀成长。作为托班教师，我们要处理好“顺应发展”和“推动发展”的关系，既要站在儿童立场，顺应幼儿的天性和需求，也要把握教育规律，创设适宜的生活环境，推动托班幼儿的身心健康发展。

互动点评·托班教养实践有方向

活动最后，闵行教育学院教研中心学前教研室鲁鸣老师就针对今天的展示活动给予点评和解析。鲁老师肯定了马桥富国幼儿园在幼儿发展优先理念下师幼之间的良好互动关系，认为教师之间的思维碰撞是促进教研走向深度聚焦和实现突破的良好方式。同时，她提出了对托班教师发展的期许，希望托班教师能够不断精进专业，在0-3岁婴幼儿教养实践方面走出自己的专业发展道路。
