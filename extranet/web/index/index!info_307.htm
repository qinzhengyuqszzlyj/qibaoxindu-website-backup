规范管理 促进优质办园——2018年新任校（园）长带教林琳导师组活动简讯

发布时间 : 2019-03-25   作者（来源）: 朱凤   阅读 : 3406

新园长进入老园所，如何快速地融入幼儿园？如何评价幼儿园的工作？面对一级园的验收，幼儿园又该如何有序高效地开展工作？如何做到既规范又优质？带着这些困惑与疑问，3月19日下午，导师林琳督学带领学员李小红、王毅、浦燕群三位新任园长来到梅陇镇中心幼儿园，一起聆听了上海市教育评估院基教处原处长叶令仪老师的专题讲座《优质幼儿园规范管理那些事》。

针对新任园长面临的诸多问题，叶老师建议大家多听、多看、多问，了解幼儿园的现状，摸清家底。她指出，园长要学会解读幼儿园已有的验收报告，从报告中读懂短板；分析情况，找出优势以及制约发展的瓶颈问题，剖析问题，找准切入口；和幼儿园的团队进行磨合，形成合力，达成共识；同时作为园长还要有脚踏实地的实干精神，才能逐步推进幼儿园的工作。

围绕如何规范管理幼儿园，叶老师结合上海市优质园等级验收指标，着重对幼儿园规划制定、环境创设、特色发展等方面进行了具体而详尽的诠释。她强调，好的环境应该是安全、卫生、整洁的，也应该是温馨、充满童趣的，能凸显幼儿的年龄特点以及教育性。同时她还提醒大家：在不同的活动区域要体现不同的功能和布局特点。

最后，导师林琳督学对本次活动进行了总结归纳，她要求新任园长们在传承与发展的过程中要处理好五对关系，即：客观了解与主观认知的关系；融入团队与坚持原则的原则；仰望团队与脚踏实地的关系；彰显特色与全面推进的关系；验收准备与常态管理的关系。

学员们深受启发，也促使大家进一步梳理和反思以往的工作，只有合理规划，有序开展，规范管理，才能使幼儿园更优质，更健康地发展。
