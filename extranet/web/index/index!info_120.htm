【学前 儿童文学项目组】关注每一个，阅见幸福

发布时间 : 2023-06-12   作者（来源）: 刘时医   阅读 : 854

3/8
←
→

 6月12日，在这个晴空灿烂、绿树成荫的季节里，区中华经典儿童文学项目组迎来了2023学年第二学期的最后一次线下主题教研活动。在项目主持人、闵行区教育学院学前教研员刘时医老师的组织带领下，中心组全体成员齐聚闵行区浦江瑞和城幼儿园，观摩了小班、大班集体教学活动并进行了主题教研。

浦江瑞和城幼儿园沈思怡老师执教大班活动《做梦》，走进神秘的“造梦工房”，阅读、循证梦的“趣”，引导大班幼儿细致观察和基于话题大胆表达。通过自主阅读，鼓励孩子发现细节，并进行比较观察和连续观察，积极表达自己的想法。

莘松幼儿园朱烨老师执教小班活动《谁的袜子》，以“寻找袜子的主人”为主线，让小班幼儿沉浸式阅读，解开一个个谜团。朱老师充分利用绘本和幼儿已有经验，引导幼儿通过观察、比较、操作等方法，发现问题、解决问题，从而体验寻找的“难”和“乐”。

教学观摩之后，项目组紧锣密鼓地开展了主题教研。教研中，项目组老师们针对活动现场大胆表达自己的想法，刘老师进行点评。随后，智慧传递环节，刘老师分享了集体教学活动中不仅要关注活动的设计和组织，更要关注幼儿的学习状况，关注每一个，提高“每一个”的获得感。

我们将继续秉持“幼儿发展优先”的理念，做会思考、懂孩子的教师。

 

撰稿与摄像：闵行区浦江瑞和城幼儿园
