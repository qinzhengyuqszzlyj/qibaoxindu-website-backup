基于精准教学，落实学科核心素养 ——区初中生命科学教研活动在上海市华漕学校举行

发布时间 : 2023-05-16   作者（来源）: 戴赟   阅读 : 650

3/8
←
→

5月10日，本学期第8次主题为“基于精准教学，落实学科核心素养”的区生命科学/生物学教研活动在华漕学校顺利举行。活动由区教研员戴赟主要策划，华漕学校莫晓连执教，莘光学校张彦主持。磨课团队包括区教研员戴赟、中心组老师张彦和宋丹婷、校学科组成员徐宗华和陆荣。全区初中生命科学教师参加了本次活动。

首先，大家观摩了莫晓连老师的《分类检索表》一课。她先通过学生课前收集的植物学习单导入新课，利用精心设计的“使用二分叉检索表”、“编制二分叉检索表”和“使用平行检索表”等三个学生活动引导学生认识不同检索表的共同特征与区别、体会检索表的功能，探索如何使用检索表这一科学工具。教学过程中，莫老师借助希沃白板技术，分享学生学习单完成情况、共同归纳使用检索表的方法，引导学生在团队中借助已有的知识储备和学习能力进行互补式学习，突破学习难点，精准地实现了每一个教学环节的目标落实。

交流研讨环节主要分教师说课、教研组建设、小组教学研讨与交流及组间交流总结四部分。莫老师的说课着重介绍了教学设计思路以及本节课的教学设计创新——以学校植物为情景导入；技术支持创新点--希沃白板的有效运用。教师以新课标中的核心素养、重要概念为落脚点，围绕两个关键问题“科学家是怎样的对生物进行分类的？”、“人们对生物进行分类的意义是什么？”对本课时教学内容进行了精心的设计。徐宗华老师介绍了华漕学校理科组教研组情况及其特色，其中生物备课组的一些亮点作业吸引了很多教师的关注和兴趣，值得大家学习和借鉴。

随后，各片组形成研讨小组，教师们从如何进行“基于精准教学，落实学科核心素养”及“融合信息技术 促进深度学习”两个方面进行专题式的研讨和交流。大家对莫老师执教的这节课给予了高度评价：充分利用校园动植物资源作为真实情境素材激发学生学习兴趣、提升学习价值，利用问题串逐层推动教学培养学生学科核心素养，信息技术助力课堂深度互动等。就这个单元中难度最大的课时，如何通过教材分析、学情分析、教学目标与策略、技术应用等的精准化实施有效教学，莫老师的课给予了大家诸多启发。同时，老师们也给出了多方面、多途径的优化改进建议。

本次教研活动，老师们再次对接新课标，学习了如何在技术的支持下，通过精准教学设计，有效驱动课堂深度互动，达成教学目标的有效落实。我们初中生命科学学科教师团队也将在闵行区智慧教育转型的浪潮中，主动积极探索技术应用，发展学科特色，落实学科素养。
