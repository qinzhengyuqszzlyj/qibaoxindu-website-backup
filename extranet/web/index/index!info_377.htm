学习“二十大” 夯实评价结果 赋能素养提升 ——闵行区小学五年级道德与法治学科研讨活动在交大实小举行

发布时间 : 2023-05-16   作者（来源）: 王玉兰   阅读 : 940

2023年5月11日下午，以“学习‘二十大’夯实评价结果 赋能素养提升”为主题的闵行区小学五年级道德与法治学科研讨活动在上海交通大学附属实验小学举行。本次活动邀请到闵行区教育学院道德与法治教研员王玉兰老师莅临指导，闵行区道德与法治中心组成员、课题项目组成员、区内东南片五年级小学道德与法治老师以及家长志愿者共同参与。活动由交大实小陈晓琳老师主持。

本次活动分为课堂观摩、主题论坛、专家点评等几个板块。

一、智慧教育入课堂，教学观摩展风采。

杨永茹老师执教的是五年级下册第一单元《公共生活靠大家》中的第3课《我参与 我奉献》第三课时。

在设计教学环节时，杨老师结合《习近平新时代中国特色社会主义思想 学生读本》《党的二十大报告》以及《义务教育道德与法治课程标准（2022年版）》等文件，对教材进行了深入的分析和解读，并通过前测问卷和观察走访的形式充分了解学情，立足于学生实际，制定了指向培养学生核心素养的教学目标。

在课堂教学过程中，杨老师先出示《德宏邻里公约》，引导学生初步理解公约内容。接着，借助智慧笔进行现场调研，根据数据反馈，让学生理解公约的作用。聆听音频，辨析明理，再修改和补充邻里公约，让学生了解在社会治理共同体中，需要人人有责、人人尽责、人人享有。最后，邀请社区志愿者奶奶进课堂，讲述社区治理过程中的烦恼和困惑，引导学生用创新、开放的方法为社区治理建设谏言支招。

课堂结束时，杨老师带领学生学习了习语金句，结合《习近平新时代中国特色社会主义思想 学生读本》中的“我们从小就要立志向、有梦想，用新理念、新本领去适应和创造新生活”，引导学生树立远大理想，用实际行动践行“习思想”。

二、思维碰撞共研讨，教育智慧齐分享

本次论坛的主题是“教育技术深层融合，夯实观察记录反馈”，由马桥文来外小宋霞梅老师主持，虹桥小学殷超、交大实小杨永茹、陈晓琳老师共同参与。

陈晓琳老师从“技术融入，创设单元评价指导新思路”这一点出发，首先介绍了在新课标和“习思想”引领下，部编版道德与法治五下第一单元的整体规划。其次，从“基于教学目标，科学制定评价规则”和“基于资源选择，以课堂教学需要为出发点”两个方面阐述了深层融合教育技术，发挥表现性评价的最大作用的方式。

杨永茹老师从“多线混融，构建智慧课堂评价新范式”与大家分享了她的课堂实践。首先，创新应用，及时反馈评价。课前任务辅学，探寻认知困惑，课上智慧驱动，促发探究学习，课后成果评价，强化道德践行；其次，融合情境，丰盈课堂评价；最后，资源赋能，提升课堂价值。

殷超老师从“资源赋能，聚焦单元学习历程新质效”这一方面着手，通过观测记录前测评价，数据资源赋能教学设计“诊断反馈学习表现，任务资源赋能课堂教学”和“更新迭代知情意行，案例资源赋能成长方向”三点内容，阐述了观察作为评价和后续改进的前提依据的重要性。

三、专家引领明方向，启思明理促提升

闵行区教育学院道德与法治教研员王玉兰老师对本次教研活动做专家点评。王老师强调，大思政课堂要具备三个方面的要求。第一是政治方向，思政课堂要坚持正确的政治方向，结合党的二十大精神、“习思想”，挖掘道德与法治课堂蕴含的时代意义和价值，充分体现学科特性。第二，价值引领，要让学生的情感尤其是道德情感参与到学习过程中，增强学生真体验，深化思想认识，生成真感悟，引导学生认同价值取向，才能使价值引领深入人心。第三，知行要求，思政教师要注重多元评价，以“知行合一”为导向，做到多角度去分析问题、解决问题，充分发挥道德与法治的学科育人功能。

习近平总书记强调：“思政教师，要给学生心灵埋下真善美的种子，引导学生扣好人生第一粒扣子。”相信在教研员王玉兰老师的引领下，闵行区道德与法治的小伙伴共商共研共成长，踏风破浪坚守初心，且行且思且研，落地学科核心素养铸魂育人，思政小课堂必会异彩纷呈，提升课堂育人实效。


撰稿：上海交通大学附属实验小学  陈晓琳

校稿：上海市闵行区中心小学  刘丹
