闵行区教育系统举办2019年青年后备干部培训：敢担当 善作为 能成事

发布时间 : 2019-12-25   作者（来源）: 曹雅明   阅读 : 4268

3/8
←
→

2019年12月17日至20日，闵行区教育系统2019青年后备干部集中封闭式培训在奉贤区委党校举行。区教育党工委书记朱雪平、教育局局长恽敏霞，区纪委监察委第七派驻组组长朱寿生，教育党工委副书记李光华，副局长马秀明，区教育学院院长朱靖、副院长龚耀昌、董雪平等领导出席，全区中小幼学校、直属单位、教育局机关的61名青年后备干部参加培训。

本期培训围绕“党性修养与使命担当、能力要素与领导者素养、基础教育改革与发展趋势、现代学校管理策略与方法、校内外教育资源的有效调适、公务礼仪与公文写作”六大方面展开。区教育党工委党群科科长林火生、教育学院校长发展中心主任俞琳在开班仪式上对培训的目标、内容及要求作解读，希望青年后备干部能通过本次培训，坚定理想信念，提高政治站位，关注改革形势，突破思维局限，努力成长为具有全局视野的新时代教育管理者。

四天的培训共安排主题报告12场、互动交流6次、分组研讨3场、实地参观学习1次，教育局领导、高校教授、区域优秀校长书记等主讲并参与活动。

华东师范大学教育学系教授宁本涛作首场报告——《坚持五育并举，践行新时代立德树人新要求》。他指出，德智体美劳相互融合，是教育深入发展的要求，更是人才成长之基。作为一名青年干部，要明确党的教育方针是培养德智体美劳全面发展的社会主义接班人和建设者，立德树人是教育的根本。

朱靖、沈珺、赵瑛群、康旻分别以《学校管理中的几个关键问题》、《“借力”助推学校发展》、《引领自觉 群体成长——教师队伍与中层干部队伍梯队建设》、《党建引领，构建学校发展的坚实壁垒》为题作现代学校治理专题报告，并与学员互动交流；何学锋、张小娟、张忆围绕青年干部成长与领导力提升这个专题，分别作《如何做一名有发展潜力的校长》、《有效运用激励机制，推动学校持续发展》、《关于学校内涵建设着力点的思考与行动》的报告。这些资深学校管理者结合自己几十年的办学实践，与学员分享现代学校治理与自身成长的经验和智慧。鲜活的案例、精辟的分析、独到的见解、深邃的思想，让大家学到很多，明白作为一个青年后备干部要重新审视，明晰自我角色定位；要与时俱进，不断锤炼，用未来的眼光把握当下；要更新管理理念，形成“在成事中成人，以成人促成事”的管理新思路。

朱寿生围绕学校办学的纪律底线与规矩，作《严守纪律底线 铸就廉洁初心》的主题报告。他告诫学员们，要领悟新形势新要求，始终保持清醒头脑，自觉遵守纪律底线，不断铸就廉洁初心，对党忠诚，为推进党的教育事业发展做出自己应有的贡献。

上师大教授田军从“遵循公务礼仪的意义、公务礼仪的基本精神、青年干部公务礼仪”等三方面对学员做公务礼仪实务培训。东方美谷园区副总经理朱豪君为大家带来了《东方美谷 全域之美》的微报告。

恽敏霞局长作《深化改革 创新治理 加速推进教育现代化》的专题报告，并参与各小组交流研讨。她回顾了近几年闵行教育在基础教育综合改革、打造教育强区等过程中所取得的巨大成就，指出了目前区域教育所面临的问题和挑战，同时提出了未来五年闵行教育的奋斗目标与艰巨任务：深化教育教学改革，推进课程教学现代化；优化教师培养机制，实现师资队伍现代化；加强教育管理改革，推进教育治理现代化等。恽局的报告高屋建瓴、深入浅出，结合自己的成长经历与管理经验娓娓道来，让学员们明白：管理能力某种程度上是管理者最重要的素养，一个好的管理者要学会协调各种资源，让资源集聚产生势能，从而为学生赋能、为学校赋能、为教育赋能。两个半小时的讲座，大家意犹未尽，表示一定铭记于心，重塑远大理想，明确发展目标，并实践于行。

利用封闭式培训的三个晚上，61位后备干部分成4个小组，围绕“新校长接管老学校，如何建立权威并维持信任”、“学校特色是否可以复制”、“如何实现高质量决策”的主题开展研讨。区教育局党政领导、教育学院正副院长，区政府教育督导室常务副主任何曙光，教育局党群科科长林火生、人保科科长俞伟珏、普教一科科长谢凯丽、学前科科长黄悦等参加分组讨论，并为后备干部答疑解惑。

结业仪式在学员连夜制作的视频短片中拉开帷幕。培训班副班长、闵行中学党办主任郭蕾主持。培训班班长、莘庄中学副校长严熙代表全体学员作总结发言。四个小组进行汇报展示：综合组的《新校长的烦恼》、中学组的《摇号之后……》、小学组的《校长遇到新政策》、幼儿园组的《新园长来了》，分别以情境表演、微论坛等形式，呈现几天学习后对现代学校治理及社会教育热点的所思所得，其中不乏关于新任校（园）长顺利开展各项工作的思辨性讨论，学校管理者接到教育新规后的应对与推进，以及在区域快速发展背景下不断凝心聚力的努力与尝试等，将学习内容融合贯通，体现了学员在思想交流与碰撞后的智慧凝练。

林火生宣布本次培训优秀学员名单，朱雪平、李光华为18位优秀学员颁发荣誉证书。

朱雪平书记为本次青年后备干部培训结业寄语。他指出，在当前闵行教育干部梯队发展不均衡、对干部有更高期望的背景下，青年干部们尤其要重视加强自身学习，努力提升管理素养，做事要有胸襟与情怀，并提出“政治站位、专业推动、提升能力、做好自己”四个方面要求。特别强调，作为后备干部要增强政治敏感度，把及时学习解读政策作为一种习惯；要重视专业发展对推动办学质量提升的关键作用，寻找专业突破口；要在实践中反思，在反思中积累领导管理素养，为闵行教育现代化建设注入活力。

本次青年后备干部集中培训圆满结束，后期还将通过全脱产挂职锻炼、参与重大活动、参加专题论坛等形式进一步加强培养与提升，期待青年后备干部们在今后更为深入的学习与工作实践中砥砺品质、历练成长，为闵行教育新一轮快速发展添砖加瓦。
