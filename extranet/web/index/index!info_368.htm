辨办学文化 研转型发展 ——闵行区小学校长培训基地第五期校长班系列活动之十八

发布时间 : 2018-11-09   作者（来源）: 曹雅明   阅读 : 1511

2018年11月8日，闵行区小学校长培训基地第五期培训班导师何学锋特级校长和全体学员相聚上海戏剧学院闵行附属学校，开展“学员校办学实践的反思与改进”专题培训活动，聚焦学校办学经验与特色，交流文化引领下的学校发展方略。

上午的主题汇报，学员校上戏附校杨慧琳校长聚焦“文化”与“发展”两个关键词，介绍了学校“教育戏剧文化”培育的基础条件，阐释了学校在面临新的机遇和挑战下以文化人的转型探索，解读了“教育戏剧文化”的基本内涵与符号表示，重点解析了教育戏剧文化方略下学校“角色场”课程的构建与实施，在对课程的具体阐述中传递了上戏附校“学戏化人、以美衍生”的教育理念。

下午的互动研讨，学员校长们畅所欲言，展开了充分而有效的“头脑风暴”——围绕“办学实践”这个核心，从学校建设、管理配置、理念渗透、育人思考、课程特色、队伍培养和文化形成等诸多方面进行了深入的交流讨论与追问辨析。杨慧琳校长在交流中陆续分享了学校创建以来诸多动人的管理故事。校长学员们不仅在发言中表达了对上戏附校八年来办学历程的认可与欣赏，更结合各学校自身情况提出建议，研究策略，明辨方向。

基地主持人何学锋校长在总结中肯定了杨慧琳校长在上戏附校近年发展中融戏剧元素于学校整体办学的不易，表达了对扑面而来的校园戏剧文化的欣赏，赞赏了学校管理团队的学习力与变革力。同时，何校长指出，学校管理是要在实践中不断优化的，管理团队的成长需要一定程度的驱动，学校教师的归属感会使学校文化向更好的方向发展，也期望上戏附校用戏剧元素让孩子的成长经历更为独特。

一天丰实的培训，学员们在办学实践的交流碰撞中对学校文化之“聚”与管理策略之“具”有了更深而新的认识。校长培训基地的管理者们也将怀着“山不厌高，水不厌深”之心，继续坚守办学理念，践行办学追求，优化办学成效。
