潜心教研勤探索，扎根课堂促提升 ——闵行区学习中心组（社会）、种子基地协同教研活动

发布时间 : 2023-05-16   作者（来源）: 顾莹   阅读 : 296

3/8
←
→

撷一束光阴，抚过心间。

教研是一场诗意的修行。

教研的芳华，沉香流年。

风有信，花正艳。缤纷的五月，明媚的阳光。踏着初夏的旋律，迈着轻快的步伐，2023年5月11日上午，在区学前教研员顾莹老师的带领下，学习中心组（社会）和种子基地教师聚集在闵行区启英幼儿园展开了以“童蒙养正 立德树人”为主题的协同教研活动。

聚焦课堂 如沐春风

今天的活动，由每小组各一位教师进行二研教学。经过专家老师及顾莹导师的指点后，在组长的带领及组员的共同努力下，每小组都进行了教案上的修改及调整，本次执教的四位教师以娴熟的教学技艺，丰富的教学经验，扎实的课堂训练，为我们带来了四节精彩的社会领域集体教学活动《起狮》、《这不是我的帽子》、《丫丫日记》、《一切刚刚好》，起到了优质的教学点拨与优化。

如琢如磨 反思优化

顾莹导师对四节活动进行了详细的指点与总结，对执教老师在活动中的亮点给予了充分的肯定，同时也针对教学过程中出现的共性和个性问题进行了探讨。顾老师从教学目标、教学设计、教学环节、课堂提问、课堂驾驭、多媒体运用等多个方面提出了思考和细节斟酌，给予老师们诸多探究及可借鉴的教学新理念和教学新方法。

教育教学对幼儿教师来说好比是无涯之海，而行者无疆，唯有积跬步，方能致远。此次教学设计研讨活动不仅是一次思想的碰撞、理念的交流，更是教师进行业务切磋、互相学习、提升自我的一个有力平台，教师在活动中得到了磨炼、提升，为老师们积蓄前行的力量。

一路耕耘一路歌，

且迎风雨且徐行。

     让我们在思考和实践中前进，

    智见教研，点亮未来。

 

摄影、撰稿：上海市闵行第四幼儿园 秦瑜蔚


闵教院学前教研室

2023年5月11日
