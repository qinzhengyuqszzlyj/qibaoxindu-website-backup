闵行区第六期（莘松中学）校长基地培训班在浦航实中开展活动

发布时间 : 2023-04-28   作者（来源）: 曹雅明   阅读 : 2657

3/8
←
→

忽而春风是美好的开始，忽而春雨使万物萌发。在这春暖花开的时节里，闵行区第六期（莘松中学）校长基地主持人张忆校长携基地学员——颛桥中学校长李卫军、马桥复旦万科实验中学校长王治华、纪王学校校长祝庆、七宝金都校长李波莅临学员金虹校长所在的浦航实中，开展了以“课程”为关键词的学习活动。君莲学校校长丁淑燕、莘城学校校长陈效、莘松中学“第四期课程领导力项目”负责人须燕蓉、闵行区挂职干部王竞以及浦航实中部分行政干部也参与了本次活动。活动特邀上海市教育学院（上海市教委教研室）副院长纪明泽指导并作专题报告。

课程是学校教育的核心。活动伊始，金虹校长以“聚焦校情 培育素养”为主题，汇报交流了浦航实中“启航课程”创建的背景、课程目标、课程设置、课程实施、课程评价与效果。立足生源特点，聚焦核心素养，浦航实中着力打造科技特色课程、体育特色课程、“芳华”课程。金虹校长表示“启航课程”的实施是学校内涵发展、教师专业发展、学生全面发展的首要路径。课程优质，学生才会优秀；课程进步，教师才会成长；课程高效，教育才会高效。浦航实中的“启航课程”为学生提供了营养餐，为教师发展搭建了快车道。

上海市教育学院（上海市教委教研室）副院长纪明泽首先对金虹校长的交流汇报进行了点评，立足校情是课程设置的根本，他鼓励金虹校长继续办好家门口好学校，积极发挥辐射引领作用，持续发力做强校，力争上游做名校。纪明泽副院长接着围绕课程领导力进行专题报告，他首先诠释了“领导力”的内涵——领导力“领”在方向，“导”是办法，“力”是各要素之间的相互作用。领导力就是专业力，专业力包含了设计力、规划力、执行力、反思力乃至生长力。纪明泽副院长还从课程体系建设、课程组织管理、深化教学改革、考试评价改革、体制机制建设、示范辐射引领等方面为校长们做了详细的讲解。同时，他也提出要校本化、高质量、创造性地进行课程建设，提出“面向人人，适合人人，人人出彩，人人成才，人人感受教育的温度”，鼓励校长们要从理念到行动转变，在新的时代，有新的担当和新的作为。

张忆校长为本次活动总结。一是校长要坚持做领路人，引领学校校本化实施国家课程，这是课程领导力的主要部分；二是校长还要坚持做贴心人，注意从普及和个性化两个视角进行校本课程建设，这样才会推动校本课程的高质量发展；三是校长更要坚持做有心人，密切关注热点，张校长结合2022年4月21日发布的课程方案，要求校长们要熟知每一门学科的改变之处及每门学科的核心素养。最后张校长强调了课堂教学改进必须与时俱进，真正做到教与学方式的转变。

课程有特色，学校才会有特色；课程多样化，学校才会多样化发展；一所优质的学校必须有一套优质的课程支撑，一套优质的课程更会促进学校优质的发展。今后，浦航实中将继续借助专家引领，加强学校课程体系的整体规划，进一步提升教师的课程领导力和课程实施的能力，立足核心素养，打造精品课程，不断提升学校的社会影响力，实现学校跨越式发展，成为家门口的新优质学校！
