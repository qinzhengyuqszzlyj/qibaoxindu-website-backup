“我”的笔，“我”的身边，“我”的生活 ——闵行区六八年级语文作文教学专题教研

发布时间 : 2023-09-22   作者（来源）: 鲍清   阅读 : 558

  时维九月，序属三秋。9月21日下午，闵行区六八年级语文作文教学专题教研活动在颛桥中学举行，北桥中学、闵行四中、友爱中学、君莲学校老师线下现场参加，其余学校通过线上云录播平台参加，本次活动由颛桥中学雷旭莉老师主持。

  

  

  活动开始，颛桥中学的两位老师呈现了两堂作文课。六年级郭健芳老师执教《慧眼慧心绘人物——写人片段写作训练》。郭老师通过课堂PK的形式，讲练结合，让学生真正参与其中，学有所获。八年级高笑笑老师执教《关键处驻足 细微处慢说》。高老师从学生习作《你是我心中的一束光》入手，聚焦细微处，教会学生生动描绘细节。

接着，颛桥中学八年级语文备课组长沈娴老师进行了以“小到微处更动人”为主题的讲座。沈老师从《语文课程标准》入手，明确学生在写人作文中主要存在的问题，并给出了指导建议。精选事例显品格，善用修辞增文采，巧借描写展个性，议论抒情来点睛。然后，沈老师通过写作任务指引、审题训练素材积累、改编式创作、范文习得等方面，展现了颛桥中学在作文方面的系列教学。

随后，闵行四中张新华老师对第一节课进行点评。张老师认为六年级的学生正处于小升初的衔接阶段，教师要回到作文的本源，培养学生的写作兴趣。要把握作文教学的“度”，注重学生真实地具体地表达。关注作文教学的时间投入，要具有针对性。友爱中学高秀萍老师从教学设计，教师的课堂表现，学生的课堂参与度对第二节课进行了高度评价。高老师的教学设计符合八年级学生的特点，并且课堂语言精简，声音轻柔，课堂评价恰当有方法。

活动最后，闵行区教研员鲍清老师进行了点评和总结。鲍老师指出两节课都给老师们带来很多启发和借鉴，八年级的作文教学交流给大家做了很好的示范。第一节课完整度高，在课堂练习时要让学生先交流，教师给与引导，再练笔。六年级的作文教学，可以起步低些、坡度小一些，即使是片段练习也要教会学生把握中心，围绕一个明确的方向来展开写作。第二节课教师课堂掌控能力好，教学内容可操作性强，但是对于“光”的理解立意过浅。八年级作文教学要帮助学生提升立意。    

 

鲍老师提示老师们，教学要紧密结合学生的生活实际，关注教材中一篇篇课文之外的“边角料”，立足学生的素养和能力的提升。

      作文教学，要引导学生关注“我”的身边，用“我”的笔，书写“我”的生活。生活无止境，语文教学也无止境！

 

撰稿：颛桥中学    谭容容
