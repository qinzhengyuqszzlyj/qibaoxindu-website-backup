落实新课标 推进教与学方式变革——闵行区第五、第六期“优秀教师培育工程”信息科技学科专场展示活动

发布时间 : 2024-04-19   作者（来源）: 朱凤   阅读 : 1058

3/8
←
→

2024年4月18日，闵行区第五、六期“优秀教师培育工程”信息科技学科专场展示活动在闵行区平南小学举行。上海市教师教育学院信息科技教研员、特级教师张汶，闵行区教育学院副院长董学平、上海师范大学王旭卿教授，上海科技教育出版社特聘首席编辑凌玲等出席活动。

信息科技骨干教师培养基地（G29）主持人、第五期“优秀教师培育工程”学员文来初中周晔老师主持活动。

平南小学张小娟校长致欢迎辞，感谢教育局与教育学院搭建平台，感谢导师悉心指导，同时介绍了学校信息技术学科课程体系的探索发展历程，并表示学校将一如既往支持教师的专业发展。聚焦新课标，四位老师分享了在推进教与学方式变革中的实践探索。白巧变开展了《设计校园投票器——探秘排序算法》现场教学并以《立足问题解决，重视教与学方式创新——“身边的算法”单元教学设计与实施》为题进行说课。通过归纳法、实验探索法，抽象总结排序规律，帮助学生探索算法背后的原理，形成思维与解决问题的模型，促进教与学的创新。实验西校张启迪分享了《编程共协——以“智能家居”项目培育小组合作素养》案例，通过接力编创实践活动，使学生在使用信息技术的同时学会与人进行合作，提升学生适应信息时代发展的关键能力。闵行区教育学院郭文波分享了《数字化环境下教师思维方式的转变》微报告，以己变应万变，适应新时代，改变思维方式、学习方式，拥抱人工智能、学习人工智能、与人工智能合作，让人工智能赋能教师专业发展。 周晔分享了《物联网海菜园》跨学科项目化学习课程建设的路径，教师需要与时俱进，把生活工作中的真实场景迁移进课堂，促成学生形象思维和逻辑思维的相互作用，帮助孩子们共性合作与个性成长协同进步。

王旭卿高度赞扬四位老师的精彩展示，她认为：新技术融入校园文化生活、融入课堂是信息科技学科的发展方向，老师们要有时不我待，只争朝夕的精神迎接新技术的革命浪潮。张汶高度肯定了本次活动，是一次有效落实学科核心素养的实践与探索。她指出：教师要转化学习内容，设计符合学生认知特点的活动，学生在具体的体验中形成直接经验，将思维可视化，帮助学生构建知识，同时通过跨学科项目化学习，提升学生创造性解决问题的能力。希望老师们不断提升专业能力，将学习变成一种习惯，始终关注科技的前沿发展，率先垂范新技术赋能教育教学。坚守教育初心，承担为未来社会培养合格数字公民的使命和担当。

董学平作领导讲话。感谢专家们一直以来对闵行教育的关心和帮助，对学员的引领和指导。通过展示看到了学员们的成长，达到了预期效果。他指出：要成为优秀教师，必须要有“志”，有“力”，有“物”，才能实现“登峰”。“优秀教师培育工程”是推进闵行区教育强区建设的重要举措，以教育家精神为引领，培养一支“有理想信念”“有道德情操”“有扎实学识”“有仁爱之心”的优秀教师队伍，共同谱写闵行教育强区建设的新篇章。

闵行区教育学院校长发展中心主任俞琳，原闵行区教育学院教师发展中心骨干培养管理办公室主任兰平，闵行区信息科技名师工作室（M20）主持人、闵行中学特级、正高级教师顾秋辉，闵行区信息科技王苇名师工作室（M19）、骨干教师周纯培养基地（G28）学员代表和信息科技顾秋辉名师工作室（M20）基地学员、骨干教师周晔培养基地（G29）全体学员一同出席了活动。
