加强师德师风建设 筑牢教育强国根基 —2023年闵行区中小幼书记、校（园）长暑期专题培训顺利召开

发布时间 : 2023-08-30   作者（来源）: 贾建芬   阅读 : 1831

3/8
←
→

2023年8月25日至27日，以“加强师德师风建设 ；；筑牢教育强国根基”为主题的闵行区中小幼书记、校（园）长暑期专题培训在闵行区教育学院举行。闵行区教育局党政领导，区教育学院领导，区教育局各科室负责人、督导室全体督学，区公民办中小幼、职成教、直属单位领导等共计600余人通过线上线下相结合的方式参与培训，闵行区区委副书记唐劲松、副区长刘艳出席活动并讲话。

闵行区教育局局长何美龙作培训开班动员，他要求全区中小幼书记、校（园）长在闵行教育高质量发展进程中，提高政治站位，敢为人先、锐意进取，以教育强区建设责任在我的大格局、大情怀为思想武装，使命担当、攻坚克难，推进学校有序有质前行。同时，怀揣本领恐慌，自觉参与培训，听专家报告、听同行经验分享，融合理论学习与实践探索，成为教育强区建设的智慧型领导。

专题报告：加强师德师风 提高政治站位

上海师范大学教授王正平作题为《立德树人：党的二十大和习近平加强新时代师德建设论述的核心要义》报告，王教授指出，要学习贯彻党的二十大和习近平总书记关于立德树人重要论述，加强师德建设，增强行动自觉。以丰富详实的案例和研究，通过几十余条古今中外关于教育、教师及德育的经典论述，提出了倡导新师德、铸筑新师魂的时代意义及新时代教师道德的基本特点，从教师的社会责任、教师的“智德”培养、教师的人类命运共同体精神等方面提出了涵养新时代教师道德的基本要求。

全国优秀县委书记孙建平博士作题为《贯彻落实教育强国精神 做忠诚担当好干部》报告，孙书记以履职过程当中的重大改革成果，如：“社保卡医保卡并轨”“城市应急管理体系创立”“虹镇老街改造”等为例，具体诠释了作为一名新时代好干部的要求，做政治上的明白人、做发展上的开路人、做群众的贴心人、做班子的带头人。

上海市教委托幼处副处长王纾然作题为《推进学前教育高质量发展的现实意义和实践路径》报告。她从学前教育高质量发展的现实意义、学前教育高质量发展的实施路径两方面剖析了学前教育面临的新局面，并就如何“构建完善的托幼服务”“如何全面建设高质量幼儿园”，帮助园长明确目标、沥清思路，形成新形势背景下学前教育发展新思考、新战略。

主题研讨：领跑新赛道 形成新经验

本次培训，围绕闵行教育强区建设，开设了两个专场的主题研讨。

闵行区德育“第一课堂”推进培训会分以下六个步骤进行：观看短视频、颁发荣誉证书、项目实施方案发布、学校分享交流、TED演讲、专家点评。

区教育局副局长王维刚从项目背景、目标、内容、范围周期、进度安排、保障措施六个方面对《闵行区德育“第一课堂”项目实施方案》作解读；七宝中学党委书记兼校长朱越、上海师范大学附属第三实验小学南校党支部书记兼校长沈珺作为参与德育“第一课堂”项目实践校，分别围绕“德育‘第一课堂’七宝中学实施方案”、“赓续血脉 ‘真火’相传”作案例分享；交大实小杨永茹、复旦万科吕晓蕊、七宝中学范丽、上海交通大学马克思主义学院李瑞奇、“大思政”课校外导师杨红霞，五位优秀教师分别以“老人与海” “百姓与河”、“人民与城”、“民族与梦”、“工匠与国”为主题展开TED演讲；华东师范大学马克思主义学院副院长许瑞芳教授作专家点评。

讲好新时代“大思政课”，培养更多担当民族复兴大任的时代新人，是习近平总书记念兹在兹的一件大事。闵行将始终把抓好思政课作为推动教育高质量发展的重要工作予以推进，以思政课改革创新为抓手，推出闵行区德育“第一课堂”项目，开展大中小学思想政治一体化建设，打造富有闵行特色的德育课堂，形成共育“大思政课”的闵行经验。

“领跑数字教育新赛道 全面推进闵行教育高质量发展专场培训”分回顾与总结、推进与发布、案例分享、领导讲话四部分推进：

区教育学院科研中心主任朱林辉对区域教育教学成果作了回顾与总结；学院党总支书记兼院长朱靖从问题的提出、支持体系架构与建设、支持体系运用实践和支持体系运行机制四个方面讲明《教育数字化转型背景下精准教学区域支持体系建设》，提出了区域精准教学支持体系的建构和实践的优化改进，为学校进一步推进提供实践路径；学院副院长龚耀昌从构建精准教学区域支持体系、优化学校精准教学的计划及实施、开展精准教学方式变革行动、实施教师精准教学能力提升工程、深化精准教学实践研究与成果推广5大维度20个方面解读《闵行区推进中小学精准教学行动计划方案》。

上海交通大学附属实验小学校长王叶婷、华东师范大学附属紫竹小学党支部书记兼校长蒋方叶、交通大学附属马桥实验学校校长、副书记巢晖、闵行中学东校副校长朱咏梅、区教育学院小学数学教研员陈培群，聚焦主题，分别围绕《阶梯式联合校本研修推进智慧型精准教学》《数字转型赋能思维成长》《融智于行 慧笔入心》《构建SIMPLE课堂 探索人才培养的闵中东校实践》《面向全体 尊重差异 技术赋能精准教学》，从不同视角分享实践智慧。

区教育局副局长潘蓓蕾在专场总结时指出，闵行教育科研取得了令人瞩目的成绩，形成了高质量研究的标识性标志性成果，希望在后续的探索实践中有更多、更高规格的科研成果呈现。同时建议书记、校（园）长结合刚发布的区精准教学行动方案，努力实践，扎实推进，开启新一轮课堂教学改进，创出学校特色，打造闵行数字教育赛道新优势。

为激发学校新活力，推进区域教育重点改革项目高质量前行，举行颁奖表彰。区教育局局长何美龙、区教育党工委副书记徐明怡、区纪委监委第七派驻纪检监察组组长商雨红、区教育局副局长潘蓓蕾为获评“党的二十大主题教学区级思政精品课”与“德育‘第一课堂’十大区级主题教育课”执教教师及荣获教育部智慧教育优秀案例学校领导颁发荣誉证书。

2023年闵行区新任校级领导以“在传承与创新中实现学校可持续发展”为主题开展岗位实务培训。闵行区纪委监委第七派驻纪检监察组组长商雨红，区教育局人事科科长俞伟珏、普教二科科长马卫锋、党群科科长林火生，为71名干部授课。商雨红结合违纪违规案件、事例，警示新任领导干部要坚定理想信念、守住政治底线、严守纪法条规、增强党性意识，自觉做到廉洁自律；俞伟珏介绍了编制问题、教师招录、局内流动、编外管理、岗位设置、职务评聘、绩效考核与分配等有关人事管理的政策法规及操作程序，对“储备教师”和“人事干部”选用提出了标准及建议；马卫锋首先向新任校领导介绍了科室的业务范围和人员分工，聚焦“加强底线思维 依法规范办学”、“项目为抓手 加强学校内涵建设”两方面，整理分享了相关教育教学政策文件，梳理了区域教学重点项目，提出了可操作可借鉴的策略方法途径；林火生指导新任校领导们要遵章办事，抓好党的基础建设工作和党的领导工作，规范执行党组织领导的校长负责制，开好两个“会”，处理好几种关系，严于律己，榜样示范，在岗位历练中成长为群众信赖的好干部。具体详实的培训拓宽了新任领导干部的管理视野，补短板、强弱项、固底板，助力其尽快站稳岗位、胜任岗位，成为一名让党放心、人民满意的好干部。

区教育党工委副书记徐明怡，区教育党工委委员、副局长马秀明，区教育局副局长王维刚，区教育学院副院长龚耀昌分别主持各专场培训活动。

领导讲话：擘画未来 开启新征程

培训期间召开2023学年教育系统党政干部大会，上半场徐明怡副书记主持，下半场刘艳副区长主持。

区教育局各分管领导就条线任务作工作布置，明确党建、德育、教学、人事、后勤保障等工作目标，布署工作重点，为学校新学期工作有序开展指明方向、提供思路。

区教育党工委书记恽敏霞、局长何美龙作工作部署，区委副书记、区教育工作领导小组组长唐劲松提工作要求。

何美龙指出，书记、校（园）长要提高政治自觉，勇于使命担当，做教育强区建设的引领者、探索者、实践者、先行者；要敢于创新，打破常规壁垒，把“12369”落到实处。明确目标，梳理区分工作的轻重缓急，扬长避短，把高中“双新”、义务教育“双减”、学前教育公益普惠等规定动作做好，把学校特色创建等自选动作做亮，对标对表，规范化、数据化、精准化，理清优势短板，不断完善人的教育，循序渐进，提高学校内涵发展品质；要加强师德师风建设，专业发展是师德的核心指标，重视师资队伍建设，确保教育教学质量落地。打造花园式学习型校园空间，压实责任，集全体教师智慧，家校协同，久久为功，强校强区。

恽敏霞强调，全区中小幼书记、校（园）长要学习贯彻落实习近平总书记关于教育的重要论述和在中央政治局第五次集体学习会议上有关教育强国的讲话精神，全面加强党对教育工作的领导，围绕闵行教育高质量发展这条主线和各项工作任务，做好做实教育强区建设的政治保障。政治建设方面，重点抓好主题教育、理论学习、师德师风建设、未成年思想道德建设。主题教育开展不划阶段不分环节，调查研究、推动发展、检视整改贯彻全过程，实现主题教育一体化推进；组织建设方面，全面推进党组织领导的校长负责制，完成工会改选、党组织领导的校长负责制的组织架构调整、集团化学区化建设中成立学校联合党委、推进中小学校党组织“攀登”计划等工作；队伍建设方面，聚焦健全师德建设长效机制、优化教师队伍结构、提升教师专业素养、强化教师保障力度这四大目标开展基础教育师资队伍建设改革试点、同时加强干部监督管理、强化人才及后备干部培养等；纪律建设方面，明确工作职责，完善制度机制，强化监督检查，开展廉政教育、谈心谈话等。恽书记提醒书记、校（园）长工作要讲究策略，有格局、有勇气、有谋略，做人民满意的“政治家”“教育家”。

唐劲松肯定了闵行教育上半年取得的卓著成效及本次培训的价值意义，并提出工作要求：

一、以孩子的健康快乐成长为中心，着力推动“五育”并举、融合育人。坚持德育为先，为学生扣好人生第一粒扣子。要从学生身心特点和思想实际出发，把思想政治工作贯穿教育教学全过程，坚定不移用习近平新时代中国特色社会主义思想铸魂育人。统筹推进闵行德育“第一课堂”和“交大-闵行”市级大思政课整体试验区建设。借助现代信息技术手段，把思政课讲得更有亲和力和感染力、更有针对性和实效性。领导干部要带头上思政课，通过开学典礼、国旗下讲话等宣讲党的教育方针，传播科学理论，弘扬社会主义核心价值观；坚持以素质教育为基础，促进学生全面个性发展。继续打造科技创新教育的特色和品牌，依托“闵行少年星计划”等平台，探索创新拔尖人才早期培养模式。同时，体育课要保质保量开好，美育要着眼提升学生艺术素养和培养学生艺术爱好，劳动教育要融入日常，开放更多社会性实践场所；坚持智慧赋能，深入推进教育数字化转型。在通过教育部“智慧教育示范区”评估验收的基础上，梳理巩固创建成果，全面推动数据驱动的大规模因材施教。推动闵智作业、智慧纸笔常态化规模应用，服务师生精准化教学、个性化学习，以数据驱动教育高质量发展。

二、以培育时代新人为目标，着力培养高素质专业化教育人才队伍。要加强师德师风建设，深入推进师德水平提升工程，开展师德“大学习”“大培训”，严格“师德一票否决制”，引导每一名教师以德施教、以德立身。组织好教师节庆祝活动，选树一批“四有”好教师，发掘宣传师德典型，讲好师德故事，提高教师的职业荣誉感和幸福感；要大力培养名师名校长。加快建设名校长名师培养工作室，切实办好闵行区第三届“教育家培育工程”和第一期“名校长培养工作室”,发挥唐盛昌、仇忠海等市区级名特校长的“头雁效应”，培育一批在全市乃至全国有影响力的校长、教师；要关心关爱校长教师，帮助一线老师解决专业发展、身心健康、安居出行等方面的“急难愁盼”问题。帮助校长教师减负，严控各类与教育教学无关的活动进校园，优化各类督查检查评比考核，关注教师心理健康问题，及时予以疏导，让校长和教师有更多时间和精力、更好状态投入到办学治校、教书育人中去。支持校长和老师创新教育思想、教育模式、教育方法，形成独特的办学风格和教育主张，推动涌现更多教育家型校长和教师。鼓励特级教师、骨干教师流动到薄弱学校、偏远地区任教，形成优质师资良性流动的格局。

三、加强党对教育工作的全面领导，把党的教育方针贯穿到立德树人全过程各方面。加强教育系统党的领导和党的建设，坚持以党的政治建设为统领，推动全面从严治党向基层学校延伸，积极稳妥推动中小学校建立党组织领导的校长负责制；营造风清气正的良好教育生态，切实抓好教育系统党风廉政建设，形成全社会育人合力，家庭、学校、政府、社会齐心协力、同向而行，文教结合、医教结合、体教结合、科教结合机制，形成育人全链条，加强正面宣传和舆论引导，大力宣传正确的人才观、教育质量观、成长观，讲好闵行教育故事，擦亮闵行教育品牌。

三天的培训，全区教育系统领导全程参与、积极投入，分组研讨时，围绕“学校教育高质量发展新思考、新行动”交流学习体会、分享学校创新之举、提出未来发展需求与建议：闵教院附属友爱中学校长严熙、上虹中学党支部书记兼校长祝庆、景东小学党支部书记兼校长王英鹰、龙茗路幼儿园党支部书记兼园长浦燕群分别代表高中、初中、小学和幼儿园学段作大会集中交流。书记、校园长们表示，闵行教育高质量发展责任在“我”，明晰使命担当、善作为、能成事，高质量完成各项工作，助推闵行教育强区建设。
