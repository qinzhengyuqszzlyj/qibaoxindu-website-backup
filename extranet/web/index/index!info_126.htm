令序当芒种，实践入课堂——闵行区“阅读与项目式学习“课程举行学年展示活动

发布时间 : 2023-06-12   作者（来源）: 丁旻平   阅读 : 1129

3/8
←
→

6月8日上午，闵行区“阅读与项目式学习“课程在交大实小举行了项目学年展示活动。

活动首先由交大实小校长王叶婷致欢迎辞。项目实施方，阅读与研究性学习联盟负责人林宇宾老师总结过去一个学年中的各项工作。

2022学年中，尽管有封控、有疫情，项目组仍和所有试点校一起，克服各种困难，为学生提供了稳定的课程输出；为了满足学生多样化的活动需求，项目组面向所有正在开设，或开设过本课程学校的学生，提供了每学期一次的线上学生活动；为了使我区教师对项目式学习有更深入的了解，并鼓励教师实践，项目组还提供了10次线上线下相结合的教师工作坊。

本次展示活动中，福山实验的陈雅婷老师和交大实小的朱丹老师作为本期教师工作坊的学员，分别带来了”草履虫印象“以及”In China, this is how we do it“两节课，以展现她们本学期的学习成果，以及对”阅读与项目式学习“课程校本化实施的思考。两位老师在课后分别进行了说课，并与人民文学出版社的王雪纯编辑，上海外国语大学赵美娟教授，原闵行区教育学院骨干教师培养办公室主任兰平，以及学院小学拓展-探究课教研员孟嬿娜一起，探讨了项目今后应如何更好地与学校校本课程衔接，为学校教师发展及校本课程提升助力。

本学期的学生活动中，共有34个小组经过不断的努力，最终完成了学习及项目制作。其中10个小组，在本次活动的关注重点“坚毅、合作、审辨、创新、关爱”方面表现尤为出色，获得学习之星称号。在本次活动中，专家们向这些小组颁发了纪念品。本学年教师工作坊中，表现优秀的教师也获颁优秀证书。

最后，闵行区教育学院副院长龚耀昌以“芒种”节气的寓意寄语项目，在学年总结“收获”的时候，勿忘进一步的思考和实践。要将“阅读”与“项目式学习”做更有机的整合，要将“教师培训”和教师的“教学方式变革“做更有机的整合，把本项目涉及的“两对关系、四个要件”，基于素养培育的目标进行深度研究和实践，将项目的价值追求落实到具体的教学情境中去。
