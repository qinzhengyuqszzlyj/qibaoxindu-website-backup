“小精灵们”的微花园

——教育装备应用于小学综合实践活动课程的探索

 

为深入推进2022版义务教育课程方案的实施，探索教育装备应用于小学综合实践活动课程，2023年6月7日下午，田园二外小以“‘小精灵们’的微花园”为主题，举办了区级现场展示研讨活动。

 

“综合实践活动如何定位与实施？”上海市教师教育学院（市教委教研室）中小学拓展-研究型课程教研员姚军老师强调，必须从学生真实的生活情境出发去发现问题，转化为活动主题，通过探究、服务、制作、体验等方式来培养学生综合素质。田园二外小依托学校“微花园”学习空间，让学生在生活情境中发现问题，充分利用教育装备，给学生提供了实践学习的工具，指向实际生活问题的解决。姚老师建议再给学生添加一些“麻烦”，当装备的问题解决后，可以腾出更多的时间让学生去思考，结合真实生活情境本身的复杂性，生成新的智慧，收获更多精彩。

上海市教育委员会教育技术装备中心曹楚斌老师就装备使用为学校和公司双方提出指导性建议。他指出：双方可以将合作作为培训自身团队的有效载体，主动“向前”一步。

闵行区教育学院龚耀昌副院长对学校“微花园”系列综合实践活动课程呈现的综合性、跨学科性、实践性给予高度赞许，尤其是活动设计在保持连续性和进阶性的同时还体现了对学生认知的进阶培养。同时龚院也对区域综合实践活动课程的实践和研究提出了新的方向。他指出：教师要尊重孩子无限的潜能，在工具和装备支持的情况下，教师能更多地聚焦活动设计对学生思维的挑战。

 

活动前，学校依托花镜社和微花园中丰富的生态植物资源，为参会教师设计了体现夏季时令元素的体验活动：多肉联“萌”、与“种”不同、特别的“艾”、清凉一“夏”。四个体验区人头攒动，吸引了参会专家、领导和老师们纷纷驻足、动手体验。

现场展研环节由田园二外小李嘉瑶老师和“肥墩墩”家族的小可爱主持。活动伊始，朱逸阳老师、周绮婷老师分别呈现了“我们的微花园”主题下的“土壤‘能量棒’”和“植物‘身份证’”两个综合实践活动。两个活动都指向第一个活动微花园养护中“问题大搜罗”的解决。

“土壤‘能量棒’”活动主要聚焦在土壤肥力不足的问题上。学生通过观察和实践，经历探究、设计、制作蚯蚓塔的完整过程，主动参与问题解决，在养护花园过程中收获自主成长，建立校园主人翁的责任与担当。

“植物‘身份证’”活动是基于大家想要认识校园植物的需求而产生的。学生通过观察、学习，讨论了植物“身份证”需要呈现的主要信息，并尝试利用信息技术制作二维码，感受现代科技带来的便利，同时利用丝网版画这一装备便捷快速的优势来实现“身份证”的统一与创新，最终完成植物“身份证”的制作。

学校综合活动课程负责人陈淑萍副校长与综合活动课程骨干教师葛飞、祝愿、朱逸阳、周绮婷组成了现场教研团队，以“我们与教育装备的故事”为主题，从“相遇——让装备走进来” “相识——让装备用起来” “相随——让装备活起来” “相守——让装备强起来”四个阶段讲述了学校探索应用教育装备的故事与装备应用的思考和展望。其间，兄弟学校积极参与了互动交流，教研员孟老师给予了现场点评。她肯定了学校立足校本构建的“微花园”学习场景，让综合实践活动真实发生，创造性地进行了跨学科主题实践活动。

 

 

“综合实践、问题解决、素养培育”，教育装备应用于综合实践活动课程的实践和探索让学生的素养清晰可见。本次活动是田园二外小课程建设和学习方式变革探索的一次具有重要意义的节点活动，也是闵行区试点“义务教育阶段综合实践活动教育装备应用的实践研究”重点课题研究工作的推进，期待本次活动给予区域兄弟学校参考和借鉴，携手推进课程改革。

 

 

——————————————————————————————————

 

中小学综合实践活动课程指导纲要：为全面贯彻党的教育方针，坚持教育与生产劳动、社会实践相结合，引导学生深入理解和践行社会主义核心价值观，充分发挥中小学综合实践活动课程在立德树人中的重要作用，综合实践活动是从学生的真实生活和发展需要出发，从生活情境中发现问题，转化为活动主题，通过探究、服务、制作、体验等方式，培养学生综合素质的跨学科实践性课程。

 

 

《义务教育阶段综合实践活动教育装备应用的实践研究》课题：义务教育阶段综合实践活动教育装备在学校课堂中的应用需要消除校企双方断点，才能知行统一，破壁合力推动学校综合实践活动前行。随着教育装备行业发展，教装企业在新技术、新装备上的发展也在不断提升。新课程背景下，教装企业已经意识到，教育装备要“从关注知识到关注育人”。教装企业正在思考如何研究符合学生个性化发展的新型教育装备产品，学校也在思考工具与装备如何结合综合实践活动内容，在跨学科、项目化学习中得以应用。
