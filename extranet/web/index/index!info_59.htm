谋划新航向 蓄力再出发 ——闵行区虹鹿幼儿园课程基地活动

发布时间 : 2023-09-29   作者（来源）: 鲁鸣   阅读 : 216

3/8
←
→

2023年9月26日，闵行区虹鹿幼儿园课程基地研究共同体开启了本学期的第一次研讨活动。本次活动由基地负责人罗琳老师主持。研讨活动由交流研讨、专家讲座和小组讨论三个板块构成。

以研促行，智慧交流

研讨前，罗琳老师首先梳理和阐述了基地组成员前期的活动开展、实践行动和经验成果等内容，对基地成员们的前期研讨和实践给予肯定和勉励，并开启本次活动的序幕。

基地一组组长陶老师，以《弹性作息，促进幼儿深度学习》为主题，从“弹性作息”的内涵、优势和实践进行了分享交流，从思、行、升的角度阐述一组成员们的践行。

基地二组组长王老师，在《优化空间 助推幼儿的主动学习》的主题分享中，从理论“空间优化”的意义、实践经验等方面进行了详细的阐述，并表达了对促进幼儿主动学习方面持续关注、研讨与践行的未来方向。

基地三组组长沈老师以《“儿童主动学习观察表”的诞生与演变》为主题，阐述了“观察表”的形成、调整和实践的行动路径。

导师指导，引领成长

基地导师对基地课程组的活动实践、经验分享给予了充分肯定，同时强调了“户外活动”开展对幼儿发展的重要意义，教师要站在赞赏幼儿的立场做“有质量”的活动，需看见儿童，看见成长。

小组讨论，反思提升

在第三环节中，基地成员以“组”为单位，进行小组间的智慧碰撞和反思研讨。大家积极探讨、各抒己见，交流与分享此次的学习心得和获得，并结合现场提炼了活动中的教师支持策略。

活动最后，虹鹿幼儿园张园长给予基地成员活动期许，期待和鼓励每一位学员带着本次活动学习的思考和所得，践行于以后的日常活动开展中。同时，依托共同学习体，发展园所间的联动和智慧共享，相互融合与激励，最终达到学员成长、基地发展、区域发展的项目研究成效。

本次课程基地活动实现了基地成员在不断实践、反思、交流过程中的专业能力提升，同时也是各基地成员在思维和理念上的又一次精彩碰撞。让我们共同致力于“幼儿发展优先”的理念落地，不断优化保教实践，为幼儿构建更加富有意义的课程。
