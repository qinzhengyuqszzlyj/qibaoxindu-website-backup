“绘美•润心”2023闵行区小学美术课堂练习作品展开幕式在美博美术馆举行

发布时间 : 2023-06-03   作者（来源）: 秦涛   阅读 : 1072

3/8
←
→

由闵行区教育学院主办，闵行区青少年研学实践教育基地美博美术馆承办的“‘绘美·润心’2023闵行区小学美术课堂练习作品展”，于6月2日下午在美博美术馆举行开幕式。

出席开幕式的嘉宾有上海市特级教师张家素、美博美术馆副馆长陈晓云，全区五十多位小学美术教师参加了活动。开幕式由区小学美术教研员秦涛老师主持。

美术课堂练习作为课堂教学重要的环节之一，是学生参与美育活动、艺术实践学习成果的体现，也是教师教学理念、教学素养、教学质量的呈现。本次作品展为师生搭建展示交流的平台，促进美术教师们对课堂教学提质增效进行不断思考与实践。

从上学期动员，到本学期的作品提交、评比、展陈，历时数月。区内各校积极响应，提交作品1071幅。经过初评与复评，最终从区内63所学校送展的作品中遴选出获得等第奖的优秀课堂练习作品400余幅/件进行展出，主要涉及小学美术教材“写生”“色彩”“构思创作”“水墨”“版画”学习内容。

 开幕式伊始，美博美术馆副馆长、上海市美术家协会会员、闵行区美术家协会秘书长陈晓云老师致欢迎辞。陈老师结合评审、策展的过程，表达了被学生们多姿多彩作品所打动，为作品背后老师们的教育热情与辛勤工作所感动的心情。相信通过园丁们的努力，美育之花会盛开于每个校园，艺术的力量也将根植于每位孩子的内心，普及他们未来的人生。

 为获得优秀组织奖的22所学校颁发荣誉证书环节后，获奖学校教师代表围绕美术课堂练习的教学实践进行了经验交流。不论是闵行小学多型态的课堂练习设计策略，实验小学规范保底、研究提效、活动促优的四校区保质均衡发展的举措，花园学校不断研究版画特色融入日常教学的实践，还是平南小学从材料入手点亮美术课堂创意、打破边界开发多样化艺术实践活动学习形式的探索，都体现了美术教师们对课堂练习重要性的准确认知，对美术课程内涵与育人价值的深入理解。

 上海市美术特级教师、市名师基地连续三期主持人、上海市美术家协会会员、闵行区美术家协会顾问张家素老师为本次活动做点评中指出，美术课堂练习作品是课堂教学效果的重要体现、是学业水平高下的重要标志。学生日常课堂练习作品走进美术馆进行展出令人欣喜，高质量的作品可见新课标背景下区域课堂教学改进的成果、区域美术课堂呈现的新面貌：美术教师引领孩子们追求美、表现美、树立用艺术表达情感的兴趣与信心；体现了美术教师课堂教学的智慧。张老师也鼓励美术教师们在单元教学实践中，思考课堂练习作品的设计与优化，以及与学科素养培育的关系。

  最后，张家素老师、陈晓云老师分别为观展老师们做展陈作品的点评、分析。从美术语言运用、美感呈现、创意激发等方面，为提升课堂练习的视觉冲击力、作品化做现场指导。鼓励将学生的兴趣爱好与课堂教学紧密地结合在一起，将审美心理与现实生活结合在一起；鼓励美术教师们多进行艺术创作，更深入了解不同艺术形式的绘制技法；提倡课堂中为学生体验水墨、版画创设更多机会，鼓励学生大胆尝试，体会优秀传统艺术的魅力。

 本次展陈的课堂教学成果寄托的是三尺讲台以美育人、以美润心的悠悠情怀，彰显的是我们的学生们幼小心灵感悟生活的种种志趣；我们将不负期望，一如既往，执着耕耘于美术教育，引领孩子们进入“至真、至善、至美之境地”。
