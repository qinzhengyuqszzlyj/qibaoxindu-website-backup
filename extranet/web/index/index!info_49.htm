【学前 个别化学习中心组】联结自然 对话幼儿

发布时间 : 2023-09-25   作者（来源）: 刘时医   阅读 : 842

3/8
←
→

2023年9月25日，闵行区教育学院教研中心学前教育研究室刘时医老师带领区个别化学习中心组的老师们在浦江镇第二幼儿园开展了现场观摩与主题教研。

观察识别幼儿

浦江二幼谈双燕老师结合大班主题活动《有用的植物》，组织实施了主题下的项目式学习活动，引导大班幼儿深入探究了驱动性问题——“哪些植物可以防治害虫？”“怎样更好地使用药草？”“干枯的植物有什么用？”孩子们通过收集植物可以防治害虫的信息，梳理了可以防治害虫的植物的种类和特征，发现间种可以防虫，在讨论种植方法和合作种植的过程中进一步了解植物之间的伴生关系。他们跟随秋天的脚步，探究干枯的植物的特征及其不同用途，制作昆虫屋、覆盖土壤、扦插盆花……在小组讨论、合作实践中感知大自然中“物尽其用”的法则，激发爱植物、爱自然的情感。

交流共享智慧

现场观摩后，谈双燕老师聚焦幼儿项目式学习过程中资源的支持，分享了《联结自然 支持幼儿——大班主题活动的优化实施》，阐释了教师在项目进展的不同阶段多途径地挖掘各类自然资源支持幼儿探究植物的种类与特征、植物的特殊作用以及植物间的互为关系。

中心组的老师们结合对幼儿活动的观察分析，肯定了幼儿园户外自然资源在问题驱动式学习中的运用、幼儿探究性学习的自主性等。同时也给出了进一步优化的建议，如幼儿学习探索的轨迹要更加清晰，合作与讨论要更凸显，学习成果的展示要更有创造性。

引航启发思考

中心组导师刘时医老师从幼儿发展、幼儿园室内外资源的运用、幼儿探究环境的营造等方面对本次活动进行了全面、专业的点评。刘老师肯定了对真实的自然资源的价值挖掘、教师在课程实施中能为每一个幼儿从容自主的探究创造机会幼儿明晰所探究的问题和任务……同时，刘老师也提出了教师要有培育完整的儿童的意识、关注幼儿前后经验的链接、基于幼儿的真实的问题推进幼儿深度思考、项目式学习中幼儿小组合作要在任务驱动下优化解决问题的方法。刘老师的点评为幼儿项目式学习活动的高质量发展拓展了新的思路。

导师引航，众行致远共成长。通过本次中心组主题教研活动，老师们对幼儿问题驱动式学习过程中的支持有了更深入、更全面的认识，激发了老师们对支持幼儿深度学习的新思考。中心组将在创新突破之路上继续前行，助力高质量课程建设。

 

撰稿与摄像：闵行区浦江镇第二幼儿园
