敢担当 善作为 能成事 ——2022-2023年闵行区教育系统优秀青年干部第三轮集中培训顺利结业

发布时间 : 2023-10-15   作者（来源）: 曹雅明   阅读 : 1714

3/8
←
→

2023年10月11日至13日，2022-2023年闵行区教育系统优秀青年干部第三轮集中培训在上海小木屋会务中心举行。闵行区教育党工委书记恽敏霞、副书记徐明怡，上海交通大学基础教育办主任、教育学院党总支书记王琳媛，闵行区教育学院党总支书记兼院长朱靖等领导出席。全区中小幼学校、直属单位、教育局机关的84名优秀青年干部参加培训。

闵行区教育党工委党群科科长林火生主持开班仪式。王琳媛书记作开班致辞，王书记介绍本次培训将紧紧围绕习近平总书记在第39届教师节提出的“教育家精神”，以及“躬耕教坛、强国有我”的各项要求，不断优化培训内容，丰富多元课程，从而开拓教师视野和激发教育教学的新思路。闵行区教育学院校长发展中心主任俞琳就《闵行区教育系统优秀青年干部第三轮集中培训实施方案》，从指导思想、培训内容、培训形式、组织与管理等方面作详细解读。俞主任指出，本期培训围绕“党性修养与理论提升、时事政治与使命担当、岗位职责与能力提高”三大方面展开，通过专家报告、专题讲座、案例分享、小组讨论等途径，涵养优秀青年干部的专业能力和综合素质。

恽敏霞书记为全体学员作开班培训动员。恽书记指出，近年来中央、市委相继出台《关于适应新时代要求大力发现培养选拔优秀年轻干部的意见》《关于进一步激励广大干部新时代新担当新作为的实施意见》，对青年干部队伍建设提出了明确要求。闵行教育立足“十四五”规划，大力培养、选拔优秀年轻干部，并纳入教育系统后备干部培训范畴，希望各位青年干部把握好培训机会，静下心来学习，沉下心来思考，严格要求自己，真正使自己在思想上、能力上、境界上有新提高。恽书记还对参加此次培训的青年干部提出三点要求：一是每位干部要适应新时代，学思践悟新思想；二是要勤于思考，锤炼过硬的思维品质；三是强化历练，实现自身更好更快成长，为区域教育高质量发展作贡献。

专题报告：提升理论，储蓄管理能量

三天的集中培训共安排四场专题报告。上海交通大学特聘教授、博士生导师俞勇开启第一讲：《科创新赛道 教育新模式》。俞教授从社会属性、心理素质、自我认同、理想信念四大方面，深刻剖析了当前教育背景下出现的部分问题学生的表现，告诉青年干部应该培养怎样的学生。俞教授还以他创办的上海交大ACM班、伯禹教育为例，强调中国教育培养的学生一定要有大情怀、大格局，这样才会有大作为。

上海交通大学人文学院副教授、硕士生导师陆岩军作《国家智慧与现代管理》专题报告。陆教授从中国传统文化治国理政、为人处世之道切入，提出作为教育工作者应有两大情怀：悲天悯人与敬畏之心。通过解读《周易》、《中庸》、《论语》、《礼记》等经典传统文学，陆教授深入浅出地阐释了中国文化中“观乎人文以化成天下”的理念。

上海交通大学教育学院教授、博士生导师，未来教育研究中心主任江丰光为学员作《未来教育趋势与创新人才培养》专题讲座。江教授从信息技术引领教育变革的背景入手，深入分析当今时代国内外教育模式，介绍了国内外关于激发学生主动学习的案例，详细论述了人工智能在教育领域的应用场景，强调人工智能重塑了教师和学生的定义，构建了新的教学生态。

上海交通大学马克思主义学院教授、博士生导师黄庆桥带来了《世界变局背景下的高水平科技自立自强之路》的专题报告。黄教授用真实的数据和鲜活的案例介绍世界变局的主要表现、经历的变化、中美之间的科技实力差异等，并从各个角度分析了中国科技发展面临的主要瓶颈问题，强调问题和困难是客观存在的，必须正视和面对。

分组讨论：头脑风暴，突破思维局限

三天的集中封闭式学习，组织管理上，小组分工明确、相互协作、责任担当；学习研讨中，大家认真聆听、头脑风暴、分享智慧。两个晚上，全体学员分成六个小组，从课程建设、教师队伍培养、打造环境育人空间、提升教学质量等多角度，围绕“教育高质量发展”开展分组讨论。上外闵行外国语中学校长吴金瑜、北外附属闵行田园高中校长陆振权、上海市实验学校西校校长章志强、交大附属实验小学校长王叶婷、颛桥中学校长李卫军、莘松小学校长张丽丽等参加分组讨论并作点评。

吴金瑜校长鼓励各位学员基于学校校情，大胆创新，助力闵行教育的高质量发展。陆振权校长向大家提出八条建议：思想先进、心态积极、眼界整体、奉献境界、善于学习、善于创新、作风务实、积极服务。章志强校长强调要清楚理解“高质量教育”的内涵，对教育的本质、学校的发展理念要有清晰的认识。王叶婷校长寄语年轻干部要明晰自己的责任与使命，对标高质量发展学校的图景，思考发挥自身的领导力和执行力。李卫军校长指出，要思考为什么提出“教育高质量发展”，要正确理解“教育高质量发展”的内涵。张丽丽校长强调教育高质量发展离不开高品质的管理，在管理过程中需要“把握边界、学会跨界、提升境界”。俞琳主任建议后备干部们从问题出发，剖析问题、解决问题，并在反思重建中提升思辨力、领导力、执行力。

小组汇报：分享智慧，彰显责任担当

第三天上午，全体学员分两场举行小组汇报。第1-3组集中第十会议室，分别围绕“校长角色在办好一所高质量发展学校中的作用”、“构建现代学校治理体系赋能教育高质量发展”、“数字化转型赋能高质量教育发展”等主题进行汇报，展现了青年后备干部智慧而具有探索性的思考，体现了青年后备干部的勇气与担当。徐明怡副书记、俞琳主任全程参与。

第4-6组集中第六会议室，分别聚焦“以人为本：高质量教育人才队伍的建设”、“高质量发展视域下高素质教师队伍的培养”、“党建引领，有效推动学校高质量发展”，以情景剧、研讨论坛、小组辩论等形式进行交流，展现了青年干部对教育教学管理的真知灼见。朱靖院长全程聆听并作精彩点评。朱院长充分肯定大家的汇报选题好，表达言之有物，与实践结合，并加入深度思考。但也建议大家还需进行系统思考，如在教师队伍建设中要有高要求且目标明确，要有制度与评价、资源与保障；作为学校管理者还要持续提升自我修养，更好地引领教师专业成长，从而进一步促进学校高质量发展。

现场学习：面向前沿，引领教育发展

培训最后，学员们来到张江高等研究院参观学习。在讲解员的介绍下，一路参观了上海交通大学张江展示厅、公共仪器平台、吕志和科学园、扫描电镜实验室、透射电镜实验室等场所，近距离了解和感受张江高研院在建设历程、定位特点和科研成果等方面的创新与成就，展现了“与日俱进、敢为人先、努力拼搏、求真务实”的交大精神。通过观看宣传片，全体学员对我国物理、化学、生命、信息等基础学科的变革性、颠覆性和引领性有了更为直观与具象的认识，深切感受到科研人员为高质量发展和高科技创新所付出的辛劳，以及坚守与突破。现场学习使学员们的知识体系得以延展、工作眼界得以拓宽，大家纷纷表示将会更加用心、用情、用力地做好教育工作，更加坚定了为党育人、为国育才的决心。

随着三天集中培训的结束，2022-2023闵行区教育系统优秀青年干部培训班的学习也即将画上圆满句号。培训班副班长、华漕学校钱佳颖主持结业仪式，培训班班长、闵行中学金寅仪代表全体学员作“思以行远奋进路 行而不辍新征程”的主题发言，对为期一年的培训活动进行回顾、总结和展望；并表示青年干部们会时刻坚守岗位，以服务大局新高度、改革创新新深度、教育为民新温度为自我要求，且行且思，行而不辍，为推动闵行教育高质量发展、办好人民满意的教育交出年轻干部的青春答卷。

徐明怡副书记、朱靖院长为学员代表颁发结业证书。徐书记并作领导讲话，她指出：本次培训既是一次学习、一次交流、一次体验，也是闵行教育党工委遴选干部、发现人才的一次机会，更是教育强国思想落地的一次重要实践。本次由闵行区教育党工委、闵行区教育局主办，上海交通大学教育学院、闵行区教育党工委党群科、教育学院校长发展中心承办的培训，非常有价值、有意义、有成效。徐书记肯定了青年干部在汇报展示中呈现的思考与实践探索，同时强调青年干部要把准方向，深刻领会国家教育战略部署；还要勇当先锋，引领学校教育的创新发展；更要行稳致远，做忠诚干净的好干部。


踔厉奋发勇担当，接续奋斗正当时。2022-2023闵行区教育系统优秀青年干部培训圆满结束。沐浴新时代、新思想，肩负新期盼、新使命，所有学员必将提高政治站位、强化责任担当、突破思维局限，紧紧围绕“学思想、强党性、重实践、建新功”的总要求，以学铸魂、以学增智、以学正风、以学促干，努力成为具有全局视野的新时代教育管理者，走好闵行教育高质量发展之路！
