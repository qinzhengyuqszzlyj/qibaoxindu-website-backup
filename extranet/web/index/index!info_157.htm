立足课堂实践 品鉴佳作韵味——初中地理浦江大教研的新尝试

发布时间 : 2023-10-20   作者（来源）: 周光明   阅读 : 468

3/8
←
→

2023年10月18日下午，地理学科浦江大教研活动在上海博世凯外国语学校举行，除了浦江片区的初中地理教师，还吸引了浦江以外的其他学校的教师来到现场。

大家观摩了尹艳铃老师执教的《金字塔之国——埃及》一课。尹老师的课堂教学以“埃及凭啥迁都”做为导入，充分调动学生的学习热情，结合多则相关新闻和背景资料，紧扣两条河流的意义展开教学，分为“地跨两洲”、“生财有‘道’”、“生存之‘廊’”三部分，通过学习探究活动，帮助学生认识国家，形成区域认知的素养，并树立正确的人地协调观。

评课环节由教研员周光明老师主持，尹老师介绍了自己的备课过程、学习单设计意图以及课程目标，也提出了一些困惑和遗憾，老师们对尹老师的课堂教学进行研讨，各抒己见，气氛活跃。

尚师初级中学的李起玲老师和协和双语的肖丹瑶老师一致认为本节课的思路清晰明了，学习单的设计起到了锦上添花的效果；上师三附中的陆虹老师认为采用一个问题作为主线贯穿一节课的做法是值得推广的，课堂的呈现效果也比较理想；交大二附中的李娇老师提出“教师在课堂上如何了解学生掌握情况”的问题，也阐述了自己对于“生存之‘廊’”这一部分的理解；浦航实验的邓倩老师肯定了本节课教师在准备图文资料上的充分程度，也针对细节问题提出了一些建设性的意见。

周光明老师提醒教师在授课时应该更多关注学生的反馈，重视对于学情的研究和课程标准的理解，在进行区域地理的教学时，要注重地图工具的运用，结合一些重要的地图进行观察、比较，设计有思维深度的问题引导学生探究，训练并落实区域认知的方法。在时间安排上，周老师建议要详略得当，教师要先做减法，在课堂巡视过程中了解、记录学习单完成的情况，对学生完成得好的、简单的、事实性的知识，不必再重复讨论；然后再尝试做加法，将更多的时间用于重点、难点的突破，针对开放性的问题进行深度的学习、讨论。

观课、评课环节之后，全体教师针对段义孚先生的代表性著作《空间与地方：经验的视角》一书进行了读书交流分享。教师们分工认领其中的一章内容进行发言，结合教学实际和自己的经历，分享书中的精彩片段和文字，思考对地理教学的有益启示。

经过一个多小时的发言、交流、讨论，老师们对于这本人文主义地理学的经典著作有了更多深刻的认识，虽然一开始有些内容会觉得深奥，甚至看不懂，但大家在体味“地方的感受价值”的同时，更能理解乡土地理内容的教学价值，在今后的教学中可以自觉选取更多的真实、具体的地方案例，拓宽教学思路。教师和学生都可以自觉结合“空间”“地方”的丰富经验来促进对生活环境的体验，丰富地理课程的学习策略。

本次活动主动参与发言的人数多，除了课堂观摩和研讨，也有事先的读书任务布置，了解段义孚先生的人文主义地理学思想的读书交流环节，在教研的内容和形式上都是一次新的尝试。大家越来越珍惜这样浦江大教研的学习机会，认真准备，主动参与，自觉思考，通过每一次片区教研活动提高自己对地理学科的理解与认识。立足课堂，自觉反思，广泛阅读，分享见解，勾画出了教师专业成长的清晰路径。
