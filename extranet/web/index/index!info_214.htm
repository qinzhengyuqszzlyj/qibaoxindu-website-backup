2020年闵行区暑期校(园)长 书记培训 ：面向教育现代化 提升改革创新行动力

发布时间 : 2020-07-19   作者（来源）: 徐婧   阅读 : 4121

3/8
←
→

7月15日至18日，2020年闵行区中小幼校（园）长、书记暑期专题培训在闵行区教育学院举行，本次培训的主题是“面向教育现代化的能力建设——提高校长改革创新课堂教学行动力，提升在线教学赋能传统教学实效”。教育局党政领导、各科室负责人，督导室全体督学，区教育学院中层及以上干部，全区公民办中小幼、职成教、直属单位正副校（园）长、书记等900余人参与线上线下培训。

为期四天的培训分为市级培训和区级培训两部分。

7月15日、16日为市级培训。全体参训者通过双向视频收看主题报告。

上海市教育委员会人事处处长李兴华作培训动员：希望通过培训进一步贯彻落实全国教育大会与上海市教育大会精神，聚焦立德树人根本任务，促进校（园）长提高改革创新课堂教学行动力，同时关注疫情引发的挑战和变革。

15日下午，复旦大学附属华山医院感染科主任张文宏作《新冠肺炎抗疫形势与学校防疫》专题报告。张教授以幽默诙谐的语言，对新冠肺炎的开始、“闷死”病毒的原理、核酸检测的机制、科学隔离的原因等问题进行了释疑解惑，分享了武汉、上海、北京等地的抗疫方案，剖析了美国疫情发展趋势，并对复学后校园的科学防疫提出了具体建议，中国将进入“动态清零”新常态。


市级专题培训分为教育信息化、家校共育、在线教育赋能传统教学、学前教育质量提升、激发学校活力五个专场。

教育信息化专场：技术赋能教育，探索模式变革。

市电教馆馆长张治主持专场活动。联合国经济合作与发展组织教育与技能司司长安德烈亚斯.施莱克尔、市教委副主任李永智分别作《学校教育中断——学校教育反思》、《周密谋划，谨慎施策，探索“四全”在线教育》的专题分享。安德烈亚斯.施莱克尔对比了36个国家在教育信息化方面所做的工作，结合上海教育变革，提出了信息化技术可以放大教学，但不能取代教师等观点。李永智对今年春季学期上海空中课堂作了回顾和思考，从疫情下全日制在线教育的主要难点、指导思想等展开，以不同视角呈现了空中课堂背后的故事，并对未来教育信息化发展指明方向，立足教育规律，实现基础应用和个性应用的排列组合，上海教育信息化，未来可期。

家校共育专场：对话后喻时代，共建合作机制。

市教委德育处副处长江伟鸣主持专场活动。复旦大学教授沈奕斐通过案例分享，概念阐释，深度剖析了后喻时代和文化反哺对教育的挑战，提出了新时代家校互动可能的方案和具体操作，确定了家校协同的核心是合作育儿的支持体系构建。建平实验学校校长李百艳以共情、共建、共治、共商、共生的“五共”对话模式，开启家校共育新机制，提出切中焦点、把握学生生长点，用父母心办教育，呵护心尖上的每一个小苹果。

在线教育赋能传统教学专场：智慧赋能教育，打造上海基础教育金名片。

市教委教研室副主任谭轶斌、王月芬，市教委教研室教研员刘达、张新宇分享了疫情期间空中课堂实录，提炼在线教学智慧，解开确保在线教学品质的“战疫方程”，探索了线上线下融合教学的可行模式。宝山区教育学院院长沈伟、静安区教育学院附属学校校长张人利、长宁区江苏路第五小学教师尹晓红分别作了《“双师模式”在线教学：宝山的行动探索》、《提高线上教育质量的学校基础》、《双轨并行，丰富学生学习体验》的经验分享。在线教育赋能传统教学，不仅是技术赋能，也是教师智慧赋能，上海市空中课堂将成为上海市基础教育金名片。

学前专场：保障教育质量，促进内涵品质提升。

市教委托幼工作处副处长王纡然主持专场活动。托幼工作处老师瞿佳杰、教研室学前特教部主任徐则民、装备中心副主任倪冬彬、信息中心学前教育信息部副主任茅红美分别作《关于推进学前教育深化改革规范发展的实施意见》、《让评价成为园所（教师）自主发展的“催化剂”》、《幼儿园装备指托南介绍》、《上海市幼儿园信息化建设与应用指南》专题报告，详细解读了从基建到人才队伍建设的具体内容和改革举措，旨在保障学前教育质量稳步提升。托幼处处长颜慧芬指出：从中央到地方，社会各界高度重视，为学前教育提供无限发展机遇，也促使其由大规模发展到内涵发展的转折，幼有善育、学有幼教，办人民满意的教育。

激发学校活力专场：创新机制制度，激发教师活力。

市师资培训中心副主任陈霞主持专场活动。市教委人事处孙鸿、市人力资源社会保障局专技处副处长陈贝艺从编制管理、岗位设置、绩效分配、招聘任用四方面做主题报告，提出三个坚持原则和“四梁八柱”工作思路。奉贤区教育局局长施文龙、上海市晋元高级中学校长季洪旭、松江区仓桥学校校长林金根、冰厂田教育集团姚健分别作《深化全员岗位聘任，激发每位教师活力》、《让评价成为教师专业发展的发动机》、《深化人事制度改革，促进教育可持续发展》、《赋能阳光教师，助力专业发展》的微报告，分享区域探索与实践经验。


协力创新，开启上海基础教育新篇章

市教委副主任贾炜作《促进上海基础教育高品质发展》重要报告。他从上海市基础教育“十四五”改革发展的历史方位、目标设定、重点任务三方面，精准定位“十四五”需全面把握新时代国家基础教育改革和发展的“四梁八柱”，精准解析新时代上海城市发展对基础教育的“品味要求”。对“十四五”期间，基础教育从“现代化”到“高品质”发展作了详细解读，提出优质均衡、融合发展、富于创造的目标关键词，并布置了优化资源配置、改革育人方式、激发学校活力、加强治理体系能力建设四方面的具体任务，建议各区、各校站得高、看的齐、把的准、做的实，成就上海基础教育“十四五”的高品质发展。


上海市教育委员会主任王平作报告。他回顾了本学期疫情防控、在线教学、中高考等工作内容，对上海教育人在疫情防控期间做出的努力和成绩给予充分肯定，并对下阶段疫情防控常态化做了工作部署。针对如何坚持把学生身心健康发展放在教育首要地位，要求上海教育人遵循学生成长特点和规律，开展全员导师，做好家庭教育指导，始终牢记要培养德智体美劳全面发展的社会主义建设者和接班人；针对如何坚持聚焦课程育人，要提高课程领导力，让课程实施产生促进学生发展的正能量作用；针对如何把教师发展作为教育发展第一资源，要为教师专业发展提供良好服务，注重对教师的人文关怀；针对如何把学习作为可持续发展动力，要学思践悟，理论联系实践，坚定政治信念，明确正确的办学方向，向时代学，向老师学，向学生学，以上海教育人特有的胸怀，咬定青山不放松，创造出更多、更好、更成功的上海经验。



7月17日、18日为区级培训。主要围绕市教委精神及闵行区教育实际，开展面向教育现代化的智慧教育，学习“四史”坚定自信等专题培训。

面向教育现代化的智慧教育：明确方向，精准定位，写好这篇“命题作文”。

17日上午的培训主题是《面向教育现代化的智慧教育》，教育局副局长乔慧芳主持。教育学院教育科研中心主任何永红就《闵行区智慧教育项目：概念解析》作专题报告，从“因材施教”的历史追踪、技术变更、数据驱动，到“个性化学习”、“差异化学习”的概念内涵，运用生动的案例分析，对闵行区“数据驱动大规模因材施教”推进作了深入浅出的解读。七宝中学党委书记兼校长朱越、闵行区实验小学校长何学锋以不同视角，结合办学实际，立足教育本身，分别对智慧教育示范区创建项目进行了思考和探索，让智慧教育项目成为学校持续转型变革的抓手和重要内核。

闵行区教育学院党总支书记、院长朱靖作主题为《数据驱动大规模因材施教》的专题报告。她从项目进展与问题分析、实施建议与管理机制两方面切入，对项目推进作了工作回顾，围绕差异化教学、个性化学习、精准化管理、智能化服务四个项目愿景，为下阶段工作提供了精准指导，强调在加强业务导向，建立数据模型、开发平台工具、创新教学模式等方面进行持续深入推进，实现教育和技术的深度融合。

结合市区两级培训内容，17日下午校（园）长、书记分组讨论关于“智慧教育如何为“双新”赋能、如何呵护和引导学生个性发展”等议题，并于18日下午进行大会集中交流。莘庄中学校长李国兴，七宝三中党支部书记兼校长程庆、蔷薇小学（上师大三附小）党支部书记兼校长沈珺、爱博果果幼儿园党支部书记兼园长唐新萍作了交流发言，

闵行区教育党工委副书记、局长恽敏霞针对智慧教育项目的规划部署作重要讲话，她将今年上海高考的命题作文类比于智慧教育项目推进，提出突如其来的疫情是带给教育的转折期，每位校（园）长要在历史转折期答好这篇命题作文，对下阶段智慧教育项目进行再破题，实现教育新发展，并针对如何结合区域规划，在推进智慧教育项目中界定学校发展目标等问题，鼓励校（园）长对结构有整体认识，对环境有整体把握，思考学校高品质发展的呈现指标和模式，走近企业，寻找差距。智慧教育不仅是时代趋势，也是未来学校不断变革创新的方向和必然路径。


“四史”教育：牢记使命 坚定信念 敢战能胜

18日上午的主题教育专场由区教育党工委副书记李光华主持。中共上海市委党校教授卜新兵作《在学习“四史”中筑牢理想信念坚定“四个“自信”》专题报告。卜教授从“孟晚舟案与华为困局、香港乱局与国安立法、中美脱钩与新型冷战、抗疫斗争与最新疫情、中印边境与中印竞技”五个热点问题切入，围绕为什么学“四史”及怎么学四史，深入浅出解析了开展“四史”学习教育是应对国内外挑战走好新时代长征路的需要，是努力创造新时代上海发展新奇迹的需要，必须明确学习目标，掌握基本方法，清醒认识历史是最好的教科书，也是最好的清醒剂，要了解历史事实、理清历史脉络、把握历史规律，才能弄清楚，理解透；只有做强做好自己，才能走好长征路，实现中华民族伟大复兴的中国梦。

闵行区发改委书记、主任胡志宏作《闵行“十四五”发展趋势展望》，他从“闵行区基本区情、“十三五”取得成效、应对新冠疫情进展情况、“十四五”发展思路、“十四五”重大任务”五方面展开，肯定了区“十三五”以来在经济实力保持领先等七方面的成效，并提出“十四五”期间，要把握好“坚持改革驱动、坚持扩大开放、坚持产城融合、坚持绿色低碳、坚持共享惠民”五大基本原则，深入落实“人民城市人民建，人民城市为人民”的重要理念，全面增强区域核心竞争力和现代化治理能力。


回顾上半年工作内容 部署下半年工作思路

18日下午，区教育局局长恽敏霞总结了2019学年工作，并提出2020学年工作思路。2020年上半年，闵行教育坚决把保障师生生命安全和身体健康作为一切工作根本前提和首要目标，严格落实疫情防控，全面部署线上教育及复学后的教育教学正常开展，同时加强学校基本建设，稳步提高公办与普惠性学前三年教育覆盖率，创建“智慧教育示范区”，坚持“五育并举”，促进学生健康全面发展。2020年是“十四五”谋划之年，闵行教育将持续贯彻落实教育部、市委、市政府和区委、区政府疫情防控决策部署，坚持常态化防控；大力推进“课程教学改革、加强师资培养管理、强化紧密型学区集团建设、创建‘智慧教育示范区’”四大工程；完成《闵行区教育改革和发展“十四五”规划》编写，为未来五年描绘发展蓝图。

区教育党工委书记朱雪平部署2020学年党工委工作，朱书记指出：要坚持在常态化防疫背景下加速推进党的建设、教育改革等重点工作，持续深入开展“四史”学习教育，推进“四史”教育进课堂、进活动、进仪式、进队伍；做好基层党组织换届工作，抓好学校组织体系建设；立足“十四五”，加强青年干部队伍梯队建设，培养懂教育、会管理、有威信，能抓党建、善于做思想政治工作的高素质专业化干部；推动创全决战、中小学思政一体化和师德建设，同时朱书记结合当前区域教育发展，对校（园）长提出三方面要求：加强学习，认清形势，做到心中有数、优化治理，担当作为，做到手中有招、严以律己，照章办事，做到胸中有谱。

四天的集中培训主题明确，内容丰实，后续校（园）长、书记们还将通过登陆市级培训平台，浏览学习资源，开展自主学习。2020年暑期校（园）长、书记培训是一次真正意义上的教育峰会，在共研共享中帮助校（园）长、书记提升改革创新行动力，以自觉的态度投身事业，以坚韧的意志攻坚克难，去帮助每一个孩子寻找到最适切的教育。
