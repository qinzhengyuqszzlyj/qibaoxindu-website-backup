落实课标要求 构建育人蓝图 丰富文化体验 —— 闵行区小学英语二年级主题教研活动于上师三附小举行

发布时间 : 2023-05-18   作者（来源）: 沈雯晴   阅读 : 1329

3/8
←
→

2023年5月16日下午，主题为“落实课标要求 构建育人蓝图 丰富文化体验”的闵行区小学英语二年级主题教研活动在上海师范大学附属闵行第三小学举行。本次教研活动由闵行区教育学院英语教研员沈雯晴老师和闵行区福山实验学校副校长、闵行区学科带头人陈敏老师参与指导，闵行区各校约150位老师亲临现场参与本次教研。

本次活动由上师三附小的常悦老师主持。活动内容包括单元整体规划、单课教学实施、教学设计说明、互动交流研讨和专家指导点评。

首先，石芊芊老师基于新课标研读，呈现了《英语（牛津上海版）2B Module4 More things to learn Unit2 Mother's Day》的单元整体规划。

随后，陈静宜老师和周颖老师分别为大家呈现了第一课时Ideas for Mother's Day和第二课时Happy Mother's Day的教学展示。

第一课时中，陈老师充分挖掘教学内容，多层次、多角度分析语篇传递的意义，巧用多模态语篇，挖掘母亲节文化内涵和育人价值，并且通过形式丰富的学习活动，激活和调动学生的感受与思维，从而选择最合适的礼物送给妈妈。学生的输出以“I hope she likes them.”收尾，既表现了对妈妈最真挚的祝福，也体现了为妈妈挑选的礼物是有思考、有依据、有温度的，从而实现从基于语篇到深入语篇，最终超越语篇的学习。第二课时中，周老师秉持学习活动观，引导学生在学习理解的过程中，获取信息，梳理语言结构；在实践应用环节中，加深学生对语言结构的理解并初步运用；在迁移创新环节中，通过视频和学生资源，将故事情境延伸到真实语境，引导学生运用所学介绍自己如何为母亲庆祝节日，回馈母爱，切实提高了用英语做事情的能力，从输入到输出水到渠成。从第一课时的节日准备，到第二课时的实施和表达，层层递进、环环相扣，最终实现单元育人目标“感悟母爱，常怀感恩之心，常行感恩之举”。

精彩的课堂过后，上师三附小的顾馨远老师和王祯滢老师从“落实课标要求、提炼主题意义、构建育人蓝图”和“优化活动设计、依托多元资源、丰富文化体验”等角度对两课时进行了教学设计说明。

紧接着，来自上师三附小的沈骁老师、福山实验的姚伟婷老师、蔷薇小学的田胖妹老师和君莲学校的何红红老师从“多元资源、核心素养、育人蓝图、学习活动”等方面对两节课进行了互动点评，提出了自己的观课感受和建议。

活动最后，闵行区福山实验学校副校长、闵行区学科带头人陈敏老师对本次活动做了点评总结。她充分肯定了两位执教教师的精彩呈现以及三附小英语教研团队的付出。陈老师基于新课标要求的落实，从“提炼主题意义，深挖育人价值”、“创设整体语境，优化教学策略”和“优化活动设计，丰富学生体验”对两节课进行了深入的剖析。

区教研员沈雯晴老师和学科带头人陈敏老师的多次指导为上师三附小教研团队提供了宝贵建议，指明了研究方向。相信在专家的指导下，我们将继续落实课标要求，践行学科育人理念，并肩走在教育教学的大道上！


撰稿：上师三附小 金晨一

审稿：闵行区教育学院 沈雯晴
