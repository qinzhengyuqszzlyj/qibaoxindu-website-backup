孩子们的自然‘食’光——记学前生活中心组实践研讨活动

发布时间 : 2023-06-09   作者（来源）: 张静   阅读 : 443

3/8
←
→

2023年6月5日，生活中心组围绕“食缘”“食行”“食话”“食思”，以“幼儿发展优先”的课程理念在天恒名都幼儿园开展了本学期第四场实践研讨活动。

一、“菌菇朵朵”微课程现场活动展示

天恒名都幼儿园的张莉莎老师向生活中心组展示了食育微课程分享会“菌菇朵朵”，小小设计师们通过自制暗房创设了秀珍菇喜爱的阴暗环境，通过安装暖灯来人为调节温度；通过小发明“按摩捶”利用细棍敲打菌棒促进菌丝的生长。他们展示“秀珍菇成长日记”，表征记录秀珍菇的心情、故事、问题和发现，秀珍菇成长过程中的点点滴滴，都可以成为我们记录的内容。“菌菇的N种吃法——体会制作菌菇美食的快乐心情”菌菇营养丰富，我们可以通过不同的烹饪手法，让它变成桌子上的一道道美味。现场“课程分享会”结束后，教研员张静老师引领中心组成员围绕“从自然食光类课程实施的角度，发现一个优点、提出一个问题、给出一个建议”这一话题展开了热烈讨论。老师们对于很多活动种幼儿的自主探究与合作赞不绝口，对于“食育的价值”“科学探究中教师支架与幼儿自主探索之间的关系”等进行了深入研讨。

二、“自然‘食’光”食育微故事分享

鑫都幼儿园的杨丽老师带来的食育微故事是“我是多么喜‘番’你”，“尖尖的是番茄的种子吗？它是怎么长出来的？我们吃下去之后会不会在我们的肚子里发芽？”杨老师尊重幼儿的兴趣和需求，开启以儿童为中心的种植活动探索之旅，让幼儿亲历种植全过程，了解植物的生长规律，解决生活问题，体验劳动的快乐。

爱博果果幼儿园的曹春燕老师带来的食育微故事是“从一颗柠檬树开始”，缘于已有的对柠檬树的种植、照料、观察经验，对柠檬树的好奇、探究兴趣等，和孩子们一起春季观察果树的花朵，秋季采摘和制作美食。

龙柏第二幼儿园的刘知非老师带来的食育微故事是“遇见小‘莓’好”，通过莓好种植、莓好采摘、莓好食刻、莓好体验，让孩子们在观察中初步了解草莓的生长过程，进行简单的表征记录，参与草莓的采摘、清洗与制作；体验与家人分享草莓的快乐。

一品漫城幼儿园的张丹玲老师带来的食育微课程故事是“小蔬菜的大收获”，当发现幼儿自由活动时第一选择是户外阳台植物角后，老师决定让他们身临其境，体验自然给予人类的馈赠。照顾蔬菜过程中孩子们体会到了劳动的快乐，采摘蔬菜让幼儿感受到通过自己努力从而收获的快乐，将采摘的蔬菜带回家和家人一起品尝，孩子们体会到了分享的快乐。

三、“自然‘食’光”里的分享与行动

颛桥第一幼儿园的李兰芬老师通过书籍查找：感受到食与人们生活的密切关系，为老师们推荐了很多有趣的书籍，让我们感受到开卷有益、育之有道，食而有味。同时也提出了“把握出发点——是为了获取食物而种植；找准落脚点——关注每个环节食育的要素”的体会。

最后教研员张静老师带领中心组的成员围绕“如何看待这些课程中的‘植物生长过程（选种—播种—照料—采摘）’与幼儿经验的生长过程之间的关系”这一话题展开了互动研讨，老师们在轻松的氛围中各抒己见。

自然食光类的课程，驱动性问题和食相关，过程中可以用食的线索去串联。植物在生长，孩子在成长，带着期待所以关注变化。食育不仅仅是吃，是生活的味道，成长的味道！



撰稿 :  天恒名都幼儿园 张莉莎
