关注每一天、关照每件事、关心每一个 ——2018年闵行区新任校（园）长带教培训金芸导师组活动通讯（1月）

发布时间 : 2019-02-18   作者（来源）: 曹雅明   阅读 : 2487

临近学期结束，各条线开始了期末成效展示和总结工作：家长开放日活动、期末家委会总结、幼儿综合能力测试分析、骨干教师展示活动、新年活动、绩效奖励发放等等，各方面事务应接不暇。师傅金芸园长告诫徒弟，越是忙碌的时候越要冷静思考和处理各项工作，做到：关注每一天，关照每件事，关心每一个。

一、联动工会，关心每一个 

为了充分发挥工会在教职工中的纽带作用，元旦前夕我们走访慰问了幼儿园的所有老教师们，从老闵行到莘庄、从一楼到六楼，我们带去了对老教师的感谢和祝福，也带回了老教师深情的厚爱。而2018年的最后一天全体教工的“挥别2018”仪式上，老师们留下了组室合作的倩影，留下来2018马桥幼儿园全体教工砥砺前行的全家福！关心每位教工让我们的心走的更近，更加便于开展工作！

二、及时跟进，关照每件事

作为园长，幼儿园各类活动的质量与现状、家长的意愿与建议、孩子的需求与发展不仅要做到了解，还要做到有效解决和落实。天冷了，孩子门口接送改为大厅接送后一度出现拥挤状况，我们采取班级分批接送、行政跟进、家志愿者维护、调整接送通道等举措，化解了接送孩子时的安全问题。组室开展研讨课我跟进观摩，参加他们的讨论，了解教研组研讨的现状和老师们的研讨水平；幼儿园重点课题开题了，我组织大家一起讨论课题研究进展，分析研究目标和内容，和大家一起制定研究的方向。师训主任联系不到讲课老师，我及时和教委联系在他们的牵头下，邀请马桥老人来园讲马桥历史。及时跟进，让许多问题能得到及时妥善的解决。

三、一日三巡，关注每一天

徒弟牢记师傅告诫：每一天都要挤时间到幼儿园的每个角落去看看，去每个部门主动了解情况，做到幼儿园每天的情况心中有数。巡视中发现幼儿午餐的盛饭工具保暖性不好，及时建议后勤制作保温袋。巡视中发现保安室的访客记录纸没有了，及时提醒总务更换……关注每一天，点点滴滴中是对细节的关注，更是园长的责任所在。

这段时间来我深刻地体会到：管理无小事，作为园长要想他人所想，想他人所未想，用细节来提升品质，用服务来打磨内涵。

 

摄影、供稿：马桥中心幼儿园   诸伟芳
