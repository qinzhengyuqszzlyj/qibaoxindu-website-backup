把握劳动教育育人导向 让“智慧纸笔”点亮劳动课程 ——闵行区初中劳动教研坊数字化项目研究教研活动

发布时间 : 2023-09-23   作者（来源）: 刘至宁   阅读 : 1277

2023年9月22日上午，主题为“数字化促劳动素养的培育”闵行区初中劳动教研坊“智慧纸笔”应用与精准教学的项目研究教研活动在交大马桥实验学校开展。初中教研坊全体成员参加了本次教研。

教研的主要内容就是研究智慧纸笔在课堂教学中的有效应用，从教学环节设计到教学实施，要体现数字化应用的有效性；并在教学中体现“劳动育美 美劳共育”的教学理念，把握劳动教育育人导向，体现教学中的单元设计和学科德育。

首先是由闵行区中小学劳动教研员刘至宁老师，开设2节公开展示课《纸艺小品——向日葵》。第一节课是以设计为主，通过向日葵的寓意和内涵，引导学生要具有阳光、温暖、积极的正能量的设计理念，再通过小组合作的形式，完成方案设计并作展示交流。其中，学生的方案设计则利用“闵智作业”平台的“智慧纸笔”进行草图绘制，教师则在平台上进行“点优”评价和“对比展示”，让学生交流设计理念，交流过程中，教师通过平台上的“思维呈现”过程来再现学生的设计过程，即让学生的设计思维可视化，助力学生的设计展示。


“智慧纸笔”的运用，大大节约教学时间，教师可以线上、线下通览学生的设计情况，及时捕捉典型设计作品，精准指导；学生也可以通过通览作品，一目了然各小组的基本设计情况，再进行学生的主动展示或推荐性展示，同时，也可以进行问题展示，平台上的作品通览功能，使展示交流环节更有助于精准评价。

第二节课则是在方案设计交流的基础上，了解学生对纸艺加工技能中的镂刻和折叠功能的实践情况，发现学生在自主学习加工技能中的问题和不足，以点拨技术要点的方式帮助学生突破技术重、难点，帮助学生完成向日葵小品的制作。因为有效的教学设计，使得学生有充足的时间完成加工、制作与美化，在作品展示环节，同学们展示了“杭州亚运”主题、“向阳而生 逆风而行”、“丰收之悦”、“秋之静美”等主题作品，设计理念积极向上，温暖并充满力量。不仅呈现作品的劳动育美特质，还呈现出学科育人的劳动魅力。


最后教师通过“月光宝盒”视频展示了优秀设计师应该用独创新颖的设计理念、精良的制作工艺、其不可替代的实用性及优美的语言表达来综合呈现最美设计，将学生的思维和方法进行迁移，引导学生进入高阶思维，为学生打开生活与未来。

两节课下来，学生作品精彩纷呈，正是因为课程的单元设计和教学设计中，准确把握育人导向，结合劳动育美的教学理念，有效地运用数字化支持，才能为学生提供有效互动和精准指导，变被动学习为主动学习，变模仿性学习为创造性学习。

课后，教研坊的老师们展开积极热烈的教学研讨，从“闵智作业”平台、教学设计、单元设计、作品呈现、美劳共育等多方面进行了学习、交流和分享。本次教研将开启初中劳动课程应用“智慧纸笔”的序曲，已经有越来越多的劳动教师开始关注和学习平台使用，为未来劳动课程教学设计和学生劳动素养培育开启新篇章。



交大马桥实验学校  胡晓翠（报道）

2023年9月23日
