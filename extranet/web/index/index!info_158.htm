【学前 东片】在对话中促成长

发布时间 : 2023-06-07   作者（来源）: 刘时医   阅读 : 855

3/8
←
→

2023年6月7日上午，闵行区教育学院主办、浦莲幼儿园承办的“基于幼儿发展优先的师幼互动”东片主题教研活动顺利开展。闵行区教育学院学前教研员刘时医老师、东片公办幼儿园保教主任、小班教研组长共同参加了本次教研活动。活动由浦莲幼儿园保教主任张迎霞老师主持。

教学观摩 记录对话

    教研前，由徐琰老师执教小班音乐活动《海底总动员》，通过情境性的游戏场景、愉悦的活动氛围、师幼生动的多元对话等形式让小班幼儿沉浸于音乐活动中，感受理解音乐，乐于用肢体动作表现音乐中的小鱼和大鲨鱼不同的音乐形象。

聚焦话题 发现对话

由教研组长吴荣老师主持，带领浦莲幼儿园不同梯队的12位教师开展了主题为《与幼儿对话——小班音乐活动中师幼互动的有效性》的现场教研。老师们基于儿童视角，围绕对话、音乐活动中师幼之间如何展开有效对话等进行深入的梳理，从对话的环境、语言运用、肢体动作三方面进行了重点的讨论，复盘教学活动中幼儿和教师的实然状态，梳理对话经验、寻找优化途径，让音乐活动中的师幼互动更鲜活灵动、更凸显园本课程的味道。

思维碰撞 拓展对话

    主题教研后，三所姐妹园的老师代表也对本次活动进行了热烈的交流发言。浦江三幼的张老师从教学活动的设计、教师教态等给予了肯定，并对基于目标的活动时间配比在环节的优化方面提出了建议。闸航路幼儿园的李老师肯定了活动前期充分的准备、场景的创设、幼儿积极的表现，并在游戏环节组织如何更开放自主方面提出了可行的想法。华师大紫竹幼儿园的吴老师发现了执教教师童趣化语言的运用、肢体动作的启发等亮点，从互动形式的丰富性方面给予了建议。

精准点评 升华对话

教研员刘时医老师对整场活动做了精彩的点评，对莲幼团队扎实的教学实践和追求优质教研的精神表示赞扬。也对本次活动提出了几点建议，在音乐教学活动中要进一步凸显音乐元素，让幼儿在游戏中不断加深对音乐的感受理解，更自主开放；同时对于音乐活动中的提问进一步精炼，紧扣音乐核心经验和活动目标提出了思考。对于教研组织方式，有待优化在整场教研中要让所有组员充分参与互动、思维碰撞，对于围绕教研目标的教研方法梳理应更加具体、可实施性强等。

通过本次主题教研活动，让我们对幼儿园的园本课程实施、教研组织有了更深入的思考，在朝着高质量幼儿园迈进的道路上更加明确方向。
