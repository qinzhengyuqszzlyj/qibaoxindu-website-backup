核心素养导向下的民族音乐教学法实践 ——闵行区小学音乐2023学年第一学期第九周教研活动

发布时间 : 2023-10-28   作者（来源）: 刘庆江   阅读 : 1621

3/8
←
→

2023年10月26日下午，闵行区2023学年第一学期小学音乐第九周音乐教研活动，在闵行区田园外语实验小学（银都校区）举行，活动主题为《核心素养导向下的教学法研究与民乐教学实践》。此次活动是本学期闵行区小学音乐高级教师教育教学风采展示系列的第二场活动。华东师范大学音乐系李茉博士，黑龙江省市三位音乐教研员，上海市音乐正高级教师、特级教师、闵行区小学音乐教研员刘庆江和全区260多位小学音乐教师参加了本次活动。

本次活动旨在探索以核心素养为导向的教学法，并结合民乐教学实践，促进教师专业能力的提升，提高课堂教学的有效性。

展示课之一由闵行区实验二小高级教师葛文君老师执教，她设计的研讨主题是小微单元背景下彝族音乐作品音乐要素感受和体验，将彝族作品《快乐的诺苏》和《阿细跳月》作为微单元，并拓展彝族文化，以培养音乐素养为目标，注重审美感知、艺术表现和文化理解。本节课主要以聆听为先，感受5/4拍的节拍韵律、主题中力度、速度变化等音乐要素感受和体验跳月基本舞步，并借助红绸带、红鼓将彝族音乐欢快、热烈的气氛推向高潮，教学过程层层递进，师生共同沉浸在狂欢热烈的氛围中。

展示课之二由闵行区实验小学的高级教师陈洁老师执教，展示了上音版音乐教材四年级第一学期第五单元《幸福年》的教学案例。陈老师通过欣赏、对比、分析和表演等多种教学方法，帮助学生了解民族乐器板胡的构造、音色，作品的人文背景，在演唱和演奏中进一步理解乐曲的特点和情感表达，并引导学生在充分的聆听感受和合奏中体验民乐的魅力。这堂课突出展示了老师的专业素养，以及学生的主体地位和合作精神，让学生在音乐中感受到了幸福与快乐，并体验了民族音乐的内涵。

   在课后的现场点评中，华师大李茉博士充分肯定葛老师善于充分调动和捕捉学生现场表现，并能灵活调整课堂节奏，学生的活动体验非常充分，她强烈感受到了孩子享受彝族音乐的快乐。李茉博士赞赏陈老师在课堂中将自己二胡专长在作品《幸福年》中发挥的淋漓尽致，学生被老师的演奏深深吸引，通过学生聆听和听辨、分组体验敲击红鼓、小钹，充分感受到洋溢着幸福的喜庆场面。来自黑龙江的教研员代表金老师用他风趣幽默的北方语系提出音乐课的核心素养到底从哪方面落实，学生对于民乐的演奏如何更加细化等角度提出了改进意见。

最后，教研员刘庆江老师也阐明要以新课标为指导和准则，学习各个年段学习内容和学业要求，建立课堂教学单元意识，制定规范单元教案，并在之后的教研活动中组织音乐教师对课标的内容进行考察。 

本次教研活动的顺利开展，不仅为教师们提供了一个学习交流的平台，也为学校音乐教学的发展和提升起到了积极引领的作用。希望通过本次活动，能够激发教师们的教学探索热情，不断打磨和精进教学基本功，切实让学生的音乐核心素养真正落地。
