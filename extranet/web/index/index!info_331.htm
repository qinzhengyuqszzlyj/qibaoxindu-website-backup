数字化助力学生思维发展 ——闵行区初中化学研训活动在文来中学举行

发布时间 : 2023-11-14   作者（来源）: 程静芳   阅读 : 467

3/8
←
→

  轰6K飞机的澎湃动力之源是什么？自然界的水经历了怎样的过程变成水杯里澄清透明的饮用水？2023年11月8日下午，闵行区全体化学教师在教研员程静芳老师的组织下，来到上海市文来中学，和学生们一起从化学的视角探秘这些源自生活的有意义的学习问题。

   文来中学葛岩老师《探寻轰6K澎湃动力之源—燃料充分燃烧》，引导学生一起探寻轰6K动力之源，解轰6K飞行过程中面临的挑战和困难，揭燃料不充分燃烧之谜。学生不断在真实情境问题的解决过程中，提升运用跨学科思维解决实际问题的能力，发展科学思维，经历科学探究，感悟大国工匠精神。

   莘城学校张晓伟老师《水“之道”——从物质分离到水的净化》，从物质的分离入手，启发学生思考利用物质间性质差异完成分离，并在自来水厂净水过程学习基础上，探究完成家用净水器内部净水过程，宏微结合感受水中物质被层层分离的原理，了解反渗透膜在海水淡化工程中的应用前景及挑战。

   有意义的学习问题，有意思的学习活动，有活力的学习思维，基于真实情境问题解决的研讨课受到与会教师的一致好评。在随后的研讨活动中，大家各抒己见，展开热烈研讨。中心组吴泾中学张晓玲老师代表中心组进行总结式评课，静安区化学教研员张彩英老师从单元设计、技术赋能等5个维度进行专家点评， 并对闵行初中化学教研团队呈现出的高质量教学研讨高度肯定。

   教研员程静芳老师在活动的最后，勉励大家在“双新”背景下，勇于探索新的教与学方式，将项目化学习理念、跨学科问题解决、数字化技术使用等融入到日常教学中，激发学生深度学习，感受化学学科价值。未来，让我们一起助力学生成长，引燃成长的智慧火花。

闵行四中 刘叶
