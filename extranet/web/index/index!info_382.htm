依法办园 化解危机 ——2018闵行区新任校园长带教培训林琳导师组活动通讯（10月）

发布时间 : 2018-10-17   作者（来源）: 曹雅明   阅读 : 2562


为了提高新园长的管理能力，从容应对在管理中存在的各类危机，并在遇到危机时能及时依法化解，更好地做好规范办园。 2018年10月12日，龙茗路幼儿园在督导室林琳督学的组织下，特邀上海市浦东新区东方幼儿园毛美娟特级园长，就如何依法办园、化解危机，对林督学带教的浦燕群、王毅、李小红等新园长以及他们的中层团队进行了两个半小时生动而形象的讲座。

毛美娟园长围绕自己在近30年幼儿园管理中的经验，结合身边的案例，具体生动地阐述了自己如何依法化解幼儿园遇到的危机，并指出做好危机管理工作需要经历以下三个阶段：

1. 危机预防和准备阶段。即在危机发生之前组建好危机管理小组，对幼儿园的各种设施进行风险评估，对老师进行危机能力处理的培训。

2. 面临危机的反应阶段。在危机发生后，作为管理者首先要平复好自己的情绪，在有效沟通的基础上分工协调、收集信息，然后制定、实施危机处理计划，最后化解危机。

3. 危机后评估和恢复阶段。危机化解后要评估危机对人的心理影响，分析和积累危机案例。

这“危机应对三部曲”看似简单明了，在真正实施过程中，却需要管理者具有“律师型”的思维与长期修炼的“智商、情商和胆商”。

培训最后，师傅林琳督学再次强调，新园长在幼儿园管理中一定要做到学法、懂法、用法，要用自己的智慧处理可能面对的各种危机，平衡法与情，做到情法相融，有情更要依法。

通过此次培训活动，三位新园长对幼儿园危机管理有了深层次的认识和了解，当然做好危机管理工作还需要新园长把危机管理策略有机融入到一日活动中， 这样才能更好地促进园所健康和谐地发展。

摄影、供稿：龙茗路幼儿园  浦燕群
