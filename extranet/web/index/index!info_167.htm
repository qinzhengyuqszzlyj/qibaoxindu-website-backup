走进户外 优化时空 —— 闵行区虹鹿幼儿园课程基地活动

发布时间 : 2023-06-03   作者（来源）: 鲁鸣   阅读 : 456

3/8
←
→

2023年6月2日上午，虹鹿幼儿园开展了以“走进户外，优化时空”为主题的课程基地活动。本次活动邀请到了正高级、特级园长洪晓琴老师莅临指导，虹鹿幼儿园基地主持人张佩敏园长、基地负责人罗老师以及所有基地成员参加了本次活动。大家共学、共鸣、共成长。

园长致辞，开启活动

张园长首先欢迎洪老师和基地学员的到来，明确了课程基地对于课程建设与研究的重要意义，并期望学员们在今天的活动中能够带着思考去观察、倾听和记录，在基地活动中有所收获。

走进游戏，观摩学习

接着，老师们一起观摩虹鹿幼儿园井亭园9个班4个年龄段的户外游戏。各班巧妙地利用户外环境为幼儿提供开放的游戏空间，真正将游戏的权力交给幼儿。在现场，老师们驻足于每个游戏区域，有的对照着《儿童主动学习观察表3.0》观察幼儿游戏，有的用手机记录下孩子们在游戏中的精彩瞬间......

聆听案例，收获经验

从游戏现场回到多功能厅，老师们聆听了两个大班的游戏故事，分别为《奇趣野战营》和《让我们荡起秋千》。胡老师和施老师生动地介绍了孩子们在游戏中主动学习的故事，以及自己对幼儿在游戏中的观察识别和支持策略，带给老师们深深的触动。

专家引领，助推成长

游戏专家洪晓琴老师对本次活动进行专业点评与指导。首先，洪老师肯定了虹鹿幼儿园在优化空间方面的实践做法。她认为“优化空间”不仅存在于管理层，同时也存在于一线教师中，因此每位教师都应当有优化意识。关于优化，她提出了四个关键词，分别是开放、合理、盘活和充分。第二，她认为游戏中幼儿非常自主，而教师也能做到以观察和记录为主，特别是在倾听了幼儿的需要后，会适宜地进行回应、给予帮助，能时时追随幼儿。第三，洪老师为大家提供了一份在游戏中的“操作宝典”，即“让孩子们把所思所想能够亮出来”。言简意赅的一句话，让老师们深受启发。

活动最后，基地主持人张园长对本次活动进行总结。张园长感谢每位基地成员的认真参与，希望老师们回去后将学习内容进行梳理，并将学习到的方法和经验运用到自己的实践过程中，不断地促进专业能力发展，为幼儿园的课程建设添砖加瓦。
