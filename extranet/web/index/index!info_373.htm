携手提升能力 强化责任担当 ——2018年闵行区新任校（园）长带教培训唐颖萍导师组活动通讯（10月）

发布时间 : 2018-11-03   作者（来源）: 曹雅明   阅读 : 2476

学校中层干部是学校各项具体工作的有力执行者，发挥着承上启下、沟通协调的桥梁作用，校长通过团队建设来实现对学校的有效管理。中层干部的执行能力和管理水平，决定着学校各项工作的顺利开展，推动学校未来发展的质量。本月，针对如何夯实学校管理的中坚力量问题，马桥复旦万科实验中学校长李曼华与师傅唐颖萍校长多次沟通、交流，分析学校中层团队建设措施，力求打造一支求实、进取、创新的中层管理团队。

一、周边校际联盟 共促团队发展

为加强马桥区域校际之间的联动，助力马桥教育发展，10月17日，马桥复旦万科实验中学李曼华校长带领部分行政团队前往同属马桥镇的姊妹学校——华东理工大学附属闵行科技高级中学，实地感受“科技”校园文化氛围，学习校园文化建设。两校领导班子就学校管理、办学特色、信息化平台构建、师资队伍培养、工作经验等方面进行了深入交流和探讨。李曼华校长提出希望能以和华理科高合作共建为契机，形成区域共同体链和学段贯通共识，打通课程通道，为学生个性化成长提供更广空间，以此实现学区内资源共享、课程共建、发展共进的目标。在本次开展的“十五届未来工程师区赛”的前期准备中，马桥复旦万科实验中学得到了华理科高科技团队的大力支持与帮助，学生们也通过自身不断训练后，在赛场上自信满满。

二、校际合作齐发展，逐梦路上创新篇

中层管理团队的能力水平，直接影响着学校的发展，但学校开办之初，马桥复旦万科实验中学行政团队整体相对较为年轻。为提升学校中层团队的管理水平，在与师傅唐校长多次沟通后，继新教师岗前培训、教研组长交流结对后，10月19日，上虹中学管理团队在唐校长的带领下来到徒弟学校马桥复旦万科实验中学，以“校际合作齐发展，逐梦路上创新篇”为主题的两校行政团队共建活动正式启动。两校行政团队分别对各自部门的工作进行了梳理和回顾，交流实践经验，碰撞思想观点，分享管理举措，最后，唐颖萍校长强调这次共建活动对两校来说都是学习、提升、互相切磋的很好机会，希望两校的管理团队利用好共建平台，经常在一起沟通、互动，引发思考，就共性的具体问题开展实质性研究，共同探究，共同进步，变被动为主动，解决自身发展中的实际问题。后续，两校行政团队针对各自部门的工作事宜经常性地互动交流，分享各自工作心得，在获得一份友谊的同时，也提高了工作效率。

三、强化沟通协作，提高团队凝聚力

中层是学校的中坚，师傅唐校长一再强调，只有高效沟通的团队，才能有高效的办事效率，徒弟李曼华校长也在不断探索规范科学高效的管理模式。马桥复旦万科实验中学每周召开行政例会，及时反馈各部门工作情况，以及针对即将开展的活动相互沟通协商。如针对近期校家委会、少代会成员及小记者团提出的学校食堂改善意见，马桥复旦万科实验中学校长带领行政团队经过近两个星期的深入调查，安全领导小组于10月21日对食堂也展开全方位自查，齐心协力，严把食品关，确保学校食堂安全，保障学生午餐营养均衡。学校活动坚持“明确责任，及时沟通”的原则，谁牵头、谁负责，随时巡视，及时整顿，相关部门协从帮助，提升条线工作效率，有效完成学校工作。

携手提升能力，强化责任担当。学校的发展离不开优秀的管理团队，在以后工作中，学校将继续多层面盘活资源，借力发力，逐渐打造一支高效、凝聚的中层管理团队，感谢师傅唐校长的精心指导与鼎力相助。

 

摄影、供稿：马桥复旦万科实验中学  李曼华
