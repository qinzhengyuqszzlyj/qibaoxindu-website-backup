以教育家精神办闵行教育--2021年闵行区校（园）长基地培训暨新任校（园）长培训开班仪式

发布时间 : 2021-11-05   作者（来源）: 徐婧   阅读 : 3618

3/8
←
→

11月2日，2021年闵行区校（园）长基地培训暨新任校（园）长培训开班仪式在浦东新区筑桥实验小学会议厅举行。闵行区教育党工委书记恽敏霞出席活动，闵行区教育党工委党群科科长林火生主持活动。

闵行区教育学院校长发展中心主任俞琳解读培训方案。她从“培训目标、培训内容、组织保障”等方面做了简要说明，围绕“学校发展规划、课程建设、教师发展、学校资源配备、学校管理”五方面明确了培训内容和形式，最后建议校（园）长注重过程管理和时间节点，在导师指导下，通过项目研究等方式，实现办学的突破性进展并呈现标志性成果。

七宝中学校长朱越、闵行区实验小学校长何学锋作为导师代表发言。他们首先感谢了教育局和教育学院领导的信任和支持，通过回忆自身成长道路明晰责任，认真研究在新时代背景下对校（园）长培养的要求，形成学习和研究的共同体，在做中学、学中做。同时建议学员们适度清空自己，善于学习，悉心观察导师和同伴学校，尝试解读导师的办学理念和导师背后的学校，形成常态化的深度研究。

颛桥中学校长李卫军、蔷薇小学校长陈晓苗作为学员代表发言。他们表示在新一轮教育改革和“双减”背景下，作为基层学校管理者，在主动作为，迎接挑战的办学过程中的确存在很多困惑，感谢教育局和教育学院提供平台，希望能通过培训学习，明确学校发展路径，实现育人质量的提升。最后表示将用勤奋和毅力，努力学习，克服困难，积极思考，遵守培训纪律，进一步凝聚思想共识，努力将学习成果转化为办学实效。

恽敏霞为基地导师和新任导师颁发聘书并作讲话。她从筑桥实验小学“梦想是责任的开始”一语引入，明确了校（园）长使命光荣、责任重大，并对学员作“学思践悟”四方面要求，充分理解“为教之道在于度，为学之道在于悟”，要注重教育管理和育人过程，提升自身专业度，并建议学员要学习导师，多走出去开拓眼界，形成自己的思考和办学实践，并在导师的引领下，学员的相互切磋中，形成更多优秀校长成长的案例，推进学校可持续发展，向教育强区迈进作出贡献。

筑桥实验小学郑校长做简要的学校介绍，并组织全体参会者参观了校园。

活动最后， 2021年闵行区校（园）长基地培训暨新任校（园）长培训班全体学员与导师举行了简单而隆重的签约仪式。相信学员在导师的精心指导和自身实践中，能尽快明确学校定位和发展方向，在传承与创新中实现可持续发展，为实现闵行智慧教育的目标共同努力！
