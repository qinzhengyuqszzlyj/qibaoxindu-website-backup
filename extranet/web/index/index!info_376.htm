多听，多看，细思量 ——2018年闵行区新任校（园）长带教培训金芸导师组活动通讯（10月）

发布时间 : 2018-10-24   作者（来源）: 曹雅明   阅读 : 1663

金秋十月，秋风送爽，马桥中心幼儿园里丹桂飘香，整洁明亮的环境、规范有序的一日活动，一切有序而井然。通则思进，接下来马桥中心幼儿园在环境、管理、内涵发展等方面该如何走？怎么走？师傅金芸园长给徒弟诸伟芳园长的建议是：多听、多看，细思量。品味着师傅的建议,10月迎来了多位专家老师的指导。

一、倾听中收获可变信息

10月10日，林琳督学带队来园指导，林老师建议幼儿来园洗手池改位置并增加水龙头数量，因为原有5个水龙头对于12个班级规模的幼儿园来说有点少，早上幼儿来园势必会有等待问题；林老师还建议对幼儿园大门口的车棚平移，用来搭建小木屋放运动器械，看来幼儿园格局的整体性、美观性、实用性也很重要。植物可以供大家观赏，并发挥教育价值，可有刺的植物可能会伤到孩子，林老师对我园“秘密花园”里植物的种植也提出了合理化建议；同时，还对幼儿园版画特色要扎根区域、依托区域特色的重要性也提出了建议。

10月16日，聆听了师训吴老师对我园师德培训主题的指导和建议，提出师德培训要与幼儿园特色课程巧妙结合的新点子，让我明确了拓宽师训内容，丰富师训形式在教师专业发展中的重要性。10月18日，马桥学区化办学颛孙主任的讲座，不仅使我了解了马桥教育的现状，更清晰了我们幼儿园的弱点，明确了发展的方向。聆听专家的指导，思量专家的建议，深受启发。

 二、观摩比较中思考差距

10月，我有幸参加了学区化姐妹园的开放活动，看他们的外环境、内环境，学习他们人文环境的设计与展示的亮点；看他们的开放活动现场，学习他们团队合作分工的精细化管理水平。我边看边思考：我们可以学什么，我们可以怎么做。

听专家的指导和建议，学习姐妹园的亮点，思变的首要准则是：规范！一切以安全为抓手，以幼儿为出发点！10月我们学得最多的是制度和法规，我们改善的重点是安全设施和设备，同时多多鼓舞老师们的干劲：选送区重点课题、一般课题、小课题全部获得批准，流动画展教师指导积极、家长热情参与。

马桥中心幼儿园的环境在悄悄发生着变化，马桥中心幼儿园的中层管理在悄悄地细化，马桥中心幼儿园老师的士气在悄悄地抬头，这些都有赖于教育局给我们搭建的学习平台，以及师傅和专家的带教与指导。

 

摄影、供稿：马桥中心幼儿园  诸伟芳
