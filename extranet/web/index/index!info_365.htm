落实课标要求 探究主题意义 践行学科育人 ——闵行区小学英语五年级主题教研活动在华师永德实小举行

发布时间 : 2023-11-17   作者（来源）: 殷素婷   阅读 : 537

3/8
←
→

2023年11月14日下午，闵行区150多名五年级英语教师相聚华东师范大学闵行永德实验小学，参加主题为“落实课标要求 探究主题意义 践行学科育人”的闵行区小学英语五年级主题教研活动。闵行区小学英语教研员殷素婷老师出席并主持此次活动。

活动前期，区学院小学英语教研员殷素婷老师多次莅临我校指导，为两位教师的教学设计、语境创设、活动推进等方面提出宝贵的改进建议。在教研员的悉心指导、教研团队的通力合作、执教教师的反思改进下，本次活动顺利开展。

首先，华师永德实小的英语团队立足课程标准，结合两堂课的教学实施，以访谈的形式对《英语（牛津上海版）5A Module4 The natural world Unit1 Water》进行教学设计阐述。姚琼霞老师和吴益飞老师通过对话展开思维碰撞，介绍了单元整体规划。接着，杨佳老师从“巧创情境，增进体验，激发探究意识”、“巧编多模态语篇，实践语言，感受主题意义”和“巧设活动，关注思维，形成文化认识”等三个方面进行了教学设计说明。

接着，青年教师谢邱燕老师和庞燕老师分别为大家呈现了本单元第三课时Making tea以及第四课时Water in my eyes的教学展示。

谢老师在第三课时中模拟真实的情景，创设了Kitty为家人泡茶的语境，引导学生扮演Kitty用真实茶具泡茶，激发学生的学习兴趣，同时也帮助学生建立英语思维，从而更好地理解和运用英语知识。在学习理解活动中，谢老师通过What do we need for making tea?和 How to make tea？的大问题引领，引导学生思考、观察、分析，了解泡茶所需物品，再进一步引导学生思考，推断泡茶步骤，从而提升学生的推断力和分析力。学生在泡茶的学习和实践体验中，学会用英语有序介绍泡茶步骤，并逐步构建运用到用水制作其他饮料中去，Water is useful!的育人目标在润物细无声中渗透在课堂里。

在第四课时中，庞老师创设了小朋友们在教室里探讨的语境。庞老师首先通过回顾前三个课时的内容，激活学生对水不同的认知。在学习理解环节，庞老师带领学生学习了解黄河的流向，让学生体会到Water is amazing！的情感。在实践体验环节，庞老师带领学生学习有关水的小实验，引导学生感受到Water is magical！的情感，加深对水的不同意义的理解。在构建运用环节，庞老师通过组织学生小组合作、阅读语篇、提炼信息，运用所学知识，制作Water in my eyes的海报并合作介绍，表达对水的意义的不同认知与看法。最后，庞老师通过多模态资源，向学生展现了水的多面性，引导学生感受到水的神奇和伟大，在享受水带来的美好时，引导学生珍惜水资源。

两位教师立足新课程标准，围绕素养导向、育人指向的理念，通过挖掘教学内容选择合适的教学策略，使用多模态语篇帮助学生更好地理解和掌握知识，实践语言，感受主题意义，培养学生的多元智能和综合素养。

互动研讨环节，殷老师和两位执教老师现场互动，交流分享了在磨课中的关键问题和改进点。在谢邱燕和庞燕两位老师的现场反思后，上戏附校的谢迪青老师、紫竹小学的陈春霞老师以及浦江二小的郑晓青老师分别从语境创设、教学资源、活动设计和实践体验等方面对两节课进行了互动点评，分享了自己的观课学习感悟和建议。

活动最后，殷老师对教研团队探究主题意义理解下的学科育人课堂实践给予了高度的肯定。随后围绕本次活动的教研主题，殷老师结合新课标，从“激活学生元认知，主题意义在萌芽”、“丰富学习体验，主题意义在生长”、“深化学习感悟，主题意义落地开花”这三方面对两节课进行了深入的剖析，为如何激发学生元认知、丰富学生阅读体验给出了建议，指明了方向。

本次教研活动给予了华师永德实小英语教研团队一次突破自我的机会，在仔细研读新课标的基础上，团队在单元规划、教学实践、教学说明等方面都做了不一样的尝试与探索。以课为本，碰撞智慧的火花；积极探索，构建不一样的精彩课堂。新课标的实施，促使我们与时俱进，更新教学理念。阅读新课标，转化新理念，探索新课堂，坚定新目标。我校教研团队将不断耕耘，智慧实践，行而不辍，与时俱进。
