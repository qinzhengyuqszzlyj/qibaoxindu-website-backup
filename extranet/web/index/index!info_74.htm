基于新课标研读，精准落实学科核心素养

发布时间 : 2023-10-02   作者（来源）: 戴赟   阅读 : 481

3/8
←
→

9月27日，闵行区高中生物学科教研活动在北京外国语大学附属上海闵行田园高级中学顺利举行。活动由区教研员戴赟老师筹划和安排，华东理工大学附属闵行科技高级中学的张甜老师担任主持，以中片教师线下，北外和南片教师线上的形式参会。

会议伊始，全体老师观看了由北外田园孟雯老师执教的《人体的神经调节》的单元复习公开课视频。该课先通过本单元的知识概念图引导学生回顾主干知识和核心概念，然后以辣椒素为大情境通过三则资料和问题链引导学生独立思考，在生生交流和师生交流中综合运用知识，习得解题方法，提升学科核心素养。接着，在老师们分组研讨后，老师们线上进行交流。

首先是执教教师说课，孟雯老师分别从空中课堂团队的备课磨课和基于线下的公开课两个方面进行阐述。首先介绍了备课过程中如何研读教材和课程标准，如何形成“知识建构•情境贯穿•综合运用”的教学思路和如何达成教学目标等，特别是平时的做题和命题经验为本节课在创设情境资料和设置题目时发挥了关键作用。在基于空课执教线下课时，孟老师基于学情在课前增加了知识的梳理环节，在课上补充了解题方法指导。孟老师感谢了帮助她磨课的区教研员、市空课专家团队、学校学科组团队，并表示本节课是集体智慧的结晶并分享了自己的一些感受。然后，教研组长冯剑锋老师对本校生物教研组团队、课程设置，实验室设施以及新入职的生物老师等进行了介绍。

接着，主持人带领大家针对“结合本节课，思考复习课中，如何更加高效有序地组织学生活动”进行教学研讨，各片教师代表发言。老师们肯定了本节课的优势：情境创设巧妙——“一境到底”完成整个单元的复习，问题链设计合理有梯度，题型多样有助于学生思维的训练；老师语言简练，和学生交流充分，善于引导、充分启发学生思考，课中带领学生一起研究课本，回归课本做的很好。此外，老师们也提出了一些需要改进的方面。上师闵分的顾苒老师建议教师在阅读题目获取信息时将主动权留给学生，七宝中学的彭石娟老师建议以大问题引发学生交流思考缓解学生的疲劳感，莘庄中学的吴红霞老师建议课程中加入视频等形式来调动学生气氛，闵行实验高中的顾荣老师建议增加行为性的调节（如吃了辣椒要喝水等），可以引导到条件反射上；讲到辣椒减肥，可以进一步延伸到这样减肥是否科学等贴近生活的例子以此来提高课程的广度和深度。华理科高的任志红老师提出了以真题带动知识点的回顾复习再引入习题的训练的复习模式。最后主持人张甜老师建议可以在学生表述环节引入智慧笔或者拍照上传希沃等技术辅助同步呈现，就可以逐字逐句分析，反馈效果会更好。

本次研讨会，以研讨促学习，以交流促提升，与会教师都收获满满，为推动本区高中生物学复习课有效性的提升具有重要意义。
