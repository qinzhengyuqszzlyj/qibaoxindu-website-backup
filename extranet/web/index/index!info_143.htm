与自然联结 与幼儿同行 ——闵行区中片幼儿园托班教研活动

发布时间 : 2023-10-18   作者（来源）: 鲁鸣   阅读 : 483

3/8
←
→

2023年10月18日，古美中心幼儿园开展了以“与自然联结 与幼儿同行”为主题的闵行区中片幼儿园托班教研活动。区教育学院学前教研室教研员鲁鸣老师、区中片幼儿园托班教师、古美路街道托班联合教研组、古美路街道学前联合体的教师们参与活动。

现场观摩·追随幼儿个性

托一班、托二班的教师们分别展示了契合“托班适应期”的室内环境和点名活动。温馨的班级环境、有趣的儿歌互动，使每位幼儿都能获得教师的爱抚、拥抱和语言互动。幼儿轻松、自在的与教师共同开启美好的一天。

托一班的何佳欣教师基于“托班适应期”幼儿的年龄特点和发展需求，以“幼儿发展优先”理念为指引，设计了托班户外圆圈活动《大自然的眼睛》。何教师借助园内自然环境，创设“鸭宝宝游农场”的游戏情境，让幼儿通过找一找、贴一贴、说一说，获得了多种感官体验，在自然、舒适的大自然中积极参与活动。

联合教研·提升互动质量

古美路街道托班联合教研组长沈丽洁教师组织开展了以“联结自然 提供个性化的互动支持”为主题的教研活动。联合教研组的教师们通过举实例、剖案例，对适应期托班个性需求进行了细致分析，加深了对“幼儿发展优先”的理解，并且达成了“因人而异 开启潜能”的共识。在随后的教研积分赛中，教师们结合自身教学经验，积极讨论、互动互研、经验共享，共同形成了“适应期托班师幼互动策略”。

片区教研·凝聚实践智慧

鲁鸣老师组织开展中片幼儿园托班教研。教师们围绕活动现场分为四组，以“个性化互动支持的策略”和“活动改进建议”为话题点展开研讨。来自颛桥镇幼儿园、东方剑桥幼儿园、晶华坊幼儿园、上师大闵行实验幼儿园的老师作为代表进行互动发言。“开放”“尊重”“追随”成为高频热词。教师们纷纷表示要不断修炼自身素养，提升自身互动水平。

最后，鲁老师针对本次活动给予点评和解析。她充分肯定了古美中心幼在“幼儿发展优先”理念下，联结自然的大胆尝试，对活动中师幼良好的互动关系提出赞赏。同时，她对全体托班教师提出期许，鼓励教师们要拥有儿童视角，做敏感的教师，为幼儿园里最柔软的群体创设良好的氛围，搭建发展的支架。

为“最柔软的群体”创设温馨有爱的环氛围，提供联结自然的教养环境，提升支持个性化需求的互动质量，古美幼人将不断前行，和孩子们一起遇见更多美好。
