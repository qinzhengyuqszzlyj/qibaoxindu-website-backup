聆听 观摩 对话 ——2018年区新任校（园）长带教培训何学锋导师组活动简讯（11月）

发布时间 : 2018-11-11   作者（来源）: 朱凤   阅读 : 1708

学校文化是办学的根基、立校的土壤。好的学校文化，能够使学校发展思路清晰、工作目标明确、共同愿景明晰、价值取向认同、师生团结和谐。作为新任校（园）长，在对学校办学基本情况全面了解的基础上，要提炼学校的文化，要实践文化立校的办学策略，不断促进学校的内涵发展。

11月8日，实验小学何学锋特级校长带领蒋红梅和李兴两位新任校长，来到上海戏剧学院闵行附属学校，参加小学校长基地第五期校长培训班的专题学习。

一、在聆听中感悟校长的教育情怀

上午，上戏闵行附校的杨慧琳校长以《“教育戏剧文化”方略下的学校转型发展》为题，作了专题汇报。汇报分为四个部分：“教育戏剧文化”破土的现实选择、“教育戏剧文化”基本的关键内核、“教育戏剧文化”方略下的“角色场”课程建构、“教育戏剧文化”深化的反思与改进。两个小时的汇报时间里，杨校长将办学的心路历程、管理的智慧策略聚焦在“文化”“课程”两个关键词，构建“教育戏剧文化”引领下的“角色场”课程实施，在艰苦奋斗与坚守超越中不断促进学校的转型发展，展现了一位成熟型校长的教育情怀和治校智慧。

二、在观摩中深化文化的整体认识

中午，何校长带领大家参观了学校的校园环境。随后，在何校长的主持下，校长们开展了互动交流。每位校长在平等民主的氛围中畅所欲言，围绕发展阶段、管理架构、理念确定、课程开发、文化呈现等办学的多个方面提问题、谈观点、论方法。校长们坦诚地研讨交流，促进了大家对学校文化的深入思考，展现出成熟型校长的表达力、思考力。

何学锋校长的总结发言高屋建瓴，使我们认识到“文化立校”的主要实施路径与策略：一是以办学规划为引领，以办学资源为保障，分阶段确定学校发展重点与办学策略；二是基于特色文化背景，打造特色课程的实践，促进学校内涵发展的实施策略；三是基于以人为本的理念，给予管理团队一定程度的驱动，增强教师团队归属感，给予学生独有的校园生活经历，共同营造学校的文化。

三、在对话中明晰发展的关键领域

在专题学习结束之后，何校长与两位学员进行了对话。何校长首先让我们分析学校的现状、学校所处的社区、校长自身的专业特长；其次让我们思考，在保持学校稳定的基础上，学校发展的关键领域和重点项目是什么，怎么来实施。通过何校长的一次次追问和点拨，我们逐渐明晰自己学校后续发展的关键领域，也确定了何校长带教的主要项目以及实施的路径。
