云聚课堂展风采 观评教研臻至善 ——九年级“作文升格暨名著导读”网上教学研讨活动纪实

发布时间 : 2023-05-17   作者（来源）: 鲍清   阅读 : 904

在区教研员鲍清老师的指导与组织下，九年级举行了“作文升格暨名著导读”语文教学网上研讨活动。全区初三语文教师网上观课，线上“云聚”，研学互鉴。

活动始于2023年5月4日，老师们登录云录播平台观课并研讨。

上海市华二附初教师曾圆执教《搭建思维训练支架，助力传统情境佳篇》作文课。曾老师紧循上海中考作文评价标准，从班级不同层次学生的习作样本中研究写作过程中思维困境的具体解决策略。学生从原版到2.0版甚至3.0版的修改过程中，学会从意（立意）、象（选材）、形（结构）、言（语言）四个方面逐步搭建有效的学习支架，将思维可视化、方法明晰化。课上，学生梳理“事件清单”，形成材料“超市”，依据选取理由与标准，提取典型事件，逐一完成梳理、整合、概括信息，对比的表格呈现选材的思维过程；选择“叙述方式”，修改“情节线”与“情感线”，确定了文章结构框架，改善了行文表达习惯；聚焦“触动情点”，展开多维度有效描写，实现思维的聚焦与发散，增强了行文表现力。通过三阶段的思维提升，辅以弹性评改支架，云梯式板书呈现特色化，其学习过程显性化、具体化、可操作化，让学生最终能言之有序、言之有趣、言之有质、言之有情，为学生中考记叙文写作提供实质性帮助。

上海市浦江三中教师段任婷执教《探索简·爱精神成长之旅》名著导读课。课堂中段老师通过搭建“简·爱”精神成长及形象评价支架，梳理其五处生活地点的独白与语言描写等清单，赏读简·爱丰富的内心世界；引导学生分析社会环境和周围人对简·爱性格的影响，探寻简·爱成长原因；指引学生通过合作探究和讨论，结合时代背景，理解简·爱精神意义。段老师运用视角转换、角度创新、思维碰撞等多种方式，将学生的思维引领到了自己的航线上，促进学生的多样表达。其搭建的名著导读支架在学生探寻理解与赏读运用之间架起了桥梁，起到了高效的示范引领作用。

语文阅读与写作中的兴趣支架是思维训练的驱动力，方法支架是思维训练的加速剂。两位老师在初三写作修改升格及名著阅读教学中都极好地为学生搭建实操性强的思维训练支架，形成了语文核心素养提升的弹跳板。相信学生们会将共性策略迁移到后续学习。

以观促教，以评促思。课后，全区初三年级的老师们各抒己见，纷纷发表了精彩的观课感言。

 

《搭建思维训练支架，助力传统情境佳篇》作文课观评留言摘录：

宋小芳

曾老师的作文课给了我很大的启发。授课语言亲切生动，导入部分以学生的思考和问题为话题，巧妙引入本节课的课堂教学目标，有效吸引学生的兴趣。教师指导的角度清晰，细节也把握仔细。学生在选材上就比一般同学高出一筹。教师在教授触动情点这个技巧时，明确方法是先确定情点，再表现情点。学生课上表现也十分优异，侃侃而谈，非一日之功，一定是老师平日多次有效的训练所达到的结果！

 
曹建明

曾老师的作文升格指导课，在初三后期指导学生作文方面值得借鉴。首先要提升的作文来自班级学生，身边的作文，给与大家的就是轻松自由亲近，就是家常。其二，四个维度，审题，立意、选材和语言，这是作文四个关键维度，任何一个维度的提升都意味着学生作文整体的升格。其三，支架的运用。作文指导必须可视化，即让学生明明白白地指导如何提升，支架就比较好地提供这方面的帮助。所以，从课堂实际来看，效果比较明显。

 
潘秀云
作文修改如何教？曾老师这节课给我们提供了范例。曾老师从事件清单入手，抓取典型事件，进行构思草拟，聚焦在“触动情点”，为学生搭建了一个明晰的思维框架，并结合几位同学的作文，从最初版开始引导学生进行修改升级，使用本班学生的例文，明确修改的重点。在曾老师的课后指导下，学生们不仅会改，改得好，还能将如此这样改的理由分条列出。那么即使换个文题，这些孩子也有甄别鉴赏的能力。整堂课对初三末阶段教师的写作教学有启发借鉴作用，值得学习。

 

徐映

曾老师运用量表，引导学生思考具体修改项目，能够让学生琢磨自己作文存在的具体问题；对典型事件、事件间的关联与组织，均有可操作性的指引，能发展学生理解的比较意识，值得在作文教学中特别是修改作文的教学中借鉴。

 
陈亮
曾老师的作文课信息量大，内容充实，指导性强，针对问题的分析解答细致到位，对学生很有指导和帮助作用。三位学生的作文各有千秋，能折射出不同的问题，较有代表性。通过这节课，相信学生能够在短时期内将自己的作文修改水平予以提高，并且能做到以情动人。选取的素材，也偏向比较少见的农村类传统文化类和亲情类，值得学习借鉴。

 

《探索简·爱精神成长之旅》名著导读课观评留言摘录：

潘秀云

段老师这节课围绕“成长”这一关键词，探索简爱的精神成长之旅。段老师带着学生梳理了简·爱人生的五个主要地点所经历的事情，用表格的形式梳理了女主人公在不同事件中获得的精神成长，让学生在课堂中对整本书的情节有了较为细致的把握，也对简·爱的成长有了初步的认识和感受。支架清晰，值得学习。

 

戴莉嫔

段老师围绕“成长”的主题，用问题链的形式铺陈，在阅读和朗读间自然地切换，引导学生梳理人物在不同情境中行为表现的差异性，感受作者的价值导向。让学生既能基于自身的阅读感受，又能阐述自己的看法和观点，进行阅读的扩展和延伸。

 

任筱伟

段老师沿着简爱的生活轨迹，抓住简·爱的语言及内心独白，并关注了身边人对简·爱精神成长的影响，带领学生们细致、深入地梳理了简·爱精神成长五阶段的表现。整节课，段老师不仅带领孩子们重温了这部作品的主要内容及其社会时代意义，而且激发了学生们的再读的兴趣和讨论的激情，思维含量极大，更为重要的是，这节课对成长中的孩子们来说，也是一次精神的盛宴。

 
单亚敏

对于整本书的阅读和复习，是我们老师在复习量过大的情形下比较棘手的问题。复习课期间，继续给学生厘清思路，删繁就简，段仁婷老师给了我们很好地师范。她主要抓住简爱生长环境和生活环境的变化，概括每个环境的特点，在每一个地方，简爱的精神方面的成长（人格的形成和爱情观的确立）。抓住了一条主线，可以分析主人公简爱精神成长的原因，以及精神成长的意义。

 
曹建明

段老师执教的《简·爱》导读课，以“成长”为探究主题，梳理主人公的精神成长，理解这个人物不断的角色突破。在这个主题之下，沿着主人公生命中的关键的四个所处环境，梳理在这个环境中主人公的所遇、所做、所见、所思，选择小说中的关键细节，进行品读，并前后关联，理出主人公的思想、情感和精神的变化，理解主人公精神世界的不断拓展和提升。段老师紧紧围绕小说，在小说的文本里抓住语言文字和相关细节，以此为基础，勾连相关内容。归纳主题式探究的主要策略和方法，既获得如何学习，又获得了精神成长。

 

为师者，惟匠心以致远，当且研且行，臻于至善。
