专家引领共研课标 理论提升实践赋能

发布时间 : 2023-11-12   作者（来源）: 高剑英   阅读 : 366

3/8
←
→

2023年11月8日下午，闵行区初中物理学科教研活动在颛桥文体中心兴都剧场顺利举行。出席活动的有七宝中学副校长刘树田、闵行区初中物理教研员高剑英及闵行区全体初中物理老师，本次活动由学科指导团邵苏萍老师主持。

新课标研读分享活动中，闵行中学附属实验中学的戴满媛老师从关注课标理念厘清变化、创设教学情境激发思维、联系生活实际学以致用三个方面着手，向大家展示了在物理课堂上创设问题情境、拓宽学生视野、启发思维宽度的一些尝试；师培附中的郭赟娴老师以实验教学为抓手，通过丰富的课堂实践案例，展示了她在实验教学中启发学生提出问题、引导学生收集证据并进行有效的交流、逐步归纳得出结论的一些做法。两位老师的精彩分享激励着我们要将新课标课程理念不断转化为自觉的教学行为，促使学生在物理课堂上变得更加灵动，思维更加活跃，有效培育学生的核心素养。

七宝中学刘树田老师为大家带来了高中力学“素描”的专题报告，他高屋建瓴地与老师们一起梳理了力学体系，将初中高中物理力学内容有效地进行衔接，帮助大家重拾本体性物理知识。刘老师用富有哲理的语言，发人深省的故事，生动鲜活的案例，深入浅出为大家介绍了从亚里士多德、伽利略、牛顿、爱因斯坦的力学发展历程；同时诠释了做人、做事、做学问的真谛，表达了他对同行物理人在努力践行“踏踏实实做人、勤勤恳恳做事、认认真真做学问”的期望。让老师们深刻体会到要想培育学生的核心素养，必须先要提升自身的物理科学素养。

最后，闵行区初中物理教研员高剑英老师对实验操作培训和青年教师实验考核进行总结与反馈，表彰了一些考核获得“优秀”和“良好”的青年教师，同时也展示了一些教师在实验操作中出现的典型错误，希望大家结合自己学校及个人的教学实际情况，内化为自己的教学行动。她期望老师们在激活课堂，启发思维，培育核心素养的道路上持之以恒地践行下去。
