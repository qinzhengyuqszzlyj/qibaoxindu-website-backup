柔活活动边界 赋能幼儿成长 —— 闵行区七宝幼儿园课程基地活动

发布时间 : 2023-06-16   作者（来源）: 鲁鸣   阅读 : 573

3/8
←
→

2023年6月6日，闵行区七宝幼儿园课程基地活动在古美中心幼儿园平吉园顺利开展。七宝幼儿园副园长汪湜莅临指导。基地共同体幼儿园的园长、保教主任及共同体老师共计50余人参与本次活动。活动由古美中心幼儿园朱修敏老师主持。


观摩教学活动

平吉园中三班的谢怡茹老师基于幼儿在菜园活动中的兴趣和表征，设计了集体教学活动《菜园趣事》。整个活动幼儿兴趣浓厚，积极参与小组合作，通过各类表征，较完整、连贯地讲述自己在菜园发生的趣事，在愉悦的氛围中提升了语言表达能力。


参观户外活动

户外活动中，一片片欢声笑语映入眼帘。孩子们忙得不亦乐乎，建构游戏、沙水游戏、角色游戏，还有小菜地丰富有趣的户外活动。孩子们自主选择，合作解决问题。教师鼓励孩子们表征记录，记录下游戏的点滴，记录下种种美好，记录下成长喜悦！


开展教研活动

中班教研组长徐英丽老师带领8位教师开展了以“以兴趣为导向 实现低高结构活动转换”为主题的教研活动。举例分享“户外活动中，通过观察幼儿的行为，如何进行低高结构活动转换”，讨论集体教学活动《菜园趣事》，教师们各抒己见，就“幼儿表征”和“教师表征”的运用，两方阵营展开激烈的讨论，提炼出有效的策略。


组织研讨互动

基地园教师们围绕“理解低高结构活动转换”展开研讨，提出了生活中各个环节都要基于幼儿兴趣，善于价值捕捉，柔活生活和教学，将“基于幼儿兴趣”的理念更好地践行和落地。

最后，汪湜老师对于活动中教师的专业素养、研讨合作方面给予了充分的肯定。同时建议教师要做到 “开心”“慧心”“好奇心”，充分了解幼儿，为幼儿提供自主选择的机会，让幼儿在最自然的状态下自由地表现自我。

七宝幼儿园课程基地将继续“基于儿童、源于生活”，从幼儿的兴趣出发，读懂幼儿的行为，让幼儿更自主自由地成长。
