齐心协力破难题 携手并肩共成长 ——2018年闵行区新任校（园）长带教培训唐颖萍导师组活动通讯（12月）

发布时间 : 2018-12-29   作者（来源）: 曹雅明   阅读 : 2533

在2018年最后一个月中，针对学校发展过程中的具体问题，马桥复旦万科实验中学李曼华校长与师傅唐颖萍校长多次沟通、交流，尤其针对学校公共卫生突发事件处理、师资队伍等问题，在听取师傅“解决问题，人是关键，制度是灵魂”等意见后，李曼华校长结合学校实际情况，进一步完善学校各项规章制度、管理体系、师资队伍发展模式等，致力以更为有效的模式推动学校发展。

一、全员齐开动，提升“战痘力”

从11月份起，马桥复旦万科实验中学出现突发水痘疫情，该病潜伏期长，传染性强，必须及时制定行之有效的隔离方案和防御措施。在师傅唐校长关心指导下，李曼华校长及时召开行政团队会议，组织以校长为组长的专门负责小组，制定水痘疫情控制方案，及时与马桥镇社区服务中心取得联系，签订《学校、幼托机构水痘发病控制措施建议书》，并邀请社区服务中心医生来校进行消毒、预防等指导，并安排任务到部门、定点到个人，校办负责隔离班级专有进出通道及值班人员安排；后勤管理中心负责隔离班级教室安排、午餐供应、教室每日消毒及向疾控中心报备病例；教学管理中心负责隔离班级的课堂教学和对得水痘学生的补课安排；学生发展中心负责向学生、家长宣传水痘的流行特征、传染途径、预防办法、每日晨检工作及安排上下学错峰时间；党支部和工会关注学校教职工水痘等疫情的知识普及和防御措施……突发水痘疫情期间，各位行政及老师各司其职，齐心协力，经过近一个多月的时间，学校现已无疫情案例出现，各项工作积极有序开展。

二、查问题定措施，补短板促提升

针对2017学年学校办学绩效评价情况，马桥复旦万科实验中学李曼华校长组织学校行政团队、教研组长、年级组长召开了2017学年办学绩效评价分析交流会。会上，李曼华校长对2017学年办学绩效分析报告进行了全面解读。李校长用详实的数据、图表分析了2017学年办学绩效指标任务完成情况，在肯定学校取得成绩的同时，也指出了存在问题，寄予新的一年中办学绩效考核的思考与改进。随后，各部门根据闵行区2017学年义务教育办学绩效指标，结合本部门工作实际，从绩效考核指标得失情况、下一步绩效考核工作目标任务及整改措施等方面进行了分析与交流。教研组长和年级组长也分别从各自条线和工作提出了一些想法和举措。此次会议，各部门、各条线立足岗位，拓展思路，提高效率，齐心协力，通力合作，为学校2018学年更好地开展工作共同出力。

三、聚焦新课堂，携手共成长

为进一步深化新课程改革，发挥骨干教师在课堂教学中的示范、引领作用，提升青年教师的专业素养，提高课堂教学有效性。12月，针对学校师资队伍分层培养，整体推进教师队伍建设，学校陆续开展了位骨干教师展示课、青年教师邀请课、见习教师汇报课、主题教育展示课等系列课堂教学展示活动。此次活动全体教师全员参加，活动开展期间，各教研组成员相互研讨，共同打磨、修改和提升，上虹中学结对师傅也纷纷给出宝贵的意见，此次活动的开展，不仅提升了老师们的专业素养，增强了学科教研组成员的凝聚力，更是推进校际之间的联系，相信老师们通过此次课堂教学活动，在自身专业成长的道路上会更进一步。

此外，学校还将课堂延伸到课外，从校内到校外，在马桥镇“悦读马桥”学区化项目引领下，在“和美与共 科文交融”的办学理念下，以马桥豆腐干人文探究室和校本课程开发为载体，为让学生真正了解家乡马桥的历史文化和风土人情，学习传承马桥非物质文化遗产“马桥豆腐干”的制作工艺，探索美食背后的科学和人文知识，学校设计开发了以“研”之有据，“豆”智昂扬为主题的马桥文化研学课程。马桥复旦万科实验中学研学负责教师带领学生们走进马桥、感知马桥，在理解中传承和创新马桥文化。

2018年，我们立足学校实际，脚踏实地，真抓实干，齐心协力破难题，携手并肩共成长。以“和美交响·艺韵校园”为主题的艺术节闭幕式暨元旦迎新活动为即将过去的一年画上圆满句号。在即将来临的2019年，有师傅唐校长的悉心指导与帮助，我相信学校将继续迎难而上，砥砺前行，凝心聚力，共同成长。


摄影、供稿：马桥复旦万科实验中学   李曼华
