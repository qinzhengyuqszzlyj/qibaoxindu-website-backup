走进名校 感受名家风采 ——闵行区实验小学校长培训基地（第六期） “特色学校考察与名校长办学经验研讨”（三）

发布时间 : 2023-05-08   作者（来源）: 曹雅明   阅读 : 2592

3/8
←
→

2023年4月21日，闵行区实验小学第六期校长培训班“特色学校考察与名校长办学经验研讨”第三场研学走出上海，来到杭州。本次考察的学校是杭州市未来科技城海创小学和杭州市钱学森学校，本次活动由基地导师闵行区实验小学何学锋校长带队，校长基地及名校长工作室12位成员一同前往参与研学。

杭州市未来科技城海创小学专场

海创小学位于杭州未来科技城核心地块，是一所高规格的公办小学，学校以“未来、科技、创新”为办学理念。学校章献明校长为全国“生态·好教育”联盟校校长、浙江省特级教师、正高级教师。

当天上午，研学团一行准时来到海创小学，基地导师何学锋校长首先代表研学团对章校长的接待表示真挚的感谢，并简要介绍了本次研学团的基本情况和研学目标。当天的研学活动分为两块，一是校园环境与学习空间建设的考察，二是校长办学思想分享。

章校长首先为大家简单介绍了学校的办学理念，随后带领研学团详细地参观了校园。从校门口中心水池的铁艺雕塑，到综合教学楼里丰富多彩的项目化学习成果展台，再到教学楼后广阔的户外空间，学校一系列以“海创”为品牌的学习活动空间设计给大家留下了深刻的印象。学员们也纷纷拿出手机记录着创意的校园环境设计，不时就各处场所和空间的设计理念和章校长展开及时互动，并对学习空间与课程建设、课堂教学的对接展开了深入的探讨。

参观结束，研学团来到会议室，听取章校长办学思想报告。报告着重从如何通过教师队伍的培养推动学校办学特色发展展开。一是加强新课标学习，严格执行国家义务教育课程方案的要求。加强教学备课上课管理，借助课堂数字化跟踪观察教师成长变化曲线。二是为教师充分搭建学习和展示平台。学校设立了“百所名校考察活动”，让老师尽可能多地走出去学习，尤其是让年轻教师形成新的教育理念，尽快成长。三是教师人人参与项目化学习资源开发。学校已经开发五十多个场地，并加大校园学习空间从校内延伸到校外，甚至在游学活动中延伸到国外，处处为学生创设真实情境中学习的机会。四是邀请各行各业顶尖专家来和学生授课，通过各类专业课程资源的引入，打开学生的视野与格局。

基地导师何校长对章校长的无私分享表示感谢，他认为海创小学在校园学习空间的打造，基于真实问题情境下的综合实践学习非常值得我们每一所学校学习，各位校长回去要能够进一步思考，如何发挥自己学校周边的资源配置优势进行学校发展规划。同时面对“双新”，无论是上海还是杭州地区，现在学校之间差别不大，关键要看谁对新课程理解的透，执行的深。


杭州市钱学森学校

杭州市钱学森学校是为了纪念伟大的科学家、思想家钱学森，弘扬科学家精神而建，建筑风格富有江南特色，融入南宋建筑风格，现为浙江省义务教育标准化学校。学校校长唐彩斌为浙江省特级教师，中学高级教师，浙教版新思维小学数学教材副主编，中国教育学会小学教育委员会常务理事。

当天下午，研学团一行来到杭州钱学森学校，受到唐彩斌校长的热烈欢迎，他先总体介绍了学校的校舍建造特点，以及学校创办的背景和发展近况。随后在学校孔慰副校长引领下，研学团一起深入参观了校园，学校古典建筑风貌、精致的园林工艺、萌动的宠物园、恢弘的大剧院、充满艺术气息的“美林教室”、史料丰厚的钱学森博物馆等等，处处让人感受到学校“文理兼修、大成智慧”的办学追求。

整个参观过程持续了一个多小时，可见学校规模之大，场馆建设之丰富，每到一处，校长们或驻足欣赏，或展开讨论，及时研讨可以借鉴的办学创意，过程中还不时进行体验。

参观结束，培训班成员来到会议室，聆听唐校长关于钱学森学校的办学实践经验分享。

唐校长着重从课程建设和实施、评价改革、以及学生身心健康发展等方面展开论述。他认为国家课程中10%的跨学科主题学习有助于学校课程的丰富度，让学生乐于学习、乐于合作。在特色教育方面，学校以科技拔尖人才培养奠基为突破口，以课内科学学科的课堂教学为基础，以科技社团为引领，通过高校“科学家进校园”项目，形成小学和中学不同阶段的科学教育系列。学校评价重点引入对近视率、脊椎侧弯率等身体健康指标的评估，关注生命教育，研究心理健康的预警机制，与科技集团家长联动，研发小软件，全方位协调关爱身心健康。与此同时，学校十分注重为学生创设优质教育资源和展示舞台，大量邀请各界名人和专业人士来学校开设讲座和授课，为学生申报大型活动展示和表演机会，并通过档案袋“为孩子的成长积累一百个瞬间”。

在报告结束后，何校长带领培训班的校长们就学校很多特色创新的做法与唐校长展开深入互动。比如，就如何在“双减”背景下通过加法打造学校特色课程，在跨学科学习中怎样深入推进创新素养培育，学生成长过程中成功体验和挫折教育的双向作用，以及小学和初中的差异和衔接等话题展开交流。


本次活动，学员们走出上海来到杭州，通过当地两所极具特色的学校的考察与学习，不仅打开了视野，看到了现代化校舍在建设过程中对学习空间和课程设置的系统打造，更是感受到了沿海发达地区在基础教育中办学思想的革新。相信各位学员通过本次研训也能更好立足本校基地和特色，创造条件在新课程理念下进一步优化办学思想，提升办学水平。
