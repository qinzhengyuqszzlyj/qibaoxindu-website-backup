闵行区“优秀教师培育工程”第六期研修班（小学音乐组）顺利开班

发布时间 : 2023-05-12   作者（来源）: 朱凤   阅读 : 781

2023年5月11日下午，闵行区“优秀教师培育工程”第六期研修班（小学音乐组）开班仪式在闵行区实验小学（春城校区）举行。闵行区教育学院副院长龚耀昌，上海市音乐特级教师、正高级教师曹晏平，闵行区实验小学特级校长、正高级校长何学锋，闵行区小学音乐教研员、正高级教师刘庆江等出席开班仪式。

闵行区教育学院教师发展中心主任刘太如主持活动。

闵行区教育学院校长发展中心主任俞琳解读方案，旨在打造一支师德高尚、专业精湛、具有理想信念的高素质专业化优秀教师队伍，助力闵行教育高质量体系建设。华师永德实小韦勇军、明强小学崔征、实验小学林青三位学员分别从研究现状、发展方向和未来需求三个方面交流了自己的情况以及对未来专业发展的期望，表示会明确自己的发展目标，积极主动争取导师在思想、业务等各方面的指导和帮助，有计划有步骤的推进学习计划，总结教学经验，提炼教学特色，形成教学成果。

曹晏平导师感谢闵行教育党工委的信任，表示会尽心尽力做好指导工作，满足学员个性化的学习需求。同时希望三位学员能建立自己更高的目标，主动钻研，扎实“三台”能力，把自己的特色、特长、特点凸显出来。何学锋校长希望学员们抓住机会，根据特级教师标准，明确培养期内的重点发展方向，强内功，争取在更高平台上亮相。

龚耀昌首先感谢导师一直以来对闵行教育的支持和帮助，并对学员提出了三点要求：一要提高自己的宏观思维能力、科学决策能力、教育创新能力等，二要在新课标的引领下，明确自己的专业发展方向，三要在新的起点上不断超越自己，争取成为闵行教育的领头羊。



撰稿 :  闵行区实验小学 林青
