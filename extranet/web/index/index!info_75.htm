沉谋研虑 探索先行 再启新程 ——闵行区义务教育学校劳动课程实施方案编制工作交流会纪实

发布时间 : 2023-07-26   作者（来源）: 刘至宁   阅读 : 2168

为了贯彻落实义务教育课程方案和劳动课程标准，推进本市劳动课程建设，结合上海市教师教育学院（上海市教育委员会教学研究室）6月中旬下发的《义务教育阶段学校劳动课程实施方案”征集的通知》要求，为保障闵行区义务教育学校切实完成本项工作，把握“一校一案”编制的规范和质量，在2023年7月26日上午，闵行区教育学院通过腾讯会议的形式，召开了学校劳动课程实施方案编制与实施工作交流会。会议由中小学劳动教研员刘至宁老师主持，副院长龚耀昌、中小学学段主任孙静贤、李群出席会议，来自全区各中小学的与会代表，包括校长、分管领导和劳动教师、学科中心组成员共计近260人参加了会议。

会议主要有三项内容，首先是学校作劳动课程实施方案的交流发言，先后有平南小学、蔷薇小学、浦江三小、浦汇小学4所小学学校、七宝三中、君莲学校、交大马桥实验学校3所初中学校和一所九年一贯制莘城学校作交流。主要是介绍了学校每周一课时的劳动课程规划和劳动周的规划情况，展现了8所学校在撰写劳动课程实施方案中的思考、课程规划与实施方法，在展现校本劳动课程特色的过程中，进一步梳理现有课程，并根据上海市义务教育劳动课程框架体系及要求，不断地开发课程，组建团队，为劳动课程的全方位育人做好准备。

其中，浦汇小学张蕊校长在发言中，介绍了《劳动实践手册》的作用发挥，体现家庭、学校、社会三位一体的劳动教育，详细规划了每周一课时的劳动课程和劳动周规划，并通过分年级实施、劳动实践记录和劳动积累，更好地关注学生的劳动素养培育和学生的健康成长。交大马桥实验学校胡晓翠老师介绍了学校劳动课程中公共项目和特色项目的具体课程菜单的设置情况及课时安排，特别是校内、校外资源的开发与利用，都贯穿在传统文化引领与现代文化结合的劳动课程整体规划上。莘城学校作为九年一贯制学校的代表，介绍了从一年级至九年级的课程菜单结构和劳动周规划，能考虑到学生的年龄特点和兴趣、能力、劳动习惯与品质、劳动观念和劳动精神的培养；在劳动周中设立专项劳动活动，供学生选择，就针对学生数量众多的大型学校而言，做到了分类、分流开展与实施。

会议的第二项内容是，教研员刘至宁老师根据各校提交的劳动课程方案撰写情况，为大家作点评并提出方案撰写与实施的指导建议。刘老师主要对每周一课时的课程规划、劳动周规划和特色项目设计三方面提出了指导建议，每周一课时的课程菜单主要明确课程内容，明确课时、关注分年级实施、对接各新课标中的十大“任务群”，同时要构建好学校劳动教师团队，组好师资保障；在劳动周规划中，要关注劳动主题、劳动任务、劳动时间与地点、校内外资源利用、劳动安全等规划要素；在特色劳动课程项目设计中，建议学校回归到闵行区近两年开展的单元设计和劳动实践活动设计，切实完善项目实施中的任务实施建议或要求，更加具体、形象、直观地表达课程。

最后，闵行区教育学院副院长龚耀昌作领导讲话，首先明确了近期义务教育各中小学校劳动课程实施方案的撰写与实施任务，强调劳动课程是实施劳动教育的重要途径，要在新课标、新形势下更加地引起重视，全程规划、全员规划、统筹学校的办学力量；另一方面，劳动课程方案的规划与实施，更能够体现立德树人根本任务的落实，体现德智体美劳“五育并举”及跨学科教学与深研，培养学生正确的劳动观念、专业的劳动能力、良好的劳动习惯和品质，并在劳动中形成和发展劳动精神，成为全面发展的社会主义建设者和接班人；最后，龚院要求劳动课程仍然要构建以实践为主的课程结构，加强学生生活和社会实际的联系，倡导丰富多样的实践方式，注重综合评价和安全保障，全面深入落实劳动课程的规划与实施。

本次会议，通过交流的形式，集思广益，沉谋研虑，为各校撰写劳动课程方案提供了有效的指导和帮助，与会的老师也进一步明确了目标，有方向、步骤地开始进一步修改、完善和优化方案。相信未来的劳动课程实施，势必探索先行，再启新程。

 

闵行区教育学院  刘至宁（报道）

2023年7月26日
