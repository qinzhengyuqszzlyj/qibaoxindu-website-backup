聚焦新课标 践行新理念 赋能新课堂 —— 2023学年区小学英语主题教研活动在田园外小举行

发布时间 : 2023-09-08   作者（来源）: 沈雯晴   阅读 : 1168

金秋九月，丹桂飘香。2023年9月5日下午，以“聚焦新课标 践行新理念 赋能新课堂”为主题的闵行区小学英语主题教研活动在上海市闵行区田园外语实验小学顺利举行。本次活动由闵行区小学英语教研员沈雯晴老师主持，特邀教育部全国小学英语教学基地主持人、上海市教委教研室英语特级教师、上海市英语教师专业委员会会长朱浦老师作专题讲座《2022英语新课标在小学英语教学中的设计与实施》。全区约300名英语教师于线下聆听讲座，其余 900多名英语教师通过直播同步参与了活动。

活动伊始，沈雯晴老师回顾了上学期闵小英的学科教研工作，对新学期教研工作提出了展望，并期待闵小英教师们从教研制度走向教研文化，从被动发展走向主动发展，从跟随式成长走向超前式成长。

讲座伊始，朱老师回顾了新课标的核心理念，再次强调了素养导向，育人指向，厘清了课程目标、课程内容和学业质量三者的整体关系，并提出现阶段实施基于主题育人意义探究的单元整体教学的必要性。

接着，朱老师从学习活动观、主题意义、语篇研读、思维品质、文化意识、学习策略这六方面带领老师们精读了英语课程标准。朱老师从学习活动观出发，逐条解读了六条活动观的主要内容，并指出通过落实学习活动观，将教学从知识导向转为素养导向，真正实现单元的育人价值，从而达成培育学生的核心素养的总目标。之后，朱老师以黄浦区复兴东路第三小学朱慧老师执教的5A Module4 Unit1 Water为例，呈现了依据单元主题意义制定学习目标学习任务的实施策略。朱老师又通过闵行区田园外语实验小学陈楠老师的课例4A Module2 Unit2 Jobs，深入浅出地展现了如何依据课标的要求确定语篇类型，如何解读主题意义和语篇交际目的，如何明确语篇内容结构和语言特点以及如何确定单元教学语篇类型和内容。接下来，朱老师以上海市七色花小学朱伊苓老师的4B Module4 Unit2 Festivals in China为例分析如何通过比较与判断、调适与沟通、感悟与内化来培育学生的文化意识。最后，朱老师指出学习策略是学习能力形成的核心组成部分，并以奉贤区奉城第二小学马丽莎老师的5A Module2 Unit3 Moving home与宝山区和衷小学范雯燕老师的5A Module3 Unit2 Buying new clothes为例，强调了板书为学生提供学习策略的功能。

朱老师的讲座妙趣横生又引人入胜，让老师们意犹未尽。他用最浅显易懂的话语向我们传递了最精辟独到的理念，为新课标在教学实践的落地指明了方向。活动的尾声，教研员沈雯晴老师对本场讲座进行了总结。她期待全体闵小英在新课标的引领下，在专家的指导下，用课程标准指导教学设计，将课程标准融入教学行为，积极推动育人方式的转型，在教育教学的道路上汲取成长的力量，让英语教学赋能，促进学生核心素养的持续发展！


撰稿：闵行区田园外语实验小学 张佳瑶 唐臻莹

审稿：闵行区教育学院 沈雯晴
