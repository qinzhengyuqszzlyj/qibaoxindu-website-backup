聚焦短板 问诊解疑 共享发展——2018年闵行区新任园长带教郁亚妹导师组学习简报(12月）

发布时间 : 2018-12-24   作者（来源）: 朱凤   阅读 : 3538

近期，在导师郁亚妹园长耐心、细致、有针对性地指导下，通过专家讲座、案例分析、实地观摩等多种类型的带教模式，使朱虹霞、杨静、王爱香3位学员在一次次的交流学习，一次次的浸润式培训中，不断梳理与思考规范管理的要素，逐步积累管理经验。

◆聚焦短板  了解“财和物的那些事儿”

“基于八项规定精神指引下的幼儿园规范管理研讨会”在莘庄幼儿园进行。区教育局计财科科长郭廷新、区教育基建装备资产管理中心主任徐伯英以及区域内近10名1-2年的职初期园长、幼儿园中层干部共同参加了本次研讨会。徐主任就校舍维修保修、设施设备等资产管理、政府采购规范操作、到家长最为关心的装修环保问题都给予了中肯的措施和建议。徐主任强调要守住底线，在学校的发展中要牢牢树立政策底线不可越的意识，勤政为校。郭科长详细解析了“八项规定”下幼儿园的财务工作如何规范开展，从年初预算到预算执行、从各种经费专款专用到资产管理等方方面面都进行了细致地解说，郭科长强调要幼儿园要健全资产管理网络，建立较完整的资产管理制度。

本次研讨会重在通过规章解读、案例分析、互动研讨等方式让每个参与研讨会的职初期园长对幼儿园规范化管理有一个全面的学习和理解。职初期园长大多优于专业，对教学规范的熟悉程度要高于后勤管理，因此安排本次“基于八项规定精神指引下的幼儿园规范管理”研讨会真正体现了学有所求、教有所需、知有所缺。依法管理是幼儿园健康发展的基础与保障，相信职初期园长在深入学习贯彻习总书记的系列重要讲话的过程中，把握中央“八项规定”精神，不断提高自律意识、正能量意识、责任意识，有效促进管理能力的提升。

◆问诊解疑  分析“管理中的为难事儿”

12月3日，3位学员参观了莘庄幼儿园的各类环境，全面细致地了解课程环境创设的思路和内涵，学员们对莘幼温馨雅致的园舍环境大为赞赏，并肯定教师们在环境创设中所秉持的“尊重幼儿”的理念。园所环境的创设，力求将环境创设与教育理念相结合，与学校文化相融合，可以不大，但要温馨；可以不奢华，但要实用；可以不繁复，但每一个细节都为孩子们考虑，从而让幼儿园成为一个培育孩子身心和谐发展的“乐园”，走进幼儿园，能感受到温馨舒适、快乐自由、安全自然。 

之后，开展了“与导师对话”问诊解疑，以问题为导向，学员们结合自身的特点与需要和导师互动。针对新园长接手老园所，如何制定三年规划，遇到某个为难的事情该如何处理等等，导师给予了非常中肯和建设性意见，同时郁园长讲述了自己的亲身经历，引导大家要尽可能开展最大范围的讨论、才能更真实地了解教师们的想法、努力继承与发扬前任园长的办园传统与建设成果。结合管理实例的问诊解疑，让一些难以捉摸的管理规律变得浅显易懂，发人深省，提高新园长处事分析的能力，积累管理经验，帮助学员们在园长岗位上更快地成长与成熟。

◆共享发展  分享“姐妹园的开心事儿”

12月10日，3位学员在导师郁园长的带领下，参加了康城幼儿园第二届“萌宝探究、奇妙科学”科技节闭幕式活动，强烈感受到了课程实施的有效性：幼儿投入的探索、教师专业的智慧。通过参加这样的活动，学员们进一步提高了课程管理的认识：管理者要鼓励教师积极探索有效、有益幼儿发展的互动策略，形成教师的专业自觉、支持教师的“需求学习”。 

作为新任幼儿园管理者，需要学习和修炼的地方还有很多，要在实际管理工作中不断学习、反思，多问、多看，尽快提高解决、处理各项园所事务的能力，成为一名合格的新手园长。
