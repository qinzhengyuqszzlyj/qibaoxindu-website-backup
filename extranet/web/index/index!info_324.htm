闵行区第五期中学校长培训基地举行结业仪式暨学员成果汇报活动

发布时间 : 2019-01-18   作者（来源）: 卞凤玲   阅读 : 2768

1月16日下午，在这个真正意义上2019年的第一个有阳光的日子，闵行区第五期中学校长培训基地结业仪式暨学员成果汇报会在七宝中学举行。闵行区教育局党工委书记朱雪平、七宝中学教育集团顾问仇忠海、上海市三女中校长徐永初、区教育局组织科科长王治华、区教育学院校长发展中心主任俞琳等领导和专家参与了本次活动，会议由培训班导师朱越主持。

培训回顾

第五期中学校长培训基地开班至今逾两年，基地依托七宝中学及七宝中学教育集团的优质资源进行全开放、浸润式的实训，旨在通过近距离感知品牌学校（企业）、浸润式体验名校文化以及深度剖析名校长的办学思想和实践等方式，开拓校长眼界，提升办学品质。在结业仪式上，六位基地学员集中呈现了各自办学成果，既有在课程教学领域的实证研究，也有着眼文化和环境育人的内涵发展，还有深挖校本特色的品牌打造，学员都将自己在基地培训期间的所思所得进行了分享，让与会的专家导师们感受到他们两年时间的提升与发展。

学员特质

第五期学员是一群宁静致远、淡泊明志、长期辛勤耕耘于基础教育的优秀校长团队。他们都很智慧，有韧性，充满拼劲；有魄力，敢变革，勇挑重担。

张忆校长，两次临危受命，勇挑学校发展重担，课程领导力改革项目极大地提升了学校办学品质和社会影响力，实现两所学校的更高层次的转型发展。

屠红伟校长，新基础教育理论的研究者和实践者，倾力为平民学子的生命成长和品格磨练创造有温度、有深度、有宽度、有高度的育人环境。

郑菊兰校长，认真务实、坚韧执着于一所新开办学校的崛起，引领区域同盟学校探索实践戏剧教育的育人模式。

梅宝红校长，豁达开朗、幽默智慧，敢于打破原有学校文化的壁垒，致力于打造青少年法治教育进校园、进课堂的课程教育实践。  

沈晓红校长，勤奋质朴，低调内敛，为两所学校的特色发展、内涵发展和优质发展不断传承与创新。

汤林校长，主动承担两校合并任务，全力构建健康和谐、主动奋进的学校文化，积极开展“自信”教育实践。

专家点评

针对学员的汇报，专家给予了精准且犀利的点评，同时更给出了未来努力方向。徐永初校长希望学员们立足校本，深耕文化，将学校特色打造与培育学生的必备品格、关键能力和价值取向结合起来；仇忠海校长则要求学员更加关注教师队伍的建设，而这不仅需要校长专业引领，更需要在人格魅力上的感召，必须要让教师有归属感，仇校长还希望学员把握教育改革的大好时机，下苦功努力让学校从特色走向品牌，也让校长更具有专业话语权。

领导寄语

活动最后，朱雪平书记用四个关键词，对学员提出了四点要求。一是“机会”。要求六位学员抓住当下初中教育教学改革的契机，进一步开放心态，开拓眼界，提升境界。二是“问题”。要找准学校问题，把握机遇，顺势而上，为学校发展争取更多的资源；要求校长更投入办学，以教育家的情怀和企业家的精神办好学校，有充分的自信和积极性；三是“行动”。指出：行动和思考同样重要，希望校长们能将许多理念更快、更有效地落地，变成可以感知的学校发展过程；四是“规范”。强调规范办学的重要性，强化学校党建工作，要守得住底线，耐得住寂寞，为学校持续发展注入动力。

就像导师朱越所说，本期基地培训从形式上已经结束了，但是整个团队更进一步的资源共享和合作共建刚刚开始，这支团队一定能坚守教育本真，脚踏实地实现教育理想，成为当下初中基础教育的研究者、引领者，成为国家基础教育的脊梁。
