持续学习 创新发展——2018年北京市昌平区中小学干部领导力提升闵行研修开班仪式

发布时间 : 2018-12-05   作者（来源）: 朱凤   阅读 : 1838

2018年12月4日，北京市昌平区中小学干部领导力提升闵行研修开班仪式在闵行区教育学院举行。区教育学院龚耀昌副院长、发展中心兰平副主任和相关老师、跟岗学校的代表，北京市昌平区进修学校尼腾副校长和学员共40多人参加。校长发展中心俞琳主任主持开班仪式。

龚耀昌副院长代表区教育学院对昌平区教育系统领导和老师的到来表示热烈的欢迎。在详细介绍了区域课程与教学的工作重点以及取得的成效后，他指出，在改革开放的洪流中，京沪两地教育都得到了长足的发展，各有自己的优势，可以相互学习和借鉴，在比较中学习，在学习中突破，在突破中创新。

尼腾副校长感谢区教育局和教育学院精心准备了本次培训，对闵行区的信息化和国际化大为赞赏。他希望学员们珍惜本次机会，思考闵行的教育教学经验对昌平教育的启示。

兰平副主任详细解读了培训方案。本次培训为期两周，分为集中培训和跟岗学习，通过理论学习和实践历练，提升学员的领导力。培训中将有5位高校教授和5位本区优秀校长和教师为学员做专题报告，围绕文化、管理、课程、评价等方面展开；跟岗学习阶段学员将去实验小学等区内6所优质的学校实地学习，通过参加行政会议、与中层干部交流、进入教室听课等跟岗学习形式，考察学校的日常管理和教学，重点在学校文化建设、课堂教学改革、干部和教师队伍等方面。

通过互相学习与交流，使两地的教师开阔眼界、相互促进、共同提升，进一步明确社会转型背景下所担负的使命与责任，实现有文化、有境界、有担当的教育情怀。
