立足“新课标”，以校本化劳动课程促劳动素养提升 ——闵行区小学劳技教研坊活动在实验小学（畹町校区）开展

发布时间 : 2023-05-25   作者（来源）: 刘至宁   阅读 : 882

春末夏初的五月，落英缤纷。2023年5月23日下午，主题为“‘新课标’背景下劳动素养的培养”闵行区小学劳技教研活动在实验小学畹町校区举行，活动由实验小学盛黎青老师主持，区劳技教研员刘至宁老师及其他兄弟学校劳技老师共同参与此次活动。


活动伊始，实验小学集团校——曹行小学的吴媚君老师呈现一堂精彩的送教课《玲珑心意刨花钥匙扣》。吴老师立足“新课标”，聚焦劳动素养，先带领学生认识工具——刨子，然后指导学生如何使用工具，最后又引导学生利用刨花制作美丽的文创作品：刨花钥匙扣。课堂中学生动静结合，有思考、有实践、有创新。

之后，吴老师向大家分享了微讲座：《刨花传神韵 木艺铸匠心——基于“刨花木艺”校本课程的思考与实践》，她娓娓道来，介绍了其学校的“刨花木艺坊”，让学生通过学习与体验，在活动中树观念，在木香里品匠心。

接着，实验小学张杰老师向大家介绍了畹町校区的特色课程《豌豆农庄》，详细介绍了课程的背景、架构及实施过程，并带领老师们参观了课程的实施基地。老师们来到屋顶花园，欣赏到了《创意园艺》课程种植的薄荷、紫藤花、玫瑰花等，看到了《中草药种植》课程种植的益母草、艾草等，还看到了《蔬菜种植》课程种植的茴香、向日葵、马兰头等，屋顶上郁郁葱葱，群蜂飞舞、香气宜人，参观的老师们赞叹不已。参观完屋顶花园之后，老师们品尝到了实小学生自己做的薄荷茶、玫瑰花茶，切实感受到实小的劳动课程在扎实推进。

在劳技研讨环节中，来自虹桥小学的张敏老师针对劳技课中拓展部分谈了自己的思路，主要分为：问题汇总、梳理选择、正反跑思考、头脑风暴、实施和验证五个板块，并以《电动小车》单元为例，进行了具体的讲解。金汇实验学校的王隽老师分享了《红绿灯模型》作品，他以材料分析入手，补充与其适应的电阻（50~100欧姆），来保证发光二极管的正常使用，配合断续性蜂鸣器，让作品在视觉和听觉多维度提升效果展示。研讨中教师们一起动手体验，设计电路图、组合元器件，分享自己经验与成果。

最后，教研员刘老师进行总结发言，她希望劳技教师在工作中勤思考、多实践、善总结、乐分享，既可以围绕自定目标，进行集体自研的方式开展活动，也可以利用其他资源对老师进行培训，拓展视野或技能，利于大家共同进步。另外，刘老师还鼓励老师在劳动课程新课标的指导下，整合校内外资源，开发校本化劳动课程，为学生劳动素养的培育积极助力。

 

实验小学  张杰（报道）

2023年5月25日
