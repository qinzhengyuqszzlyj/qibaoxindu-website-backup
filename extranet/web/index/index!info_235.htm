聚焦“有向开放”的同课异构迸发别样精彩 —— “区小学美术教研活动”在实小畹町校区举行

发布时间 : 2023-05-26   作者（来源）: 秦涛   阅读 : 570

  • 实小欧阳丹军老师执教二年级《巧妙的编织》

  • 实小程果老师执教二年级《巧妙的编织》

  • 评课研讨，各校青年教师积极交流收获与思考

2023年5月25日下午，主题为“立足单元视野，优化课堂活动”的区小学美术教研活动在实验小学畹町校区举行。活动由区小学美术教研员秦涛老师主持，全区150多位小学美术教师参与了此次活动，实验小学畹町校区范向华副校长也亲临现场关心指导。

本次活动承担研讨课任务的是本学期实小“蒙正杯”成熟型教师教学评优活动中呈现亮点的“老”教师——欧阳丹军老师，以及首次在区级公开课的平台亮相展示的年轻教师——程果老师，均来自实小畹町校区。她们基于本次蒙正评优活动中四位参赛教师对“二年级第二学期第七单元‘感受民间艺术’单元中《巧妙的编织》一课”这一相同内容的不同设计与实施呈现丰富多样的差异资源的前移基础，以及后续尽可能保留智慧、优秀的个性设计，卷入备课组、教研组、学科组团队进行反思重建研磨后，最终呈现了聚焦“有向开放”迸发别样精彩的《巧妙的编织》一课的同课异构教学。

两位老师个性鲜明，欧阳丹军老师彰显了老教师的沉稳朴实、程果老师则颇具青年教师的活泼灵动。通过她们同课异构的教学，《巧妙的编织》一课呈现了“相同素材不同设计促成落实素养的欣赏、家常个性两种情境促成乐于表现的内驱、不同维度有向开放促成整体深入的思维、相同要求贯穿情境促成殊途同归的评价”的精彩。两节课都把民间编织工艺“巧妙”贯穿始终，学生对“编织”的学习，不仅只限于技能，更通融了对编织工艺独特魅力的认同、对创意编织的思维、对规律美的感知，从而达成目标、落实核心素养。

研讨课后，实小二年级备课组成员莘松校区的张菲菲老师作为代表，对该单元研究的由来和背景以、单元设计思路、《巧妙的编织》一课的两套设计方案，进行清晰且详细的整体介绍和说明。

随后的互动评课环节，来自上戏附校的谈易平老师、二实小的刘秦老师、实小景城校区的吴晓凤老师、紫竹小学的金诗雨老师分别结合自身的实践思考、学习收获等，从材料的运用、资源的捕捉、活动的开放、思维的培养等角度进行互动评课，肯定了亮点、也提出了建议，也得到了区内老师们的共鸣！

最后，区教研员秦老师分析了“感受民间工艺”学习内容的教学要点，并对本次教研的课堂呈现与团队研究进行了细致而深入的点评，对实小美术团队日常始终如一坚持研究单元教学、践行单元教学，形成扎实、深入的单元教学撰写和设计能力给予了高度评价。并鼓励和建议各校美术教研组长或备课组长行动起来，学习新课标，树立素养导向意识，带领老师们加强单元教学的日常研究，加强促进学生主动学、积极思维、素养形成的学习活动设计与实施的实践。
