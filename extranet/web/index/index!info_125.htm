践行初心 踔厉奋发 办人民满意的教育——闵行区校（园）长、书记学习宣传贯彻党的二十大精神

发布时间 : 2022-11-12   作者（来源）: 朱凤   阅读 : 891

3/8
←
→

11月11-12日，闵行区校（园）长、书记学习宣传贯彻党的二十大精神开班启动仪式在闵行区教育学院隆重举行。闵行区教育党工委书记恽敏霞做动员部署，教育局局长何美龙传达市、区关于党的二十大精神学习宣传贯彻要点。

动员部署：坚定信念  勇毅前行

何美龙局长传达了市、区关于党的二十大精神学习宣传贯彻要点：坚持以习近平新时代中国特色社会主义思想为指导，进一步增强忠诚拥护“两个确立”的思想自觉、政治自觉和行动自觉；把学习宣传贯彻党的二十大精神作为当前和今后一个时期的首要政治任务抓紧抓好抓到位，迅速兴起学习宣传贯彻党的二十大精神热潮；深入领会“五个牢牢把握”；认真对照党的二十大明确的任务要求，结合实际进一步细化、实化、具体化，切实把党的二十大精神贯穿到工作全过程、体现在各方面。团结奋斗，用苦干实干奋力书写中国式现代化闵行实践的时代篇章。

恽敏霞书记做动员部署：一要深入学习宣传贯彻党的二十大精神，汲取推进教育改革的精神力量，全面准确学习领会党的二十大精神，提高政治站位，加强系统谋划，推动闵行教育高质量发展。二要紧密联系教育系统实际，扎实推进党的二十大精神学习宣传贯彻各项任务，坚持示范带头、突出日常经常、聚焦校园主阵地，以高度的政治责任感和使命感，认真组织、周密安排。三要坚持围绕中心、服务大局，把党的二十大精神贯彻落实到教育改革发展各方面、全过程，落实中小学校党组织领导的校长负责制，切实抓好干部队伍和教育人才队伍建设，围绕教育强区建设目标任务，深入推进各项教育改革。

专题报告：深刻领悟  深入理解

闵行区教育学院党总支书记兼院长朱靖主持专题学习活动。

浦东干部学院教授董德兵 、吴涛，市委党校副教授卜新兵分别作了《夺取中国特色社会主义新胜利的政治宣言和行动纲领》、《以中国式现代化全面推进中华民族伟大复兴》、《学习党的章程 牢记使命担当》专题报告。聚焦深刻认识党的二十大的重要意义，准确把握党的二十大的主要内容，贯彻落实党的二十大的实践要求三个方面进行了深刻阐述。党的二十大报告为新时代发展中国特色社会主义，以中国式现代化全面推进中华民族伟大复兴一切重大理论和实践问题给出了答案、指明了方向、提供了遵循。学习领会党的二十大精神，必须坚持全面准确，深入理解内涵，准确把握外延，要原原本本、逐字逐句学习党的二十大报告和党章。深刻领悟“两个确立”的决定性意义，增强“四个意识”、坚定“四个自信”、做到“两个维护”，提高政治判断力、政治领悟力、政治执行力，深刻领会习近平新时代中国特色社会主义思想的世界观和方法论，深刻领会以中国式现代化全面推进中华民族伟大复兴的使命任务。

一天半的动员部署、专题学习，意味着教育系统学习宣传贯彻党的二十大精神全面启动。校（园）长、书记纷纷表示，将以党的二十大精神为强大动力，全身全心全情投入教育事业，立足当下，谋划未来，着眼高质量发展，坚持为党育人、为国育才，办人民满意的教育，让闵行的每一个孩子健康快乐的成长。
