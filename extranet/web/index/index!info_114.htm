聚焦育人方式变革 促进优质内涵发展 ——闵行区第六期小学校长基地（田园外小）研修活动在鹤北小学开展

发布时间 : 2022-11-25   作者（来源）: 曹雅明   阅读 : 878

3/8
←
→

和煦初冬日，研修进行时。2022年11月23日下午，闵行区第六期小学校长基地（田园外小）研修活动在鹤北小学开展。基地邀请了嘉定区教育学院副院长、正高级教师花洁莅临指导。活动由上海市特级校长、闵行区田园外国语小学校长兼书记、基地主持人赵瑛群主持。

参观校园，感受美好环境

首先，在鹤北小学蒋红梅校长的带领下，花院长、赵校长以及研修班学员们漫步校园，查看“智慧学习舱”、校园主题版面、“小不点”农场等校园环境，聆听“小不点”讲解员介绍秋季综合活动成果，走进学科专用教室、藏书室等了解设施设备。整洁小巧的校园、精致灵动的版面彰显着鹤北小学“精致和谐、阳光育人”的核心文化。

汇报总结，体悟内涵品质

下午1:30，在赵校长的主持下，开始汇报总结。赵校长代表基地热烈欢迎花院长莅临指导，也表达了希望在专家引领下进一步提升学员办学水平、提炼学员办学思想的愿望。

随后，基地学员蒋红梅校长向与会嘉宾汇报了担任校长岗位以来的实践与体会。蒋校长简单介绍了鹤北小学的基本情况，然后分“缘起、实践、展望”三个部分，介绍了基于“美好教育”理念下，聚焦育人方式变革、促进优质内涵发展的办学实践之路。

蒋校长结合先后担任鑫都小学、鹤北小学校长的办学经历，秉持“在可为处有所作为”的工作思路，融入学校把脉诊断，身体力行推进变革，分“聚焦核心文化，塑造美好育人理念；聚焦课程方案，塑造美好育人课程；聚焦课堂教学，塑造美好育人课堂；聚焦校园环境，塑造美好育人环境”四个部分，重点介绍了聚焦育人方式变革的四个关键维度的行动路径。并结合前天鹤北小学接受的五年一次的办学水平综合督导的初步评估意见，展望了“教育，让人的生命更美好”办学愿景。

互动交流，分享所思所感

活动的第三个环节是互动交流，学员们分享所思所感。

七宝明强二小的郭芳校长认为，蒋校长来到鹤北小学的时间不长，却在方方面面提升了细节，关键是思想的科学定位。有了正确的思想引领才会有发展的动力源泉，成就老师，成就学生。汇报的四个方面有传承，更有提升。从物到人的全面思考，尤其是育人课程和育人环境的变革引发了思考。学校发展进入快车道，期待后续打造品牌特色。

浦江二小的赵美华书记认为，蒋校长通过回顾鑫都办学的经历和介绍鹤北办学的时光，详细地阐述了不同阶段基于现状的思考。“可为处有所作为”这句话令人感动，对办学理念的更新提升体现了教育的特质，是一种过程，有自己对教育的深度理解。

列席基地活动的安师附小马吉副校长认为，鹤北小学小巧而精致、齐全，尤其是充分运用好了每一块墙的墙面布置，让每一块墙都发挥了教育的作用。特别是少先队工作的展示墙，面面俱到，责任工作落到实处，体现了精细化管理水平。

导师点评，传授办学经验

赵瑛群校长对蒋红梅校长的汇报进行了点评。

赵校长认为，鹤北小学是家门口的好学校，蒋校长在不长的时间内，把新一轮的办学成果进行提炼和特色创生，非常不容易。蒋校长在传承和创新中有效实现了育人方式的变革，重点构建了德育特色，并基于“新基础教育”理念和新课程标准打造“阳光课堂”，是综合环境的营造。鹤小呈现出新的气象面貌。蒋校长对“美好教育”的实践，将会呈现美好的课堂、美好的文化、美好的意境和精神状态。

赵校长也期望蒋校长通过学校管理、课程建设、校园环境营造等，为办学理念的落实建立一个统一清晰的标志体系，从课堂到教师到学生，最终是要实现校长、师生团队和家长的美好愿景。

专家把脉，引领校长发展

最后，花洁副院长进行把脉诊断，为学员们引领赋能。

花院长首先以黄金圆环理论的思维方式，从why 到how到 what的思维模型肯定了蒋校长的汇报总结。花院长认为，优秀的人会带动整个学校的良性发展。学方式方法一阵子，学思维方式一辈子。一个校长应该以怎样的思维模型来做一件事，这很重要。黄金圆环理论是领导者的思维方式，作为校长，要学习并运用领导者的思维方式。

随后，花院长以蒋校长的汇报为案例，为学员们作了精彩的指导。

以学校育人目标在原有的基础上放大拔高了，又有具体的十大核心表现将目标落地为例，阐释了“经营促发展”的理念；以学校中层管理设置为例，阐释了KPR指标管理与OKR使命驱动；结合办学特色建设，阐释了行动路径的关键维度和关键要素，把一个单一要素最大化，以终为始；结合教师队伍建设阐释了人生商业模式，能力×效率×杠杆，要抓好青年教师的关键期，要助推高端教师的发展。

花院长作为一名在教育领域奋斗了36年的前辈，用激昂的语言，用丰富的阅历，用清晰的逻辑，向学员展示着作为一名即将退休的教育专家的人生感言：一辈子在学习，一辈子在挑战，一辈子充满热情。


活动最后，赵瑛群校长对花院长的精心指导和精彩报告表示感谢，同时也要求每位学员进一步消化和内化专家的思想精髓，思考与行动并行，努力践行办人民满意的教育。
