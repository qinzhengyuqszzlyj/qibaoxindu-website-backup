适度支持 提升初中融合课堂教学实效——鹤北初中承办区级初中学段随班就读教研活动

发布时间 : 2023-05-26   作者（来源）: 谭宏   阅读 : 544

3/8
←
→

2023年5月23日，闵行区“适度支持 提升初中融合课堂教学实效——初中学段随班就读教研活动”暨“交大—江川”学区石筱菁特级校长工作室活动在鹤北初级中学举行。闵行区特教教研员谭宏、闵行区特教指导中心主任、启智学校石筱菁特级校长、闵行区特教指导中心聂佳妮老师以及全区初中及九年一贯制学校随班就读负责老师、“交大—江川”学区石筱菁特级校长工作室全体学员参与了本次活动。

鹤北中学心理教师俞洁老师首先进行了说课，俞老师从教材分析、学情分析说起，阐述了为何选择《注意你的“注意”》这个主题，以及如何根据班情和学生情况进行分层教学设计。随后，大家一起观摩了俞老师的现场教学——《注意你的“注意”》，俞老师通过丰富的课堂活动帮助学生了解分心的原因，掌握避免分心的方法，从而不断提高学生注意力水平。

鹤北中学政教主任张丹老师向与会者介绍了学校“多维赋能 共享公平教育”的融合教育工作。学校从“坚持理念先行，营造良好氛围”“推进1+1+1模式，优化融合教育师资”“落实特殊需求 关注融合教育时效 ”三个方面不断推进学校融合教育实践与研究，让每一位孩子享受公平教育，不断提升学校融合教育的质能和效能。张老师通过一个个鲜活的故事案例，生动展现鹤北教师对特殊需求孩子的关心关怀和因材施教。

随后，俞老师对课堂进行了反思重构，与会老师们进行了精彩的点评交流。交大二附中林老师首先肯定了俞老师切合学生特点的选题，导入通过一个小游戏一下子巧妙聚焦了学生的注意力，授课过程中多次关注特殊需求学生，适时进行鼓励，学生在老师的关注和鼓励下，课堂中注意力明显增强。特教中心聂老师则表示在初中阶段关注学生心理健康是极为重要的，做好初中特殊需求学生的社会适应性，心理课程发挥着重要意义。区教研员谭宏老师肯定了俞老师在教材和学情分析方面的细致分析，同时也告诉大家，做好学情分析是做好融合教育的基石。同时课堂设计中老师充分发挥了视觉、听觉等不同的感官，游戏、小组合作展示等形式丰富，重点聚焦。如果能够抓住课堂生成资源，利用生成资源再次聚焦本课重点“注意力”，及时与学生对话与提炼，那么学生受益会更大，课堂目标达成度会更高。

最后，闵行区特教指导中心主任、启智学校石筱菁特级校长肯定了我校教师在融合教育工作的辛勤付出，同时也对区域融合教育提出了希望：党和国家越来越受重视融合教育，让普特孩子在融合教育中共同发展，让特殊需求孩子更好适应集体生活，共同学习，相互理解和尊重，将来更好地适应社会生活是我们教育工作者的使命。这需要学校构建普特融合课程，需要教师不断提升对特殊需求孩子的适度支持和精准服务的本领，各方联动，协同推进。

本次教研活动给予进一步明确了融合教育前进方向，始终把融合教育放在学校工作的重要位置，进一步深入学习，积极研究实践，提升学校融合教育的质与能，成就每一个孩子的出彩人生。
