> 干部培训

敢担当 善作为 能成事 ——2022年闵行区教育系统部分副校（园）长、后备干部培训顺利启动

发布时间 : 2022-08-26   作者（来源）: 曹雅明   阅读 : 1676

3/8
←
→

为进一步加强闵行教育校级干部梯队建设，2022年8月22日，闵行区教育系统副校（园）长、后备干部培训开班仪式在紫竹国际教育学院举行。来自我区中小幼学校的29名副校（园）长、62名后备干部参加开班仪式。

闵行区教育党工委副书记李光华出席，并对学员提出三点要求：一要珍惜来之不易的培训机会，在意识上高度重视、行动上积极参与、纪律上严格要求；二要处理好三组关系：学校工作与培训学习的关系、师徒之间的关系、学员之间的关系；三要正确对待晋升，机会总是给予有准备的人，要以平常心态去对待。

教院附中副校长杨希、闵行中学德育副主任金寅仪分别代表副校长班、后备干部班学员作发言。表示一定不负领导厚望，严格培训纪律，积极参与，学以致用，在工作中真正做到牢筑思想根基，勇挑责任担当；表示要在“好教师”到“好干部”的转型过程中，把握学习机会，积极思考、勇于实践、突破自我。

教育学院校长发展中心主任俞琳对本次培训的安排作详细解读。教育党工委党群科科长林火生主持开班仪式。

闵行区教育党工委书记恽敏霞为学员作《新时代优秀干部的使命和担当》首场报告。恽书记回顾了近10年来闵行教育的厚积薄发与发展变化，让学员们深切体会到闵行在由教育大区向教育强区转型过程中所取得的丰硕成果；同时指出了闵行教育在现有干部队伍、师资队伍建设上面临的问题和挑战，主要表现为与教育现代化建设目标存在差距，对新时代、新理念、新发展的理解和适应还不够。紧接着，恽书记围绕“干部的使命与担当”，结合自己的成长经历与管理经验，从目标任务、管理方法、学校文化建设、团队合作原则、新时代干部素养等五个方面，为年轻干部提出要求，并指出学校管理既要讲原则又要讲感情，学员们倍受启发，进一步明确了成长方向。

上海市教育功臣、原七宝中学校长仇忠海围绕“学校领导艺术”，给全体学员带来一次深刻的教育观念与教育情怀的洗礼。大家从仇校长的学习之路体会教师肩负的使命，从仇校长的为师之路感受校长的教育情怀，更从他的办学之路看到仇校长独到的教育思想。作为新时代的青年干部，大家表示要学习仇校长满怀热情的教育情怀、与时俱进的创新精神、以人为本的管理艺术，勇于追求教育的真善美，让教师获得成就感和幸福感，让闵行的每一个孩子健康快乐成长。

市委党校教授沈士光为学员们带来了《提高青年干部政治能力》的专题报告。他从三方面阐述了青年干部提升政治能力的要素：一要切实增强青年干部的政治判断力，二要切实提高政治领悟力，三要切实提高政治执行力。沈教授结合案例，给学员们介绍了很多具体的“妙招”,比如：养成目光如炬的眼力，学会辩证看待问题等。沈教授的报告高屋建瓴、深入浅出，进一步帮助学员明确新时代优秀年轻干部应不断提升自身政治能力。

专题报告后，学员们围绕“新时代优秀干部的使命与担当”这个主题，先分成7个小组开展讨论，再以班级为单位作集中汇报。

紫竹小学蒋方叶、世博小学张玲、师培附中蔡晓玲、上师闵幼郭丽娜，文绮中学许灵均、陈轶，明强小学姜丽霞、交大实小杨永茹、七宝中心幼儿园陈丽君九位学员，结合组内成员的讨论和分享，以及自身工作实践作集中汇报。表示年轻干部们一定牢记立德树人根本任务，不忘初心、坚守“大任”，责任担当、笃行“己任”，通达创新、敏于“善任”；以学思践悟加强理论修养，以勇立潮头彰显岗位本色，以成人成事打造优质队伍，争做新时代优秀年轻干部。 

七宝中学党委书记、校长朱越对副校长学员的汇报作点评。朱校长明确提出，新时代优秀干部的使命和担当就是为党育人、为国育才，培养德智体美劳全面发展的社会主义事业接班人。并围绕四个问题和学员们展开探讨：当上级指派去薄弱学校担任校长，你是否能义无反顾地接受？当面对种种诱惑，你是否能做到不计名利、坚守信念？当面对问题或失败，你是否能知难而上？当有人需要你去打破底线处理问题时，你是否会有底线和担当意识？朱校长饱含深情、情真意切地告诉年轻学员，优秀干部面临困难与挑战，应知难而上一如既往，面对风险与失败，应做好自己做强自己，发挥志同道合的力量，用扎实的专业功底找到适切的解决办法，办一所百姓认同、学生向往的好学校。

闵行中学党委书记、校长乔慧芳对后备干部的汇报作点评。乔校长告诉学员们“闵行教育看见每一个全力以赴的教育人，闵行教育用优质生态激励每一所热腾腾成长的学校”，期望学员们能够践行“自然人”到“社会人”的转换，审视“使命”与“担当”的含义；并努力提高三种品质：勇于尝试不断成长、情绪稳定善于反思、宽厚温暖善于合作；进而形成两种能力：一是持续学习，形成认真学习、善于学习、分享学习、带动学习的连锁反应；二是智慧践行，要有目标意识、问题意识、过程意识。坚定理想信念，坚持正确方向，履行时代赋予的使命，做敢担当、善作为、能成事的优秀年轻干部。

李光华副书记、潘蓓蕾副局长分别作讲话，对本次干部培训的精心策划，以及学员们的学习态度和精彩汇报给予高度评价。李书记指出，作为一名学校管理者的使命就是：把学校办好，把学生教好，敢于作为，勇于承担责任，用行动证明自己的使命和担当。潘局长对学员们提出殷切期望，希望学员们在培训和工作中汲取养分，寻找同行者，重点探索如何积累管理素养，从一个人到带领一个团队，再到带领团队走向高度，达到“敢担当、善作为、能成事”的自我实现需要。
