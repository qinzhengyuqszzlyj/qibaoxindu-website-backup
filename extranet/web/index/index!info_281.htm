趣玩户外 聚焦分享 ——“幼儿发展优先”低结构分享马桥富卓专场

发布时间 : 2023-05-24   作者（来源）: 顾莹   阅读 : 728

3/8
←
→

2023年5月24日，在区教育学院教研员顾莹老师组织带领下，闵行区马桥富卓幼儿园开展了以“幼儿发展优先”理念下幼儿户外低结构活动分享实践的主题教研，南片幼儿园的骨干教师也共同参与了此次活动。

首先，马桥富卓幼儿园进行了户外低结构游戏展示活动，游戏中幼儿充分挖掘材料的特性，利用各种低结构材料进行组合探索，想象模拟出了“我心目中的小学”、“高速公路”、“农家乐”、“游乐场”、“野餐”等不同的情境游戏，充分展现出幼儿对游戏的投入及自主能动性。其次老师们回到多功能厅集中观摩了大四班沈婕老师的游戏分享活动视频，沈老师通过不同的方法策略让幼儿之间有话可说、有问题可解决，形成了有效的师生互动、生生互动。

接着，瞿萍园长的微报告“放手游戏，回归本真”，用生动的语言带领大家从空间和材料的视角去欣赏了马桥富卓幼儿园宝贝们在游戏中的“花头精”，瞿园长深度解析了幼儿园教师如何合理划分场地、有效提供材料、实时观察并全面解读，从而针对性推进户外游戏，展现了马桥富卓幼儿园户外游戏中的多面性和创造力。

随后，王琳老师带领项目组老师们，围绕沈老师组织的户外低结构游戏分享，开展了“趣玩户外，聚焦分享——户外游戏分享中的策略与方法”研讨交流。研讨中教师们基于儿童立场，结合分享中教师引导语进行剖析解读，从“观察幼儿游戏行为、分析教师的教育支持策略和提出合理化的建议”三个方面进行交流与探讨，老师们积极发表想法，提出优化建议，形成教研共识。

为了能与南片骨干教师们有更好互动，形成思维碰撞深入交流，教研员顾莹老师组织开展了现场研讨活动，共分为三个环节，环节一：每个老师用两个字评价沈老师的游戏分享活动，表达欣赏与感受；环节二：现场互动，客人老师与执教老师对话，以问交流；环节三：分组讨论，梳理本次游戏分享活动可以优化的地方。这样的教研形式充分调动了每一位教研参与者的积极性，以建设性的建议双向奔赴，达成教研的最终成效——共同成长。

最后，教研员顾莹老师对本次活动做了精彩的点评和总结。顾莹老师支持沈婕老师在户外游戏分享中走动式分享的大胆尝试，肯定了马桥富卓幼儿园近两年来对幼儿户外游戏的深入探索精神。同时顾老师也提出，游戏分享中老师要学会思考如何聚焦话题，引发生生互动的高阶思维与表达。

每一次尝试都是新的起点，我们将继续不断探索、创新户外游戏的支持方式，创设游戏环境、丰富游戏材料，给予幼儿自主自由的户外游戏空间，同时努力搭建教师展示的平台，让师生在户外游戏的实践研讨中不断探索、成长！
