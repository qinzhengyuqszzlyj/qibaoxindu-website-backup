趣“研”自然，助力“童”行 ——闵行区第四期幼儿园园长培训基地现场交流研讨活动

发布时间 : 2022-12-02   作者（来源）: 曹雅明   阅读 : 1380

2022年11月25日，闵行区第四期幼儿园园长基地培训班在导师郁亚妹园长的带领下，走进基地学员校交大闵行幼儿园。通过现场课程观摩调研，解答课程实施和管理等方式，培养学员全面思考、统观全局的思维品质，积淀管理经验，提升园所办园质量。

在交大闵幼于莉园长的陪同下，郁亚妹导师和学员们参观了园所环境，观摩了两个特色活动。郁园长针对现场观摩及于莉园长的介绍，和学员们进行了深入的研讨交流，为闵幼的课程实施提供了很多“金点子”。

              深挖“田趣”、浸探菜园

导师郁园长带领着学员们来到菜趣园，观摩孩子们在菜园子里摘菜称重、提色提香、除草养护、枯叶创想等活动，对这些独特的活动形式给予了肯定，也提出了更有新意、更有深度的课程架构和场景的调整。

来到探趣工坊，这里的孩子们又是另一番热闹的场景，加工蔬果、稻草制作、收货种子等，充分运用植物的特性，挖掘各种有趣的玩法，基地学员们们对材料和玩法产生浓厚的兴趣，同时，基于儿童的立场，导师带领着学员们探讨了很多利于幼儿操作和能力发展的课程内容。

              优化场地、玩探自然

随后，又观摩了孩子们的户外游戏，特别留足沙水游戏，对沙、水、泥的场地设置和幼儿活动路径郁园长提出了优化的建议。

              彰显课程活力、趣探自然魅力

基地学员于莉园长对闵幼近期课程特色实施进行了介绍，主题为：让孩子在自然中成长。于园长对“自然”的进行了深入的解读，自然是一个地点：亲身体验、直接感知；自然是一堆材料：就地取材、亲手操作；自然是一种氛围：自主探索、自由撒欢。

最后，导师郁园长给予了精彩、细致地点评，郁园长指出：要让课程内容充满着幼儿对生活的体验和回味的呈现；让课程的样式充满着教育的浪漫和对生活生命活力的彰显，要让孩子成为课程建设的主体！基地学员们在聆听和交流互动中受益匪浅，我们将一路“童”行，让课程跟随孩子们成长的脚步不断优化、不断完善。
