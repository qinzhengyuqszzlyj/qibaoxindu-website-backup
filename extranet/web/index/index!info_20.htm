新时代优秀年轻干部的使命担当——2024年闵行区教育系统优秀年轻干部培训班第三次专题培训

发布时间 : 2024-05-17   作者（来源）: 朱凤   阅读 : 1045

3/8
←
→

在推进闵行教育高质量发展和教育强区建设的新征程中，作为闵行教育的中坚力量，年轻干部承载着极为重要的使命与重任。5月15日，2024年闵行区教育系统优秀年轻干部培训班第三次专题培训在闵行区教育学院举行，闵行区教育党工委书记恽敏霞、副书记徐明怡、区教育学院党总支书记、院长朱靖、仇忠海校长、张人利校长、朱越校长等近40位领导、专家出席活动。

恽敏霞作领导讲话。她希望学员们坚定理想信念、牢记初心使命、不断修炼自我、主动担当作为，以实际行动诠释对党的忠诚、对党的教育事业的忠诚。并提出了三点要求：一、明方向，深化思想淬炼、加强政治历练、努力成为政治上的“明白人”，不断增强政治判断力、政治领悟力、政治执行力。二、敢担当，敢于在教育改革发展的关键事、关键时中迎难而上，攻坚克难、闯关夺隘，确保党的教育方针在教育实践中得到全面贯彻和深入落实。三、善作为，加强岗位历练，提升化解矛盾的能力，脚踏实地、坚韧不拔、埋头苦干，推近区域教育高质量发展。

培训分小组汇报和集中汇报两个环节，聚焦“五育融合，助力学生身心和谐发展”这一主题，中小幼10个小组71位学员进行分组汇报。围绕“坚定的政治立场、正确的教育思想、清晰的逻辑思维、流畅的语言表达、厚实的专业功底、较高的综合素养”等方面对学员做综合评估，经过专家推荐，莘庄二幼唐英、中福会浦江幼儿园黄璟珉、平南小学殷克清、景东小学江俊芝、上师三附小南校王晓涛、 浦江一中沈磊、七宝鑫都孙艳婷、华理科中王丽娜、七宝二中翁宁蓁、莘庄中学秦承勇10位学员代表小组进行集中汇报，颛桥镇幼儿园金丽莉主持集中汇报。学员们通过文献研究，深刻解读了五育融合的目的意义、价值追求，培养德智体美劳全面发展的社会主义建设者和接班人。结合校情学情，通过课程、项目、活动等载体，分享了五育融合的路径和策略，培养全面发展的人。同时也提出了实施过程中的困境。

专家们肯定了学员的汇报有价值，有思考，有实践，思维激荡，见解独到。思想引领、提高认识，立德树人是教育的根本任务,五育融合是实现立德树人的有效途径；躬身实践，知行合一，转变育人理念，突出实践担当,对标对表政策，提炼课程建设的理念和目标，落实五育融合的实践智慧。同时建议：五育融合实践要向下扎根，要立足学科，深度学习，体现学科的优势；汇报内容要立足岗位、立足自身成长，形成独特的思想，追求表达的思想力；要以小见大，要有观众意识，平实、朴实、真实最能打动人！

作为闵行教育系统的年轻干部，珍惜这个伟大的时代机遇，以更加昂扬的斗志、更加务实的作风投身到教育改革创新中去。激发想为动力、提升善为能力、鼓舞敢为勇气，鼓足精气神、练就铁肩膀，创造闵行教育高质量发展后继有人、接续奋斗的好局面。

后续还将有三次这样的主题汇报。
