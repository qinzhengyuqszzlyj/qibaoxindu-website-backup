四年级语文区级教研活动在虹桥小学举行

发布时间 : 2023-09-22   作者（来源）: 杨献荣   阅读 : 369

运用现代信息技术，提升习作导评实效

——小学四年级语文区教研活动

 

9月21日下午，“运用现代信息技术，提升习作导评实效——统编语文四年级上册第二单元习作教学研讨”区级教研活动在虹桥中心小学举行。闵行区教研员杨献荣老师莅临指导并主持本次活动，全区四年级语文老师全程参与了本次线下和线上直播活动。

学思融通，精彩纷呈

活动伊始，全体教师一同观摩了虹桥中心小学刘丰霞和殷超两位老师的课堂教学，他们分别执教第二单元习作《小小“动物园”》的指导课和评改课。

刘丰霞老师课堂各环节层次分明，运用有详有略、先概括后具体的写作方法，借助思维导图的形式，帮助学生思考和梳理家人和动物的相似之处，让学生感受到家人的可爱、家庭生活的趣味以及表达的乐趣。

殷超老师在指导课的基础上，深入推进，运用智慧笔使学生能够在班级同学面前修改、分享自己的习作，通过其扎实的教学功底和有趣的活动设计将教学内容完美地呈现出来，让学生体验表达的乐趣，点燃学生思维的火花。

交流研讨，共促提高

课后，在杨老师幽默风趣的主持下，进入研讨环节。首先由两位上课老师就教后心得做了分享，杨老师与两位上课老师精彩互动，思维的火花在彼此脑海中碰撞，“教”与“研”也得到了进一步深入与延伸。杨老师肯定了两位执教老师对于智慧笔运用的阶段性成果，在教学中创设学生相互交流、互相评改的环节，打造高效的真实课堂。同时给予两位执教老师鼓励和建议，使两位老师对自己的教学有了全面的认识和改进方向。

接着，实验小学中高级语文教师王冬华老师对两位老师的习作课进行了评课。习作指导课循循善诱，由三个问题引起学生对家人与动物相似点的思考；习作评改课有的放矢，运用智慧笔创设真实情境，从点到面讲评学生的习作。

最后，杨老师对本次活动进行点评，通过杨老师深入浅出的剖析，老师们认真倾听、用心记录，老师们明白了习作课中蕴含的智慧，以便将其精髓渗透到日常教学之中，真正提高习作教学水平。

教师成长于课堂，唯有经历才会成长。此次教研活动虽然结束了，但课堂教学没有停止，杨老师的亲临指导，对于今后的语文习作教学提供了方向性指导意见，“运用现代信息技术，提升习作导评实效”打下坚实的基础！

  

 

                                             虹桥中心小学供稿
