校本教研助力教师专业成长助推学生素养发展

——闵行区高中政治教研活动在上海师范大学附属中学闵行分校举行

 

2023年10月26日下午，闵行区高中政治教研活动在上海师范大学附属中学闵行分校（上师闵分）举行，本次活动也是上师闵分“MF-PTTI”式循证教研实践研究的系列活动之一。与会老师围绕”情景设计与学科核心素养培育关系“的循证问题展开研讨活动。区全体政治教师参加了此次活动，活动由区高中政治中心组成员、上师闵分文综组教研组长朱林英主持。

首先，大家观摩了上师闵分的张世敏老师的公开课：《从石库门看上海文化——必修四第三单元综合复习》。张老师首先由“中国邮政·上海民居”照片导入石库门这一情境，课前给学生提供丰富详实的关于《石库门的诞生与发展》的阅读材料，让学生在了解石库门诞生与发展的基础上对石库门是如何发展而来的与石库门发展的过程中反映了哪些文化现象的问题进行思考，帮助学生提升学科阅读理解能力以及提升透视真实情境背后蕴含的学理及其现象之间的内在有机联系。接着，通过提供材料二《建筑改造利用项目——新天地》，引导同学对“新天地”的改造案例进行辨析，进而探究对待传统文化的正确态度，领会中华优秀传统文化进行创造性转化、创新性发展的重要意义，培养学生的科学精神。最后，以为进博会布置“石库门”展区进行布展方案的设计为情境，帮助学生综合运用学科知识，提升公共参与的素养，提升文化自信。

随后，上师闵分政治教研组做了课题介绍并进行组内研讨。朱老师向各位与会的老师们介绍了循证教研的由来。”MF”一方面是“上师闵分”简称的缩写，另一方面又指“Master、Faculty”即由专家和教师组成的同伴互助式团队，“PTTI”代表着Problem、Tool、Test、Improvement四个方面。循证教研由基于教学现状确定真实可研究的问题即提出循证问题；搜集证据、评价证据质量；筛选、提炼证据后，进行教学设计；通过课堂观察工具评估教学效果；生成新的证据；进行改进几个过程构成。王悦老师对政治组教研的问题——情境设计与学科核心素养的培育进行了背景资料的介绍。《普通高中思想政治课程标准（2017年版）》指出，高中思想政治课程是落实立德树人根本任务的关键课程，以培育社会主义核心价值观为目的，是帮助学生确立正确的政治方向、提高思想政治学科核心素养、增强社会理解和参与能力的综合性、活动型学科课程。思想政治学科核心素养就是看学生能否运用学科内容应对各种复杂社会生活情境的问题和挑战。学科内容也只有与具体的问题情境相融合，才能体现出它的素养意义，反映学生真实的价值观、品格和能力。因此，本次教研以一节教学公开课为例，探究如何在真实情境中，培育学生学科核心素养。刘谷老师分享了政治组前期证据的搜集与评估。围绕情境设计与学科核心素养培育之间的关系搜集了来自知名学者和核心期刊相关的参考文献，并从中选取了具有主要参考价值的核心文献，从教学情境的分类、情境教学的要素、情境的结构化以及情境实效的落实四方面入手，聚焦如何区分选择复杂情境，如何在实践中优化、处理情境，如何将情境实效落于学科核心素养的培育三个角度。通过这些证据的整理与分析为教师构建设计合理的优秀情境、在实践中优化处理情境以及如何在课堂中通过真实情境培育学科核心素养提供了极为重要的理论支撑。云丹阳老师就在实践中发现的新证据进行了补充。在具体操作时既要对情境进行精准性的应用与把控，做好对信息比较丰富和复杂的情境的划分和引导，又要在情境中融合多元的探索方式，激发学生的辩证思维能力，还要能依据情境的落实情况而创设新问题，落实参与感和互动感。最后，张世敏老师分享了本节课的教学设计思路、教学目标及与循证教研问题之间的关系。以文化的传承和创新为载体，既完成对必修四第三单元对正确认识中华传统文化、弘扬中华优秀传统文化、理解文化交流互鉴的意义的知识复习，又以石库门为情境主题探究真实情境背后所蕴含的学理和学科核心素养的落实。

学科组各位老师结合本课课例，依据描述与分类、解释与论证、预测与选择、辨析与评价四重维度开展现场研讨。朱林英老师认为，张老师的这堂课依据情境所设计的问题层层深入，就是遵循着从描述与分类到辨析与评价的逻辑线的，整堂课既体现了对学生科学精神的培养，又很好地通过问题设计将学科任务融入其中。王悦老师认为，张老师的课堂基于文化载体，以主题式的情景形式推进，很好地呈现了简单情境到复杂情境的结构化建构。并且对于同一个问题能够在不同的情境基础上进行推进，生动的体现了课堂从以知识为导向向以素养为导向的转变。刘谷老师从课堂教学情境创设与学科素养的养成角度进行了分析，张老师的课堂做到了情境选择的普遍性与典型性兼顾，启发性与思考性并存，开放性与封闭性得宜。云丹阳老师认为张老师的课堂充分体现了学生的主体性地位，注重对学生的启发和引导，鼓励他们的多元探索，以真问题调动学生运用核心素养来切实分析和解决问题，对年轻老师的帮助非常大。

接着，兄弟学校的老师们对本课展开了热烈讨论。闵行中学的周莺老师认为张老师能够给高三年级开设一堂如此精彩的复习课非常不容易，整堂课的课堂充盈着一种轻松感。巧妙的情境设计充分落实教研组所设定的教研活动目标。上海中医药大学附属浦江高级中学傅志连老师认为本节课的最大亮点在于情境的设置与问题的追问，是难得的不悬浮，以教材知识点为落脚点的高效、完整的课堂。华二紫竹的孙红艳老师评价本节课成功做到了仅仅用一课时完成了整个单元的复习任务，依据情境把问题推进不留痕迹的将整个第三单元的知识串联起来。北外田园的王明睿老师充分肯定了课堂中辩论环节的设计，这个环节充分展现了对学生核心素养的培育。闵行实验高中的王首强老师在充分肯定本节课的基础上，提出课堂导入环节可以提前让学生准备，课堂小结环节尽量再多展开一些的建议。华二紫竹的郑培立老师从石库门素材选择拓宽到了整个海派文化，帮助大家打开思路，拓宽方向，为讲授文化部分的内容提供了很多启发性的素材。

教学即研究，教师在教学中不仅要低头走路，也要抬头看天，要以研究的态度反观自己的教学，只有在不断的研究中，在不断的反思和改进中，才能让自己的教学之路走的更稳、更远。本次教学研讨活动，是上师闵分政治组关于循证教研课题的阶段性成果展示，给与我们如何开展校本教研，创新改进教学提供很好的启示。
