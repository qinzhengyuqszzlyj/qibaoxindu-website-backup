从理论性“教书匠”走向反思性“实践家” ——闵行区学习中心组（社会）、种子基地活动协同教研

发布时间 : 2023-05-31   作者（来源）: 顾莹   阅读 : 256

3/8
←
→

春花落尽，夏木成荫。由闵行区学前教研员顾莹老师组织，学前一组种子基地教师协同学习中心组（社会）共同教研以“童蒙养正·立德树人”为主题、以幼儿可持续发展为教育导向、以教师的教育专业发展为研究根本，活动于2023年5月29日在上海市启英幼儿园如火如荼地开展并生发。

观摩分享共研学

上午，由黄仁敏、徐雪菲、王文、张懿芸四位老师分别执教了大班《换个角度说规则》、《小青花》、《党员》、《三个和尚》的偏社会领域的集体教学活动。

《换个角度说规则》中幼儿通过游戏的多次体验与探讨，感知到规则的重要性，尝试和伙伴们共同制定规则。活动《小青花》从绘本青花瓷小猫入手，带领孩子们感受中国经典风景的壮阔与美丽。活动《党员》让幼儿感知到有着为人名服务、勇于人先这些优秀品质的党员就在我们身边。《三个和尚》活动让幼儿从传统民间故事中感知团结的重要性，找到有益团结的好方法。每一个活动的独特和其对幼儿成长的意义价值让大家受益颇多。

良师引领促成长

下午，专家徐老师对四个活动进行了细致的指导与总结，肯定了教材对于幼儿的德育价值和活动方式的多元，同时也针对活动环节及目标制定中出现的共性问题提出斟酌和思考。风趣幽默的语言、一针见血的专业引领和另辟蹊径的独特视角使得在场学员深受启发，碰撞出更多的灵感火花。而后顾莹老师肯定了学员们精益求精的态度以及对专业永不止步的追求。

教而不研则浅，研而不教则空。此次活动让成员们在融洽和谐又深刻的研讨中有所思、有所获，为专业道路上的每一步前行，打下夯实的基础。

 

摄影、撰稿：上海市闵行区马桥富国幼儿园 沈嘉玮
