敢担当 善作为 能成事 ——2021年闵行区教育系统优秀年轻干部挂职锻炼培训班集中培训开班

发布时间 : 2021-12-16   作者（来源）: 曹雅明   阅读 : 1750

为进一步提升区域优秀年轻干部的综合素养和实践能力，提高他们的政治站位和责任担当，2021年12月15日，闵行区教育系统优秀年轻干部挂职锻炼培训班集中培训活动在上海市陈云纪念馆举行，闵行区教育党工委书记恽敏霞、教育局副局长乔慧芳出席。七宝中学党委书记、特级校长朱越，实验小学党委书记、特级校长何学锋前来指导。

党群科科长林火生主持开班仪式并作开班动员。他指出，此次集中培训是对为期一年左右的年轻干部挂职锻炼培训新模式探索的总结和分享，全体学员将围绕党的建设、教育改革、学校办学绩效等方面开展深度学习，形成思想共识，树立教育自信，锤炼党性，提升素养。

教育学院校长发展中心主任俞琳从“培养目标、培训内容、呈现形式”三方面对培训方案进行解读说明。介绍了本次培训将通过系列专题报告，帮助学员拓宽视野，完善管理知识结构，提高作为学校领导的综合素养；通过学员讲堂、专题研讨、结业论坛等形式，聚焦挂职过程中的人与事，帮助学员提升思想境界，提高思维品质；通过实行班委会领导下的“班委+小组组长”自我管理模式，帮助学员在完成任务的过程中感受组织的力量，感悟机制制度建设的价值意义。

鹤庆幼儿园王贝贝代表26名挂职干部作发言。感谢教育党工委、教育局、教育学院为年轻干部提供的宝贵学习机会，她从“开眼界、树格局、立思想”三方面阐述了挂职过程中的体会和收获，并表态在集中培训的三天时间里，将与小伙伴智慧同行、思维碰撞，博观而约取，厚积而薄发，将所学内化于心外化于行，用实践和青春为闵行教育贡献力量。

恽敏霞书记围绕《闵行教育的现在与未来》为学员们作首场报告。她指出闵行教育将围绕建设高质量教育体系，以教育评价改革为牵引，统筹推进育人方式、办学模式、管理体制、保障机制等一系列改革。21世纪是人才发展和人才竞争的时代，作为闵行教育的优秀年轻干部，要珍惜这样的学习机会，把握国家“十四五”教育改革发展战略，瞄准上海教育现代化2035和“十四五”教育发展目标。更要立足新时代的历史方位明晰责任，坚守教育初心，坚定为党育人、为国育才信念，在推进各项教育改革中攻坚克难、真抓实干，成长为政治过硬、品德高尚、专业精湛、治校有方的学校管理者。

下午，学员们一起参观了陈云纪念馆，并且聆听了徐建平老馆长《庆建党百年，传伟人精神，讲“四史”故事》的专题报告。近距离了解和感受一代领袖陈云从20岁入党，在70多年的革命生涯中，不忘初心，牢记使命，为共产主义事业奋斗一生的故事。陈云同志“不唯上，不唯书，只为实”“交换，比较，反复”的坚定理想信念，以及求真务实的作风、艰苦朴素的公仆情怀，让学员们深受启发。

接下来的学员讲堂，闵行中学华佳齐、闵行四中刘叶、紫竹小学沈雯珺、田园外小何敏四位挂职干部，分别围绕《“双新”“双减”背景下的学校作为》，分享了“双新”推进学校内涵发展、指向学科课堂变革的教学研究与实践的有效方法；并以处理信访的经历向大家展示了“双减”背景下家长社会的纠结状态；阐述了“双减”背景下，应该还学生一个喜欢的校园、一个精彩的课堂、一点闲暇的时空，还教育本应该有的样子。

何学锋校长、朱越校长对学员讲堂进行点评。何校长指出“双减”背景下学校的作为是一项系统工程：第一要学习和深思，研究透中央文件的具体内容；第二要梳理和分析，了解学校个体存在什么问题，面临的困难和挑战是什么；第三要整体性地策划和设计，并有序智慧地推进，建立系统思考和策划的思维品质。朱越校长提出，不管处于哪个教育改革阶段，对管理者而言，都要成为一所学校问题的发现者、研究者、解决者，要有敏锐的问题意识和解决问题的目标意识，站在学校和学生的立场，秉着实事求是的态度对待教育，并解决教育过程中出现的问题。

乔慧芳副局长作点评并讲话。她指出，作为闵行教育未来的学校管理者，要牢记立德树人的教育初心，强化学习并注重消化吸收，然后构建愿景，形成共识。要敬畏规范，因为质量来自于精益求精的规范，教育管理者要做好各个领域里的规范。要彰显个性，用实践智慧彰显教育个性，尊重教师、善待教师，因为教师的模样就是学生未来的样子。

晚上，全体学员分成中学、小学、幼儿园三组，围绕挂职锻炼体会展开了一场特殊的“头脑风暴”。大家站在学校管理者的角度，聚焦校长领导风格、管理思维修炼、干部结构优化等核心词，结合自己挂职期间的所见所闻、所学所思充分交流，既有对导师的评价，也有挂职单位与原工作单位管理上的对比分析，以及自己在工作中遇到的疑问、困惑和对未来的工作设想。

乔慧芳副局长，朱越校长、何学锋校长，以及林火生科长、俞琳主任等参与晚上的小组讨论。领导专家对学员们挂职期间的工作态度，善于观察与思考、总结与感悟，能够用比较与分析来审视自己工作的做法表示肯定。强调青年干部要站在未来管理者的角度提高站位、保持定力，要形成问题意识、研究意识，在挂职实践中要重塑自我，做到心胸开阔有韧性。并建议大家按照“现状分析、优势不足、最大触动、收获提升”的思考路径进行总结和凝练，深入挖掘培训的意义和价值，真正做到“学、思、行”相结合。

在后续两天的培训中，颛桥中学校长李卫军、申莘/鑫都小学校长康旻、莘松中学校长张忆将为学员分享自己的办学实践，以及学校管理中基层学校党组织的作用等。学员们还将结合近一年的挂职锻炼，以及本次集中培训作第二次学员讲堂和分组汇报。我们期待培训能取得圆满成功，助力年轻干部成长为敢担当、善作为、能成事的好干部。
