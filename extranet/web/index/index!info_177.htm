走进社会大课堂 ——社会实践类综合实践的活动课程的设计与实施

发布时间 : 2023-06-02   作者（来源）: 马丽敏   阅读 : 703

2023年5月30日下午，主题为“走进社会大课堂——社会实践类综合实践的活动课程的设计与实施”研讨活动在线上召开。此次活动是“基于‘双新’推进的上海市普通高中综合实践活动课程实施研究与实践项目（第二期）”系列研讨活动之一，也是继去年11月22日之后，闵行与普陀的又一次跨区联合教研。两区项目核心成员、项目校研究团队、各高中综合实践活动课程教研（备课）组长、任课教师代表以及对活动感兴趣的初中老师相聚云端共同参与，活动由上海市普陀区教育学院中学研究型课程教研员钱铁男老师主持。 

本次活动传承了区域联合教研的良好传统，对标市项目研究任务、落实区域教研导引作用，也是对本学期“上海教研在线”网络教研主题的回应，是上一期教研活动的延展。来自两区四所学校五位老师依次围绕主题作经验分享：

上海音乐学院附属安师实验中学乔宗君老师介绍了学校是如何立足学校特色，开发 “音乐采风”系列研学活动，组织在音乐方面有专长以及对传统文化有浓厚兴趣的学生深入少数民族地区，在专业指导教师带领下，分组实地调研、收集民族特色音乐、改编形成创新作品。活动带给学生的不仅仅是音乐素养和文件知识的累积，更是厚植民族情怀的种子。

上海市曹杨第二中学孙英俊老师阐述了对“走进社会大课堂”主题的理解，并在此基础上介绍了学校“格莱珉研学实践”项目设立与实施情况，该板块与“重走大师路”江村社会实践考察活动以及“博雅西部行”甘肃三农考察社会实践活动一起构成学生“了解三农——体验三农——服务三农”三部曲，螺旋递进、多维有序，综合提升学生社会责任。

上海市晋元高级中学姜炜老师从“大思政课”视域出发，解读了多项政策和文件内涵，在培育学生家国情怀、提升实践服务能力上，大思政“行走课堂”与综合实践活动课程殊途同归，能彼此助力。为此学校结合原有红色之旅寻访考察活动的组织经验，策划实施“光辉历程、使命传承”南北双线研学考察活动，强化学生爱国信念与责任担当。

上海市闵行中学陈珂老师以自己带领学生赴新虹桥国际医学中心开展场馆学习的经历为例，细致阐述了过程中如何通过“问题-议题-课题”逐步推进，在发挥学生主观能动性的同时亦不缺失教师的指导作用，成功打通社会大课堂与学科小课堂，让学科与综合实践活动互为支撑、协同育人。陈老师也对如何将此类教学模式迁移至日后更多活动，以丰实学校人文实践系列课程，形成激发学生潜能的综合实践活动课程体系做了进一步思考与展望。

随后，上海市闵行中学乔晓岚老师分享了自己参与开发和实施《博物馆之旅》课程的历程，从带领学生找寻生活中的历史——寻根老闵行，到与不同学段不同学科老师共同探索基于不同博物场馆的跨学科教学和生涯教育，再到寻访老校址、保护老建筑，呼吁传承闵中文脉……虽然师生游走的场馆一直在变，课程也在不断迭代更替，但探寻文脉、立德树人的初心不曾改变！

在自由研讨环节，老师们就如何合理规划课时、统筹安排学生出游，场馆的选择、学生过程性及终结性评价设计等实施细节做了进一步沟通交流，让部分尚处于起步和探寻阶段的老师增强了信心，也为部分学校整体规划学生寻访活动打开了思路。闵行区中学两类课程教研员马丽敏老师分享了自己学习五个案例后的感悟，并对如何从校园小课堂走向社会大课堂，如何从活动提升为课程，如何从研究型课程转型对接综合实践活动课程的实施要点和注意事项做了梳理，也对目标细化过程中的难点与重点做了提点，并引导学校要规范课程管理文本，架构起三级网格体系。普陀区中学研究型课程教研员钱铁男老师总结活动，并对网络教研工作要求做了补充强调，分享了一些优质评论，也希望这些意见和建议会成为我们下次教研的起点和切入点，使研究不断深化！
