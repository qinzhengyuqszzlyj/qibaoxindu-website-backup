快乐游戏 共同成长——2018闵行区保教主任初级研修班一组培训简讯（4月）

发布时间 : 2019-04-30   作者（来源）: 朱凤   阅读 : 2061

4月23日，2018闵行区保教主任初级研修班一组学员在金色幼儿园进行建构游戏专题培训，虹鹿幼儿园、七宝中心幼儿园、诸翟幼儿园、鹤庆幼儿园的相关老师，以及本次研修班五组的学员共计50余人参加了活动。

七宝中心幼儿园副园长兼研修班一组导师汪湜主持活动。

此次活动共分为三个环节：观摩中大班混班建构游戏、反思互动交流、小组专题研讨。

观摩环节，学员们们提出了各种相关的问题，如：“建构游戏需要导入环节吗？”、“中班建构专用活动室需要提供大型的玩具吗？”、“六边形的积木适合放在大班建构室吗？”……金色幼儿园保教主任侯灵杰一一解答了组员的疑问：建构游戏的导入基于孩子的基础并给于充分的游戏时间；中班提供大型搭建玩具是考虑到与小班角色游戏、表演游戏、建构游戏进行衔接，让幼儿在搭建中进行角色游戏；六边形积木利于大班幼儿创造性行为、逼真表现能力的发展。

互动环节，金色幼儿园的四位老师根据观察要点和幼儿行为进行交流：观察要点的制定来源于前期对幼儿游戏的观察；观察形式多样、目的性强，既关注幼儿建构技能的推进，又关注了幼儿自评、他评的能力发展。互动中有老师提出：“小中大班幼儿的建构技能运用何种方法推进？”、“不同年龄段的建构专用活动室如何提供材料？”，侯灵杰做了相应的回答：建构技能需要在师生互动中逐步推进，通过问题引导帮助幼儿梳理搭建中的方法，引发同伴经验的迁移；同时根据实际操作现状，有目的地调整材料的投放策略，满足不同年龄段的需求。

研讨环节，组员们就“建构游戏活动室是否需要桌面游戏的区域？”、“小班需要建构游戏活动室吗？”等问题展开了激烈的讨论。最终大家得出的一致答案是：看幼儿的游戏需要，切勿“应试”教育。

本次活动多方位展示了建构游戏的组织与实施过程，学员们收获满满，以“儿童的视角出发”思考各类教育教学问题，不断促进保教工作的时效性！
