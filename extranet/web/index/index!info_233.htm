【学前 个别化学习中心组】经验联结 项目驱动

发布时间 : 2023-10-30   作者（来源）: 刘时医   阅读 : 443

3/8
←
→

2023年10月30日，闵行区教育学院教研中心学前教育研究室刘时医老师带领闵行区个别化学习中心组的老师们在闵行区康城幼儿园开展了现场观摩与主题教研。

现场观摩，真实场景启思考

康城幼儿园陈华平老师结合幼儿园园本特色大活动“尚礼节”以及大班主题活动《我是中国人》，开展了主题活动优化实施“尚礼节展厅布置我做主”。大班幼儿围绕驱动性问题，以“书画之缘”、“探秘‘蓝’文化”、“童‘画’皮影”、“品味茶香”四个小组，在小组讨论、合作实践中探秘、感知中国传统文化的博大精深，激发起幼儿满满的爱国情怀，感受作为中国人的自豪感。在长程的探索中，幼儿生发出各种问题：如“怎样把一口染缸的洞口堵住？”“书画的颜料用植物和水果能制作成功吗？”“怎么样才能让我做的皮影动起来？”孩子们通过收集信息、实验探索、分享交流、再调整计划、验证评价策略，主动解决问题。

交流研讨，答疑解惑促提升

现场观摩后，陈华平老师聚焦 “了不起的儿童”“追随的教师” “给力的家长”，分享了《尚礼节展厅布置我做主——主题活动的优化实施》过程中的收获与困惑，阐释了教师在项目进展的过程中如何基于幼儿的真问题，推进和支持幼儿的深度学习。

中心组的老师们结合对现场活动的观察分析，一致认为园本课程与主题活动的柔活、倾听幼儿对主题活动的兴趣与想法、过程中幼儿自主探究、专注学习的品质等亮点多多，同时也提出了进一步优化的宝贵建议，如：结合园本的礼仪特色，聚焦幼儿学习探索的礼仪小点的渗透，以及多元材料提供进一步激发幼儿深度探索的可能等，从而更好实现每一位幼儿的深度学习。

专业引领，扎根深发助升华

中心组导师刘时医老师从幼儿发展、链接主题核心经验、工具材料的支持等方面对本次活动进行了全面、专业的点评。刘老师对教师在活动组织和实施中的教育观、儿童观、资源观的努力践行给予了充分肯定，并点赞主题推进过程中的家园共育……与此同时，刘老师也从主题核心检验的链接、项目小组的“真合作”等提出了进一步优化的建议，以促进幼儿的全面发展，培育完整儿童。刘老师的点评为主题活动优化实施的推进拓展了新思路。

学无境，思不止，行不尽。本次中心组主题教研活动，立足于实践研讨，扎根于幼儿发展，为老师们后续设计和组织实施主题活动注入了新的思路，也提出了新的挑战！中心组也会将教研成果落实到幼儿的一日活动中，更好地为幼儿园课程建设的高质量发展蓄力赋能。
