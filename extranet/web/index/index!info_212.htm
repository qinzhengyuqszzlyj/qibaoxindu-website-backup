沪滇情深 共谋发展——2020年云南省香格里拉市部分中小学校长、教师赴闵行培训开班仪式

发布时间 : 2020-10-11   作者（来源）: 朱凤   阅读 : 4458

10月10日，2020年云南省香格里拉市部分中小学校长、教师来闵行跟岗培训开班仪式在闵教院附属友爱实验中学举行。区教育学院骨干教师培养与管理办公室主任兰平、闵教院附属友爱实验中学校长黄超群、华师大附属紫竹小学校长张计蕾、香格里拉市部分中小学校长、教师等参加。教育学院校长发展中心俞琳主任主持活动。

兰平解读方案，根据学员的需求，分别从校长和教师的岗位职责制定培训内容，聚焦课程与教学、学校管理等领域，进一步帮助学员牢固树立立德树人根本任务，转变观念、提升认识，不断提升教育领导力与学科教学能力。跟岗学校会提供最好的师资，通过一对一、一对多的带教形式，让学员更深入的了解学校、了解学生、了解上海的教育。建议学员们能跳出教育看教育，整合多方资源，拓宽眼界；改进学习方式，勤于笔耕；在比较中学习，在学习中感悟，在感悟中成长突破，同时注意安全。

互动交流环节，黄超群、张计蕾两位导师表示将带领学校团队，根据学员实际确定研修方案，全面开放各条线工作，尽力为学员提供所需资源。香格里拉市第三中学李正钧校长、金江镇小学杨聪老师作为学员代表发言，感谢闵行教育提供的优质平台，让沪滇两地心相连，情相牵，并表示会珍惜机会，增强学习使命感、严肃纪律，确保学习实效性，将所学因地制宜转化，让“种子”生根发芽。

本次培训近三月，以跟岗培训为主，观摩、考察、交流为辅的培训方式推进，促进学员开阔教育视野、更新教育理念，优化教育行为，提高教师管理能力和执教水平。凝心聚力，共谋发展，共同助力香格里拉打赢教育精准扶贫攻坚战。
