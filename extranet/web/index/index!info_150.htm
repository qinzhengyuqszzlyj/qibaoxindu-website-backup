心有所信 方能行远 ——2021年闵行区教育系统优秀年轻干部挂职锻炼培训班集中培训顺利结业

发布时间 : 2021-12-23   作者（来源）: 曹雅明   阅读 : 4246

3/8
←
→

2021年12月15日至17日，闵行区教育系统优秀年轻干部挂职锻炼培训班集中培训在上海市陈云纪念馆举行。闵行区教育党工委书记恽敏霞、副书记李光华，教育局副局长乔慧芳，党群科科长林火生，教育学院校长发展中心主任俞琳等出席。

本期培训围绕“面向教育现代化、走向现代学校治理、青年干部领导力修养”三大板块，共组织专题报告5个、学员讲堂2期、挂职体会交流3场、结业微论坛3组。七宝中学党委书记、特级校长朱越，实验小学党委副书记、特级校长何学锋，莘松中学党委书记、特级校长张忆，莘庄幼儿园党支部书记、特级园长郁亚妹，颛桥中学校长李卫军、申莘/鑫都小学校长康旻前来指导。

专题报告：拓宽视野，完善管理知识结构

陈云是伟大的无产阶级革命家、政治家，中国社会主义经济建设的开创者和奠基人之一。本次集中培训安排在陈云纪念馆，对年轻干部来说也是一次很好的学习机会。在参观陈云纪念馆后，老馆长徐建平为学员作《庆建党百年，传伟人精神，讲“四史”故事》的专题报告。大家近距离了解和感受一代领袖陈云在70多年的革命生涯中，不忘初心，牢记使命，为共产主义事业奋斗一生的故事。陈云同志“不唯上，不唯书，只为实”“交换，比较，反复”的理想信念，以及求真务实的作风、艰苦朴素的公仆情怀，让学员们深受启发。

恽敏霞书记为学员作《闵行教育的现在与未来》的专题报告。她指出闵行教育将围绕建设高质量教育体系，以教育评价改革为牵引，统筹推进育人方式、办学模式、管理体制、保障机制等一系列改革。21世纪是人才发展和人才竞争的时代，希望年轻干部们要珍惜这样的学习机会，把握国家“十四五”教育改革发展战略，瞄准上海教育现代化2035和“十四五”教育发展目标。更要立足新时代的历史方位明晰责任，坚守教育初心，坚定为党育人、为国育才信念，在推进各项教育改革中攻坚克难、真抓实干，成长为政治过硬、品德高尚、专业精湛、治校有方的学校管理者。

“走向现代学校治理”板块的三场报告深入浅出、操作性强。李卫军、康旻、张忆三位校长分别以《调试内外资源，凝聚教育合力》、《基层学校党组织的作用与作为》、《“技工贸”和“贸工技”之争带给我的启示》为题，聚焦学校干群关系、教学质量、教育资源、学生情况等方面，分享了现代学校治理的智慧；还从党员管理、党费收缴、组织生活等方面，介绍了做好党支部标准化、规范化建设，建立“书记·校长”良好合作关系的经验和方法；让学员们明白在学校发展中找准突破口、建立共同愿景实现“低开高走”，通过教育教学变革的深度实践与研究实现“优质发展”，通过学校氛围和专业引领实现“同向而行”的管理策略。

学员讲堂：智慧分享，锤炼学员思维品质

本期培训组织开展“学员讲堂”两场。闵行中学华佳齐、闵行四中刘叶、紫竹小学沈雯珺、田园外小何敏四位学员，围绕《“双新”“双减”背景下的学校作为》展开第一次学员讲堂，分享了“双新”推进学校内涵发展、指向学科课堂变革的教学研究与实践的有效方法；阐述了“双减”背景下，应该还学生一个喜欢的校园、一个精彩的课堂、一点闲暇的时空，还教育本应该有的样子。华二附初刘蕾、莘松中学花卉、虹鹿幼儿园潘玲、华漕中心幼儿园侯惠林四位挂职干部，围绕《和谐校园建设的途径和方法》展开第二次学员讲堂，分享了指向学校文化、育人环境、管理制度、课程文化、教师队伍等多角度、多维度构建和谐校园的实践与思考。

何学锋校长、朱越校长对第一场学员讲堂作点评。何校长指出“双减”背景下学校的作为是一项系统工程：第一要学习和深思，研究透中央文件的具体内容；第二要梳理和分析，了解学校个体存在什么问题，面临的困难和挑战是什么；第三要整体性地策划和设计，并有序智慧地推进，建立系统思考和策划的思维品质。朱越校长提出，不管处于哪个教育改革阶段，对管理者而言，都要成为一所学校问题的发现者、研究者、解决者，要有敏锐的问题意识和解决问题的目标意识，站在学校和学生的立场，秉着实事求是的态度对待教育，并解决教育过程中出现的问题。张忆校长、郁亚妹园长对第二场学员讲堂作点评。指出在和谐校园建设中要有发现和解决问题意识，坚持眼中有事、心中有人的原则，关注环境育人、队伍建设、制度管理等着眼点，促使教师与学生自主成长、自信发展，进而实现和谐校园。

李光华副书记、乔慧芳副局长参与学员讲堂并作点评讲话。李书记勉励学员要在有经验校园长的办学实践中汲取成长动力，指出和谐校园的核心是人际关系的和谐，希望大家能从思想引领、制度规范、榜样示范三方面营造和谐校园。乔局指出，作为闵行教育未来的学校管理者，要牢记立德树人的教育初心，强化学习并注重消化吸收，然后构建愿景，形成共识。要敬畏规范、彰显个性，教育管理者要做好各个领域里的规范，用实践智慧彰显教育个性。

挂职体会交流：头脑风暴，梳理学校管理智慧

集中培训的两个晚上，中小幼三组分别举行挂职锻炼体会交流活动。中学组9位学员交流了自己在挂职期间的所见所闻所感，在对挂职单位与原工作单位管理风格进行对比分析的基础上，阐明自己的发现和思考；并提出自己工作中的困惑和问题，以及对未来工作的展望。小学组9位青年干部围绕挂职锻炼体会展开了一场特殊的“头脑风暴”，大家站在学校管理者的角度，聚焦校长领导风格、管理思维修炼、干部结构优化等核心词充分交流。8位幼儿园挂职干部聚焦挂职中的人和事，对两个学期的挂职经历作深入反思，并从一些管理案例中抽丝剥茧，梳理管理智慧。

乔慧芳副局长、朱越校长、何学锋校长、郁亚妹园长，以及林火生科长、俞琳主任等参与晚上的小组讨论。领导专家对学员们挂职期间的工作态度，善于观察与思考、总结与感悟，能够用比较与分析来审视自己工作的做法表示肯定；生动分享学校复杂管理问题应对之策的同时，引导青年干部要有稳定的价值取向，要提高站位、保持定力，形成问题和研究意识，在挂职实践中要重塑自我，做到心胸开阔有韧性；并建议大家按照“现状分析、优势不足、最大触动、收获提升”的思考路径进行挂职锻炼总结和凝练，深入挖掘培训的意义和价值，真正做到“学、思、行”相结合。

结业论坛：直面问题，彰显年轻干部实力

视频短片《凝心聚力 笃志前行》拉开了结业仪式的序幕，班长范丽对本次培训活动作总结。在副班长李腾蛟的主持下，全体学员分成中、小、幼三组开展微论坛。幼儿园组的《让管不再被动，浅谈办园管理新视角》，围绕在办园过程中如何处理各方关系，解决棘手问题，做出正确决策，引领学校发展等方面展开。小学组尝试从管理机制建构与学校高质量发展之间的关系，探讨了“机制创新视角下学校的高质量发展”。中学组聚焦正确处理和化解学校管理与发展过程中可能遇到的突发事件，尝试了“学校突发事件应急管理机制的探索”。

恽敏霞书记对三组学员的微论坛表示充分肯定。她称赞学员认真学习的劲头、直面问题的勇气，以及理性思考的火花；并希望学员们将所学所思反复琢磨、内化升华，进一步拓宽视野、提升境界，完善管理知识结构，提高作为学校领导的综合素养，逐渐形成自己的思考和办学实践。

为期三天的集中培训开拓视野、激荡思想，也是学员们团结互助、蓄力储能的三天，在领导和专家的引领下，所有学员都实现了一次从外到内的蜕变，学思践悟、历练成长！我们相信作为闵行教育未来的优秀年轻干部定能不负所托，在深化教育强区的建设中贡献自己的力量。
