面向可持续发展的园所管理 ——记闵行区第四期幼儿园园长（莘庄幼儿园）培训基地结业仪式

发布时间 : 2023-06-21   作者（来源）: 曹雅明   阅读 : 1601

3/8
←
→

2023年6月21日上午，闵行区第四期幼儿园园长（莘庄幼儿园）培训基地结业仪式在闵行区实验幼儿园顺利举行。闵行区教育局副局长王维刚，区教育学院校长发展中心俞琳主任、曹雅明老师，以及莘幼教育集团园长与中层干部共同参与本次活动。

基地学员颛桥一幼园长蔡爽主持结业仪式。整场活动牢牢把握习近平总书记在中共中央政治局第五次集体学习时的讲话精神，围绕教育高质量发展，通过园长论坛、微报告、案例分享等形式，呈现园长学员们在“面向可持续发展的园所管理”中的思与行。

历程回顾，汗水收获成长

短片《共筑幼教梦 扬帆再起航》带领大家一同回顾两年来培训基地的学习经历与建设成效，讲述了学员们在导师郁亚妹园长带领下，以耕耘收获成长，以实干笃定前行，以情怀构筑愿景的培训历程。

园长论坛，抓牢管理核心

园长论坛紧密围绕“高质量发展”这一核心词，对“如何实现‘人’的价值，以推动园所可持续发展”的问题展开了讨论。论坛发起人莘庄幼儿园吴晓君老师和基地学员荷花池世博幼儿园邱雯菁、颛桥一幼蔡爽、上海交通大学闵行幼儿园于莉、启英宝盛幼儿园张雯四名园长围坐畅谈，分享了园所在用人、选人、激励人过程中的故事和引发的思考。

导师赠书，传递美好祝福

论坛后举行了温馨的赠书仪式。导师郁园长将《做有思想的园长，办有理想的幼儿园》的园长基地建设集锦，以及莘庄幼儿园课程领导力成果《珍视对话，让课程经历充满智慧》赠予学员们，希望学员们朝着更美好的高质量办学共同努力。

报告分享，凝练实践智慧

微报告环节，浦江一幼园长朱虹霞、马桥富国幼儿园园长蒋慧莺从园本课程建设、家园共育两方面分享了自己的办学思想和特色。

朱园长以《让幼儿运动沐浴阳光——“阳光运动”特色课程的传承与创新》为题，分享了幼儿园通过“设组织-探路径-强辐射”让特色课程萌发新的“生长点”的创新实践经历。

蒋园长分享了《依托“云上国学苑”课程，构建家园协同共育新格局》报告，从“云上”家长学校的功能定位、课程目标、家长课程内容体系等视角，解读了园所面向高学历家长的家园共育新路径构建之路。

案例讲述，共享管理经验

数字故事《让校园洋溢办学智慧》以莘庄幼儿园的一个班级实例，向大家阐释了导师郁园长在高质量办园中，园长、老师教育主张的反思体悟与智慧举措，她倡导学员们要在办园主张的引领下，为园所每一位孩子与教师的发展不断注入能量。

吴泾一幼郁蓓园长作为一位美术特色园所的新任园长，她在《让教师的行为带有研究的样态》的案例分享中，介绍了她如何以课题研究转变教育形态，小步递进、以点带面突破教育行为，从而转变老师观念，优化园所美术课程建设。

莘松二幼原继恩园长立足老学校、老教师的园所情况，以“直入心灵，从根本改变”为主题进行汇报，围绕“老教师的观念如何转变？”“老教师的工作积极性如何调动？”等问题，与大家分享了老教师们的教育困境和突破策略。

颁发证书，寄语殷殷期望

俞琳主任宣读结业词，祝贺七位园长顺利结业，并表示希望年轻有为的园长们能够笃学不辍、精进不休，带领园所走向更高层次发展，并为学员们颁发了结业证书。

基地导师郁园长感谢教育局与教育学院的信任，作为导师，她告诫学员们“办一所幼儿园易，办一所好幼儿园难”，并指出办好一所幼儿园，园长需要基于规划发展，让学校充满领导力；根植于校园文化的变迁，让校园充满生长力；追求学校办学质量，让管理充满持续力。同时，郁园长呼应习总书记办好人民满意教育的要求，对学员们提出了“三个要求”：一是希望持续凝练办学思想，跟随时代脉搏，有办学敏锐；二是希望创新实践办学举措，关心团队需求，有办学热情；三是希望勤恳努力地关注办学日常，实现质量倍增，有办学定力。“天下难事必作于易，天下大事必作于细”，未来持续为闵行教育高质量发展贡献自己的力量。

领导讲话，领航未来方向

王维刚副局长感谢基地导师、校长发展中心在繁重的工作中和疫情特殊背景下依然把培训工作做到位，同时肯定学员们在办园过程中的付出和努力。他指出，园长是学前教育的财富、核心和中流砥柱。在日常管理中，一要认真思考，有自己的教育理想与办园目标，带动园所团队迈向优质；二要身先士卒，带领园所形成一种良好的风气；三要深入研究，关注工作细节，理解术与道之间的关系。并希望学员和参会园长们不断努力，做一个有使命感的平凡人，更好地服务于幼儿，服务于闵行教育的质量提升。


虽然培训接近尾声，但学员们定将坚守教育初心，为了每一名幼儿的全面发展匠心筑梦，为了高质量、可持续发展的闵行教育扬帆再起航！
