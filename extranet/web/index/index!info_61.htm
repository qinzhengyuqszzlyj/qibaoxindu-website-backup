逐梦新时代 引领新征程 助推高质量 ——闵行区第六期中学校长（莘松中学）培训基地结业仪式

发布时间 : 2023-06-28   作者（来源）: 曹雅明   阅读 : 2394

3/8
←
→

2023年6月27日下午，闵行区第六期中学校长（莘松中学）培训基地结业仪式在莘松中学春申校区举行。闵行区教育党工委书记恽敏霞、区教育局党办程思嘉、区教育学院校长发展中心主任俞琳、校长发展中心曹雅明出席，进才北校校长、上海教育学会初专委主任金卫东作现场点评，并作精彩报告。五位基地学员、闵行区新任校长、挂职副校长以及莘松中学管理团队部分成员参加了本次活动。活动由基地导师、上海市特级校长、莘松中学校长张忆主持。

活动围绕《习近平总书记在中共中央政治局第五次集体学习关于教育强国建设的重要讲话》的精神，通过学员汇报交流、专家点评的形式，呈现学员们在“探索建设教育高质量发展的路径”中的思与行。

活动伊始，张忆校长首先介绍了第六期校长培养基地（莘松中学）的培训课程、培养路径等情况，简要回顾了两年来和学员们一起走过的培训学习历程。

学员汇报

祝庆校长以《融通、重构：创建校园德育新生态》作主题汇报，首先回顾了自己在基地的学习经历，然后从对教育新空间的再思考、优化学校德育生态的实践、对“知、行、思”的价值思考等方面进行了分享。

李波书记以《课题驱动管理变革，工作重塑助推学校高质量发展》为主题，着重从“精准把脉，明晰高质量发展目标”“工作重塑，探索高质量发展路径”“打造品牌，引领高质量发展方向”等方面分享了学校近年来的变革实践与初步成效。

李卫军校长从创建自己的办学思想为切入点，做了《以正向教育办学思想推动学校高质量发展》主题汇报，从正向教育办学思想溯源、正向教育办学思想的内涵阐释等方面进行了详细的阐述。

王治华校长做了《党建引领 探索“大思政”教育新高地》的主题汇报，从“党建引领“大思政”方向，构建‘1+3+5’育人体系”“党建加强“大思政”培训，激发教师使命担当”“党建抓好‘大思政’教学，落实课堂立德树人”“党建助推‘大思政’品牌，创新育人发展新格局”等方面分享了本校的实践探索。


金虹校长基于校情，以《以师资强校 以质量兴校》为主题进行汇报，从“践行三大工程，统一思想认识”“教师培养重举措，梯队建设有路径”“着眼多维评价，激励教师发展”等方面分享了本校的探索与实践。

专家点评

金卫东校长首先指出了学员们值得学习的地方：一、均展现有良好的职业精神和专业素养；二、均聚焦在高质量发展的目标指向和教师队伍建设的具体抓手；三、均基于学校实际，基于问题导向，基于发展谋划，开展或项目或课题的实践研究，有思考、有实践、有提炼。同时，金校长提出了几个商榷的问题：一是宏观政策与校本思行问题；二是学校导引与教师内化问题；三是培训汇报与成果发表问题。

随后，金校长与大家分享了自己的观点，作为若干共勉：一要厘清大政方针引领下的教育哲学；二要把握师资队伍建设中的未来走向；三要研析基于校本实践上的理实逻辑；四要悟透校长专业成长域的路径选择。

领导讲话

恽敏霞书记充分肯定了莘松中学基地为区域教育做出的贡献，并给学员们提出了希望与鼓励：一是区域会为每一位校长提供更多学习和更高层次的带教平台，鼓励更多的初中校长成为杰出人才，而且能够形成自己的教育思想；二是希望大家在工作中要有换位思考的意识和专业管理的方法，要有一个非常清晰的角色意识，理清各个角色所承担的不同任务，能与各部门携手并进；三是在今后的工作中立足于本职工作，锐意进取、积累沉淀，共同促进基础教育高质量发展。

颁发证书

导师祝贺五位学员顺利结业，并邀请恽书记和金校长共同为学员们颁发结业证书。

基地培训虽然告一段落，但这也意味着各自在成长过程中又站在了一个新的起点，相信学员们定将坚守教育初心，以孜孜不倦的精神，引领学校发展，办优质的教育，为建设教育强区、教育强国贡献力量。
