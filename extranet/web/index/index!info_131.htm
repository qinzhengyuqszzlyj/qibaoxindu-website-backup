不忘初心担使命 “蝶魅”校园展风采 ——闵行区第六期小学校长基地（田园外小）研修活动在明强二小开展

发布时间 : 2022-11-10   作者（来源）: 曹雅明   阅读 : 978

3/8
←
→

2022年11月9日下午，闵行区第六期小学校长基地（田园外小）研修活动在明强二小开展。本次活动是该校长基地研修活动的第一站，邀请了上海市教育委员会教学研究室党总支书记纪明泽莅临参加。闵行区督导室责任督学阚爱华、闵行区第六期小学校长基地研修班的学员们全程参与此次活动。上海市特级校长、闵行区田园外国语小学校长兼书记、基地主持人赵瑛群主持活动。

活动前，在郭芳校长的带领下，研修班的校长们参观了校园环境，从蝴蝶长廊的标本参观，到田园特色的蝴蝶园观赏，再到充满蝶韵的环境浸润，研修班的老师们感受了明强二小浓郁的蝶文化办学特色。

下午1:00，活动正式开始。赵瑛群校长代表基地热烈欢迎导师纪明泽书记的莅临指导，也表达了希望在专家指引下提升校长学员办学水平和办学思想的期盼。

随后，基地学员郭芳校长向与会嘉宾汇报了明强二小办学实践之路。郭校长先从办学规模、办学特色、文化建设、教师队伍、办学成绩等方面整体介绍了学校办学情况，呈现了学校近几年来取得的新变化与新成果。随后，她又从确立学校文化生长点、明晰管理变革延伸点、挖掘项目发展融合点、找准专业发展突破点四个方面详细介绍了学校在办学文化凝练、课堂教学研究、特色课程开发、教师梯队发展、学生成长实践、教育政策落实、线上教学管理等方面的探索与实践，展现出学校蓬勃发展的良好态势。

赵瑛群校长对郭芳校长的汇报进行了点评。她认为明强二小十年来发展取得的丰硕成果离不开郭校团队的艰辛付出与精诚合作；五育融合下的“五蝶文化”的开发也颇具特色。学校运用信息技术打造的智慧学习空间，也带动学生主动创造资源，实现了学习资源的交流与互动。

基地学员们也分别进行了交流。鹤北小学蒋红梅校长从学校转型角度，表达了对郭校长给明强二小带来巨大变化的敬佩。浦江二小赵美华书记从校长影响力角度，阐述了一所好学校离不开一位好校长，一个好学校也会创造一位好校长的观点，肯定了“新基础”教育成事成人的逻辑思维在二小发展方方面面的渗透；昆阳路小学夏权书记从校长办学思想的角度，围绕郭校汇报的四个“点”和四个“新”，表达了自己的收获和感想，他表示从二小办学特色和办学成绩中可以看出郭校长作为一名“新基础”教育人的办学追求和扎根实践。

最后，基地导师纪明泽书记进行了专家点评。纪书记首先肯定了校长基地相互学习、智慧共享的研修机制，让校长学员们凝聚力量，共同发展。随后纪书记针对郭芳校长的汇报内容，结合当前教育改革政策要求，对基地学员进行了精准培训与专业指导。他肯定了郭芳校长是一位有思想、有思路、有办法、有追求、有品位、有责任的校长，能够站在文化的角度治理学校，实现以德育人，以文化人的办学目标；能以专业的领导能力带领团队在短时间内取得飞速发展；接着又从教师梯队建设、项目团队研究、学校环境生态、课程开发与改革、校长专业发展等多个方面对学员进行了细致入微的专业培训，以幽默风趣的语言让在场所有学员醍醐灌顶。

活动最后，赵瑛群校长对纪明泽书记的精心指导和精彩报告表示感谢，同时也督促每位学员将今日所学带回学校，站在新时代的起点上，踔厉奋发，攻坚克难，努力办好人民满意的教育。
