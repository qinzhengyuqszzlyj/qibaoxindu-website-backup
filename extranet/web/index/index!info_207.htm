运用class量表 促进师幼互动 ——“幼儿发展优先”理念下低结构活动的实践与研讨

发布时间 : 2023-06-01   作者（来源）: 顾莹   阅读 : 1048

3/8
←
→

5月31日上午，以“运用class量表 促进师幼互动”为主题的“幼儿发展优先”理念下低结构活动的实践与研讨在上师闵行幼三楼多功能厅举行。闵行区教育学院教研中心学前教育研究室教研员顾莹老师以及中片幼儿园的骨干教师们共同参与了本次活动。

现场观摩 走进游戏

户外场地，全体教师观摩了中四班小朋友的建构游戏现场。孩子们通过自主的选择材料、自由结伴、运用创造想象和思维进行有目的地设计搭建。游戏中孩子们动手动脑，结合自己已有的生活经验进行创意构思、巧妙搭建，并通过对不同建构材料进行观察、操作，运用延伸、叠高、架空、对称、中心点支撑等方法进行搭建，孩子们专注、认真地沉浸在游戏中，共同领略建筑的美。

接着，老师们观摩了中四班王老师的分享交流。王老师基于游戏观察，抓住游戏中孩子们的闪光点，引发了孩子们积极的分享与交流，观摩老师对孩子们在分享中的问题讨论、大胆想象、提出建议以及王老师在分享中给予孩子的情感支持和适时捕捉有价值点的提问等方面给予了充分的肯定。

反思研讨 智慧分享

游戏现场观摩后，全体教师在三楼多功能厅倾听完中四班王老师的现场反思并观摩了一场以“基于class量表 提升师幼互动质量做法”为主题的微教研活动。老师们结合现场分享活动，围绕class量表中的情感支持和教育支持对师幼互动进行了深入的研讨，大家在互动中质疑解惑、集思广益，不断反思优化，做到智慧共享。

通过整个研讨的思维碰撞，老师们对于让幼儿主体性在建构游戏中得到最大的表现，游戏中的自主性得到最大的激发，游戏中情感需求得到最大的支持，完成一个逐渐深入逐渐递进的探索过程都有了深层次的思考。

互动对话 专家点评

教研员顾莹老师组织观摩老师们与执教王老师进行了面对面的对话，老师们表达了观摩后的收获和优化建议，肯定了孩子们的建构水平、协商合作能力。同时，老师们也对分享交流中师幼互动、教师追问以及价值点的捕捉提出了优化建议。

最后，顾莹老师对游戏现场进行了深入精准的点评，肯定了游戏中幼儿发展优先理念的体现，同时对幼儿园在提升师幼互动质量方面加入class量表这一积极尝试表示鼓励，希望老师们能不忘初心，关注到每一位孩子，给予每一位孩子机会，把尊重和欣赏落实到每时每刻，看到孩子们的闪光点，让孩子们在开放、多元、真实的活动情景中获得感性体验，教师要提供支持促进幼儿的深度学习，让幼儿成为游戏的主人，建构更大的梦，推动“幼儿发展优先”的理念更好地践行和落地。
