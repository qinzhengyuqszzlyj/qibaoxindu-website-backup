聚焦劳动课程方案 推进精准教研实施 ——劳动学科新学期教研活动在闵教院附属友爱实验中学开展

发布时间 : 2023-09-08   作者（来源）: 刘至宁   阅读 : 822

2023年9月7日下午，闵行区中小学劳动技术/通用技术学科新学期教研活动在闵行区教院附属友爱实验中学举行。本次教研主题是“双新背景下劳动素养的培育”，活动由闵行区教育学院劳技学科教研员刘至宁老师主持，全区近100名中小学劳技教师参加了教研。活动主要内容分为四个环节，一是上学期学科教研工作小结，二是新学期学科教研工作安排与布置，三是各类获奖表彰及颁奖仪式，四是中心组短会。

会议开始，刘至宁老师表扬并感谢各位老师能主动积极地参加本次教研，表示能感受到老师们对本次教研地足够重视。刘老师首先总结回顾了上学期劳动学科的教研情况及取得获奖成果，特别是在课堂教学、教学基本功、教学类课题已及高中通用技竞赛的做的成果汇总，肯定了相关老师们的努力和付出，期待新学年大家的积极努力一定会有收获和成长。在区学科平台搭建上，刘老师还展示了2023年新春劳动获奖展评的获奖情况，期待每所学校老师们都能积极鼓励学生关注自己的劳动作品并能积极参评，希望新学年能有更多的优质作品获奖。

接着，刘老师带领大家回顾了2022学年劳技\通用技术的教研活动的掠影与教研情况：有新学集中教研、七宝中学市级课题研讨、实验西校项目化学习公开课研讨、诸翟学校和浦江三小区公开课研讨、各学段教研坊活动、浦江片区活动、各项目研讨花絮以及暑期线上活动，为广大劳技教师展现了丰富的学科教研态势和良好的教研氛围。

总结师为了未来更好的前行。刘至宁老师在新学期工作计划中，结合市级学科教研工作，对新学期的工作确定了目标，重点推进数字化转型教育，要求各校继续完善“一校一案”劳动课程实施方案，做到打造方案、精准教研，完善劳动课程。重点解释了2024学年计划实施义教学校劳动课程实施方案中各年级的课程实施标准。介绍了数字化转型资源建设团队人员，成立上海市“三个助手”资源平台建设项目组，旨在搭建2024年上海市劳动课程资源平台，72项公共项目上传，便于每一位劳动教师取用。其次，劳动技术课程要努力做到精准教学，技术赋能，深化主题教研。对本学期各项教学评选活动、区级展示活动、学科研究和即将迎来的高中通用技术学科竞赛、高二选必课程的“空中课堂”拍摄的相关工作、义务教育劳动课程案例研究、学科调研、论文评选、区级公开课申报情况、各学段教研坊活动安排等做了部署，为了激活学科教研共同体自主教研，促进教师专业发展，确定了本学年各学段学科中心组名单及各学段教研坊负责人，期待新一届学科团队有更大的作为。参加活动的老师认真聆听，在学习中所获颇丰。会上，还公布了新一届中心组成员名单，发放了新一届中心组成员聘书并合影留念。

会上进行了颁奖仪式，各学段各类获奖教师代表予以了表彰并进行了合影留念。对被选进老师《共同体项目研究》新书中的案例、单元设计等作者进行了赠书仪式，感谢她们的积极参与和辛苦付出。与会后个别教师还与教研员进行了亲切互动。

本次活动在充满生机活力的教研氛围中圆满结束，讲座使广大教师能够更好的把握学科发展方向，劳技学科工作总结未来展望都使得广大教师在鼓励与鞭策中不断思考、不断钻研，在学科团队的互帮互助中，踔厉前行，不断创新。

 

 

闵行区教育学院附属友爱实验中学  邓珊珊（报道）

2023年9月8日
