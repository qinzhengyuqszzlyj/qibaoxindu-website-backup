深思互融 对标共进——2021 年闵行区教育系统优秀年轻干部挂职培训班主题论坛活动纪实

发布时间 : 2021-05-26   作者（来源）: 曹雅明   阅读 : 1147

3/8
←
→

为进一步激励年轻干部比学赶超、担当作为、砥砺品质，近期，闵行区教育系统优秀年轻干部挂职培训班分别在七宝外国语小学、七宝中学举行了分组主题论坛。闵行区教育局党工委书记朱雪平、区教育局党群科科长林火生、区教育学院校长发展中心主任俞琳出席，七宝中学校长朱越、实验小学校长何学锋、七宝中心幼儿园园长丁春燕应邀为本次论坛做点评。

教育评价事关教育发展方向，有什么样的评价指挥棒，就有什么样的办学导向。为深入贯彻落实习近平总书记关于教育的重要论述和全国教育大会精神，完善立德树人体制机制，扭转不科学的教育评价导向，提高教育治理能力和水平，26名优秀年轻干部分成学前、小学、中学三个小组，就“教育评价改革背景下学校的作为”展开研讨，围绕各自学段特点、立足学校工作实际各抒己见。

 幼儿园组

幼儿园组8位学员围绕教育评价改革背景下园所的“所为和能为”，“变与不变”“成长和超越”“推进幼有善育”等角度阐述了自己观点。专家导师丁春燕园长进行了点评，肯定了八位挂职干部对教育评价改革核心问题的领悟，鼓励大家跳出个人的条线工作，站在更高的视野上看问题，由点及面，寻找差距，迎难而上。 

小学组

小学组9位青年干部聚焦“五育并举”、师生成长等方面，探讨了教育评价与提升教师与人能力、赋能学生成长的内在联系，在介绍学校的经验和做法的基础上又进一步谈了自己的思考和体会。何学锋校长作点评，重点对如何进行发言表达进行指导。他指出：作为学员，首先要学会对会议性质进行判断，然后展开对论坛主题的解读，还要考虑发言的站位以及倾听的对象，再确定发言的思路和内容。发言时尽量聚焦一个问题，并讲清为什么聚焦这个问题？还要分析这个问题在区域以及学校的现实情况如何？然后再清晰地表达自己的思考和观点，最后也可以提出自己的困惑和思考。学员们受益匪浅。

 中学组

中学组9位青年干部从提出工作中的问题和困惑为出发点，围绕为什么评价、评价什么等问题，立足学生发展，从学校机制体制、教育教学、师德修养等方面阐述了评价改革如何助推学校治理能力的提升。在点评环节，朱越校长肯定了各位干部的实践与思考，并提出了几点建议，她认为后备干部正处于学科专业的发展期，也处于行政治理能力的成长期，建议大家要立足本职专业，基于学科的母体思维逐步养成自身的管理思维；立足本职岗位存在的真问题去做研究；除了做好本职岗位外，还要向更高层面学习。 

朱雪平书记全程参与了三个小组的分组研讨，朱书记指出此次挂职培训的出发点在于丰富大家的经历，从而获得思想的转变，他希望各位年轻干部要提高站位，找到改革的根源；强化问题意识，学会追问不同问题；提升领导素养，积极学习学校治理。并就挂职过程及未来工作中将要遇到的难点进行了具体的解读与指导。
