拾趣“传统文化”，让户外表演更加有“戏” ——“童蒙养正，立德树人”大班幼儿德育实践研讨活动

发布时间 : 2023-11-11   作者（来源）: 顾莹   阅读 : 640

3/8
←
→

 

“扣好人生的第一粒纽扣”是习近平总书记对广大青少年儿童的殷切希望。只有德“立”住了，人才能“树”起来。

2023年11月7日上午，闵行区安宁路幼儿园开展了“童蒙养正，立德树人”大班幼儿德育实践研讨活动。区教育学院教研员顾莹老师、安宁路幼儿园杨园长以及南片姐妹园的老师们共同参与了此次活动。

一、源于近期的、感兴趣的游戏内容，充分尊重并进行专业判断

游戏前期，基于孩子的近期热点，围绕大班“爱祖国、爱家乡”与孩子们共同收集游戏素材并协商投票。有“七宝皮影戏”、“游上海”、“中华舞龙”、“玩转石库门”等。

教师深入分析、充分解读，对其内容进行筛选。如“中华舞龙”与“皮影戏”是中华民族的传统非遗文化，在表演“龙”和操作“皮影”的过程中感受中华文明的创造力从而对自己是中国人感到骄傲。

基于孩子在听、念“上海话”童谣的煽动下，开启“侬好，上海之旅”打卡活动。通过吃吃、看看、逛逛等使孩子对上海小吃、景点、建筑有所了解。“东方明珠”、“迪士尼”成为了孩子们心中代表性较强的上海地标。户外场地给到孩子更大的创造空间，激发了孩子们对家乡的归属感。

教师在充分尊重幼儿想法的同时进行了有效的价值判断，最终形成了“娃娃游上海”、“中华舞龙”、“皮影戏-老鼠嫁女”的户外表演游戏内容。

二、支持持续的、创造性的表演过程，充分享受户外游戏的快乐

幼儿对大自然有着天然的亲近感，对自然界充满好奇，幼儿在户外尽情的感受风、感受太阳......从而激发孩子们的创造力和想象力。

高低的山坡、弯曲的迷宫树丛让“中华龙”的造型不断发生变化；充满乐趣的迪士尼花车巡演等，让更多同伴看到自己的表演，为孩子创造了很大的舞台展现自己。户外“皮影戏”利用在光影的变化、得天独厚的户外场景，激发了游戏新的情节、新的对话和新的表达方式。

孩子们充分体验到了户外表演游戏带来的快乐。

三、推进“知”“情”“行”阶段发展，充分内化传统文化的内涵

如何通过户外表演游戏内化情感？安幼课程组团队开启了沉浸式微教研《中华优秀传统文化在表演游戏中的内容选择和支持性策略》来推进从“知”到“情”最后形成“行”的过程发展。通过“趣味辩论”、“围炉解析”“多维度分析”提炼适用于大班幼儿表演游戏的支持性策略。如：区域互动策略、自然渗透策略、思维表征策略、多媒体交互策略、“3+1”分享策略等支持幼儿在户外表演游戏中主动挖掘，从而进一步萌发爱国、爱家乡的情感。

四、专业的，深度的互动评析，充分提升户外表演游戏的实施

 活动尾声，顾莹老师分别就孩子的游戏以及现场微教研给予了高度的评价，如：孩子了展现自信、愉悦、松弛、从容的游戏状态；教研内容深度剖析有理有据。同时提出了指导见解：“给予同频共振五感体验的机会”、“注重细节关注品质发展”等帮助老师指明了后期实施的方向。

拾趣“传统文化”，让户外表演更加有“戏”。相信大家在经后的工作中能孕育更强大的课程意识；定期复盘，发现孩子、看见自己。把“为党育人”的信念根植心中！
