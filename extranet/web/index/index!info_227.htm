资源为先 小课本拓展大教材 ——区小学道德与法治教研暨智慧教育研讨活动在闵行区田园外语实验小学举行

发布时间 : 2023-10-29   作者（来源）: 王玉兰   阅读 : 585

3/8
←
→


2023年10月26日下午，小学道德与法治课堂“大资源的开发与应用实践”主题教研系列活动之“资源为先 小课本拓展大教材”闵行区小学三年级道德与法治学科研讨活动在上海市闵行区田园外语实验小学金都校区举行。

本次活动中，闵行区教育学院道德与法治教研员王玉兰老师莅临指导，田园外语实验小学赵美华书记、道德与法治学科教导何艳丽、闵行区三年级道德与法治老师以及来自云南、黑龙江的骨干教师代表一百多人共同参与。本次活动由区中心组成员北桥中心小学姜英老师主持。

一、课例展示，多维呈现

朱桢初老师执教的是三年级上册第二单元第6课《让我们的学校更美好》中的第二课时《我为学校出点力》。课前，朱老师通过智慧笔收集学生数据，进行学情分析，了解学生想从哪几个方面来为学校出点力，并据此进行分组，并由学校自编歌曲《颛桥》导入课堂。课中，结合校情学情，通过小组合作探究的形式，引导学生发现学校有待改进的地方，并提出1-3个自己的看法或切实可行的建议，完成行动方案，交流分享改进意见，为学校建设出谋划策。与此同时，“行走的思政课”光华研学方式，将地区文化资源的传承与发展同学校文化学习与生活相融合，激发学生自主建设校园的热情，为更美好的校园献计献策，将关心集体落实到实际行动中，培养校园小主人的责任意识。

二、论坛分享，智慧齐聚

本次论坛的主题为“资源为先，小课本拓展大教材”，由田园外小郑榆川老师、朱桢初老师和北桥小学姜英老师共同参与。

首先，由郑老师进行“活动园地展示”部分——“我们的学校生活 智慧美校园”的分享。此次分享以“精心设计 巧用资源 智慧评价”为主题，从单元内容设计、单元活动实施、单元活动评价和单元活动经验这四个角度展开阐述。明确了“道德修养”“法治观念”和“责任意识”为本单元的三大核心素养。依次以“童手绘校园”、“童心话师恩”、“童慧支妙招”三个系列活动进行串联。

接着，朱桢初老师从本课的目标制定；如何在教学中提升学生责任意识和道德修养；以及渗透地域文化资源的教学策略这三个方面进行了智慧分享。学生作为校园的小主人，走进校园的各个场所，实地探访调查学校里有待改进之处，培养了他们热爱集体、积极参与集体活动的意识。与此同时，合作学习的学习方式贯穿了整个系列活动，学生一起挖掘问题，完善调查报告并落实行动，从而提升个人道德修养。智慧笔的运用，使得在课堂上能够更为便捷地采集学生数据，并实时呈现结果，从而更好地把握学情。紧接着，朱老师还提出构建道法课堂厚植中国“根”的教学关键在于渗透地域文化的教学策略，采用“以情生根，以心培根，以魂伸根”三种方法，使以地域文化立意的道法课堂达到“有情、有味、有魂”三大效应，向学生传递源源不断的正能量，促进完成“立德树人”的新时代教学任务。

其后，姜英老师在交流中总结了如何在思政课程中以“资源为先，小课本拓展大教材”的实施途径：思政课程中应以资源为依托，充分开发本土资源，包括教材、社会资源和网络资源，以拓展教学内容，增加教学活力，提高思政课堂的效益；同时，思政课程应与现实生活紧密结合，使课堂内容更加生动和富有启发性。通过具体案例和资源类别的介绍，她展示了如何充分运用这些资源来支持思政课程，促进学生的思想政治素养，并强调了红色资源的重要性，用以引导学生明确人生方向，培养精神成长。

三、教师互动，集思广益

观课后，云南和黑龙江的骨干教师代表纷纷发表了自己的独到见解。来自云南昭通市的研修团队代表教师表示朱老师的课堂别具特色：“智慧笔”和“闵智学堂”在本节课中发挥了技术赋能教育的功能；此外，朱老师立足“政治认同、道德修养和责任意识”三个核心素养，挖掘学生身边资源，推动他们爱校园、爱家园、爱祖国的情感提升。

来自黑龙江的代表老师肯定了本堂课的三个亮点：一是以学生为本设计教学活动。二是对教材资源进行合理的拓展。三是关注道德与法治学科的生活性与核心素养的培养。并且表示本次活动对于后续自身开展教研活动提供了参考，受益匪浅。

四、专家点评，启思引路

闵行区教育学院道德与法治教研员王玉兰老师对本次教研活动做专家点评。她认为朱老师所呈现的是一堂素养导向下的课，让我们看到了真实而复杂的教学情景：学生经历了科学的调查、取证、协商、判断、决议，这是一个作为校园小主人开展民主管理的复杂过程，同时又与学生日常生活中的真实经历紧紧勾连。

接着，王老师结合本堂课，就如何设计一堂素养导向下的道德与法治课堂这个问题，为在座老师进行了指导。王老师强调在进行教学设计时要关注正确的价值导向，明确要培养学生的关键能力是什么，以及希望学生拥有怎样的必备品格。她指出，要让学生在综合实践活动中获得学科涵养。如本堂课中，朱老师从学生的数据入手，分组讨论，在讨论和互动中，进行了有价值的追问和组际的补充。在此过程中，培养了学生的批判性思维。教师应用一种对比的思维来观察课堂和学生的成长变化。经过本课的学习，孩子最大收获在于通过有效、规范、民主的渠道充分发挥了小主人的作用，在他们的心中播下了一颗议政、参政的种子，培养了学生的使命感和责任心。

思政教育始于心，落于行。通过此次教研活动，老师们在分享中进行思维碰撞，在党的二十大和习近平新时代中国特色社会主义思想的精神引领下，在闵行区道德与法治教师将在区教研员王玉兰老师高屋建瓴的指导下，大家对于道法课程的提质增效都有了新的收获与感悟。同时，通过此次教研也与来自黑龙江和云南的骨干教师们建立了深厚的友谊，携手共探智慧教育背景下的精准教学。教而不研则浅，研而不教则空。随着教改的不断深入，教师们将不断注重学习，博采众长，多一份智慧，多一份创新！
