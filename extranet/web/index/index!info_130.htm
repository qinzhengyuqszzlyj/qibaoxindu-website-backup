双减背景下的课堂教学改进——数字化赋能物理课堂 ——闵行区初中物理南片教研活动

发布时间 : 2023-10-15   作者（来源）: 高剑英   阅读 : 535

3/8
←
→

2023年10月11日下午，以“双减背景下的课堂教学改进——数字化赋能物理课堂”为主题的闵行区初中物理教研活动在上海世外教育附属浦江外国语学校顺利举行。本次活动闵行区南片初中物理教师出席，由华东师范大学第二附属中学附属初级中学的陈凤玲老师主持。

听课环节中，浦江世外的王佳盛老师和浦江二中的张露老师应用智慧笔助力精彩课堂，改进课堂教学策略，提升物理课堂的教学效果。王老师的复习课《光的反射和折射》课程设计精炼而深入，她通过生活化的案例和情境，激发了学生们对光学知识的兴趣和思考。张老师以情景创设引入《大气压强3》的教学，课堂气氛亲切自然，借助直观的实验，帮助学生们理解大气压强的相关知识，并通过丰富的实例，引导学生利用所学知识对生活中的现象做出解释。

两节课后，上海世外教育附属浦江外国语学校的孙校长发表讲话。孙校长强调了教学研究的重要性，并鼓励教师们积极参与，共同提高。陈凤玲老师随后介绍两位了上课老师，并为她们整个过程中的的用心准备和最终课堂的精彩呈现点赞。

评课环节，老师们聚焦数字化赋能物理课堂，着重点评两节研讨课的目标达成情况。文绮中学的陶醉老师认为两位老师的教学设计各有特色：王老师通过巧妙例题设计，深入浅出地帮助学生梳理光的反射和折射的知识脉络图；张老师则是借助实验，帮助学生逐步理解大气压强的影响因素。浦江三中的陈齐斌指出老师都能灵活运用智慧笔，及时的发现学生的问题，及时跟进并高效的利用生成资源。最后莘松中学的刘忠伟老师《单元视角下的物理核心素养教学》主题评课，两位老师单元知识结构化，并通过生活情境引入，进行安全教育，用数字化及时的评价帮助学生更好的学习。总之两位老师为我们展示了两节师生互动精彩、课堂内容丰富、逻辑性强的物理课，让人意犹未尽。

活动的最后，陈凤玲老师对本次活动做了总结，并提出这场教研活动不仅为教师们提供了一个观摩、讨论和学习的平台，也为他们提供了一个深入理解双减政策、提升课堂教学效果的机会。教研，目光在远方，在未来；教研，脚步在当下，在现在。
