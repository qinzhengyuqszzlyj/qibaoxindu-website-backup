庆丰收做美食 爱劳动享健康 ——闵行区小学劳动课程教学展示活动

发布时间 : 2023-11-15   作者（来源）: 刘至宁   阅读 : 796

前言

近年来，闵行区实验小学挖掘资源、创建劳动教育的条件，以《义务教育劳动教育课程标准》为依据，以“能自理，会服务，创造幸福生活”为学校劳动教育基本目标引领下，积极开发和完善劳动教育课程，开设以“蒙娃小农庄”为特色的劳动教育项目。

活动报道：

2023年11月14日下午，我校承担了主题为“庆丰收做美食  爱劳动享健康”区级小学劳动课程教学展示活动，畹町校区盛黎青、郭鑫两位老师以“双师型”教学方式执教了一堂别开生面的劳动课——《南瓜“变变变”》，活动由闵行区劳技教研员刘至宁老师主持，嘉定区劳技教研员严海平、教育局普教一科分管劳动教育的张美琴老师莅临观摩指导，全区小学劳技教师共同参与。

本次劳动课程为畹町校区《豌豆农庄》系列课程的一部分。活动伊始，学校通过一个课程视频向大家介绍了《豌豆农庄》课程的开发与实施情况，学校为“豌豆农庄”开辟了专门的场地，并开设了综合实践课程“趣农庄”，主要包含“一方水土养一方人”、“花草的神秘力量”和“花园创意多”三大板块课程内容，引导学生接触大自然，探究科学，学习劳动本领，培养学生热爱自然、热爱生活的品质。

接着，盛黎青和郭鑫两位老师共同执教了《南瓜“变变变”》一课，该课属于“一方水土养一方人”的教学内容，是以学校小小农艺师们的种植成果为食材，完成一次以“南瓜”为主原料的美食制作。课堂开始时，教师们先引导学生利用平板电脑查找食品学习资料，以小组合作的方式，观看和阅读推送的资源包，学习南瓜食品多种加工方法。过程中，学生自主选择食材和工具，合理分工，在其乐融融的气氛中齐心协力地完成了以“南瓜”为主原料的多种美食，有的蒸了红糖南瓜馅饼，有的做了南瓜松饼，还有的做了奶香紫薯南瓜团等。

课堂结尾时，同学们还对自己的作品进行摆盘和装饰，并向其他小组同学进行了美食介绍。

最后，闵行区劳技教研员刘至宁、嘉定区劳技教研员严海平和闵行区教育张美琴老师分别对本节课进行了点评，刘老师从三个方面谈了自己的感受：一是课程的延续性和内容的深化，将长时课程和短时课程相结合，从春种秋收到烹饪与营养，再到健康与关爱，并与“粮博会”和社区服务活动对接；不仅体现了生产活动、生活劳动和服务性劳动“三大”劳动课程内容，而且体现了劳动课程的“出力流汗”、“动手实践”、“劳动创造美好生活”等劳动课程理念，是学校自主构建特色劳动课程的典型案例；二是课程场地的开放性和多样性，特别是关注食品安全和安全使用工具，劳动场域设置在小食堂，分为学习区、料理区和烹饪区，科学合理；三是课程充分体现了学生的主体性，给学生充足的动手实践时间，教师给予精准的指导与服务，使劳动效率提高，作品完成度高。中心组成员金汇实验学校王隽作为代表老师进行了评课发言，他认为从今天的劳动课程教学来看，学校真正得为师生创设劳动资源，才能有这样的教学呈现。

针对本节课，严老师提炼出了三个“真”：真实的劳动情境、真实的劳动体验和真实的劳动过程。接着严老师的话题，张老师补充了另外两个“真”：真心喜欢和真心做事。几位专家一致肯定了本节课能有效落实“五育”并举，促进学生劳动素养的发展，同时也希望实小的劳动课程在《新课标》的指引下，迈向更大的“空间田野”，结出累累“劳动硕果”。
