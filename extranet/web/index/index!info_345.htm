核心素养培养导向的教学实践——九年级第二次数学教研活动

发布时间 : 2023-11-17   作者（来源）: 张胤睿   阅读 : 778

3/8
←
→

2023年10月18日，闵行区初中数学九年级第二次教研活动在七宝中学附属金都实验中学顺利举行。本次区教研活动的主题是《核心素养培养导向的教学实践研究》。200余位专家和老师齐聚我校二楼报告厅，认真观摩两位老师带来的精彩的复习课教学，分别是七宝金都的刘冬梅老师执教的《相似三角形章节复习》与上师三附中的蒋赢利老师执教的《相似三角形中的分类讨论专题复习课》。

刘冬梅老师的《相似三角形章节复习》一课，通过对简单图形进行添加条件梳理相似三角形判定的知识。紧接着，通过三次添线，逐渐将图形变为复杂图形，从复杂图形中分析基本图形，寻找相似三角形进行定性分析，从而巩固了相似三角形的判定。整节课堂，刘老师由简单图形到复杂图形，由定性分析到定量计算，由空间形式到数量关系，由给定数值进行计算到给定字母建立函数模型，遵循学生的认知规律，充分发挥教师的引导作用，调动学生的学习积极性。

蒋赢利老师执教的《相似三角形中的分类讨论专题复习课》主要针对问题中的不确定因素培养学生的分类讨论思想。首先，蒋老师由一道开放的作图题激发学生专题复习和探索的兴趣，以问题唤醒学生的策略性思维。接着以一道动点题目帮助学生建立知识、方法、思想、过程之间的联系，提升学生的思维品质。


第二环节是评课环节。

对于刘冬梅老师的课堂，七宝三中的邹毓老师这样说：“刘老师的课堂最精彩的点在于以不变应万变，引导学生从复杂图形中分解出简单图形进行思考问题。另外，课堂采用磁贴的方式很新颖，之后我都想在自己课堂上引入一些类似方法，为孩子枯燥的九年级的学习生活增加一些乐趣！”

对于蒋赢利老师的课堂，尚师初中的吴中卿老师这样说：“蒋老师的课堂，最吸引我的就是有激情，用自己的声音感染学生，在借班上课的条件下依旧能够调动孩子们的积极性，以学生为主体，让学生充分发挥自己的主观能动性，积极在黑板上板演，引导学生充分理解对于该类问题为什么要分类、如何分类。”

第三环节，闵行区初中数学中心组的鹤北中学的秦志强老师给我们做了《相似章节解题策略》的干货分享。秦老师由几道极具代表性的例题向我们依次分析了相似三角形的证明问题和相似三角形的计算问题。其中证明问题例题中，将等积式化为比例式，关注线段所在的三角形（三点定形法），证明其相似必要时，用相等线段替换，或者等比过渡。计算例题中，秦老师提供了边角关系分析和图形关系分析两种思路。


活动最后，张胤睿教研员对本次活动进行了精彩总结：教学过程中要重视引导学生掌握条件和结论的互相转化，寻找可知和需知之间的关系，多角度思考，不要陷入思维定势。最后一句话总结此次教研：初三的复习课不是题目的堆砌，要求教师要在平时的教学中培养解决数学问题能力，关注提升学生核心素养。
