教学数字化转型，推进学科新发展——2023学年第一学期初中科学教研活动

发布时间 : 2023-09-18   作者（来源）: 徐婧   阅读 : 517

3/8
←
→

2023年9月13日，以“教学数字化转型 推进学科新发展”为主题的闵行区初中科学教研活动在龙茗中学开展，活动由闵行区龙茗中学胡继国老师主持，区初中科学教研员徐婧和区初中科学教师一起参与了本次活动。

首先，徐婧老师作题为“数字化支持初中科学学习活动的区域推进”主题交流。徐老师提出，今天的科学课在教学数字化转型上已经逐步走向“常态化”和“深度融合”的阶段，建议教师们要密切关注教学数字化转型领域的相关内容，将理论转化为实践，取长补短、借鉴学习。随后，徐老师介绍了上海市“赛·课堂”教学系统和闵行区“智慧纸笔”两个数字化平台在发展学生证据意识、探究能力、科学思维水平以及核心素养培育等方面起到了重要作用，并以实验西校周永国老师的《动植物与大气间的气体交换》一课为例，讲解了利用数字化手段，辅助学生科学学习的过程，建议教师后续应进一步思考如何利用数字化实现教学过程最优化。

文来中学游胤涛老师作“数字化支持初中科学学习活动的实践案例”主题交流，游老师以《测量》和《观察蜗牛》两课为例，介绍了自己巧妙借助数字化等实验设备，赋能学生科学学习活动的过程，充分体现了借助其他数字化手段，助力科学学习活动的设计，实现教学过程的优化。

接着，徐婧老师在教学数字化转型、用科学大观念指引探究教学等方面进行了2022学年第二学期工作通报，表彰了上海市中小学实验教学说课活动和闵行区中青年教师教学评选活动的获奖教师。

最后徐老师对2023学年新学期工作进行了布置，明确指导思想和工作思路，继续推进数字化转型，在学科日常教学研究工作等方面开展学科实践，并对区级公开课和教研活动进行了相关要求说明。

本次教研活动以技术赋能精准教学，为新学期的工作指明了方向，为教学工作积蓄了前行力量，相信闵行区初中科学教师会在合作的氛围中携手奋进、稳步提升！
