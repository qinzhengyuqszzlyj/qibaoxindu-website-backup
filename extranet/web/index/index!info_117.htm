平凡与精彩 传承与开拓 ——闵行区实验小学校长培训基地（第六期）“学员校办学实践分享与研讨”活动（二）

发布时间 : 2022-11-25   作者（来源）: 曹雅明   阅读 : 2743

3/8
←
→

闵行区实验小学校长培训基地（第六期）校长班于2022年11月15日在景东小学和闵行区第二实验小学举办第二场“学员校办学实践分享与研讨”活动，当天活动由基地导师、上海市特级校长、正高级教师何学锋主持，学员校校长及管理层代表、云南跟岗校长团参与研修。

景东小学专场

当天上午，一行近三十人来到景东小学，首先在王英鹰校长的带领下游览了校园风貌和各类场馆。基地导师何学锋校长和各位学员对学校珍藏丰富史料的校史陈列馆、专业的网球场表示称赞，同时也对学校局部校园环境的改进提出了建议。

王英鹰校长以《蕴平凡  续创造  更精彩》为主题作汇报。报告先总地详实梳理了学校的历史沿革和发展脉络，阐述了学校“让平凡更精彩”的办学理念形成过程与背后的价值取向，再从“五年发展规划制定”、“新基础基地校创建”和“校本课程建设”三个校长管理故事的叙述出发，生动鲜活的讲述了学校在教师梯队建设规划中的因人制宜，新基础研究带给教师个体的发展效应，以及如何围绕网球课程打造学校特色文化的思考与实践。

基地导师何学锋校长在听取报告的过程中，及时和学员们展开互动追问。他提出，在讲述学校管理故事时，怎么讲好教师个体发展故事，整体架构教师发展个案与常规梯队培养机制之间的关联，如何借助校长述职对办学作系统思考和阐述，以富有逻辑而感性的方式与老师作交流特别值得深入研究。

研讨环节，学员校长们就王校长的报告发表了自己的感受。大家认为，三个故事的分享非常精彩，既讲出了老师的故事也讲出了学校办学的故事和校长自己成长的故事，在故事分享中能充分感受到学校的办学理念，看到校长直面问题的决心，和坚定的办学信念。同时，学员们也联系自己学校的校情，一起探讨分享了学校特色课程怎么进一步发展。

何校长总结到，各校校情、校长经历不一样，分享探讨非常有价值。他认为王校长作为景东小学的一个“老”人，既有有利的熟悉一面，也容易有惯性思维，能立足于校长的使命和担当，在传承中创新很不容易。何校长对学校特色项目的拓展提升、办学理念内涵的丰实，从怎么和百年老校的文化、新课标的要求、新时代学生发展的需求等方面作深入指导。


闵行区第二实验小学专场

当天下午，校长班来到闵行区第二实验小学鑫都校区，康旻校长首先带领各位校长参观了校园，观看了正在进行的校园四季寻秋主题活动，基地导师何学锋校长适时与康旻校长交流校园环境布置和建设方面的建议。

康旻校长以《纸上谈兵到躬身入局——新校长一年手记》作主题汇报。报告从“背景与起点、挑战与机遇、造势与酝酿、步入并轨启航、总结和反思”五个部分展开，分享了作为新校新校长在“撤二建一”成立闵行区第二实验小学中的成长心路。基于原来两校办学差异和一体化办学的目标要求，学校从新基础理论学习与校服校徽设计等方面开始孕育文化价值认同，从中层岗位设置与聘任、绩效考核方案制定等完善教师队伍建设体系，从与实小骨干教师柔性流动和成立学科专业委员会等推进课程教学实施专业水平提升。

在交流环节，学员校长们认为，康旻校长开办新校工作开展得有条不紊，厘清事理、凝结人心，充分体现了管理的智慧和魄力；善于教育价值挖掘和引领，引导教师全员卷入蓄势待发；善于抓住一体化办学的优势，成立学科专业委员会引领学校教学改革。

基地导师何学锋校长认为康旻作为一名新任校长，对一年来的工作作了全面梳理和小结，系统研究和解决了建校之初的优势、困难和矛盾点，运用整体性思维，从校标设计、舆论宣传中传递学校的价值导向，得到了学校和社会的积极认同，他希望学校能够在高质量发展之路上不断做实做优。


学员学习体会

景东小学王英鹰校长：

本次的基地培训活动，一位校长站在新时代教育变革背景下讲述百年老校“平凡之路再出发”的故事，一位校长基于两校撤二合一讲述“缔造新学校，播种‘新基础’”的故事，故事中，我们观察和体会校长、老师们的“工作日常”与“生命日常”，看变化，也看不变。 “变”的是工作智慧，发展步伐，“不变”的是浸润与影响校园生活的学校文化。校长既要读懂自己，也要读懂老师，读懂学生，通过主动选择和积极实践，把握和创造新的确定性和新的“不变”。

闵行区第二实验小学康旻校长：

无论是新学校还是老学校，每一个校长都会因为学校的独特性而呈现自我独特的思考和实践，从而实现自我独特的成长。王英鹰接手的是一所颇有历史底蕴的老学校，作为一名从景东成长起来又反哺景东发展的“坐家女”，王校长勇于跳出习惯性思维，边继承边反思到再发展，使学校既保留底蕴又焕发新枝，既尊重历史又面向未来，给予我很多启示。作为“新手上路”一年的自己，通过向何校长及基地伙伴的汇报，也很好的反思了一年来的成长，我觉得自己与不同年份的校长伙伴们呈现出一个个生动的“新校长成长样本”。最大的感受是永远不要只是站在自己的立场急于抱怨处境，更要冷静地站在学校的立场思考其成因，首先要学会接纳学校的独特性，让自己融入其中。

田园二外小蒋宛萍校长：

今天走访的两所学校呈现了各自独特之美。王英鹰校长三年后重回百年景东，围绕“学校发展规划的制定”、“新基础基地校创建”和“学校网球特色发展”三个关键节点事件，讲述了她的思考立场和管理策略，展现了一名富有办学自信和治校智慧的学校管理者的思与行。二实小的康旻校长作为一名“空降兵”，一年来基于对两校办学基础和校情差异的理性分析，通过整体规划和高质量实施顺利完成了撤二建一，让我们看到了康校长在面对一项项迫切而具体的问题时直面挑战的韧性、开放坦诚的心态和科学严谨的管理。一天的走访信息量大，何校长和学员校长的互动与点评让我受到了很大的启发。

曹行小学徐建国校长：

经过一天紧张而充实地学习，我觉得，作为一名校长，首先要成为学校发展的总体规划者。校长要有明确的治校思路，具备整合融通的思维方式，充分梳理、分析校内外各方面情况，利用好各方资源，为学校发展赋能助力。其次，校长要成为师生生命成长的价值引领者。校长要在读懂学校，读懂师生的基础上，努力构建和谐的学校生态系统，促进每一位师生的主动健康发展，不断增强每一位教工的安全感、归属感、成就感和幸福感。

汽轮科技实验小学吴怡校长：

王英鹰校长，在办学实践中，努力坚守在传承中发展，在发展中创新。康旻校长在管理中不断萌发管理智慧，坚持做有情怀的教育。通过两位校长的管理故事分享，让我体会到关键的事件为学校创造了“解冻”的契机，更为学校的“改变”提升搭建了共识平台。作为校长，更要抓住每次事件带来的发展机遇，安排、设计、引发全校行动，智慧引领教师主动践行所悟到的经验与教训，将自身复盘所悟所感付诸实践，推动学校内涵的发展。


本场学员校办学实践分享与研讨活动，让我们看到了新校长在办学实践中探索的勇气和智慧，也促进了各校在新时期对办学特色发展的传承与开拓的思考。
