【学前 东片】“音”领互动——探小班音乐活动中师幼互动的有效性

发布时间 : 2023-10-11   作者（来源）: 刘时医   阅读 : 712

3/8
←
→

2023年10月11日上午，闵行区学前东片“基于幼儿发展优先的师幼互动”主题教研活动在华师大闵行永德实验幼儿园顺利开展。活动由闵行区教育学院教研员刘时医老师莅临指导，东片姐妹园保教主任、教研组长共同参与。

                  观摩教学  引发互动

教学现场由陈书雯老师带来小班音乐活动《我爱洗澡》，活动贴近幼儿生活，旨在情境中感受音乐的快慢变化，跟随音乐快乐地律动。活动中，陈老师能够及时捕捉幼儿的兴趣与需求，给予幼儿充分的表达与表现空间，让幼儿在家庭式游戏氛围中感受音乐活动的趣味并大胆表现。

聚焦问题  真情教研

“教师如何支持幼儿感受与表现能力的发展？”“还可以优化哪些引发良好师幼互动的教育契机？”带着这些思考，大教研组长毕井伟老师带领园内教师团队，围绕主题《“音”领互动——探小班音乐活动师幼互动的有效性》展开教研。永德幼教师们基于活动中幼儿的表现行为，结合《幼儿园办园质量评价指南》，运用激励式、追随式、挑战式师幼互动策略，针对活动中的真问题深入分析，梳理总结出在音乐活动中可以运用的师幼互动具体做法。

思维碰撞  集思广益

观摩活动的姐妹园老师们在现场也展开了激烈的讨论，围绕教学和教研活动分组阐述了各自想法。

浦江宝邸幼儿园姜黎黎老师、中福会浦江幼儿园武晨瑶老师、吴泾三幼杨祎老师作为三组的教师代表发言。老师们认为活动选材适合小班幼儿年龄特点，主题鲜明、贴近生活；幼儿情绪愉快，教师重视每个幼儿在活动中的表现；活动中运用的材料十分童趣，环节上层层递进等。几位老师对陈老师在过程中的师幼互动、生生互动、一对一倾听、集体欣赏等方面给予赞赏，同时提出了分段式欣赏音乐等方面的建议。

智慧点评  专业引领

刘时医老师对本次活动进行了细致而深入的专业引领，对永德幼执教教师和教研团队的表现表示肯定。刘老师对教学活动中幼儿的表现与主题经验的紧密贴合、活动中教师尊重每一个孩子的实然状态所体现出的教师专业素养而点赞。

同时，刘老师也指出：第一，要关注具身性经验；第三，要关注每一个幼儿的学习与发展；第三，要提炼互动策略，形成操作指引。

本次主题教研，以一场生动有趣的小班音乐活动聚焦师幼互动，以教学真问题引发教研真互动，为教师们搭建了教科研训一体化的平台，促进教师对自身专业发展的对照和思考。

永德幼将继续践行“幼儿发展优先”的理念，从“有质量”向着“高质量”不断努力！

 

撰稿：华师大闵行永德实验幼儿园
