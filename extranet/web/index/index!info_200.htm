变化教学场域，激活真实劳动的新活力 ——“新课标”背景下特色劳动课程的开发与实施

发布时间 : 2023-06-01   作者（来源）: 刘至宁   阅读 : 1035

2023年5月30日下午，闵行区“义教”一体化劳动课程教学研讨活动第一弹，在浦江三小拉开帷幕。本次教研的主题为“‘新课标’背景下特色劳动课程的开发与实施”，首先是浦江三小梁书宬老师开设了一堂别样的劳技公开课——《“给葡萄瘦瘦身”——葡萄疏果技法》。本节课的特点，一是属于校本特色劳动课程开发，二是教学场域先在教室完成学习交流与任务布置，再到户外校园葡萄廊下进行劳动实践和技能体验，是一节别开生面的劳动课。

 

梁老师在教室教学的15分钟中，以问题设计和提供资源包来激发学生自主学习，充分调动学生积极性，在师生和生生之间的交流互动中的激活思维，学会新知，学会葡萄疏果的原理和技法，了解了“疏果”是为了成熟时的葡萄甜度、颗粒大小等达到最佳，需要在葡萄成熟前去除一部分葡萄果粒；同时掌握了“544332”的疏果口诀，并完成小组任务分工与合作；在后20分钟里，老师组织学生到校园内的葡萄长廊下，以分组实践、分层疏果的形式，让学生们化身为“小果农”的身份，进行亲自操作实践；同时，再分别体验园艺师、摄影师、观察员的角色分工，还有安全员对“登高”同学的安全保护，大家各负其责，轮流体验角色任务，在完成劳动任务后，每位学生都体会到了劳动的快乐和辛苦，并表示以后要尊重他人劳动成果，珍惜劳动果实。

 

课后的说课评课环节由闵行区劳技教研员刘至宁老师主持。梁老师在说课中主要介绍了本节劳动课的设计要等到葡萄长果的季节，要等待时令；另外也是基于学校有一条特色葡萄长廊的校本优势，创设和建立的劳动课程；让学生亲历劳动，学会知识与技能的迁移，让真实的劳动在真实的生活中得以真正体现。

其他兄弟学校的老师们也交流了对本节课的看法和建议，对于这次将特色校本课程融入劳动观念、劳动能力、劳动习惯和品质、劳动精神等核心素养，都有不同程度体现。虹桥中心小学张敏老师在发言中说道，本节课在安全使用工具、合理应用疏果技法、小组有序分工、技术思维培养等方面都体现了“新课标”下以实践为主线的课程结构，课程内容设计也生动有趣，非常适合培养学生的劳动素养。

 

活动当天有幸请到上海市劳动课程特级教师、正高级教师吴强老师，吴老师对本课也给予了高度的评价，他指出本节课是基于“新课标”下的典型劳动课程，基本能够落实核心素养目标，符合“四真”课堂的特点，即在真实的情境中，解决真实的问题，有真实的实践，并且通过过程性评价完成了真实的评价；吴老师在点评中还建议教学中的数字化转型，不能被数字化“主导”，要为教学目标实现而作为载体或工具，有目的、有控制地使用；另外还可以尝试以数据建模的形式，对评价标准的要素进行赋值，再通过建模建立来完成更高阶的评价体系，这还需要教师进一步去学习和研究。

随后由吴老师为大家做题为《什么是“双新”课程？》的专题讲座，用生动形象的比喻诠释了“新课标”背景下教师的教学方法如何转型，对核心素养下的教学特征、核心素养下的新观点与难点、核心素养下如何培养关键劳动能力和品质等方面进行解读，特别是在学科核心素养教育实施的知识与技术载体、教学评一体化设计、学习支架的设计、学业质量评价体系的设计这四个难点做了深入浅出地分析和指导，建议使用学历案或学习单，加强学习指导。本次讲座使与会教师更深入了解新课标下的劳动课程的实施要求，也进一步思考未来劳动课程的设计与教学方向，大家纷纷表示受益匪浅。

 

最后，教研员刘至宁老师为本次活动做了总结，首先对本节课教学场域的变化设计给予肯定，教室内适合集中学习，教室外适合技术实践，并且非常符合校园文化特色；教师在创设真实的劳动情境中，基于真实的校园生活，引导学生学会分享劳动成果和劳动快乐；课程内容选择因地制宜，宜农则农，注重时令特点和校园特色资源利用；教学中强调亲历劳动，亲身参与，亲手操作，手脑并用，通过探究的方式获得劳动体验，习得劳动知识与技能，使学生更好地感悟和体认劳动价值，培养劳动精神。

 刘老师还建议在单元设计上，可以先进行小单元设计，再到大单元和大任务设计，并基于时令特点进行分阶段教学实施，逐步完成时令性、季节性劳动课程的设置，从种植、养护、疏果、采摘到加工制作等园艺技术环节来进行小单元设计，最终形成校本特色劳动课程。刘老师还鼓励广大劳技老师多以真实的劳动情境出发，以单元教学设计为引领，多关注校本特色劳动课程和教师擅长的劳动课程，积极开发并完善“新课标”的特色劳动课程，为未来的劳动课程教学及校本劳动教育课程体系构建做好必要的课程储备。

     会后吴老师和浦江片劳技教师及部分劳技教师合影留念。

 

浦江三小  粱书宬（报道）

2023年6月1日
