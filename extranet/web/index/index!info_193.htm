“以材激趣 快乐适应”——闵行区东片幼儿园托班教研活动

发布时间 : 2023-10-27   作者（来源）: 鲁鸣   阅读 : 298

3/8
←
→

2023年10月25日上午，由闵行区教育学院主办、闸航路幼儿园承办的“以材激趣 快乐适应”东片幼儿园托班主题教研活动如期举行。闵行区教育学院学前教育研究室教研员鲁鸣老师、黑龙江省东宁市进修学校教研员刘炳芝老师、黑龙江省穆棱市教师进修学校教研员杨丽静老师、闵行区东片幼儿园托班教师以及浦江镇托班联合教研组的老师们，共同参与了本次研讨活动。

观摩现场

活动现场，富有自然野趣的游戏环境、丰富多元的游戏材料、充足灵活的游戏时间，充分满足了幼儿游戏和探索的需要。在活动中，托班幼儿情绪稳定，自主、自由、自发地穿梭于娃娃家、积木建构、玩色、骑行、拍泡泡、投球、小菜园、爬笼等活动区域。教师适时发起小组游戏，既回应了幼儿的游戏兴趣，又适时拓展了幼儿经验。

现场教研

观摩结束后，我园李单英老师组织浦江镇学区化托班联合教研组的老师们开展了“以材激趣 快乐适应——引发托班幼儿快乐游戏的户外材料投放”的现场主题教研活动。每一次教研中的互动都是思维的碰撞，每一次的碰撞都会有新的收获。通过集思广益，教师们对引发托班幼儿快乐游戏的的户外游戏材料投放有了新的认识和想法。

互动交流

闵行区教育学院学前教育研究室教研员鲁鸣老师组织现场的老师们开展了分组研讨活动，各园所教师对闸航路幼儿园托班丰富的游戏材料和当前托班幼儿的发展水平给予了高度评价，同时也基于今天的活动现场提出了新的思考和建议。

活动最后，鲁鸣老师对本次活动给予了高度的肯定，同时，结合本次教研主题“以材激趣 快乐适应——引发托班幼儿快乐游戏的户外材料投放”的价值进行了深层剖析，也提供了后续的思考方向，例如：如何调整材料以保证幼儿的容易接近、乐于参与，在具备丰富材料的活动区域中教师站位如何优化以凸显三位一体，如何对个别幼儿的需求给予合适的回应等。

我园将一如既往地关注托班幼儿的个体差异，满足不同幼儿的发展需求，不断优化材料以促进幼儿的学习探索，在最大程度上支持每一位托班幼儿的全面发展！
