资源为媒 小课堂衔接大课堂 ——区小学道德与法治教研活动在闵行区马桥文来外国语小学举行

发布时间 : 2023-09-15   作者（来源）: 王玉兰   阅读 : 597


“云天收夏色，木叶动秋声。”仲秋伊始，丹桂生香。2023年9月14日下午，以小学道德与法治课堂“大资源的开发与应用实践”为主题的闵行区小学五年级道德与法治学科研讨活动在上海市闵行区马桥文来外国语小学举行。

本次活动闵行区教育学院道德与法治教研员王玉兰老师莅临指导，马桥文来外国语小学王勤蓉校长、张翀宇副校长出席，闵行区道德与法治教研组长、五年级道德与法治老师200余人共同参与。活动由区中心组成员浦汇小学王晓丹老师主持。

一、智慧教育入课堂，“思政金课”展风采

宋霞梅老师执教的是五年级上册第一单元《感受家乡文化 关心家乡发展》中的第一课《我们当地的风俗》第一课时。

在设计教学环节时，宋老师结合《习近平新时代中国特色社会主义思想 学生读本》《党的二十大报告》以及《义务教育道德与法治课程标准（2022年版）》等文件，对教材进行了深入的分析和解读，并通过前测问卷和观察走访的形式充分了解学情，立足于学生实际，制定了指向培养学生核心素养的教学目标。

在课堂教学过程中，宋老师以习语金句“中华优秀传统文化是中华民族的精神命脉。要努力从中华民族世世代代形成和积累的优秀传统文化中汲取营养和智慧，延续文化基因，萃取思想精华，展现精神魅力。”为核心思想，以马桥当地的特色风俗为载体，展开习语金句的深入学习和深度感悟。

二、“活动园地”共研讨，“智慧论坛”齐分享

本次论坛的主题是“资源为媒，小课堂衔接大课堂”，由马桥文来外小张倩老师和宋霞梅老师、浦汇小学王晓丹老师共同参与。

张倩老师向大家分享了“小小肩担起大责任”的单元活动介绍。她从单元活动的“设计、实施、评价、经验”这四方面阐述了五上第一单元活动的设计意图。以“小小风俗推广者”、“小小民艺传承者”、“小小家乡建设者”这三个单元子活动为推手，充分挖掘马桥地区的资源，丰富思政课教育内涵，又从单元表现性评价任务出发，关注学校小课堂和社会大课堂的衔接，丰富学生体验与感受，促使学生情感态度价值观有温度的提升。

宋霞梅老师则从本课的课堂教学实践出发，以“场馆资源引马桥风俗”、“实物资源探马桥生活”、“现场资源扬文化美德”这三个角度向老师们进行了说课。最后通过本班学生对于这节课的评价、感悟，让我们知道了通过实地探究的研学活动，学生们对于中华优秀传统文化产生了归属感和认同感，树立了文化自信。

王晓丹老师以“媒介”一词为指引，介绍了本单元的教学活动是如何借助资源这一媒介，有效推动思政小课堂衔接社会大课堂。她以单元问题“如何关心家长发展”为导向，提出新时代的思政课应当“拿进来、走出去、请进来”，把握精准教学，培育学生核心素养的全面发展。

三、专家引领明方向，启思明理促提升

闵行区教育学院道德与法治教研员王玉兰老师对本次教研活动做专家点评。王老师强调，思政课堂需要立体思维，带有任务驱动，从而引导学生进行深入学习。教师则需整合单元概念，加深学生的分析和应用能力。

思政课堂需具备三个方面的要求。第一，教师应对本土资源如数家珍，亮出鲜明的地域特色；第二，学生是学习主体，学生立场要鲜明，正如叶澜教授说的：把课堂还给学生。第三，教师应对学生的呼应、点评到位，提升每个环节的价值引领，把握住本课的核心素养提升点。

王老师强调，新时代要上好思政课，必须建立立体思维，从横向上打开校内、校外、社区等广阔视野，将众多典型资源引入课题；从纵向上设计深度学习，以任务驱动的学科活动，既要让学生在活动的过程中看到基本的事实、学会基本的技能，获得广泛而深入地理解，同时还要获得本学科教材中本体知识的重要概念，理解概念与概念之间的关系，形成意义建构，把日常生活中随机的事实整合到单元，提升学生的分析、综合、评价素养。

相信在教研员王玉兰老师的引领下，闵行区道德与法治的小伙伴共商共研共成长，将继续以强心、用心、专心、钻研术业，一步一脚印打造创新性思政课堂，让每一个孩子都能在我们的课堂中有所经历、有所体验、有所收获，真正落实立德树人的根本任务。
