数据驱动智慧教育的实践探索——闵行区“智慧教育”校园长研修班（第二期）培训

发布时间 : 2020-04-23   作者（来源）: 朱凤   阅读 : 2336



4月22日，闵行区“智慧教育”校园长研修班（第二期）第三次在线培训如期举行，本次培训邀请重庆市巴蜀小学校副校长李永强、苏州市教育质量监测中心主任兼党支部书记罗强担任主讲，他们分别从学校和区域的角度分享了数据驱动智慧教育的实践探索。来自区域内的相关领导，“智慧教育”项目申报学校的正副校园长、部分研训员、部分骨干教师等两百多人参与了培训。区教育学院科研中心主任何永红主持活动。

李永强做了《泛在学习的巴蜀样态》专题讲座，围绕如何推进一所学校的智慧教育，分享了巴蜀小学校的经验。通过梳理教学成果奖，寻找到改革的切入点和突破点；以教育部泛在课堂共同体项目为契机，探索泛在课堂，突破传统课堂时空限制，促进学校教育与在线教育融合创新，让学生随时随地进行泛在学习；化危为机，疫情期间精心设计巴蜀公开课，形成泛在学习的样态；借助有效的评价工具，将情景化描述变成数据化，将散点的数据系统化形成数据链，使评价为教育赋能。校长们的情怀使命、创新、梦想、行动，成就了巴蜀的样态。

罗强分享了《高品质监测促进苏州教育生态优化》，介绍了苏州教育质量监测的整个系统设计和操作办法。质量检测是一项系统工程，有目标有指标有工具有数据有模型有分析有应用，质量监测的目的是为了让学生更好地成长，让学校更健康的发展，让区域教育生态更健康。质量检测是一个持续的、科学检测过程，检测结果可视化。

如何有效利用学校中每天产生的数据，发挥数据的教育生产力作用，不仅需要数据收集、数据整理、数据分析，更需要合作研究，共同推进智慧教育建设。
