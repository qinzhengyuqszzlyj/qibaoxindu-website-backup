从儿童视角看托班幼儿发展 —— 闵行区0-3岁幼儿教育专家讲座顺利举行

发布时间 : 2023-05-23   作者（来源）: 鲁鸣   阅读 : 894

3/8
←
→

2023年5月22日，闵行区0-3岁幼儿教育专家讲座在线上顺利举行。上海市教师教育学院学前教育教研员王菁女士受邀做了以“从儿童视角看托班幼儿发展”为主题的精彩讲座。活动由闵行区教育学院教研中心学前教育研究室鲁鸣老师主持。全区学前教研员以及来自公、民办幼儿园的保教主任、托班教师200余人齐聚会场，共同聆听讲座。

一、幼儿发展优先是指导托班教养工作的基本理念

讲座之初，王老师回顾了幼儿发展优先理念的产生背景和内涵要求，进而分析0-3岁幼儿发展及其教养工作的特殊性。王老师指出，小年龄群体能够通过教师创设各类环境以及提供的活动实现整体发展、主动发展和差异发展。每一位教师应对托班幼儿的发展规律有所认知，认识到0-3岁幼儿教养更强调成人提供温暖的陪伴和安抚，提倡教养融合，相信幼儿可以通过成人为其放慢节奏而实现“自己来”，进而实现个性发展和潜能开发。

二、幼儿是有强大能力的学习者

幼儿是柔弱无助的，还是有强大学习能力的？王老师通过一系列国际范围内有关0-3岁幼儿心理行为实验、脑科学研究结果为大家展示了0-3岁幼儿的强大学习力。同时，王老师还呈现了教师们熟悉的幼儿游戏互动场景，让大家认识到，沉浸于“躲猫猫”游戏反映出幼儿具有了物体永存的认知，且长时记忆能力和抓取能力都在不断发展；对开、关灯的不断尝试是幼儿对开关控制灯亮已建立了因果关系认知，且手眼协调和手部控制水平不断提升；对积木垒高和推到的执着是幼儿感知地心引力的好奇心萌发，以及对事物控制的满足感和自我效能感不断获得。由此，王老师希望老师们充分认识到，0-3岁幼儿有着强大的学习力，每一位教师应建立对0-3岁幼儿教育工作专业性的敬畏心。

三、托班幼儿的教养是以幼儿发展优先为指导的教养实践

王老师基于实践观察和系统梳理，围绕以下问题进行了深入讲解。

托班幼儿喜欢怎样的环境？王老师指出，托班幼儿处于依恋感强烈、自我中心、情绪不稳、秩序敏感和语言爆发期，需要教师为其提供安全温馨、丰富有序、适度开放和自然可互动的环境。具体来说，班级环境整体色调应体现家庭化，全面均衡地兼顾到室内和户外环境、生活和游戏环境以及幼儿和成人使用，同时还应方便观察、动线合理，从而最大限度满足幼儿需要，支持幼儿发展。

入园适应期如何度过？适应期是幼儿在托班发展阶段中最困难的时期。如何度过适应期对每个托班教师都是极大的挑战。王老师娓娓道来，指出保教人员是幼儿的“重要他人”，良好养育关系的建立是幼儿发展的根本所在。教师应尊重、理解、接纳幼儿所有的情绪。幼儿园层面应采用渐进性的适应策略，创设适宜的过渡环境，作息安排上考虑由初期的亲子陪伴过渡到幼儿独自在园，帮助幼儿平稳度过适应期。

托班活动如何组织？托班活动的组织应基于班级幼儿的情绪发展来安排。其中小组游戏贯穿于幼儿整个游戏活动中，可以根据幼儿兴趣在小组范围内适时开展。圆圈活动则是教师根据幼儿已有经验和发展水平，有目的的设计活动，组织全班幼儿一起游戏。不管是小组活动还是圆圈活动，教师都应关注到活动中的每个个体，让幼儿在充满信任关系的环境中学习。

四、托班工作开展是“幼儿发展优先理念”下基于区域发展的适宜性实践

活动最后，蔡志刚老师基于区域实际，结合讲座内容对区域托班工作开展情况做了总结，并提出要求。蔡老师指出，当前区域托班实践面临托班开设体量大、教师经验不足以及园所对托班课程研究与思考有限的现实挑战，因而区域层面给予托班工作越来越多的重视，支持幼儿园从园长层面到教师层面共同思考如何优化与实施托班课程。蔡老师对后续托班工作开展提出两点期望：第一是区、园合力，形成托班研究共同体，共同优化托班课程建设与实施。各幼儿园应基于区域现场和幼儿园实际情况思考如何根据自身特点做好托班工作，比如如何在教养医融合背景下更好地适应托班幼儿，如何做好家园工作，如何进行高质量的师幼互动等等。第二是坚持幼儿发展优先，基于幼儿视角开展各类活动。教师应进一步地做到认真观察，对幼儿的细微变化具有高度敏感性，并能够把握每个幼儿发展大纲，发现托班幼儿个体的发展里程碑，在心理层面真正走近幼儿，实施幼儿喜欢的活动，建设有意义的托班课程。
