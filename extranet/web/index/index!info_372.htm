老辈传道解惑 新人技艺传承 ——2023年闵行区中小学劳动教师水仙花雕刻技术培训活动

发布时间 : 2023-11-17   作者（来源）: 刘至宁   阅读 : 919

2023年11月16日下午，闵行区中小学劳动教师水仙花雕刻技术培训活动在上海市七宝实验中学开讲。本次活动由七宝实验中学朱怡之老师主持。这是“七实中杯”“玉玲珑奖”区水仙花雕刻造型活动展评中，不可缺少的一个环节。水仙花雕刻无论是作为传统技艺，还是作为新课标中农业生产劳动，都是课程重要的组成部分。让更多的教师学会雕刻技法、学习造型养护，也是推广我区水仙花雕刻课程的一种有效途径，也为我区参加上海市水仙花作品展评活动，提供技术支持与指导。往年的培训对象以中学教师为主，今年吸引了不少小学劳技教师前来学习，壮大了我区课程实施的教师队伍。

此次讲座的主讲人是来自诸翟学校的孔凡云老师，他是中学高级教师，也是我们区学科中心组成员。他和他学生的作品多次在市级和区级水仙花展评活动中，荣获一等奖，积累了丰富的经验。孔老师从水仙鳞茎各部分的名称开始介绍，详细讲解了“蟹爪水仙”的雕刻过程、步骤以及养护要点，并透露了在雕刻过程中的一些小诀窍：如雕刻前，水仙鳞茎的顶芽最好长长一点，那样雕刻花梗会更加方便；在削叶缘时，手指从后面压一下叶芽，使得叶芽之间的距离变大一些，更容易切削等。他以自己以往参赛作品为例，教大家做出特殊造型、养出长而洁白的根须等，还与大家分享他的参赛经验。

接着给我们讲解的是明星学校孙守思老师，他也是我们劳动学科的高级教师。他利用实物投影，现场教授大家“茶壶”形状的雕刻方法。听说他是在“掏心式”刻法的基础上做了自己改进与创新。从选择鳞茎到雕刻工具，再演示整个雕刻过程，让在座的“学员们”惊叹。孙老师的方法与原来的雕刻方法相比，简便很多但造型同样美观，大家直呼收获颇丰。

然后，参加培训的老师开始动手实践，现场雕刻。孔老师、孙老师以及中心组的老师们成为了现场指导教师，不仅是手把手的演示。还有对于鳞茎造型的设想讨论、花苞的判断等积极与“学员们”互动交流，“学员们”学得认真，“老师们”教得仔细。

最后，区劳技学科教研员刘至宁老师总结了此次活动，并给我们提出了几点希望，希望老师们坚持每年参加培训，在精进雕刻和造型技艺的同时，积极开展课堂教学。希望有更多的学校和学生参加区级和市级的展评活动。希望我校成为教师培训基地，以水仙花雕刻和养护为主题，根据新课标要求，依托单元教学设计，开发出较为完整的且具有七宝实中特色的劳动课程。

此次讲座接近尾声，“学员们”还围着孔老师和孙老师不停地请教，希望孔老师把讲座的资料发到群里，大家都表示还要回去再学习、消化吸收今天的内容，以便更扎实地练习、掌握讲座内容。而孔老师和孙老师两位前辈，更是倾囊相授，细心有耐心的指导，真心为青年教师的技艺传承提供助力。本次培训活动不仅给教师们提供了学习交流的机会，也为区水仙花作品展评活动作了动员，更为水仙花雕刻劳动课程的深入开展提供了前进动力。

  

七宝实验中学  朱怡之（报道）

2023年11月17日
