> 干部培训

敢担当 善作为 能成事——2022年闵行区教育系统副校（园）长、后备干部培训班 举行第二轮集中培训

发布时间 : 2022-10-31   作者（来源）: 朱凤   阅读 : 1193

3/8
←
→

区域教育高质量发展，需要一支政治过硬、品德高尚、业务精湛的青年干部队伍。2022年10月26日至27日，闵行区教育系统副校（园）长、后备干部培训班全体学员在闵行区教育学院举行第二轮集中培训。

 

领导讲话：高瞻远瞩，坚定信念走向未来

闵行区教育党工委书记恽敏霞亲临现场并作讲话，她认为作为一名优秀干部需要具备以下三点：一用“同理心”管理，急群众之所急、想群众之所想，用心用情用力实去做好管理工作；二要“多元化”学习，希望学员们以培训为契机，向专家学习、向同伴学习、向实践学习，做到学有所思、学有所获、学有所成；三应“多渠道”读书，希望学员们精读泛读各类书籍，向书本世界学习，适应教育教学的变革，补足短板，完善自身管理体系。

 

专题报告：智慧赋能，夯实管理知识结构

交通大学余明阳教授、浦东新区进才实验小学赵国弟校长、市西中学董君武校长分别做了《品牌的成长与建设》、《学校发展，最应关注的几个要素》、《建构新型教学范式的实践研究》专题报告。余教授就品牌话题进行了梳理和分享，从城市定位到学校定位，从品质提升到品牌创设，在传承与创新中打造属于自己的学校品牌。赵校长立足于凸显学生的理念、符合实际的目标、全面发展的课程、理顺人心的管理、幸福工作的教师五个要素，以理论与案例相结合的方式，深入浅出的分析了这些关键要素对学校发展的重要性。董校长从“优势学习，促进学生发展”、“思维广场，撬动教学变革”、“网学课堂，再造教学流程”，“面向未来，整体因材施教”四个方面分享了他的管理实践与探索。学员们深刻认识到：要明确新时代学校管理者和引领者的使命和担当，深入思考教育的本源，树立“只要找到路，就不怕路有多远”的决心，不谋全局者，不足谋一域。

 

学员讲堂：学思践悟，锤炼管理思维品质

聚焦《现代学校管理机制与制度建设》，副校（园）长培训班和后备干部培训班分别开展“学员讲堂”，日新实小徐为和晶城中学张静分别担任两场讲堂的主持人。

副校长班日新实验学校徐为、交大实小黄萍、世博小学张玲、康城学校谌海兰、颛桥幼儿园唐春丽结合各自学校实际，分别做了《正向激励触摸学校管理的“温情”》、《创新校本教研机制  护航教师专业成长》、《以ATM管理模式推动学校机制改革》、《现代九年一贯制学校教学管理机制与制度建设的思考与探索》、《“颛”注均衡  “幼”见成长》智慧分享。后备干部班浦江三小秦燕琼、莘松小学杨婷婷、闵行四幼顾慧闵、文来初中徐枢清分别做了《在变革中立序 与融合中共长》、《管而有理 张驰有度》、《创新学校管理机制  丰富学校发展内涵》、《有序运作  有效保障》交流。在政策引领、理论探析的基础上，聚焦各具特色的学校管理机制及制度，阐述现代学校管理要以人为本促动学生主动发展，做到有深度、有精度、有力度更要有温度；创新学校管理机制，凸显人本性、民主性、开放性、科学性、发展性，从而实现学校高质量发展。

 

专家点评：高屋建瓴，精细指导助力成长

上闵外校长吴金瑜、上师三附小校长沈珺分别对副校长班和后备干部班学员做点评。两位专家肯定了学员们从不同的侧面，基于自己的思考、结合学校特色，呈现《学校管理机制与制度建设》的主题。吴校长对成为优秀的管理者提出三个要求：一要有一段扎实的教育管理经历，二要有厚实的教育管理理论知识，三以前两点为基础，再针对学校特点、师生特点、管理者个人特点形成非常有针对性、有效的以学生发展为本管理机制。沈校长希望学员关注以下几点：关注制度价值的重要性，努力做到“有迹无痕，若有若无”，体现人本关怀的学校制度。关注校园文化的重要性，用文化引领员工的行为，营造以生为本、以师为本的学校文化，助推校园前行成长。

闵行区教育学院院长朱靖作指导，通过四个问题：制度的本质、目的是什么？制度的内在逻辑是什么？制度的类别是什么？所有制度的制定中最难的是什么制度？引发全体学员的思考。强调制度制定要重视人的尊严，遵循人的发展规律，实现人的价值，并希望作为管理者一定要不断学习先进的管理理念，以大教育背景为支撑，同校园管理实践紧密结合，在实践中创新，在创新中发展。

闵行区教育局调研员刘兆利希望后备干部不辜负组织重托，珍惜现有平台，积累管理经验。增强大局观念、增强法治思维、增强危机处理的能力，办人民满意的教育。
