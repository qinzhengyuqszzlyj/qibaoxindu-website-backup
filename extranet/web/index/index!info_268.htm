基于新课标，探索新课堂----第15周小学体育区级教研活动

发布时间 : 2023-05-25   作者（来源）: 顾佳伟   阅读 : 342

3/8
←
→

2023年5月24日,明强小学开展了以“基于新课标、探索新课堂”为主题的闵行区小学体育教学研讨活动。明强小学姚凤校长、闵行区北片学校的体育教师、松江九亭五小的体育教师们参与了本次活动。活动由明强小学体育分管教导陆一老师主持。

首先，陆一老师针对本次活动的立意进行了解释：“新课标的修订要求老师们更加注重学生对体育健康知识的学习，体育理论课是我们对这一板块进行的最新探索，如何基于水平二学生的现有知识，让他们学会正确规避运动伤害是我们设计本次教研的初衷，我们期待学生通过理论课的学习和实践课的具体应用来综合提升学生体育核心素养。”

第一节课是由王怡元老师执教的三年级体育理论课“我的热身我做主”本节课的重点在于让学生明理。通过游戏体验和互动问答的形式让学生从身体热身、心理热身两个方面明白热身的作用；通过 “10秒把脉法”让学生测量并记录自己运动前、运动中以及运动后的心率变化，从实践的角度理解心率的作用以及如何通过心率衡量和调控自己的运动内容；通过知识迁移和游戏互动的形式让学生理解人体结构并学会如何针对教学内容进行针对性的重点热身。整节课从明理到验证，再到应用，层层推进，让学生理解如何做到“我的热身我做主”

第二节课是由管曦老师执教的三年级实践课“正面助跑，屈腿跳高”。课堂以问题链的形式让学生学习“正面助跑，屈腿跳高”的内容。首先，从上一步单脚起跳过低高度的障碍到助跑几步正面助跑屈腿跳过较高高度的障碍。其次，在此过程中，基于学生屈膝上体不到位的问题，给出针对性的应对策略。最后让学生体验具有一定远度的正面助跑，屈腿跳高。使学生在会做动作的同时，理解动作的物理学原理。在新课标的指导下，综合活动以游戏的形式锻炼学生的体能：不同形式（匍匐前进、铅笔滚等）的行进方式，各式各样的挑战任务（摆垫子、拆盲盒、翻对子等）充分调动了学生对于体能的学练积极性。

两节精彩的课堂引入入胜，老师们在看课的过程中积极讨论。

课后，两位老师通过自己的视角分别对两节课的设计思路以及磨课历程进行了现场说课。

两节课的呈现以及说课时对课堂的解读激发了观课老师们的热情回应，老师们在评课环节纷纷表达了对两节课的认可，同时也为我们提出了很多更好的建议。

最后，姚校以“探索性”和“常态化”为关键词对本次活动做出总结，即肯定了本次教研对新课标的勇敢探索，同时期望我们能将新课标的理念常态化，落实到每一节课中。
