资源为本 小方案融入大社会——区小学道德与法治教研暨智慧教育研讨活动在闵行区实验小学（莘松校区）举行

发布时间 : 2023-11-11   作者（来源）: 王玉兰   阅读 : 903

“生命无彩排，防患于未然”，在第32个全国消防日，小学道德与法治课堂“大资源的开发与应用实践”主题教研系列活动之“资源为本 小方案融入大社会”闵行区小学三年级道德与法治学科《安全记心上》研讨活动在上海市闵行区实验小学（莘松校区）举行。

本次活动中，闵行区教育学院道德与法治教研员王玉兰老师莅临指导，实验小学莘松校区吴红霞执行校长、尤兆蕾副校长、学生工作部主任陈静老师、陆慧玲老师、杨晨艳老师，闵行区道德与法治教研组长，三年级道德与法治老师，闵行区小学道德与法治中青年教师教学竞赛参加复赛的38名教师，共150余人共同参与。活动由实验小学莘松校区道德与法治教研组长郭惠杰老师主持。

一、精彩课堂 提升素养 

实验小学莘松校区明珠老师执教的是部编版小学道德与法治三上第三单元第8课《安全记心上》中的第二课时。教学伊始，明老师由孩子们之前参加的 “出行小达人，游泳小达人”等系列活动引出今天的话题——争做“消防小达人”。紧接着她出示火灾新闻，引导学生猜测事故损失，感受火灾危害大，寻求解决办法。在此基础上，明老师以点带面，创设真实生活情境，结合学校的平面图，指导学生自主绘制逃生路线图，现场演练逃生。在强烈的体验与参与中，学生不仅感受到防患于未“燃”的重要性，还学会了如何解决问题安全逃生。

二、专家点评 引领成长

教学结束，明珠老师结合自己的教学，围绕单元规划的设计、教学过程的设计、教学资源的运用等几个方面进行了详细地说课。

结合明老师的课堂，王老师进行了精准的点评。首先，王老师肯定了明珠老师的这节课，她认为这是一节精准教学的高质量课。紧接着，王老师围绕“高低有度 长程有接 内外有联”三个关键词分析了明老师的这节课。“高低有度”体现在高处着眼，明老师抓住习总书记讲话中关于安全问题的金句，同时又从低处落实，将课标中和本单元相关的核心素养以及学业质量标准的要求进行细化分层，使得整个单元的规划高低有度。“长程有接”体现在一个单元的课程中，明老师有序的、有进阶的，螺旋上升的开展单元教学整体育人的活动，进行系统长程规划。“内外有联”则体现在单元规划时，明老师将教材内部的内容与我们大社会外部的丰富多维的资源链接起来，将绘本资源、国家的法条、消防站的实地考察有效结合。除此之外，王老师还指出要上出一节生动有效的思政课，我们需要无限逼近真实，极限挑战潜能。王老师以明珠老师的课为例，从核心概念的树立、真实情境的建立、教师情感地投入、教学环节的系统性等几个方面手把手地告诉大家该如何“无限逼近真实，极限挑战潜能”。最后，王老师还表达她的期望，希望我们老师能够从明珠老师的课堂中汲取一些经验和做法，并且迁移到我们的日常教学中。

三、高效讲座 指明方向

为了进一步帮助大家更好地上好思政课，王老师又贴心地为大家做了一场“以赛促研 精准教学”的讲座。王老师指出，要上好思政课，一定要建立四个理念：首先，要深化改革、落实课标，明确素养导向；其次，要规范文本，系统设计，具备单元视野；再次，要懂得资源开发，发挥学科特色，做到知行合一；最后，要上的有声有色、对学生进行道德启蒙，引导学生深度学习。如何将这四点运用到教学活动中呢？王老师结合具体的案例，从“习思想、新课标、大单元、通教材、强活动、重生成、大资源”这七个方面对大家进行了指导。王老师深入浅出的讲解，好似拨云见日，不仅打开了老师们的视野，还为大家设计思政课指明了方向。

与大雁齐飞，目之所及皆是广袤天空。通过这次教研活动，大家不仅仅观摩了一堂道法课，更学会了如何去设计教学道法课。山高自可登临上，水深不妨涉足知，在未来的日子里，愿老师们学其所用，用其所学，带领着我们的学生共同成长。
