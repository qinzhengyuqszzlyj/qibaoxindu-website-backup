牵手姐妹园 带教促成长——导师殷蓓琳组10月带教月报

发布时间 : 2018-10-25   作者（来源）: 朱凤   阅读 : 2149

为帮助新任园长尽快进入角色，推进新园各项工作的顺利开展，10月8日下午，导师闵行四幼殷蓓琳园长和学员鹤庆幼儿园吴兰园长分别带领本园的优秀总务主任和保教老师共11人，一起现场指导闸航路幼儿园的后勤管理与三大员的规范操作。

在闸航路幼儿园陆学红园长的带领下，大家现场查看保健室、观察室 、教室盥洗室、保育员操作室、食堂等环境，并一一进行指导。鹤庆幼儿园吴政权老师以其精湛的专业技能，结合实际边看边指导，让保健老师们进一步明确了一些标签的使用方法以及规范流程中容易出现的问题；对于一些有困惑或争议的问题也进行了指导，使后续各项工作能更加规范。同时她又分享了关于一日活动操作规范化的经验，在分享与互动的过程中，吴老师细致入微的讲解，结合实践有理有据的分析，使我园的总务主任与保健老师明确了工作重点、明晰了责任，受益匪浅。特别是当吴老师一对一地帮助新手保健老师熟悉每一张表格时，更是从心底里感到一种温暖，让新园感受到了牵手姐妹园的幸福感。

殷园长和吴园长分析了闸航路幼儿园的园情，对三年发展规划进行了针对性的指导。从一所新园的实际出发，仔细分析现有的优势与不足，目标清晰、定位准确，循序渐进，不急于求成，不眼高手低，一步一个脚印，踏踏实实地去做好当前的工作。

导师带教，不仅仅只是针对一名新园长的带教，而是对一所新园所全覆盖的带教，在这个过程中，姐妹园资源共享，优势互补，让新园所真正受益。

金秋十月，应导师殷蓓琳园长之邀，我们还参与了四幼主题为“弘四幼新乐，做四有教师”的第四届教工文化节活动，共同分享四幼新乐文化的硕果。


供稿：吴兰、陆学红
