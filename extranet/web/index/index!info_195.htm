——2022年第二学期闵行区中片初中物理教研活动

2023年5月24日下午，闵行区中片初中物理教研活动在晶城中学顺利举行。活动内容聚焦在如何精准诊断学情，关注教育数字化转型趋势，创新实验发展等话题。活动由颛桥中学肖俊老师主持，闵行区中片全体物理老师参加。

本次活动的第一环节是由晶城中学的王琴老师和李欣老师带来的两节高质量公开课。王琴老师以温差风扇的演示实验激发学生的好奇心，通过设问、猜想等方法引出课题。引导学生从已有的机械能概念出发、通过类比的方法建立内能的概念。课堂中利用具有创新性的演示实验和精心设计的学生自主实验帮助学生理解抽象的内能知识。李欣老师则在课前充分调研，通过思维导图和学生自我评价了解学生学情，聚焦学生困惑。重在解决学生解决s-t 图像问题薄弱点、困惑点等，李老师注重引导学生得出解决这类专题的一般方法，此外课堂中智慧笔的使用进一步提升了教学效能，达到了精准教学的目的。

       评课环节给老师们带来了思维的碰撞、教学智慧的启迪。莘松中学的童瑜意老师和七宝鑫都中学的李亚洁老师一致认为两节课都对本校学生的学情做了深入调研，注重科学方法的渗透，有效落实了学科核心素养。主题评课环节，莘松中学袁晨玉老师认为王老师创设实验情境，激发了学生的探究兴趣、提升了探索思维，课堂中注重课时关联，体现了单元教学思想。师培附中的郭赟娴老师从“整合”为高效复习“筑基”， 反思”为高效课堂“插翅”为主题对李欣老师所执教的课进行了深度评析，认为本节s-t图像专题复习课给青年教师起到了良好的示范作用。

关注教育数字化转型趋势，注重实验创新。作为一名物理教师，我们应该积极探索和研究，提升学生的创新意识和实践能力。
