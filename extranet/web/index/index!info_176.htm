基于新课标 探索新课堂 ——闵行区第9周小学体育研训活动

发布时间 : 2023-10-26   作者（来源）: 顾佳伟   阅读 : 406

3/8
←
→

重阳节后，大地流金。10月25日下午，以“基于新课标 探索新课堂”为主题的闵行区小学体育研训活动在闵行区实验小学畹町校区举行。闵行区小学体育教研员顾佳伟老师，闵行区实验小学体育学科主任、特级教师瞿伟华老师、远道而来的黑龙江省牡丹江市穆棱市体育教研员尚衍军老师和穆棱市骨干教师赵欣及全区小学体育教师出席了本次活动。活动由闵行区实验小学畹町校区体育主管张培培老师、莘松校区体育主管陆薇老师共同主持。

来由·感悟

活动伊始，张培培老师与陆薇老师向莅临此次活动的老师表示欢迎，并对活动情况进行了简要介绍。本活动两节课的主教材内容取材于陈敏晔老师参加市青教赛并获一等奖的10节模拟课，通过对模拟课的梳理、细分、改良，形成了支撑类教学的大单元内容，选取其中两节课，完成由模拟课到实践课的转换。过程中，团队老师们也感受到大单元设计对老师综合能力的考验，以及模拟课更关注教师教的过程，而实践课是灵活运用、教师能力、学生状态、课堂效果的整体展现，在解决了诸多问题与挑战后，最终在活动中呈现出来。

实践·真知

接着，由闵行区实验小学畹町校区蔡超老师与莘松校区许嘉伦老师分别展示了“四年级：支撑与悬垂”大单元的“支撑移动”第6课次与第7课次。在内容主题上，蔡老师的课聚焦于手和脚在不同方向的移动练习，许老师则重点放在脚搁不同高度的支撑移动，通过器材变化让学生感受重心变化情况下的支撑移动练习。两节课都设置了不同的教学情境，利用了多媒体、光电球等信息手段，激发学生学练趣味。另外，在教学环节难度提升的同时，由单人练习到双人合作到多人合作的设计，评价手段也与练习组合内容相结合，不仅身体动，更要脑动、心动，激发了学生的团队荣誉感和努力拼搏的信念感，体会到积极参与带来的成就感，充分发挥体育学科育人价值。

互评·共进

展示结束后，两位老师当即对课堂呈现进行了反思总结，表示应更加关注到每一位孩子，加强师生间的互动交流，整体氛围会更热烈，更有利于达成预期目标。明强小学王老师表示，许老师的课，从一开始就调动了学生的积极性，在主教材环节通过器材的灵活变化展现出了教学内容的难度递进，并且在教学评价中，非常注重评价的仪式感，让学生知道如何评，并为了争星而努力奋进，展示出非常有效的激励引导作用。复旦大学附属闵行实验学校的盛老师提出，蔡老师的课有很多值得学习的地方，如在支撑移动练习时，改变移动的身体部位和方向，都能体现学习难度的提升；学习过程中学生角色的转换，从练习者到帮助者到裁判员等，能通过不同角色体验大大激发学生的学习兴趣与参与的热情；体能游戏的最后一个环节，在比赛的同时又完成了体育课器材的回收，对于体育教师的课堂组织环节有实用性的借鉴意义。

突破·创新

活动尾声，闵行区小学体育教研员顾佳伟老师对本次活动进行点评与总结。顾老师表示，两节课在已有“主菜”的基础上，细化“配菜”和“佐料”，时更需要脚踏实地的实践，大胆创新。在提倡信息技术赋能的当下，我们更需要认真探索，积极实践，在不断尝试和突破中创新出融入课堂的方法。在后续展示课研讨活动中，还需要每位老师关注到体能练习是否多元化，练习时交流指导是否更有针对性，放松练习的设计如何才能更给课堂“画龙点睛”等细节，只有走好每一步、走稳每一程，才能以不变应万变，在体育课堂中呈现出更多的教育智慧。
