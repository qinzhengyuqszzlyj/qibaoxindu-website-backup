互联互通 协同发展——吴江区中小学管理者赴闵行跟岗研修开班典礼

发布时间 : 2019-11-26   作者（来源）: 朱凤   阅读 : 3447

3/8
←
→

11月25日，为期两周的苏州市吴江区中小学校长影子跟岗研修班开班仪式在北京外国语大学附属田园高级中学举行。闵行区教育党工委副书记李光华、党群科科长林火生、吴江区教育局副局长周萍、组织科科长王芳等相关领导、导师、学员，近30人参加仪式。闵行区教育学院校长发展中心主任俞琳主持活动。

李光华致欢迎词：在长三角一体化国家战略的大背景下，闵行和吴江两地教育各有优势，各自精彩，同时，介绍了五位导师的成长经历及所在学校在内涵发展方面的特色和优势。建议五位中小学校长以大气谦和的姿态欢迎学员，开放学校各管理领域，期待学员有所收获。

周萍代表吴江区教育感谢闵行教育的支持和帮助，吴江区将以长三角基础教育一体化发展试验区为契机，充分借助上海市的优质教育资源优势，提高本区校长的政治素质、管理能力等。希望学员们尊重导师，以问题为导向，制定学习方案，学以致用，改进管理方法和思路。

吴江区教师发展中心副主任李红仙宣布聘任导师名单、组织科科长王芳为导师颁发聘书。

北外附属田园高级中学付宗亮书记、颛桥小学唐为民校长、北桥中学梅宝红校长、颛桥中学李卫军校长、鑫都小学蒋红梅校长五位导师纷纷表示吴江市是一个教育发达的地区，有许多经验值得学习。他们将根据学员的实际，量身定制培训方案，全面开放学校的各条线工作，通过外在的“牵引力”和“助推力”共同激发学员的内驱力，在相互学习、相互交流的过程中，共同成长、共同收获。吴江中学徐峰副校长代表学员发言，表示会珍惜机会、遵守纪律，争做专家型校长，努力提升理论是实践能力，并将新理念带回学校，为学校和教师服务。

在后续的培训中，聚焦“规划学校发展、营造育人文化、领导课程教学、引领教师成长、优化内部管理、调适外部环境”校长标准六个模块，结合学员实际需求开展跟岗学习。在比较中学习、在学习中突破、在突破中成长，在长三角一体化大背景下，紧跟时代、顺应潮流、主动融入、协同发展，共同打造两地高质量的教育强区。
