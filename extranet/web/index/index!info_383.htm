规范办学 智慧管理 ——闵行区举行职初校（园）长、书记专题培训暨2018年新任校（园）长、书记培训（第二批）启动仪式

发布时间 : 2018-09-27   作者（来源）: 曹雅明   阅读 : 6556

2018年9月26日，以“规范办学、智慧管理”为主题的职初校（园）长、书记专题培训暨2018年新任校（园）长、书记培训（第二批）启动仪式在区教育学院举行。闵行区教育局党工委书记朱雪平、教育局党工委纪委书记朱寿生、区教育学院副院长（主持工作）朱靖、区教育局各科室长等领导出席。区教育局组织科王治华科长、区教育学院校长发展中心俞琳主任分别主持培训活动。

第一部分： 师徒结对、牵手同行

新任校（园）长、书记一般都是从教育教学业务强手中提拔上来的，如何从点线的工作转换到学校的综合管理？如何成功起步、平稳过渡、致力发展？这是许多新任校（园）长面临的新任务和新挑战。

2018年新任校（园）长、书记培训（第二批）启动仪式上，王治华科长首先介绍了本期培训的目的与意义、导师选聘的原则。俞琳主任解读了为期一年的培训方案内容。一品漫城幼儿园李沈君园长、鑫都小学蒋红梅校长作为学员代表进行发言，表达了对组织培养、对导师带教的感谢之情，并表示将珍惜机会、勤奋学习、务实工作，引领师生发展。莘庄幼儿园郁亚妹园长、实验小学何学锋校长作为导师代表进行发言。郁园长希望用陪伴成长的方式进行带教，对新园长提出了三个要求：喜欢园长的岗位、制订合理的目标、处理多重的关系。特级校长何学锋将用个性化的方式进行带教，希望新校长做到主动学习、按需学习、学中思考、学中提炼。

朱雪平书记和朱靖院长为杨静霞、郁亚妹、丁春燕、林琳、殷蓓琳、何学锋、康旻、查建生、屠红伟、李国兴十位经验丰富的校（园）长、书记颁发了导师聘书，十八位学员与导师签订了带教协议。

随后，朱雪平书记作讲话。他结合闵行区经济社会发展的背景以及新老校（园）长更替的现状，指出了闵行教育需要一支有思想、有作为的校（园）长队伍的现实，并对导师和学员提出了要求和期望：希望导师就“平稳起步、厘清思路、扎实管理、获得认可”四个方面重点进行带教，帮助学员转变角色、胜任工作；希望学员在“适应新环境、谨慎用评价、稳步求发展、倡导正能量”这四方面虚心向导师请教，与导师牵手同行，尽快适应新岗位，持续提升管理素养。

第二部分：实务培训、提升素养

接着，区教育局各职能科室的八位科长，为30位职初校（园）长、书记开展了“基于问题的学校日常管理解析及问答”的微讲座，从各个科室的职责任务出发，为学员补足短板、助力发展。

人保科俞伟珏科长介绍了编制与招录、教师流动、职务评审、绩效考核与分配等有关人事管理的政策、程序，强调了人事管理的重要性。计财科郭廷新科长介绍了财务管理的四个基本方法：一是提高认识，重视财务管理，依法实施经济行为；二是明确设置财务管理的岗位，清晰责任；三是建立健全财务内控制度，规范操作流程；四是定期召开学校经济工作会议，提高预算决算的科学性。基建装备资产管理中心徐伯英主任从基本建设、校舍维修、专项申报、合同管理四个方面，介绍了相关的程序和操作方法。核算中心潘胜泉主任从绩效工资、差旅报销、资产审核、经费管理等四方面介绍了财务最后关口的操作要求和注意事项。

组织科王治华科长从校长职级、党建工作两个方面介绍了相关工作。学前科黄悦科长从幼儿园的课程建设、安全管理、内涵发展等方面作了介绍和指导。普教二科马卫锋科长从招生毕业、内涵项目、特殊教育等方面强调了内涵发展的重要性。普教一科谢凯丽科长通过回顾一年中组织开展的各级各类活动及项目课程，运用大数据分析，帮助校园长梳理出基于校本，学校可以有针对性地参加德、艺、体、卫、科等各类活动，进一步促进学校的内涵发展，并就后续工作提出了五个要求。

八位科长的微讲座从实务管理出发，从学员问题出发，让学员更深刻认识到：规范办学是让学校“活下去”的基本要求，智慧管理是让学校“强起来”的条件保障。

第三部分： 廉洁教育、保持清醒

实务培训后，区教育局党工委纪委书记朱寿生为学员们作党风廉政教育微报告。朱书记介绍了十九大以来，党风廉政建设的新要求、新任务，结合相关典型案例分析了学校管理重点领域廉政教育的紧迫性和重要性，并提出了两个方面的具体任务：一是坚持对标对表，把握形势任务，保持清醒。要求学员带动学校干部、教师加强学习教育，加强监督检查，吸取教训，防微杜渐。二是坚持率先垂范，坚守职责，在纪律和规范方面落到实处。要求学员在守纪律讲规矩上做表率，在讲师德重品行上做表率，在带好队伍强化作风上做表率。

本次专题培训为期一天。后续，学员们将在导师的引领下，以集中专题培训与师徒结对带教相结合的方式，开展基于问题解决的办学实践探索研究，帮助学员厘清办学思路，明确发展定位，在传承与创新中实现学校健康、稳定、持续发展。
