化危为机，助力智慧教育纵深推进—— 闵行区“智慧教育”校园长研修班（第二期）培训启动

发布时间 : 2020-04-09   作者（来源）: 朱凤   阅读 : 2334


4月8日，闵行区“智慧教育”校园长研修班（第二期）首次培训在线进行。教育局局长恽敏霞、教育学院副院长龚耀昌、该项目申报学校的正副校园长、部分研训员、部分骨干教师等共两百多人参与了培训活动。

闵行区教育学院朱靖院长主持活动并解读《闵行区数据驱动大规模因材施教推进计划与实施建议》。朱院长从项目背景，项目设计、项目实施三个方面详细阐述了“智慧教育”的推进计划与实施建议，区域将围绕创建目标，明确数据驱动的大规模因材施教的内涵要素、实施路径、表现形态、成果形式，在区校联动框架下开展大规模因材施教的实践研究，建构基于大数据的评估与诊断模型，建立智慧教育实践研究长效机制，形成智慧教育的经验和成果。同时，建议校园长通过突破与改变，升级智能教育环境、转变课堂教学模、完成管理体系转型，形成智慧教育新生态。

龚耀昌作题为《以项目为载体，扎实推进智慧教育》微报告：诠释区域开展智慧教育项目图谱建设与学校开展项目申报工作的目标与意义，并聚焦典型项目案例给出“项目推进与成果形成”建设意见：把握疫情后学校教学形态的发展方向，紧扣“数据驱动”和“行为改变”，聚焦核心教育命题，在课程教学的关键环节展开探索，在不断的实践、研讨、提炼中形成成果。平阳小学朱红校长和复旦万科实验学校金悦校长分别以《“五育”背景下智能化健康教育课程建设的思考》、《线上线下聚合力，智慧教育促成长》为题，从不同视角分享学校在智慧教育创建工作中的思考与实践探索。

恽敏霞作《疫情后的在线教学和智慧教育》专题报告。恽局围绕“瘟疫与人类文明的共存发展历史、疫情下的在线教学、智慧教育创建的前景与展望、教育现代化中校园长的责任与担当”四方面阐述观点：人类文明进程从来都是曲折艰难的，我们要学会从科学与历史的角度正视疫情，与致病微生物和瘟疫共生共存是人类社会发展至今的真实经历，我们需要深省疫情将给这个世界带来的巨大影响；在做好当下开展的在线教学中，作为学校教育管理者，要明确我们正在经历什么，学会总结得失，线上教学一定会带来传统教学的变革；5G+智慧教育探索创新应用需要借助多方专业力量，同时也依托学校、校长、教师的智慧，在共同探索中明确智慧教育的内涵与实施路径；教育现代化就是以公平、均衡、优质为教育取向，为此，我们需要牢固树立立德树人根本任务，明确信息技术在优质资源共享、促进教育公平，提高教与学效率、提升教师专业化水平，推动全社会终身学习等方面的不可替代作用。校园长应适应全媒体时代带来的新变革、新要求，帮助每一位教师提升网络教学能力，在教育改革中勇担当、敢负责。

本期培训共六次，通过项目引领，专家理论指导、校园长案例分享，助力学校教育管理者加深对智慧教育内涵与外延的理解，并开展区校联动下的智慧教育实践探索。
