科研赋能教育教学——闵行区第五期“优秀教师培育工程”专题报告系列（四）

发布时间 : 2021-09-27   作者（来源）: 朱凤   阅读 : 1362

9月26日，闵行区“第五期优秀教师培育工程”专题培训在区教育学院如期举行，浦东教育发展研究学校发展中心副主任王丽琴做专题报告《科研赋能教师写作》，校长发展中心主任俞琳主持活动。

本次培训分为案列分享、专题报告、互动交流三个环节。

七宝中学柳英华、明强小学刘依婷两位老师分别分享了题为《科研伴我成长》、《疑中生趣、探中解惑》案列，剖析了课题的来源、课题的研究过程以及成长与收获，聚焦教育教学中的真问题，理论先导、实践佐证、调整优化的迭代式发展，实现科研增值。随后七宝中学杨睿、文绮中学徐虹、华漕学校王玉敏三位学员分别代表综合组、文科组、理科组提出了课题研究过程中的困惑。

王丽琴分析了学员们在科研过程中提出的高频词，围绕“选题焦虑、成果焦虑、方法焦虑、理论焦虑”量身定制专题报告。以“三次拆解”来回应学员的困惑，一、先拆解一个结题报告——“以终为始”，把握课题研究的要件，二、再拆解一个课题下的系列论文——“论文之四有”，重中之重是有核心观点！、三、最后拆解研究报告、专题论文里的课例、案例——“课例”“案例”如何有机嵌入科研成果。她结合大量的案列，深入浅出的分析课题研究的要素及相应的内容、论文和课题间的关系是过去式和未来时的区别等等。

聆听了王主任的报告后，闵行中学韩海坤、文来实验学校李松涛、实验小学毕明芳三位学员代表表示报告接地气，逻辑性强，过程方法可操作性。既有理论的高度，实践的厚度，又有思维的深度，架构了理论和实践的桥梁，疏通了课题研究过程中的堵点，解构了课题研究中的难点，重新点燃了学员们课题研究的信心。以教育家精神办闵行教育，善于修炼学术，仰高钻坚，博学经史，理念独具，风格独树。
