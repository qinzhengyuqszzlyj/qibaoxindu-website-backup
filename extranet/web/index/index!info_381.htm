议题问题课题兼容 研学研教研评贯通 ——闵行区中小学学科教研活动展示（初中物理专场）暨闵行区重大课题《学评一致性视域下课堂形成性评价的设计与实施》中期汇报

发布时间 : 2023-05-15   作者（来源）: 高剑英   阅读 : 746

3/8
←
→

2023年5月10日下午闵行区中小学学科教研活动展示（初中物理专场），暨闵行区重大课题《学评一致性视域下课堂形成性评价的设计与实施》中期汇报在文来中学隆重举行。

本次活动由初中物理青师六期学员孙铁楠、史小亮主持。出席活动的有上海市物理教研员，上海市特级、正高级教师汤清修；闵行区教育学院院长朱靖、副院长龚耀昌，教研中心主任朱诵玉、孙静贤，闵行区教科研专家；区初中物理全体教师。

一、探索新教学——展现智慧纸笔支持下的形成性评价

本次主题汇报活动在形成性评价概念阐述与辨析后拉开帷幕，首先由青师六期学员徐佳琳执教了“电功率复习专题”一课。徐老师以智慧笔为媒介推动课堂形成性评价的展开，以扎实的教学功底、优雅的教姿教态与有效的形成性评价手段呈现了一节评价诊断-反馈-激励学生学习的精彩课堂。

二、研问在课堂——引入学生评议开拓形成性评价增值空间

课堂聆听观摩后，迎来了辩论式评课环节。围绕“学习活动的设计与组织是否体现了学评一致性视域”与“课堂的形成性评价是否有助于达成学习目标”的正反辩题进行激烈的思辩，来自七宝三中、莘光学校、北桥中学、上师大三附中四所学校的教师代表就如何拓宽评价和反馈方式的深度、广度进行了精彩发言。徐佳琳老师博采众长进行反思型说课，针对教师代表提出的问题作出回应，并就“课堂生成资源”和“学生问题的持续跟进”进一步反思说明。

三、演绎式反馈——编制情景剧呈现阶段研究历程与深度思考

回顾课题研究过程，凝聚着每一位专家导师的研精覃思，对于青师六期的学员来说更是自我修复与不断提升的过程。青师剧场以情境演绎的方式再现了课题研究历程，以课堂现问题—导师拨云雾—实践促研究—成果改课堂的剧幕，将形成性评价在教学中的实践与研究分享给每一位老师，生动呈现了研究驱动的共享学习机制。

四、专业化指引——高位引领精准剖析点拨智慧可望集腋成裘

活动尾声，汤清修老师对主题展示课中信息手段的使用、辩论式评课的价值以及青师班的研究模式三个方面进行了深入浅出的点拨和指导。同时提到辩论式评课以独有的正反方形式促进了深度化与证据性评课。此外汤老师肯定了青师班以课堂实际为出发点的研究模式，期待这样的青年教师培养方式推广到全市范围。

龚耀昌副院长以“研什么”、“怎么研”、“为何研”三个发人深省的问题切入，为全区下一步智慧型精准教学指明了方向。最后，朱靖院长为本次活动做了总结。朱院长提出汇报中体现了“智慧笔使用”和“形成性评价”两个难点在实际教学中的运用，为区域内的突破性探索打下了扎实基础。同时指出以教研训一体化的方式组织主题式的教研活动，能助力闵行教研活动质量的提升，有效带动区域内教育教学质量。

本次活动得到了市、区专家领导的鼎力支持，高屋建瓴的精彩分享开拓了学科研训的视角。借助这一平台，青师六期学员们展现了自身的风采和努力，同时更坚定了以智慧的探索扎根于课堂，将在研究的路上，永不止步。
