比较·学习·突破·提升 ——2018年闵行区教育系统中学后备干部班集中培训顺利开展

发布时间 : 2018-11-07   作者（来源）: 曹雅明   阅读 : 2586

10月29日至11月2日，2018年闵行区教育系统中学后备干部集中培训在青浦区陈云纪念馆和杭州育才中学等地进行，本次培训的主题是“坚定信念 锤炼品质 做新时代的优秀干部”。闵行区委组织部副部长汪丹，区教育局党工委书记朱雪平、局长恽敏霞，党工委副书记李光华、纪委书记朱寿生、教育局副局长李啸瑜、区教育学院党总支书记（主持工作副院长）朱靖、教育局副调研员刘兆利等领导出席，来自区域各中学的30位后备干部参加培训。

为期五天的培训分为集中学习和实地参访学习两部分。

集中学习：在比较中学习  在学习中突破  在突破中成长

10月29日至31日为集中学习，多位领导、专家为学员们呈现了一场场精彩的报告和讲座。

10月29日，在教育局组织科王治华科长的主持下，闵行区教育系统中学后备干部集中培训开班仪式在陈云纪念馆举行。教育学院校长发展中心俞琳主任从本次培训的目标定位、学习内容与形式、培训制度等方面解读了本次培训方案，并对全体学员提出了相关要求。王治华科长希望学员们在培训中努力提升、不断完善，早日成为学校的排头兵。

朱雪平书记在开班仪式与结业典礼上作重要讲话。他要求学员们要有全局意识，主动了解教育教学发展的最新动态；要善于团结，明辨是非，争取各方力量，为学校发展营造良好氛围；对待工作要有追求、有激情，并能感染、带动教师。在听取了学员的汇报交流后，他肯定学员们的学习激情、团队合作意识及积极主动的思考和有成效的思维提升，希望学员们在今后的工作中明确党对教育的领导，明晰培养目标，把握大形势，并努力做好自己，在成事中成人。

闵行区委组织部副部长、区社会建设工作党委书记汪丹为学员们作了《不忘初心 感受信仰之力》的专题报告。汪部长从行动的哲学、创造的哲学、务实的哲学、人本的哲学四个角度，结合生动的案例向学员们阐释了马克思的哲学理论在新时代的现实意义，并提醒学员们要将理论与实践紧密结合，用信仰指导行动，提升工作效能。

恽敏霞局长为学员们作题为《综合改革背景下闵行教育的现在和未来》的专题报告。作为闵行教育的掌舵人，恽局长高屋建瓴，如数家珍地向学员介绍了闵行教育的体量、资源、发展趋势和目标愿景。她指出，闵行教育的目标是要建设一批好学校，培养一支好队伍，尤其强调师德提升和后备干部素养提升的重要意义，她谆谆嘱托学员们要认真学习，扛住压力快速成长，为闵行教育优质、均衡、快速发展贡献力量。

教育局党工委副书记李光华、纪委书记朱寿生、副调研员刘兆利分别作了《新时代青年干部成长》、《学校党风廉政建设》、《学校财经纪律解读》的微报告。三位领导明确指出，干部成长的第一要务是要有梦想和信念，要经历“学、悟、炼”的过程，做好当下，赢在坚守。同时，要保持对纪律的清醒认识，要秉公用权、依法办事，要敬畏组织、敬畏群众、敬畏法纪，在各种利益诱惑面前要保持定力，在是非面前要有辨识能力，干净做事、清白做人。

上海外国语大学闵行外国语中学吴金瑜特级校长、协和教育集团总校长卢慧文、蔷薇小学沈珺特级校长分别作《中高考改革背景下的机遇与挑战》、《整合各类教育资源，助推学校健康可持续发展》、《“借力”助推学校发展》的专题报告。为学员们分析了中高考改革的本质内涵，以及我们该有的行动；让学员明白当今社会并不缺乏教育资源，关键要学会取舍与科学整合；指出只有“用心做、坚持做”才能做好一件事，“用情做、智慧做”才能事半功倍。陈云纪念馆原党委书记马继奋用生动、精彩的语言，翔实、丰富的史料向大家介绍了陈云同志的革命事迹，以及他“不唯上、不唯书、只唯实，交换、比较、反复”的重要思想，希望学员们学习陈云同志学以致用的学习作风、调查研究的工作作风、廉洁朴素的生活作风，以及“不唯上、不唯书、只唯实”的思想作风。

本次培训坚持发挥学员的主体作用，实行班委会领导下的“班委+小组组长”自我管理模式。每天的培训主持、影像拍摄与制作、通讯撰写、日常事务管理等均由学员自己完成。集中培训压力大、任务重，但学员们准备充分，变压力为动力，每天利用早、中、晚课余时间交流、学习。有序的组织管理，精细的任务分工，营造了团结协作、积极向上的班集体氛围。学员们珍惜每一次的学习机会，专家报告后，纷纷与领导、专家进行互动交流，各小组还围绕专家报告的主题进行了三场“学员讲堂”，大家打开思路、分享感受、深化认识，提升对教育的理解。结业典礼中，学员们分别用TED演讲、情景剧论坛、圆桌会议等多样化的形式，针对“新时代、新政策、学校新发展”、“陈云思想在学校师德师风建设中的运用”、“自媒体时代学校如何应对舆情”、“中高考改革给学校带来的机遇与挑战”四个议题，结合几天来的学习体会向领导和同伴作了交流汇报。

区教育局李啸瑜副局长、教育学院朱靖院长对结业汇报作了点评。两位领导肯定了四组学员的表现：汇报交流设计有核心思想，能够学以化用，个体表现放松、真实，有思想。同时希望后备干部从道、法、术、器四个层面作思考，在专业站位、政治站位、实践站位三个方面更高要求自己，实现提升。

实地参访学习：在参观中学习 在学习后思考 在思考中沉淀

后面两天主要是实地参访学习。杭州学军中学紫金港校区，硬件条件先进，师资力量强大，科研设施完备，管理严格，学风浓厚。继余林副校长报告之后，学员们在学校信息化智慧指挥中心，围绕校园文化建设、学校教育教学管理、特色课程开发，特别是学军中学基于信息化背景下的教学改革、云录播课堂、学校物联网等与余副校长进行了深度交流。

来到杭州育才中学，聆听了郜晏中校长两个多小时的精彩报告，学员们深深感动的同时，更多的是敬重与佩服——这是一位兼具教育家情怀和企业家睿智谋略的校长：18年前初创时，在没有资金支持、没有生源保障、没有固定场所的艰难条件下，郜校长和他的领导团队硬是凭借“样样落实，天天坚持”的治学、处事、为人之道，形成了“稳定优异的质量，朴素实效的德育，独树一帜的教学”三大学校品牌，“一身正气”的第一价值观，“抓得实，盯得牢，学得好”的教学特色，更是闯出了一片民办教育的新天地。学员们被震撼的同时也清醒地认识到：唯有坚守与坚持才是教育成功的不二法门。学员们还来到了杭州武警士官学校，感受不一样的学校管理，感叹学校钢铁长城的严谨与威严，并通过心理拓展活动进一步加强了团体凝聚力。

五天的集中培训结束了，30位闵行区教育系统中学后备干部在其中比较、学习、突破、提升，积蓄能量，迅速成长。学员们必将以终为始，始终牢记本次培训的经历，不断坚定信念，锤炼品质，做闵行教育新时代的优秀干部，为闵行区由教育大区走向教育强区而不懈奋斗！
