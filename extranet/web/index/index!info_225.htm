传承传统文化 用生活劳动传递爱与温暖 ——2023闵行区中小学新春劳动作品展评掠影

发布时间 : 2023-05-29   作者（来源）: 刘至宁   阅读 : 1147

随着2023年闵行区中小学生“创造美好家居  喜迎新春佳节”劳动作品展评活动的圆满落幕，涌现出很多富有传统文化内涵的劳动作品，也体现了学生在新春佳节之际，与亲朋好友一起回归到真实生活中，用真实的生活劳动传递着爱与温暖。

本次展评在区云录播平台上的评优活动里进行线上展评，共上传作品82件，获奖率近100%，以学生自主策划、自愿参与为主。本次参展的劳动作品在内容上主要体现了花卉盆栽、彩灯装饰、写春联、贴春联、挂灯笼、水仙花及插花造型、创新小手作等各种方式进行居家环境的布置和装饰，目的是让学生在节日快乐祥和的氛围里，传承传统文化，更能真正理解劳动的价值、劳动的意义和劳动的美好。

在诸多新春劳动作品中，值得关注的是陪伴家人的亲情式劳动，占比作品的90%左右，体现出用劳动去创设喜气和乐的新春家庭氛围。比如学生与家人一起包饺子、蒸面鱼、做糯米圆子；还有的亲朋好友在一起写书法、贴春联、挂红灯；还有的学生跟着外婆学种花、与家人一起摘草莓、做插花等，用劳动去陪伴家人，一起分享劳动成果，从每一位孩子脸上的幸福笑容就可以看出，新春劳动给他们带来的幸福与快乐，每一件劳动作品都陪伴他们过一个美好祥和的中国年。现将劳动作品展评中的部分获奖作品做一个分享与展示，看看孩子们的笑脸，一起感受劳动的快乐与美好。

经区中小学劳动技术学科中心组及核心成员的投票评选，分别评选出小学组一、二、三等奖获奖作品25件；初中组一、二、三等奖获奖作品54件，获奖名单已公布。详见链接：

https://jyxy.mhedu.sh.cn/extranet/web/index/index!edunoticeInfo.action?id=ff80808186bb2eed0187b132189b1cde）
