“智”领未来 技术赋“能” —— 闵行区中学智能创造研修坊培训活动在闵行区梅陇中学开展

发布时间 : 2023-06-30   作者（来源）: 马丽敏   阅读 : 609

闵行区中学智能创造研修坊是一个旨在帮助教师提升动手实践能力以及智能控制类课程开发和实施能力的平台。研修坊的学员学科背景多元，有来自闵行区各所初高中的负责科技创新类课程的教师，来自青少年活动中心的指导教师，还有上海市史坦默国际科学教育研究中心的研究员。6月29日上午9点，研修坊学员们齐聚上海市闵行区梅陇中学“为乐”科技实验室，开启了第一期智能创造研修坊的最后一次活动。

闵行区梅陇中学沙越老师首先为大家介绍了学校科技社团自主设计的六种常用传感器，带领老师们探究传感器的工作原理和制作方法。

随后，上海市史坦默国际科学教育研究中心的张伽蓝老师通过声控氛围灯、防盗震动车灯等生活中常见的场景将“新鲜出炉”的传感器组合运用，为大家解密了生活中的智能作品的设计方法。

下午，闵行区科技指导专家丁志康老师为学员们带来了满满干货，针对前期大家在智能作品设计和制作中遇到的实际问题，丁老师重点解读了继电器模块和语音模块在智能作品中是如何运用的，并分享了自己在长期实践过程中的宝贵经验。

一整天的研修坊活动安排非常紧凑，但学员们都沉浸其中，学得聚精会神，玩得不亦乐乎。

随着此次培训的落幕，第一期研修坊暂告一个段落，学员们在系列活动中收获满满，但仍然有许多有趣的课题值得我们挖掘，让我们期待下一次的相聚，继续同行在智能创造研修之路。
