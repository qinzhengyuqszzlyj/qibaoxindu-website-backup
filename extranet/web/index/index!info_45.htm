新课标视域下小学英语单元整体教学实践活动——闵行区优秀教师培育工程小学英语学科专场展示活动

发布时间 : 2023-09-25   作者（来源）: 朱凤   阅读 : 617

3/8
←
→

金气秋分，桂子飘香。9月22日，闵行区优秀教师培育工程小学英语学科专场展示活动在田园外语实验小学举行，主题为“新课标视域下小学英语单元整体教学的实践研究活动”。闵行区教育学院副院长董学平、上海市教师教育学院小学英语教研员朱浦、祁承辉、田园外语实验小学赵瑛群校长、区教育学院校长发展中心主任俞琳等出席了活动。区小学英语教研员沈雯晴主持活动。

伴随着田园外语实验小学孩子们美妙的歌声，开启了本次活动，共四个篇章。

第一篇章：述·育人缘起。七宝外国语小学王晓燕和颛桥中心小学张佳萍阐释了项目的研修背景和主题，七位学员在朱浦导师的指导下，持续深入探索单元整体教学实践研究，收获成长。第二篇章：研·育人变革。通过短视频分享了学员们的研修故事，见证成长。明强小学邓茜和闵行区中心小学陈雪萍做研修专题分享，围绕主题，以多模态语篇为抓手，从理念到实践进行阐述，提升学生对语言的学习。第三篇章：展·课堂生长。课堂是实践理论，实现专业成长的主阵地。上海师范大学附属闵行第三小学李腾蛟和区教育学院殷素婷进行课例说明，田园外语实验小学高晟进行了课堂教学展示。凸显了学思结合、用创文本的学习活动观。第四篇章：越·追光前行。祁承辉老师充分肯定了本次活动的意义和影响，通过案例分析总结了成功的三大要素：基因、坚持、创新。利用“基因”，顺势前行；坚持探索，持续前行；敢于创新，开拓前行。朱浦导师以“他们是谁？”、“他们来做什么？”、“他们靠什么成长？”分析了学员的已有成绩，专业方向和发展路径，鼓励学员们耕耘理论、沉积思路、定义路径，实践课堂，积淀成长。他表示会继续支持闵行教育，做专业的伙伴、学科的同行、成长的知己。

董学平做领导讲话。他首先感谢导师的付出，也肯定了学员的成长。专业展示本身也是一种培训方式，为学员成长提供内动力，推动教学实践的梳理与凝练，推动研修成果的彰显与固化，推动优秀经验的传播与分享。期待每一场专业展示都能有规格、有品质、有内涵，为闵行教育营造教学研究与专业成长的浓浓氛围。他表示在各级专家、领导的支持下，将进一步深化闵行教育强区建设，为更多教师的专业发展搭台、鼓劲、喝彩。 期待学员们呈现一场场精彩的“学术”盛宴。

闵行区第七届小学英语学科带头人、骨干教师、第八届小学英语学科骨干后备教师观摩了现场。
