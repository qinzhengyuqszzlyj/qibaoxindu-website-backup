新课程背景下上好音乐课的观念更新与方法跟进 ————闵行区2023学年第一学期小学音乐区级教研活动

发布时间 : 2023-09-15   作者（来源）: 刘庆江   阅读 : 1257

3/8
←
→

2023年9月14日星期四下午，2023学年第一学期闵行区小学音乐区级教研工作会议在闵行区实验小学畹町校区五楼梦想剧场举行，教研员刘庆江老师和全区小学音乐教师全程共同参加此次活动。全场活动分为三个部分。

第一部分是闵行区6位新进音乐教师的基本功汇报环节。本年度新进的音乐教师，分别是毕业于上海师大，华东师大等音乐学院的硕士研究生。年轻的教师们给活动带来了女生二重唱《山楂树》，小提琴独奏《花儿为什么这样红》、钢琴弹唱《思乡》、三弦独奏《十八板》和歌舞表演《九儿》等丰富精彩的节目，展现新进教师的专业素养和综合表演能力，为闵行区小学音乐教师队伍注入了新的力量。

第二部分是闵行区田园外国语实验小学的音乐教师曹露露介绍英国公立学校的音乐教育情况。曹露露老师以《英国公立小学音乐教育的课程标准及基于MMC下恒拍练习的方法》为主题，除向全区小学音乐老师们分享了英国国家音乐课程的总目标、两个不同关键阶段的分段目标、儿童音乐素养发展的四个关键领域及不同阶段的培养内容外，曹老师还分享了在示范音乐课程里建议的节拍练习的三种方法：身体打击乐(Body Percussion)、音乐图(Musicograma)、打击乐(Percussion Instruments)，并与现场的老师们进行了声势律动和演奏打击乐器的互动，让老师们深刻直观地感受到了三种不同方法结合所带来的课堂效果。

第三部分小学音乐教研员正高级、特级教师刘庆江老师作新学期的区小学音乐教研工作报告。刘庆江老师以《新课程背景下上好音乐课的观念更新与方法跟进》作主题讲座，明确新学期的教研工作主题与方向。刘老师从新课标描述新观念，新观念下的新方法，新观念面对的新挑战，对未来的期待与展望等四个方面，结合自身几十年的教育、教研的经验，要求全区音乐教师在新课程的背景下，持续学习，更新教学观念，整合课程资源，持续推进大单元的教学设计，加强音乐学科的学科知识与技能的递进式的教学，培养学生的音乐学科核心素养与能力。

全区教研工作会议之后，教研员刘庆江老师和区音乐学科中心组的老师们在五楼会议室召开了新一届区学科中心组会议。会议中，刘庆江老师对新的小学音乐中心组教师进行了教研工作的分工，明确每位中心组教师的工作职责与任务，勉励中心组教师要有团队意识，服务精神，协助做好本学期的各项区级教研工作，同时，要立足自身专业特长，引领全区音乐教师的专业发展。会议到当日下午五点一刻结束。
