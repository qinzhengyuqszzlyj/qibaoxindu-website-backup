提升管理智慧 打造优质园所 ——闵行区第四期幼儿园园长（七宝中心幼儿园）基地培训结业仪式

发布时间 : 2023-08-27   作者（来源）: 曹雅明   阅读 : 1197

3/8
←
→

秋风渐起夏未尽，2023年8月22日上午，闵行区第四期幼儿园园长（七宝幼）基地培训结业仪式在闵行区梅陇梅锦幼儿园顺利举行。

闵行区教育党工委书记恽敏霞，区教育局党办副主任陈妍出席了本次活动，基地学员梅陇梅锦幼儿园园长李芸芳主持结业仪式。

教育兴则国家兴，教育强则国家强。整场活动以“全面建设高质量幼儿园的思与行——提升管理智慧 打造优质园所”为主题，紧紧把握习总书记在中共中央政治局第五次集体学习时的讲话精神，以建设高质量幼儿园为目标，对园所的智慧管理进行了总结与提炼。

 

回首过往，是收获，是圆满

微视频《匠人匠心 打造高质量教育》回顾了本期园长基地两年当中开展的活动过程与学习经历。短短的片段，长长的回忆，一幅幅画面结成了基地活动的生动画卷，一组组数据展现了基地培训的丰硕成果。

 

思想交锋，是感悟，是提升

园之发展，管理先行。浦江瑞和城幼儿园黄静园长围绕“提升管理智慧 打造优质园所”发起精彩论坛，浦航幼儿园施春美园长、颛桥镇第二幼儿园黄蔚艳园长、颛桥镇田园都市幼儿园乔雅萍园长、梅陇梅锦幼儿园李芸芳园长、马桥启英幼儿园沈磊园长，共同分享了自己担任园长以来的管理经验。“党建引领”、“立德树人”、“转变角色”、“打开格局”……高质量办园之路上，基地学员们总结出各具特色的管理秘籍，诠释了各自对教育管理的理解与感悟。

 

深度耕耘，是挑战，是蜕变

专题汇报环节，颛桥镇田园都市幼儿园乔雅萍园长讲述了深耕园所管理的历程，她带领全园师生员工坚实步伐，通过“7654321”个改变，实现了园所环境的焕然一新；通过激发成长内驱力，实现教师专业性的稳步提升。正所谓步步坚实求发展，精心管理促蜕变！

 

传道授业，是欣慰，是期许

“非我而当者，吾师也”，基地导师丁春燕园长肯定了六位基地学员在理论水平、实践能力、创新意识以及办学思想方面的提升与成长，也直言不讳指出了小小的“缺憾”。丁园长对基地学员提出了几点殷切期许：作为高质量园所建设的领航人，要有教育情怀和教育智慧，会埋头做，也时常仰望星空；做好发展规划，想到前，做到前；通过课程建设实现立德树人；做好人文关怀，善于用人；勤于学习，积累内涵，提升课程领导力。

 

点睛之语，是教诲，是祝福

毕业了，带着美好的祝福，恽书记和丁园长为基地学员隆重颁发了结业证书。

恽书记对本次活动和基地两年来的学习历程给予了赞许，充分肯定了基地六个园所稳步扎实的发展，称赞了六位园长在论坛中对于“管理”这一论题的感悟和总结。恽书记强调，启蒙教育是基础教育的排头兵，任务难而重。如何建设高质量的教育，办好人民满意的教育，她给出了三点中肯的建议：顺势而为，遵循孩子的成长规律和教师的发展规律；有目标感，制定共同的愿景，克服困难，凝心聚力；重谋略，有格局，要“勇谋善断”，更要做到“硬话软说”。最后，恽书记祝福六位园长“越来越有园长的样子，成为教育家”。

 

歌声传情，是欣喜，是期待

毕业期满，是终点，更是起点。活动的最后，六位基地学员用一曲《最好的未来》传递对未来更好的憧憬。

“闵”而好学，“行”稳致远。基地学员将不断地在工作中汲取奋进的力量，提升管理智慧，为闵行教育高质量的发展贡献自己的绵薄之力。
