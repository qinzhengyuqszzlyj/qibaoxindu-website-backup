质量立校，提升管理智慧和能力 ——闵行区第六期小学校长（田园外小）基地举行研修结业活动

发布时间 : 2023-06-21   作者（来源）: 曹雅明   阅读 : 826

3/8
←
→

质量是学校发展的核心，校长作为学校的掌舵人，需要不断提升管理智慧和能力，才能更好地推进学校的高质量发展。2023年6月19日下午，闵行区第六期小学校长（田园外小）基地研修结业活动在田园外小金都校区举行，旨在推动质量立校意识，提升管理智慧和能力。闵行区教育党工委书记恽敏霞、党群科科长林火生，教育学院校长发展中心主任俞琳出席活动。上海市教师教育学院副院长纪明泽、上海市静安区大宁国际小学校长徐晓唯莅临指导，奉贤区教育学院附属小学校长何哲慧连线互动。上海市特级校长、闵行区田园外语实验小学校长赵瑛群主持活动，基地学员、何哲慧特级校长工作室学员及田园外小管理团队共同参与了活动。

本次活动以《习近平总书记在中央政治局第五次集体学习的重要讲话精神》导学拉开序幕，并以视频的方式回顾了两年紧凑又美好的培训时光——专家指导，学校访问交流，以及互动研讨、读书分享、专题交流等等，展现了田园外小校长基地丰富的培训课程，专业的导师团队以及和谐融洽的合作关系。

基地学员闵行区昆阳路小学党支部书记夏权、浦江二小党支部书记赵美华以TED演讲的方式，分享了《脚踏实地，行稳致远》和《精耕细读，课程育人》两篇学校管理案例。夏书记围绕学校发展目标制定教师培养目标，以中华优秀传统文化为引领，搭建更多的发展平台，使学校办学质量得到加速度进步。赵书记立足“耕读树人，健康灵动”的办学理念，汇报了学校主题式综合活动课程实施情况，探索了如何强化整体的课程内容与育人目标一致性，推动学校高质量发展。

基地导师赵瑛群校长带领两位基地学员：闵行区明强二小校长郭芳、鹤北小学校长蒋红梅，与三位“何哲慧特级校长工作室”学员围绕“教育高质量发展”，呈现了一场精彩的主题论坛。学员们结合个人办学经历，从学校管理者的角度，围绕教育高质量发展的目标、内容、实施路径阐述了五个方面的认识，进一步激发学生发展潜能、构建教师队伍成长、赋能学校办学管理。

林火生科长和俞琳主任为基地学员颁发了结业证书。两年的培训让各位校长学员收获了教育的新理念，在学校管理中不仅要做教育思想的领跑者，更是要成为新时代的教育楷模。

徐晓唯校长作专家点评，并分享了自己的所思所想。她谈到校长之于学校，犹如灵魂之于躯体，欲强教育先强教师，欲强教师先强校长。校长在研修过程中“研”的是分析、探求管理之道，“修”的是管理见识、立意和素养。校长们要在“优秀走向卓越”的过程中，完善师生维度的高质量发展，实现为党育人，为国育才的根本任务。

纪明泽副院长从宏观层面、中观层面、微观层面三个层面强调了高质量发展是全面建设社会主义现代化国家的首要任务，校长队伍是基础教育改革发展中具有引领性、推动性、保障性的重要资源力量。并期许在座的校长们能够坚守优质学校发展理念，助推教育高质量发展的创新之路。

恽敏霞书记对即将结业的校长学员们提出了更高要求，要建设高品质课程、建立高素质队伍，要加强高标准建设、推进高智能数字化、提升高效能治理。恽书记表示有思想的校长才是真正意义上的好校长，书记、校长要成为管理中的“政治家”、“教育家”，让学校成为每一位教师幸福成长的地方，每一个孩子乐于快乐学习生活的地方，这才是教育高质量发展的内涵要义和实践路径。

两年的研修，忙碌而充实，感恩遇见，铭记美好。感谢专家们的高位引领，感谢领导们的保障支持，为学员校长的专业成长之路点亮一盏盏明灯。感谢学员伙伴们的相帮互助，让学习一路欢笑、一路歌，行而不辍，未来可期。如今的终点亦是明天的起点，希望校长学员们永葆教育初心，做新时代的教育楷模，在追求高质量的教育道路上行稳致远,砥砺前行。
