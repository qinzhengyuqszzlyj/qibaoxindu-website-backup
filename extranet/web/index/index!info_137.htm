小课本拓展大教材 ——区小学道德与法治教研暨智慧教育研讨活动在闵行区华坪小学举行

发布时间 : 2023-10-16   作者（来源）: 王玉兰   阅读 : 821

3/8
←
→


2023年10月12日下午，小学道德与法治课堂“大资源的开发与应用实践”主题教研系列活动之“资源为先 小课本拓展大教材”闵行区小学二年级道德与法治学科研讨活动在上海市闵行区华坪小学举行。

闵行区教育学院道德与法治教研员王玉兰老师莅临指导本次活动，华坪小学徐黎校长、闵行区二年级道德与法治老师近100人共同参与。活动由区中心组成员江川路小学周萍老师主持。

一、 “思政金课”展风采，精准教学育素养

陆叶老师执教的是二年级上册第一单元《我们的节假日》中的第3课《欢欢喜喜庆国庆》第二课时。课前，陆老师充分利用江川场馆设施资源，鼓励学生去实地探访汽轮厂厂史馆、文化中心、电机厂劳模墙，了解厂史、走进劳模。课中，引进汽轮厂党建中心王漫姐姐、劳模过国庆、劳模国庆加班和劳模精神等人力资源、知识资源和精神资源，通过新旧对比、互动交流、对比反思等方式，引导学生学习劳模精神，鼓励学生合理安排节假日生活，用实际行动继承和发扬劳模精神，激发学生参与建设祖国的意愿，厚植家国情怀，培育政治认同核心素养。

二、“活动园地”巧学习，智慧论坛齐分享

本次论坛的主题是“资源为先，小课本拓展大教材”，由华坪小学陆洲老师、陆叶老师、顾燕红老师，莘庄镇小文朝霞老师和江川路小学周萍老师共同参与。陆洲老师向大家分享了“水宝宝假日秀”的单元活动介绍。她从单元活动的“设计、实施、评价、单元精准教学实践”这四方面阐述了二上第一单元活动的设计意图。以“乐享暑假” “巧度周末”“喜庆中秋”“欢度国庆”这四个单元子活动为推手，充分挖掘教材资源和江川地区资源丰富思政课教育内涵，又结合三条单元学习思路和推进单元精准教学的实践探索，丰富学生体验与感受，促使学生情感态度价值观有温度地提升。陆叶老师从本课课堂教学实践出发，着重交流了资源为先，实施精准教学的四条途径“突破时空限制，设施资源共享；恰当选择提炼，精神资源引领；勾连生活实践，人力资源共情；优化呈现方式，知识资源转化”。

顾燕红老师立足华坪小学智慧教育实践，从创意环境打造到精准教学实践，分享了华坪小学在技术赋能课程，助力精准教学中的亮点和生长点。文朝霞老师交流了莘庄镇小在实施本单元教学中，结合莘庄地区资源探索出“单元问题链推进活动任务；资源新路径提高育人效果；表现性评价培育核心素养”三条单元教学策略。周萍老师以观察者的角度，分享了华坪小学在实施同一单元教学中，结合江川地区资源探索出的“活用素材性资源，建构生活性课堂；巧用条件性资源，培育实践性课堂；妙用课程外资源，创生情感性课堂”的经验。思政小课本和社会大教材是相互联系、相互支持、相互作用的。对本地区资源进行挖掘、转化与利用，让资源先行，可以使两者有机统一，有效弥补课程资源的不足，还可以创新教学的内容与形式，促进思政课堂变革，从而彰显大思政课作为落实“立德树人”根本任务关键性课程的独特价值。

三、专家引领明方向，学思践悟促提升

闵行区教育学院道德与法治教研员王玉兰老师对本次教研活动做专家点评。她认为华坪小学的教研创新了三个高度：单元高度、智慧高度、思政高度。华坪小学整个道德与法治教研组对课标和教材都有深度的解读，对江川地区资源的开发也有自己独特的思考，课堂呈现出了丰盛的一个课堂资源，单元活动设计的分享和整个论坛高品质、高质量。

这场教研活动提供了三个非常宝贵的借鉴和思路。第一、同单元异构的教研创新。华坪小学依托江川地区老闵行工业发展的历史，莘庄镇小依托莘庄地区的乡土文化、革命文化，分别进行了以单元为整体的活动方案设计，基于课标开展任务串、任务链、表现性评价等进行广阔教育视野下的教学实践。 第二、智慧笔应用的扎实开展。大单元教学设计实施过程中，教师利用智慧笔把握学生的思想状态、内心需求，以及对事物核心概念的价值判断，实施精准教学，提高学生学科素养和品格。第三、大思政资源的开发创新。充分挖掘学校、区域的历史、地理、人文资源等思政课“大资源”。充分调动社会力量和资源，将区域“大师资”请进来，推动思政小课堂与社会大课堂相结合。

华坪小学徐黎校长全程参与了本次活动，她评价这是一场高质量的区级教研活动。她非常感谢教研员王玉兰老师一路对华坪的支持和指导，感谢兄弟学校的帮助，感谢街道和学校共建单位的全力支持，肯定了学校教研团队在课堂实践中的创新与思考。此外，她指出这次活动的开展为学校管理带来了新的思考和方向，未来将精准定位学校特色，精细设计校园活动，精益化管理教学团队。老闵行正以崭新的姿态，迎接新征程。活动结束，到会老师都兴致勃勃地赶到川上美集，感受新时代下江川人民的幸福生活。

在党的二十大和习近平新时代中国特色社会主义思想的精神引领下，闵行区道德与法治教师将在区教研员王玉兰老师的带领下共研共享共成长，跳出课本看课本，努力将思政小课本拓展成社会大教材，打造地区资源鲜明、学生立场鲜明、价值引领鲜明的思政金课，让每一个孩子在课堂中共情、共生、共成长，充分彰显大思政课落实“立德树人”根本任务的独特价值！
