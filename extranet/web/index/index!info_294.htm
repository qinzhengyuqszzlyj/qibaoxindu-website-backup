创生激活生命成长的校园新生活 ——2018年闵行区新任校（园）长带教培训赵瑛群导师组活动通讯（4月）

发布时间 : 2019-04-28   作者（来源）: 曹雅明   阅读 : 1737

草长莺飞，最美人间四月天。4月，学校四季综合活动在如火如荼地开展，整个校园沐浴在和煦的春风中，生命潜滋暗长，涌动着勃勃生机。4月，也是学期工作全面推进的关键阶段，作为颛桥镇学区化大家庭中开办未满一年的新成员，师傅赵校长经常提醒我要顺势而为，借力发展。

一、四季综合活动，找回丢失的“自然”

“新基础教育”共生体“学校四季综合活动”研究暨“学校日常生活中的学生发展”第六次全国现场研讨会在田园外小举行。作为长期开展研究探索的实践者，有幸参与了下午的论坛分享活动，在师傅的指导下担任论坛主持人。

叶澜教授在2018年5月的常州会议上提出了“依‘教育所是’而行，达‘自然而然’之境”，我们的教师论坛从目前各校实施“校园四季综合活动”的现状和困难入手，聚焦对“自然”一词的理解，梳理了五个层面的内涵：1）四季交替的自然世界；2）生命成长的自然规律；3）学习过程的自然节律；4）学校空间的自然气息；5）师生关系的自然互动。随后，结合各校实践案例进行了分享，有结合“春主生发”开展“春之律动，乐享健康”的亲子综合活动，制定个性化家庭健身计划，唤醒机体内的生命节律。有结合班级进行班本化创新实践的冬季之“一份别样的温暖”活动。还有以“前移后续”的研修方式，让班队活动成为四季活动重要节点有益探索。贯穿活动前后的重心下移式设计、实施、反思，让四季综合活动成为了创造学校教育新生活的重要组成部分。

教育的意义不只是在未来，它就在当下创造生命成长的、丰富的学校活动之中，有清晰的价值认同，有正确的思维方式，有以身立学的教育志向，不断实现自我的超越，逐步走向“生命.实践”教育学追求的教育最高境界——自然而然。

二、整合学区资源，激活共育的“因子”

颛桥镇以“联动传承、资源共享、优势互补、特色发展、共同提高”的实施策略推进学区化办学，作为学区内最年轻的一所学校，除了依托一体化办学的力量外，我校始终秉持开放创新的学校文化，依托紧密型学区建设，探索协同共育机制。

面对家长对优质课程的课外和起步阶段学校师资力量不强的供需矛盾，我们萌发了挖掘学区内兄弟校特色课程资源的想法，对各校的办学特色做了梳理，盘点了特色场馆资源、特色教师资源、特色社团资源等，并对资源进行清理，锁定了适合低龄段学生体验和学习的特色资源，形成了我校向外借力的“预选菜单”。

活动设想得到了镇教委和兄弟校的大力支持和积极响应，“学区化研学课程”得以顺利实施。通过请进来、走出去的方式，让一年级学生接触到了更为广阔和丰富的课程，获得了难忘的学习体验。鉴于特色场馆的特殊性，我们采用走出去的方式，进入了颛桥小学和田外外小的校园，两所学校各具特色的校园环境让孩子们流连忘返。在颛桥小学“安全体验教室”的真实场景中，孩子们近距离观察了火灾时的情境，了解了火灾来临时应该怎么处理现场并做好自救。来到田园外小的“纸循环创新实验室”，跟随着大哥哥大姐姐置身于纸的世界中，了解了四大发明“造纸术”，学习折纸。更为丰富的是初中学生送来的学校特色课程和家委会的共享课程，课程内容各不相同，既有知识学习，还有互动体验，孩子们透过课程的学习，不仅建立了良好的友谊，还对各校的特色有了深入的了解，感受到了颛桥镇深厚的教育底蕴。兄弟学校的学生也受聘成为了我校研学周的首批学生导师。

一周的研学活动虽然已经落下了帷幕，但是颛桥镇学区化“做强地方特色，提高地区教育质量，办家门口好学校，造福于颛桥地方百姓”的教育宗旨正在落地开花，颛桥镇深厚的教育底蕴，正在让更多的适龄儿童享受“公平而有质量的教育”。
