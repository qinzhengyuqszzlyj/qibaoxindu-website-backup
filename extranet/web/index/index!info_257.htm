在创新与发展中实现学校可持续发展 ——闵行区举行2019新任校(园)长培训启动暨2018新任校(园)长培训结业仪式

发布时间 : 2019-11-14   作者（来源）: 曹雅明   阅读 : 3284


2019年11月12日，闵行区2019新任校（园）长培训启动暨2018新任校（园）长培训结业仪式在田园二外小举行。闵行区教育党工委书记朱雪平、教育学院党总支书记、院长朱靖，教育党工委党群科科长林火生等领导出席，教育学院校长发展中心主任俞琳主持活动。

短视频“规范办学 智慧管理”，再现了2018年闵行区新任校（园）长带教培训的过程，以及通过一年的集中培训与导师精准指导，学员在实践中的反思与感悟。梅陇中心幼儿园园长金素芬、华师大紫竹幼儿园园长徐臻、鑫都小学校长蒋红梅、田园二外小校长蒋宛萍、金汇高中校长余长辉代表学员作精彩发言，分享了自己在导师引领下，通过“融入学校、实践探索、理清方向、博采众长”的递进式目标，立足“引领力”，智慧融合，稳扎稳打，探寻学校发展路径，平稳过渡谋发展的办学实践与思考，赢得在场学员和导师的阵阵掌声。

何学锋、赵瑛群、杨静霞、林琳、郁亚妹、殷蓓琳、丁春燕、李国兴等8位导师针对每位学员情况和办学特色做点评，充分肯定了学员一年来的进步与成长，鼓励他们办学有创新，行动有思考，责任有担当；并建议学员办校要以人为本、和谐发展，要眼中有事、心中有人，学会情感管理、恩威并施，成为风险控制的能手。与会导师高屋建瓴的智慧分享，让区域新任校园长收获良多。

特级校（园）长何学锋、赵瑛群、杨静霞还针对区域新任校园长培训工作提出了合理化建议，觉得导师带教要有一定载体，通过承担项目或任务历练学员、促进学员校中层团队的成长；在集中培训与导师带教的基础上，可以为新任校（园）长提供更多学习交流的平台，或建立2-3年新任校（园）长培养的回访机制、“3-5年青年校长沙龙”等，邀请资深校长现场把脉，或让新任校（园）长呈现青春风采、展示专业水准，为闵行教育培养、储备更多优秀干部。

林火生科长和各位导师为2017-2018新任校（园）长颁发结业证书，朱雪平书记、朱靖院长为“优秀学员”颁发荣誉证书。

2019年新任校（园）长带教培训又将启动，朱雪平书记、朱靖院长为2019带教导师何学锋校长、汤军红园长、郁亚妹园长颁发聘书，并作领导讲话。

朱靖院长从科学规划教师发展、有效落实课程实施等几方面，对导师和学员提出了宝贵建议。指出：学校的可持续发展依赖于教师的专业成长，在把握好教育方向的同时，要努力提升教师信息素养，关注课程实施的途径和方法。作为领导者，校长要树立学术权威、道德权威和管理权威，进一步提升校长领导力。

朱雪平书记对导师们为闵行教育发展作出的贡献表示感谢，并希望新任校（园）长要学会研究导师的办学思想，处理好“单一带教与团队提升”、“急于求成和稳步发展”的关系，回归教育本质，把握好立德树人根本任务，坚持为党育人、为国育才，在开放办学中，坚守初心，不迷失自我，把握好时代脉搏，为闵行教育多做贡献。

活动最后，2019年新任校（园）长带教培训班学员与导师举行了简单而隆重的签约仪式。本次培训为期一年，分为集中专题培训和导师带教培训两部分，相信这些新学员在导师的精心指导和自身实践中，能尽快明确学校定位和发展方向，在传承与创新中实现可持续发展，为实现闵行智慧教育的目标共同努力！

 

供稿：教育学院校长发展中心    摄影：田园第二外国语实验学校
