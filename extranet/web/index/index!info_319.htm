潜心问道 匠心筑梦——闵行区第二期幼儿园园长培训基地结业仪式

发布时间 : 2019-01-21   作者（来源）: 朱凤   阅读 : 3991

1月18日下午，闵行区第二期幼儿园园长培训基地结业仪式在莘松幼儿园举行。区教育局党工委朱雪平书记、组织科王治华科长、学前科黄悦科长,区人民政府教育督导室督学林琳、区教育学院校长发展中心俞琳主任等领导参与了本次活动。活动由基地学员韩懿雯主持。

活动总结：潜心问道  匠心筑梦

基地主持人上海市特级园长杨静霞进行了两年培训活动的工作总结。她指出：依据学员起点高后劲足的特点，基地着力提升学员的管理智慧与专业素养，实现学员领导与管理能力内在持续性提升，成为区域园长队伍中的骨干力量，在此基础上基地采取了相应的培训策略：1. 潜心问道，强化办园使命，做有思想的管理者。基地组织学员问道导师，优化办园思想；问道名师，拓展教育视野。2. 潜心实践，提升关键能力，做有智慧的践行者。聚焦园所顶层设计力, 互研规划，培育园长的大视野；聚焦日常问题解决能力，案例剖析，培育园长大智慧；聚焦团队合作力，互动指导，培育园长大胸襟。通过缜密设计的培训课程与培训方式，使学员们教育思想更加坚定，教育方向更加明确，教育观念更加先进，教育手段更加科学，教育资源更加丰富。最后导师对学员们的未来发展提出两点希望，一是 希望学员能践行“听得见炮声的”管理，深入一线了解基层。二是希望学员做强自己、做厚自己,厚积薄发。

学员收获：坚守信念 坚持实践

学员们分别以“礼物、味道、那一天、十年”四个篇章，讲述自己的管理故事，分享办学经验和智慧。

陈蓓琳：管理的最高境界其实就是信仰管理，教师应该是富有理想、坚持信念、拥有信仰的人，重建教师的教育信仰，对提升教师的职业幸福感、实现教育目的，以崇高的人生追求践行自己的教育理想，具有不可替代的意义。

韩懿雯：加入新团队近半年，带领中层们携手共同完成了学期初的既定目标，在各种活动中合作默契、信任感日益增强。这些收获得益于基地带给我的“智慧礼包”——“思维、视野、方法、资源、互助”。

谢贤群：以规范管理为基本，以人文管理稳定人心，充分运用各方资源支持，有效的管理风险。感恩基地和导师给予的智慧支持，提升了我刚柔并济，盘活资源的管理智慧。

龚悦华：课程优化促进了孩子的成长，提升了教师的专业成长，基地学习给我树立了追求“平凡但不平庸”的教育信念，引领我努力构建社会支持系统。

丁春燕：对于一所近70年历史的老校，传承、发展与创新是一项极其复杂的系统工程，过程中需要有价值理念的提升、管理重心的下移、活动过程的创生以及发展动力的内化等，而要推进与实现这样一系列的变革，除了要有整体系统的顶层设计之外，极其重要的是过程中需要有一系列中间节点性力量的有力支撑。

顾伟华：面对不同教师产生的不同问题，园长要站在教师的角度去看待问题、思考问题、才能更好的解决问题。善于等待、学会信任，萌发教师主动发展的意识，这样的管理才是更有味道的。

杨莉娜：管理的背后是文化与制度的思考，横向比较同类园所的文化与制度，纵向思考办园理念到文化诠释，围绕文化，制度，队伍，课程，后勤，安全等，让理念与文化，制度与管理一脉相承，互伴左右。

唐新萍：园长一定要有大视野大格局，眼界决定境界，思路决定方向，只要找准了方向，就不怕路有多远。导师带着我们聚焦园所顶层设计力，在观、听、研、悟的过程中，培养我们用“格局意识”来引领工作的思路和方向，进行园所发展的顶层设计。

张佩敏：园长要有担当意识，用专业的教育理念信服团队、用人性化的管理激励团队、用富有内涵的校园文化凝聚团队，让每一个人在平凡岗位上发挥聪敏才智，成就自己的教育梦想。

孙纯薇：在充分调研的基础上，制定符合不同园所实际的新三年发展规划。实施中调整三个园区的教师结构，启用年轻富有朝气的骨干担任行政，加强三园区之间行政的柔性流动，使园所走上了均衡优质发展的道路。

陆萍：作为园长要有一双识人的慧眼，有一颗真心助人的心，把合适的人放到合适的岗位，为教职工提供更多适合他们的平台，关心关注她们的成长。

周莉：借助《“幼儿快乐玩色彩”园本课程构建的实践与研究》课题，历练教师的能力和智慧。从引外力到修内功，一次次将课程这架风筝送上新的高度，留下了一条从瓶颈中挣脱，在低迷中拉高、在迎风中翱翔的攀升轨迹。

领导讲话：勇于探索 不断前行

朱雪平书记肯定了基地两年的培训成效，感谢导师的努力和付出。他用三个关键词对学员提出了要求。一是“争创一流”，要求学员向着最高的目标去奋进，努力培育争创有影响力的一流园长和园所的信念。二是“专业引领”，要求园长提高自身的学术素养，形成自己的专业品质；成事中成人，培育教师的学术专业，制定好教师专业发展的规划，形成专业氛围，引领教师在实践中提升；专业渗透，要求学员遇到任何事情都要用专业的标准解读，才能有效的解决问题。三是“坚守规范”。强调规范办园的重要性，守住底线，依法办事，程序合法。希望学员们只争朝夕，齐头迈进，集群发展，为闵行由教育大区走向教育强区注入动力。

两年的培训经历，激活了学员们的管理智慧，虽然培训结束了，但学员们会坚守教育初心，为了每一个孩子的全面发展匠心筑梦！
