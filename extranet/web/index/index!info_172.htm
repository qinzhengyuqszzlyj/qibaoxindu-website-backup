聚焦核心素养 变革教学方式 ——闵行区小学自然教研活动在新梅小学举行

发布时间 : 2023-10-26   作者（来源）: 徐婧   阅读 : 566

3/8
←
→

2023年10月24日，闵行区小学自然教研活动在闵行区新梅小学举行，本次活动围绕“聚焦核心素养 变革教学方式”的主题展开，新梅小学校长陈红、区教研员徐婧老师、黑龙江边境县的教师培训者和全区小学自然教师一起参与了本次活动。活动由区教研员徐婧老师主持。 

首先，由新梅小学金怡晨老师执教小学自然科教版三年级《生物的启示》单元中的《奇妙的壳》。金怡晨老师以蛋的视角看世界引入，围绕“母鸡会不会压坏鸡蛋”、“小鸡会不会‘鸡死壳中’”以及“薄壳结构有哪些优点”三个问题，以问题链的形式，层层推进，引导学生在实践的过程中探究蛋壳的承重本领以及其内外承重能力的区别，并利用相关资料介绍了解薄壳建筑在生产生活中的广泛应用。教师充分运用信息化设备，拍摄微课对学生使用学具进行实验探究提供指导；在此基础上利用智慧纸笔实时呈现学生实验记录，帮助培养学生从大量的数据或根据实验现象初步归纳得出科学结论的科学思维。

随后，执教教师从“立足单元析教材”、“聚焦素养说学情”、“以生为本展流程”和“结合理念说特色”四个方面进行了说课。随后，全区小学自然教师先进行了分片研讨，再进行了集中研讨。浦汇小学王雯清、明强小学蔡亚男、蔷薇小学陈佳咪、紫竹双语阎玮、上师三附小南校秦能等几位老师对本节课进行了点评。老师们一致认为执教老师精心设计制作教具体现其在教学设计中的思考；拍摄微课指导学生进行实验探究以及利用智慧纸笔呈现学生实验记录，凸显了信息技术在实验课堂中的优势。同时，教学中老师也非常关注学生的思维过程表达，注重培养学生科学思维，锻炼学生实验技能。 

最后，华理梅陇副校长沈意指出本节课的设计有较高的规范性、逻辑性和层次性，很好地落实了本节课的主题教学，并对执教老师的自制学具表示认可。教研员徐婧老师首先肯定了本节课中各活动的衔接具有较强的逻辑性；结合活动主题，整体教学过程体现核心素养；利用自制教具、智慧纸笔以及希沃投屏实现了技术赋能。最后，徐老师鼓励与会老师在今后的教学中，不仅要关注探究活动的组织，还需要关注对学生科学观念的培养。
