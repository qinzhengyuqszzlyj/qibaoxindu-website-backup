目标导向下的学生课堂活动设计

发布时间 : 2023-05-17   作者（来源）: 徐婧   阅读 : 668

3/8
←
→

5月16日，浦江校际联合自然学科教研活动在世外浦江开展。活动由浦汇小学王雯清老师主持，区自然学科教研员刘永亮老师和徐婧老师以及浦江各校自然教师参与了本次活动。

首先，参会教师观摩了由上师大三附小南校秦能老师执教的《搭纸桥》和世外浦江王寅辉老师执教的《水的压力》录播课。秦老师通过介绍桥的基本功能及式样，让学生通过小组合作体验纸桥的承重本领与式样有关。王老师通过真实情景引入，引导学生充分探究影响水的压力大小的因素。随后，两位上课教师分别从教材分析、教学重难点、教学过程及反思重建等方面进行了说课。

在教学研讨环节，王雯清老师和夏玲凤老师予以称赞。秦老师的课在教学行规上，对学生的纪律和实验要求有着清晰的指示。王老师的课在教学难点的突破上，精心设计多种体验活动帮助学生，多维度给予学生体验方法。

紧接着，由江川路小学夏玲凤老师带来的《审慎预判，小步慢走》的课题撰写主题讲座，夏老师从课题开题论证，制定行动计划，分步落实实施，及时阶段评估4个维度进行介绍，为浦江镇的自然老师带来了前沿的科研理论指导。

最后，教研员刘老师和徐老师对本次活动作专家点评。两位教研员指出本次教研活动紧扣“目标导向下的学生课堂活动设计”主题，在基于单元整体设计下落实学生核心素养。同时也指出，可以结合多维度去评价激励孩子，落实精准教学，教评一体。
