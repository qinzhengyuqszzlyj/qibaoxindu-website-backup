万里扶桑晓色微，青葱田园友伴随——2023年中日高中生研讨交流活动顺利举行

发布时间 : 2023-07-10   作者（来源）: 丁旻平   阅读 : 548

3/8
←
→

7月3、4日，2023年中日高中生研讨交流活动于北京外国语大学附属上海田园高级中学顺利举行。

为了推进中外青少年交流，优化多语种人才早期培养，闵行区教育学院在上海教育国际交流协会指导下，策划举办了本次活动。来自我区北京外国语大学附属上海田园高级中学和上海外国语大学闵行外国语中学的24名学生，以及来自上海日本人外籍人员子女学校高中部的12名学生，一同参加本次活动。

活动在北外田园陆振权校长的致辞中拉开序幕。上海教育国际交流协会副秘书长林萌、上海市教师教育学院关键语种教研员郭侃亮、闵行区教育学院副院长龚耀昌也在开幕式上对活动的举行表示了深切的祝福。

在两天的活动中，中日高中生除了参观北外田园文创工作坊、体验日语学习游戏、完成非遗香囊制作之外，还共同参加了华东师范大学日语系乔颖副教授所做的《中日文化渊源探究》讲座，并在东华大学日语系劳轶琛主任的指导下学习了《如何撰写课题报告》。随后，中日高中生们，在各活动带队教师的帮助下，按活动前期确认的“衣”、“食”、“住”、“行”课题小组分组，展开了各自的课题研究。

在活动最后一天的下午，各小组结合本组成员才艺特长，向专家们展示了他们在“中日文化的交流与传承”这一大命题下，各自课题方向上的探究成果。华东师范大学外语学院关键语种教育研究中心主任尹松教授、上海外国语大学原副校长谭晶华教授，为同学们的课题成果进行了点评。谭校长指出，本次活动有着很好的氛围，大家热情洋溢、活力四射、多才多艺。中日高中生围绕共同的主题，查阅了很多资料，学习了丰富的知识，在互相研讨的过程中也增进了彼此的友谊。日本学生讲中国文化，中国学生讲日本文化，大家都增进了开口说外语的勇气和自信。希望大家今后能更有勇气，更自信地开口，学好日语。

在颁发活动纪念证书之后，本次活动圆满闭幕。
