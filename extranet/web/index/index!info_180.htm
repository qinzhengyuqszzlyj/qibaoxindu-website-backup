追随孩子的脚步 读懂孩子的语言 —— 闵行区七宝幼儿园课程基地活动

发布时间 : 2023-06-02   作者（来源）: 鲁鸣   阅读 : 637

3/8
←
→

五月的天，绿荫冉冉，浅夏悠悠，草木温润，阳光正好。万物都在初夏中生长，幼儿园的孩子们也在阳光下快乐地、自由地成长着。2023年5月29日上午，闵行区第三轮幼儿园课程基地（七幼基地）活动在闵行第四幼儿园顺利开展。七幼丁春燕园长带领下的闵行区第四期幼儿园园长基地（七幼基地）的园长、七幼课程基地的共同体教师，以及虹桥中心幼骨干基地教师，共计50余人参与本次活动，活动由闵行四幼周靖老师主持。

现场观摩 趣玩自然

在观摩小中大三个年龄段的户外活动现场中，我们看到了认真投入、发现探究、专注坚持的孩子们。

认真投入：沙水有天然的亲和力，是大自然赐给儿童最天然、最有趣的自然游戏材料，小小的一粒沙看似简单，但在孩子们的探索下，却变得充满趣味，透过玩沙，我们看到了幼儿内心充满智慧和创造的世界。 

发现探究：创玩材料，激发了幼儿的想象力和创造力，探究游戏中用自然物材料，发现创造不同的玩法，并不断调整自己的游戏行为，在感受游戏乐趣的同时，切身体会到与同伴相互交流、共同游戏、友好相处的愉悦。

专注坚持：自然材料充满想象和创造，孩子们用自然材料和辅助材料探索并创造眼中的世界。他们在游戏中学会分享，体验成功与失败，也体验了在自然环境中游戏带来的愉悦。

案例分享 实践共研

骨干教师金筱庠和青年教师沈思媛，分别分享了精彩的游戏案例《水道渠成》和《载人火箭》。从中能够看到，教师尽可能地给予幼儿充分的游戏自主权，最大程度的放手、最小程度的介入，相信每一个幼儿都是积极主动、有能力的学习者，最大限度地支持和满足幼儿通过直接感知、实际操作、亲身体验获取经验的需要，让幼儿能毫无顾忌地投入到真游戏中。当游戏的自主权完全还给孩子，我们会惊喜的发现儿童天生就会玩并创造自己的最近发展区。

柔活教学 自主探索

“竹”够有趣分享会中，青年教师范佳敏老师善于发现偶发的教育契机，能抓住活动中幼儿感兴趣的事物，基于孩子们的兴趣和已有经验，尝试将个体经验分享为集体经验。及时给予有效支持，将户外游戏中低结构材料生成高结构活动。

活动中，范老师能充分尊重和保护幼儿的好奇心和探究兴趣，发现和支持幼儿有意义的学习，采用小组或集体的形式讨论幼儿感兴趣的话题，鼓励幼儿表达自己的观点，提出问题、分析解决问题，拓展提升幼儿游戏中的经验；采用自评、他人、同伴互评、教师评价的多元评价方式，帮助幼儿建立新经验以及自信心、游戏的热情。在师幼互动中，范老师尊重并回应幼儿的想法与问题，通过开放性提问、推测、讨论等方式，支持和拓展每一个幼儿的学习。

现场教研 智慧碰撞

在现场教研《藏在表征里的游戏印迹》中，区骨干教师、区骨干后备教师、教研组长、青年教师们都能以幼儿发展优先，立足幼儿视角，结合现场观摩的幼儿表现行为、自身户外游戏实践中的所思进行了一场智慧的思维碰撞。教研主持人孔蓓婷老师对教师们现场的智慧发言进行及时地总结和提升，并梳理成思维导图，助推教师们对于户外游戏中表征的思考以及后续的支持。

在户外游戏中，幼儿自由奔跑、自主创想，在游戏中获得经验，在快乐中获得发展。我们教师要做的就是立足儿童立场，充分发挥观察者、支持者的作用，推动幼儿持续自主发展。愿我们每一个老师都能全面理解幼儿，看到更真实、生动的学习与发展，支持幼儿不断探索、不断收获、不断成长！
