聚焦素养 技术赋能 深化单元

——闵行区中学美术教学研讨活动

2023年6月6日下午13：00，为持续推进闵行区中学美术指向素养的单元教学实践研究活动，闵行区初中教师在区教育学院美术教研员宋运芬老师的主持下，于上海市实验西校开展了主题为“聚焦素养 技术赋能 深化单元”的教学研讨活动。

课堂教学展示

活动伊始，实验西校美术教师陈海扬老师执教了六年级《湖石之魅》这一课，本节课是本单元7课时中的第3课时。陈老师通过分析湖石的特征、作用与造景布局，引导学生着重理解与区分湖石瘦透漏皱的美与魅，同时陈老师将湖石的寓意结合学校文化与精神，并将人文内涵与品格很好的渗透进课堂，落实了核心价值观中的审美感知与文化理解，使学生在探索中领略苏州园林的美。

课堂练习以小组合作的形式完成超轻土湖石作品，结合上节课的剪刻漏窗作品，各小组都完成了精美且完整的湖石微景观。这样的教学实践不仅促进了学生的综合素养发展，也为他们今后的艺术学习打下了坚实的基础。

在线讲座

吉林省教育学院美术教研员王贺教授为老师们开展了主题为《漫画、绘本的构思与创作》的讲座，王教授以问题串的方式，结合自身创作经验，向老师们介绍了漫画创作的关键要素与必要的流程，通过以图讲理的方式，带领大家领略故事漫画的特色与形式美感，提升漫画形象思维。

讲座后，宋老师对美术教师们提出了期望与要求，漫画是义务课标绘画领域的重要内容之一，希望老师们提炼漫画、插画中的造型元素、形式原理等关键知识，并应用艺术创作活动中，引导学生运用漫画创作思路，在创作过程中，培养学生审美感知、艺术表现、创意实践和文化理解的能力。

研讨交流

评课环节，老师们充分肯定了《湖石之魅》这一课中陈老师对于单元的设计、课程的环节设置、知识点的把握以及教师对于苏州园林的热爱。朱帆老师认为这节课以“整体——局部——整体”这一项目化设计渗透到学校的方式，值得肯定，同时可以加强教学环节中手绘线稿湖石与制作湖石的关联性。夏天然老师对本单元课程不同门类的融会贯通表示值得学习，可以尝试不同工具或套材，体现湖石不同质感的线条及作品呈现的生动性。吴震老师就本节课人文内涵的融入与核心素养的培育表示了肯定，并建议可以由学生自行总结学习成果，将学生自主贯穿始终。

最后，宋老师就本节课对提出了四点总结与思考：一老师对育人价值的理解，二教师对教材分析的理解，三教师对教学设计的理解，四教师对教学实施的理解。并鼓励教师们继续在课堂实践中，利用单元教学设计，深入探索新课程标准下的核心素养的落实。
