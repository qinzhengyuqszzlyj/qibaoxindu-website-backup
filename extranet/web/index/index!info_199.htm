支持幼儿经验连续性生长的集体学习活动设计与组织 ——闵行区北片幼儿园主题教研活动（星辰幼儿园专场）

发布时间 : 2023-10-27   作者（来源）: 张静   阅读 : 398

3/8
←
→

2023年10月25日上午，闵行区星辰幼儿园开展了 “支持中班幼儿经验连续性生长的集体教学活动”北片幼儿园主题教研星辰专场。闵行区教育学院学前教研室张静老师主持研讨现场，参加本次观摩研讨的是北片公办幼儿园及七宝联合体的中班教研组长们。

l 识别分析 聚焦儿童经验

首先观摩了青年教师张圆圆老师执教的中班运动集体教学活动《独木桥大冒险》，本次活动源于2号户外自主运动场地的平衡走，教师结合幼儿的问题及发展需要，设计了本次的运动集体教学活动。活动通过营造走过小河上交错高低独木桥的游戏场景，从“挑战多种不同方法通过平衡木”、“挑战多种方式通过平衡木上固定障碍物”、“挑战提升障碍物难度”三方面经验的梳理，帮助孩子在已有的平衡经验上，丰富多种通过有障碍窄道的方法，锻炼身体的平衡能力，以及动作的协调性和下肢的力量。启发幼儿递进性地思考，在运动中提升探索、创新能力，体验挑战的快乐。

中班教研《运动中，支持中班幼儿经验连续性生长的集体学习活动组织》的活动现场，组长带领老师们通过运用关键词提炼、提问质疑、问题延伸等方式，梳理幼儿的能力及品质，教师的支持策略，进一步思考教师的支持策略对于幼儿经验连续生长的价值和作用。

l 共思解疑  碰撞教育智慧 

在张静老师的带领下，老师们分成3组围绕主题展开讨论，并对活动提出了自己的思考与困惑。现场气氛热烈，老师们大胆表达自己的观点，互动间不断激发出新的智慧火花。

最后，张静老师从教师要如何转化孩子的经验，充分分析幼儿的核心经验，将动作和路线建立链接，提升背后的意义是要鼓励幼儿成为聪明的运动者。

通过此次活动，星辰幼儿园的老师将运用儿童视角，从儿童立场出发，进一步思考运用什么样的支持策略作为“中介物”，有意识地关注儿童经验的连续性，促进和引领幼儿经验螺旋式上升发展。
