聚焦问题 按需施教——2018年闵行区新任园长带教丁春燕导师组学习简报（3月）

发布时间 : 2019-04-01   作者（来源）: 朱凤   阅读 : 2076

一、观摩学习，解决共性问题

游戏是幼儿最喜欢的活动，对幼儿早期发展起重要作用。目前幼儿园都在实施角色游戏这一内容，但在开展过程中，大家都存在着共同的困惑。如：角色游戏环境的创设、材料的提供、教师的角色定位等。导师丁春燕园长根据学员的共性问题，制定了本次培训的内容。

大家观摩了四个班级的角色游戏现场，分别位于教室、走廊、专用活动室中的角色游戏活动区。幼儿们结合自身经验，自主划分区域、选择材料、自由结伴，快乐投入地进行游戏，呈现了“原生态”的游戏现场。

利用三种不同空间创设角色游戏环境，打开了大家的思路，明确了幼儿活动空间的创设，理解了“一室两用”环境创设的理念，通过有序摆放材料让幼儿快乐游戏。

二、交流碰撞，启迪管理方法

丁园长认为：管理见仁见智、各有方法，但讨论能开拓思路，提供解决问题的方法。学员就工作中的问题向导师请教，如：“教师在大活动中请假怎么办？”、“中层不配合怎么办？”等等。大家一起分析问题，列出解决问题的办法，再基于园情选择最合适的答案。她提醒我们：“讨论有其借鉴之处，但不一定适合各自的单位，需慎思慎行，具体问题具体分析。”

三、送教上门，全面带教指导

3月28日，丁园长带领七幼骨干教师团队来到古美幼平吉分园，对平吉分园的环境创设、各类活动的实施进行了全面指导。她对小菜园、幼儿园大环境、活动室创设、厨房以及保育员操作室设备的配置等作了详尽的指导，同时在经费使用上要做到：心中有底，紧急必须的先做、每月按照预算核算等原则进行有规划性的使用。以团队形式的带教不仅为平吉园后续工作指明了方向，同时也提升了新园长的整体思考能力。
