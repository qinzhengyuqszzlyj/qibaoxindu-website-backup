坚守初心 走向超越 ——闵行区小学校长培训基地举行第五期校长班结业仪式

发布时间 : 2019-01-15   作者（来源）: 曹雅明   阅读 : 2435

小雨初晴好天气，冬风有力百花开。2019年1月14日下午，闵行区小学校长基地第五期培训班结业仪式在实验小学景城校区举行。参加活动的有区教育局恽敏霞局长、区教育学院朱靖院长，第五期培训班全体学员及实验小学教育联盟部分校级管理人员。会议由区教育学院校长发展中心俞琳主任主持。

活动总结：坚守初心  走向超越

首先，上海市特级校长何学锋进行了两年培训活动的工作总结。他指出本期校长培训班具有两个特点：一、培养对象定位高：选拔部分有教育理想和追求的、有一定办学经验、有专业发展潜力、学校办学成效明显的校长；二、培养目标定位高：着力提升学员的管理智慧与专业素养，使学员的理论水平、实践能力、创新意识、办学思想有突破性提高。针对这样一批校长学员，该设计怎样的系列性培训课程？怎样的培训方式能够激发这样一批校长的内在学习自觉，又能帮助校长提升专业素养和管理智慧？作为基地主持人的他在反复斟酌下，为学员们策划了学习研究的三种策略：1.对话名校长：深度解读，感悟教育家办学的精神与智慧；2.寻求新突破：系统反思，聚焦学校发展的新视野新路径；3.强化团队研修：凸显团队精神和共同价值追求。通过缜密设计的培训课程与培训方式，使学员们在培训期间个人专业得到成长，学校管理获得突破，发展方向开辟了新路径，既夯实了原有的基础，又迈上了更高的台阶。

交流互动：百花齐放  百家争鸣

紧接着，每位校长学员对自己经过本期培训班系列培训后，在办学理念、管理方式、团队建设、教师发展等领域的新认识、新发现、新智慧等方面进行了交流，交流中处处闪现出这些有教育情怀和理想的校长学员们的成长与亮点。

王勤蓉：学校管理千头万绪，校长不可能面面俱到，要善于抓住学校管理中的主要矛盾和核心问题，科学决策。要真正站在教师立场，真心关注教师的生命成长，精准分析队伍现状，整体策划发展路径，才能真实推进教师队伍发展的良性循环。

杨慧琳：坚持“登高望远，为知而行”的办学思路，加强学习与感悟和思考与实践。培训班学习中感触最深的是思辨的品质，学会从多方面系统地思考办学问题，也经常思考教育问题、人的问题，这是校长幸福的最高境界。

张丽丽：“鸡蛋”从内打破是生命；“人生”从内打破是成长。通过办学追求，提升内生力，实现新突破。一个拥有内生力的自动化团队，是真正支持学校走向未来、突破自我的强大力量。

薛 琼：学校发展要成就人，成就的不仅仅是学生，还应该有教师，只有让每一个教师都能够看到进步的可能，都能够有自我发展的空间和平台，教师才能够有成就感、获得感和归属感。真正让学校发展惠及每一个教师，让每一个教师感受到成长的幸福。

叶 萍：通过聆听名校长们在岗位中的快乐和辛苦，感受到学校管理文化的外显和内隐，懂得自己在办学过程中不足和挑战，学会以长程思考学校发展，思考“人”的发展的管理理念。

唐晓安：以解开“社会满意度”之结为抓手，寻因归因,找准问题，聚焦问题,精准突破，主动有为,研究提升。由此，家校互动默契，形成工作合力，管理的方式发生了转变，提升了各条线工作的主动性与开拓性，促进了各领域工作的新生长。

朱 红：一直在思考：学校发展之“困”为何形成？“突围”的路径与方法如何形成？也产生了如下思考与顿悟：真问题，一定是具有科学逻辑的思考，专业理性的反思中产生的问题，是基于学校发展的未来思考与重建。因而以真实的研究，平实的解决，务实的实践，迈向了教育幸福之路。

姚 凤：坚信“做强自己是最好的辐射”。从新基本功到和美课堂，做强教师的凝聚力和发展心；从小校长到大校长，做强校长的自信心和自省力。从“德育统筹”、“师资优化”、“校区自强”、“项目深化”、“规划蓝图”、“重心下移”等做强新自己。

交流中，何学锋校长逐一点评每位学员的办学成绩和研究实践之路，同时也颇感欣慰地说：两年的校长基地培训历程，作为基地主持人的他特别强化团队的意识，从课程设计到具体的培训过程，始终采用团队研修的方式来展开。两年来，因共同的课程、聚焦的话题、互动的思维、一样的追求，让第五期校长班逐渐聚合为一个具有共同价值追求，能给予相互支持、信任、欣赏与鼓励的独特校长团队！

领导讲话：坚定追求  达人成己

教育学院朱靖院长以曾是一名校长的身份，跟学员们分享了她的成长经验和心得体会：1.校长要有自己独立的思想和坚定的追求；2.校长懂得把握时机，用大项目引领学校发展，出成果出名师，出名校长；3.善用外力，善用资源，来提高学校知名度和影响力；4.维持学校的稳定，团队的稳定非常重要。

教育局恽敏霞局长充分肯定了大家通过培训所取得的成果，实现了四个一：有一个做法、有一点突破、有一项成就、有一种感悟；她也赞赏了本期“提高型”小学校长培训班所做出的卓有成效的工作，为以后对新任校长、后备干部的培训探索了一条新路。她还勉励大家在新形势下要有新活力，新创造，要做有灵性的校长，办个性化的学校。她表示办学有很长的路要走，办教育的过程中要成就教师，校长也需要成长感和成就感。只有在百花齐放、百家争鸣中，才能推动闵行教育向更高层面发展。局长的讲话让校长学员们更加明晰了今后的奋斗方向和成长空间。

本期培训班学习结束了，可学校发展的研究实践步伐始终在路上。围绕“如何办好学校，如何达人成己”，大家坚信“千里之行，积于跬步；万里之船，成于罗盘”这个道理，只要坚持正确的办学方向（目标），坚守初心，持之以恒，夯实细节，就一定能成就梦想，走向超越！
