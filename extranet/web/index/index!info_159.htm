拓宽视野 提升教育领导力——云南昭通市中小幼部分校长、教师来闵行学习圆满完成

发布时间 : 2021-12-03   作者（来源）: 朱凤   阅读 : 1596

3/8
←
→

贯彻落实闵行区和云南昭通东西部协作要求，11月4日至12月4日，闵行区教育学院发展中心协助教育党工委党群科为云南昭通市中小幼部分校长、教师共63人开展为期一个月的培训，旨在优化育人方式，提升育德水平，促进校长、教师开拓教育视野、更新教育理念，成为新时代好校长、好教师。

拓宽视野  提升教育领导力

11月8日—10日为集中专题培训。首日举行了简短隆重的开班仪式，教育学院骨干基地办公室兰平主任解读方案，永善县上善中学祖万章老师代表教师发言，表示会珍惜机会，认真学习，不辜负家乡领导和人民的期望。永善县知临中学洪茂容副校长作为领队，感谢闵行区教育局、教育学院精心准备的同时，表示将带领学员全身心投入到学习，争取学有所获，为昭通教育的发展贡献自己的力量。校长发展中心主任俞琳主持活动。

开班仪式后，教育学院董学平副院长、普教二科马卫锋科长、教研中心孙静贤和李群两位副主任、思政一体化研究中心秦书珩主任、信息中心刘太如副主任先后作专题报告，六位专家聚焦闵行区教师队伍建设、中高考改革的闵行实践、教育质量监控、绿色指标评价、思政和信息化教育等，帮助学员拓宽视野，提升专业素养与能力。

岗位实践 提升教育执行力

11月11日—12月1日学员们前往学校跟岗实践，北外田园高中、七宝鑫都中学、颛桥中学、田园外小、颛桥小学、鑫都小学、田园二外小、天恒名都幼儿园八所学校承担任务，为学员提供个性化跟岗实践培训内容。

校长跟岗学习内容：校长的价值领导力、教学领导力和组织领导力、校本课程的开发和利用、校本研修的有效开展、教研组和年级组建设、教师团队建设和专业化发展、学校文化建设与特色建设、学校和教师的教科研管理、学校的德育管理和班主任建设等。

教师跟岗学习内容：上海市各级各类相关文件、课程与教学专题活动、日常学科研讨、校本研修的有效开展、教研组和年级组建设、教师团队建设和专业化发展、学校文化建设与特色建设、学校和教师的教科研管理、学校的德育管理和班主任建设等。

跟岗学校为每位学员配备导师，学员们可以观摩课堂，参加教研、旁听行政会等，近距离感受闵行学校的文化、理念、制度、方法等，感悟闵行教育在教学改革和创新过程中的创新智慧。

总结反思 逐梦路上再出发

12月1日，“云南昭通市中小幼部分校长、教师来闵行学习”培训班结业典礼在北外田园高中举行，教育党工委李光华副书记、党群科林火生科长、北外田园高中陆振权校长参加了仪式，俞琳主持活动。

永善县实验中学全洪平、永善三中李梅、团结小学伍霞梅、民族小学李邦玉、城南小学刘绪平、溪洛渡小学刘代琼、新拉幼儿园谢蝶、绥江一中李俊龙八位老师分别代表跟岗小组进行汇报。学员们回顾了一个月的学习经历，感谢跟岗学校精心制定培训内容，全方位开放学校资源，通过浸润式学习，感受闵行教育高质量体系建设的同时，反思自己的教育行为，表示将通过比较差异、发现差距、寻找答案、及时总结，不断反思，把学到的、悟道的、想到的、适合的坚持运用到今后的工作中，为昭通的教育源源不断注入活水，最后洪茂容副校长作培训总结。

李光华副书记感谢昭通校长、教师来闵行交流，闵行和昭通各有优势、两地相互学习，相互促进。他指出，想要成为一名成功的教育者和真正的教育人，要有教育情怀和梦想，有自己的办学思想和理念，坚守初心，不断实践，相信昭通教育在大家的努力下定会精彩纷呈。最后林火生科长和陆振权校长为学员颁发结业证书。

贯彻习近平总书记系列重要讲话精神，推动中西部协同发展，以教育实现精准扶贫。
