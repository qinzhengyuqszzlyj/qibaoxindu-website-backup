实地指导提专业要求 答疑解惑悟管理内涵——2018年闵行区新任园长带教杨静霞导师组活动通讯(11月)

发布时间 : 2018-11-30   作者（来源）: 朱凤   阅读 : 2028

走进园所  规划发展

11月2日，导师杨静霞特级园长带着邱雯静、曹燕菲和李沈君三位学员走进学员校一品漫城幼儿园，观摩集体教学活动。杨园长指出：幼儿发展是园所发展的重中之重，作为园长，除了行政工作以外，还要孜孜不倦提升专业水平，园长“姓教”的能力不能丢。

导师和学员就一品漫城幼儿园活动室创建、操场布局、师资队伍管理、幼儿发展等内容进行了实地交流。针对该园“趣味阅读”特色课程，该园李沈君园长提出她们将着重打造阅览室。在此过程中，大家一起出谋划策，对阅览室的位置、内部设置等提出自己的想法。

李园长还提出：随着幼儿人数的不断增加，可供做操的区域不足，杨老师对操场的整体布局以及后续调整方向提出了可行性建议，导师思考的全面性以及对细节和安全的关注，都让我们受益匪浅。

答疑解惑  思维碰撞

11月14日，我们来到虹桥中心幼儿园，观摩虹桥镇学区化教研展示活动，聆听幼教专家黄琼老师的点评。她指出：教研的内容要适宜，重点要清晰；特质要凸显等，让我们进一步明晰了成功的教研需要具备的要素。

三位学员就自己在管理中的问题与导师互动。针对新园长接手老园所，如何制定三年规划这一问题进行了深入的探讨：要尽快了解园所原有的情况，根据问题制定后续方向；要找出前一轮三年规划及终结性评估，再制定新的规划；要听听中层团队及老师们的想法......

杨园长肯定了大家的做法，也提出了自己的看法：一、三年规划是学校全面发展计划，想清楚要办怎样的学校；二、接手老园所，如何在传承的基础上创新，让学校不断发展；三、规划中涉及的发展目标等，应该是三年内幼儿发展、教师发展、幼儿园发展的总体目标，设置要具体，具有可操作性和可检测性；四、文本格式规范，文字表述清楚精炼。

在思维碰撞中，我们深深体会到管理真的不是一件容易的事，园所的方方面面都要考虑周到，作为新手园长要学的还有很多。
