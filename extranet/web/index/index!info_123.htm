学习“二十大” 建构新生态 赋能素养提升 ——闵行区小学三年级道德与法治学科研讨活动在华师永德实小举行

发布时间 : 2023-06-12   作者（来源）: 王玉兰   阅读 : 818

3/8
←
→

“时雨及芒种，四野皆插秧”。夏雨阵阵润新芽，悦收丰芒盈希望。2023年6月9日，及时雨后，以“学习‘二十大’建构新生态 赋能素养提升”为主题的闵行区小学三年级道德与法治学科研讨活动在华师闵行永德实小举行。本次活动邀请到闵行区教育学院道德与法治教研员王玉兰老师莅临指导，闵行区道德与法治中心组成员、课题项目组成员、区优青小组成员及区内中片三年级小学道德与法治老师共同参与。活动由永德小学丁丽莉老师主持，永德小学张计蕾校长、党支部书记翁春花老师、工会主席陈继红老师出席。

课堂观摩现亮点

丁宛婷老师执教的是三年级下册第四单元《多样的交通与通信》中的第12课《慧眼看交通》一课。

在设计教学环节时，丁老师结合《习近平新时代中国特色社会主义思想学生读本》《党的二十大报告》以及《义务教育道德与法治课程标准（2022年版）》等文件，对教材进行了深入的分析和解读，并通过前测问卷的形式充分了解学情，立足于学生实际，制定了指向培养学生核心素养的教学目标。

在课堂教学过程中，丁老师精心设计了 “问题观察员”“妙招发现者”“智慧创想师”等活动，引导学生逐步达成标准要求，并通过联系生活，发表学生的真实感受，了解交通发展带来的“利”，对祖国未来交通发展充满信心。在“问题观察员”这一环节，丁老师通过图片、音频、数据采访等多种形式，调动学生多感官分析、交流交通发展带来的“弊”，并引导学生思考如何处理交通发展问题。在“妙招发现者”这一环节，丁老师通过组织小组合作学习，带领学生对“国家从国情出发想问题、作决策、办事情”有所了解，并通过分析、交流交通发展问题及措施，增强学生的安全素养和责任意识。在“智慧创想师”这一环节，丁老师激发学生为祖国交通发展出谋划策的意愿，小组合作设计一份未来交通，帮助学生增强环保意识，树立起绿色、循环、低碳的生态文明建设理念。同时在活动过程中，对于三大核心素养的达成，丁老师巧妙融合了自评、师评、生评的综合式过程性评价。

课堂结束时，丁老师引导、鼓励学生向其他展示小组的创想学习，从而借鉴、优化自己的创意积极参与校级评选活动，再思再绘，感受在新时代新生活中的美好未来。践行习语金句 “人民对美好生活的向往，就是我们的奋斗目标。”

主题论坛共生长

本次主题论坛围绕“技术赋能提素养 综合评价促成长 智慧共研新生态”，分别由三位老师展开分享交流。

永德小学丁宛婷老师以“化过程评价 赋能素养提升”为核心，着眼于本课的活动任务，以及评价设计与成效展开分享。丁老师以活动任务为导向，抓住本课设计中的三大活动项目来“现身说法”，举例“问题观察员、妙招发现者、智慧创想师”来生动阐释如何融评价于教学过程，实现教学评一致；如何让技术赋能评价，实现智慧教育发展；如何让评价基于项目，实现深度学习。随后，丁老师深度总结、解读了如何通过自我评价、教师评价、组际互评等多方评价结合，来促进学生的自我监控、反思、调节，并帮助学生学习更好、更有效的方法，从而内化为自身的行为，实现“教学评一体化”，反思生长原点，迭代素养提升，进而促进学生的知行合一，更好地落实立德树人。

永德小学杜鸿艺老师依据“综合评价见发展 协同育人促成长”这一论点，抓准本单元的三大核心素养，从“精心设计多元活动项目”与“综合运用多种评价方式”两个角度出发，在单元视野下深入解读了如何赋能评价探索、促进知行合一。杜老师提到我们应围绕发展学生核心素养，依据指向核心素养的表现性评价标准，设计开展序列化活动，并通过探索增值评价、强化过程评价、改进结果评价，三管齐下，来发挥评价的引导作用，利用好绿色生态的智慧评价方式改进结果评价，使育人资源呈可持续发展状态，使素养发展智能化，提高育德效能。

浦汇小学王晓丹老师从“赋能优质迭代，促进区域共享”这一论点出发，从如何实施智慧单元教学，如何促进学生全面发展，如何实现赋能优质迭代三个角度思考问题，并以本次课程展示为例，进一步发掘了同一主题区域共研、跨学科教学联动、信息技术共享的优势。让我们切身感知了通过信息技术推进优质教育资源建设、开放与共享，以数字化转型推动教育走向公平均衡、优质高效、个性灵活和智慧绿色，以促进实现大规模因材施教的重要性与必然性。

理念引领启未来

闵行区教育学院道德与法治教研员王玉兰老师对本次教研活动做专家点评。首先，王老师充分肯定了丁宛婷老师在课堂上的呈现，丁老师的每一个教学环节都体现出非常清晰的价值引领和目标指向，在如此面广、量多、内涵深厚的思政课堂上，真正践行了“不求面面俱到，但要突出重点”的理念。王老师更以本课教案为模板，引领着我们进一步体悟新教案“新”亮点，即在单元视野下，强调问题导向下的任务活动设计，突出教学评一致性特点，为我们今后的思政课堂找准方向、把舵启航。与此同时，“不仅层层落实，更要夯实经历”，“不仅写实记录，更要自信分析”，“不仅价值引领，更要知行合一”，王老师的“四不、四更”如连珠妙语，字字铿锵、发人深省。让在座的老师们更投入地思考，如何为学生搭建更丰富多层次的阶梯递进式的学习路径，如何与学生建立多种深层的有效对话、多维的交流以及深刻的价值体验，如何指导学生具有德性的追求真善美的行为。让学生真正在思政课堂中，真正在各种各样的经历和体验中催发德性的生长，真正有所获得。

伴随思考，王老师也对更好学习“二十大”，建构新生态，赋能素养提升提出了以下建议：多用归纳法，少用演绎法。在课堂之中，我们由生活实例去引入，由生活经验引发学生的道德认知、道德情感和道德行为的提升，进而归纳出道理，以提升学生道德认知和认同感。深化表现性评价要点。在思政课堂中，更强调学科核心素养，围绕社会调查、人际交往、价值体验为评价要点。

临近活动尾声，王玉兰老师对2022学年第二学期闵行区小学道德与法治教研工作作出的总结却再度掀起了现场的高潮。只听王老师细数了下半学年以来取得的种种工作成效，并从“课堂教学、课程建设、教研活动、问题与反思、工作安排与建议”五大板块展开，深度详析了课堂教学层面 “真、深、实” 的趋向，课程建设中“大命题、大课程 、大先生、大熔炉、大课堂” 的体现，以及教研活动中 “目标时代化、主题精准化、内容丰富化、方法智能化”种种实现，更是对工作中显现出的问题精准提出了以下三点反思改进：一在于教学设计，解读新课标的能力有待加强，在制定教学目标及评价目标时还不够清晰准确。二在于课堂教学，需持续关注学生深度学习的发生及个体需要，及时有效解决学生的问题和疑惑。三在于作业和评价，练习与反馈应发挥巩固所学、深化理解、拓展延伸的作用。评价时，要多鼓励积极思维和坚持努力的学生。

最后，王老师对本学期的阶段性评价工作作出了指导性统筹安排，同时对现场全体老师的学期工作给予高度认可，并寄予了暖心的鼓励与关怀。相信在教研员王玉兰老师的引领下，闵行区道德与法治学科教师将继续以强心、用心、专心、钻研术业，一步一脚印打造创新性思政课堂，让每一个孩子都能在我们的课堂中有所经历、有所体验、有所收获，真正落实立德树人的根本任务。
