敢担当 善作为 能成事———2024年闵行教育系统优秀年轻干部培训班开班仪式报道

发布时间 : 2024-04-13   作者（来源）: 朱凤   阅读 : 1910

3/8
←
→

4月11-12日，2024年闵行教育系统优秀年轻干部培训班开班仪式在上海交通大学李政道图书馆举行。交通大学教育学院党总支副书记陈鹏，闵行区教育党工委副书记徐明怡，交大教育学院教务与实习办公室主任谢嘉祖等出席。

徐明怡做领导讲话，感谢交大教育学院对本次培训的大力支持。她指出：培养选拔优秀年轻干部是加强领导班子和干部队伍建设的重要基础性工程，希望学员们坚定理想信念、牢记初心使命、不断修炼自我、主动担当作为，以实际行动诠释对党的忠诚、对党的教育事业的忠诚。并提出了三点要求：第一要坚定理想信念，锤炼政治品格。必须旗帜鲜明地讲政治、树大局，为全面贯彻党的教育方针，落实立德树人根本任务，培养担当民族复兴大任的时代新人提供坚强的政治保障。第二要坚持学思践悟，磨砺过硬本领。不断更新知识结构和教育理念，坚持理论联系实际，不断提高解决实际问题的能力，以创新思维推动教育改革和发展，以实际成效检验学习成果和工作能力。第三要坚持锤炼自我，敢于担当作为。立足岗位主动担当，切实增强责任感，拿出切实有效的举措，敢于担当、善于作为、勇于开拓创新。

陈鹏做开班致辞，本次培训围绕习近平总书记关于“教育家精神”的重要论述，以“躬耕教坛，强国有我”的各项要求，精心设置培训课程，不断优化培训内容，激发学员们教育教学新思路。区教育学院校长发展中心主任俞琳从学习内容与形式、组织管理与要求两方面进行了方案解读。本次培训始终抓住一条主线：深入学习贯彻习近平总书记重要讲话和全国两会精神，落实习总书记关于教育强国的重要论述及考察上海重要讲话精神，全面学深学透习近平新时代中国特色社会主义思想理论，涵养优秀年轻干部的政治素养和专业能力。七宝中学金亚伦代表全体学员发言，感谢区教育党工委、教育局为学员们搭建的学习平台，表示会珍惜机会，充分发挥主动性和能动性，把“学思想”摆在首位，通过互帮互助形成学习共同体，学以致用，为闵行教育高质量发展注入强大动力。

闵行区教育党工委党群科科长连军主持开班仪式。

开班仪式后，开启了2024年闵行教育系统优秀年轻干部专题培训的序幕。交大马克思主义学院黄庆桥教授做了《领袖智慧与领导决策》的专题讲座。通过分析领导的智慧和决策，引导年轻干部思考什么叫领导？始终秉持不怕吃苦，活到老学到老，格局要大站位要高。教育学院徐斌艳教授做了《新时代教师专业发展多元路径》的专题讲座。聚焦“教育强国”、“课程改革”、“数智时代”对新时代教师专业发展诉求的提出背景进行深入剖析，提出理解学习、理解自我、参与合作共享的多元发展路径。人文学院陆岩军教授以《人文素养与中国智慧》为题，从中国传统文化治国理政、为人处世之道切入，提出作为教育工作者应有两大情怀：悲天悯人与敬畏之心。通过解读《论语》、《礼记》等经典文学，深入浅出地阐释了中国文化中“观乎人文以化成天下”的理念。资深媒体人何老师分享了《认识和做好舆情管理工作》主题报告。何老师结合具体的案例，诠释了宣传思想文化工作事关党的前途命运，事关国家的长治久安，必须牢牢掌握舆论场的主动权和主导权。

本期学员294人，体量大、经历差异大，为期9个月的培训，聚焦“党性修养与理论提升”、“时事政治与使命担当”、“岗位职责与能力提高”这三个模块开展线下培训；围绕“党的政治理论、全面从严治党、党风廉政教育、深入贯彻党组织领导的校长负责制、党建工作交流、师德师风建设、干部素养提升、教师队伍建设”这八大模块展开线上培训，依托交大教育学院、市区委党校、国家教育学院等高校教育资源，邀请专家教授、基础教育名家、区域相关领导、名校长等开展指导，加强思想淬炼、政治历练、实践锻炼、专业训练，助力学员们成为闵行教育改革发展的“开路先锋、事业闯将”。奋楫笃行，臻于至善；行而不辍，履践致远。
