数字化转型中的科学学习活动 ——闵行区小学自然教研活动在七宝文来学校举行

发布时间 : 2023-11-10   作者（来源）: 徐婧   阅读 : 552

3/8
←
→

2023年11月7日，闵行区小学自然教研活动在闵行区七宝文来学校举行，本次活动围绕“数字化转型中的科学学习活动”的主题展开，七宝文来小学部教务处主任褚卓炼、全区部分小学自然教师一起参与了本次活动。活动由明强二小沈翠红老师主持。活动分为课堂观摩、教师说课、单元设计四人说、分组研讨、专家指导五个板块。

板块一：课堂观摩

七宝文来唐晓凤老师执教科教版自然二年级《昼夜与四季》中的《白天与黑夜》一课。唐老师围绕生活中常见的白天与黑夜的变化，基于真实的生活情境进行科学探究。首先利用信息对所探究的问题做出科学的解释，发现白天与黑夜场景下的活动不同，培养实证研究能力；接着通过模拟被太阳照射着的地球，知道白天与黑夜的不同来自太阳光的照射，初步建立建模的思想；最后按小组进行有依据的猜测，引导学生对自己的猜测进行说明并形成辩论形式的解说，最终发现地球自转产生白天与黑夜，自转一周是一天。并借助“赛课堂”，在系统加持下将学生的任务单从纸质转变成线上操作，用“拖一拖”、“涂一涂”、“拍一拍”等形式更方便、更高效地进行课堂活动，能及时将学生的作品和评价出示在主屏上，提高了教学的效率。

板块二：教师说课

由执教教师从“挖掘文本说教材”、“关注差异谈学情”、“依标扣本说目标”和“突出主体说方法”、“立足发展讲过程”五个方面进行了说课。上海的学生身处北半球中低纬度地区，对白天和黑夜的存在与更替感知清晰，初步具有了探究科学知识的能力，会对生活中的现象产生很多的问题，为本节课的教学提供了较好的探究基础。在技术加持下，玩转课堂，基于真实的生活情境进行科学探究，通过活动与活动之间的联系，以培养学生的科学素养为宗旨，对学生进行科学启蒙，关注核心素养的形成。

板块三：单元设计四人说

由基地学员郭智娟、唐晓凤、严诗帆、金佳豪四位老师主讲沪科教版自然二年级第一学期的第四单元《昼夜与四季》。老师们从解读课标，析单元教学内容；立足学情，设单元学习任务；整体设计，定单元学习路径；技术赋能，实现教学评一体四个方面展开《昼夜与四季》单元设计说，最后邀请一个学生小组展示他们创作的“昼夜与四季科普剧”，同时向在座的各位专家和老师抛出问题：如何在低年级的教学中去落实科学思维方面的核心素养？

板块四：分组研讨

在评课板块，听课老师们随机分组、人人交流，并以组长代表汇报的形式总结评课。

第一组代表顾佳斌老师认为整个单元大任务的设计体现了“理论—实际”的逻辑关系，大任务中科普剧的脚本设计也贯穿在每堂课中，例如本节课所呈现的固定句式，既帮助低年级学生有效的梳理思路，同时促进科学素养的提升，另外赛课堂数字化教学的加持，使教师能够实时给予评价，调动学生学习积极性以及整个班级的举手频率。第二组代表宋隽波老师从创设真实情境、活动体验丰富、数字化技术赋能课堂等三个方面进行评课，认为唐老师的本节课是非常值得探索的一堂课。第三组代表沈一鸣老师认为唐老师给了二年级学生足够的学习支架，如一个个问题链的引导、实验操作微课视频、数字化评价单、句式引导等都助力学生恰到好处；三个活动中的“拖一拖”“画一画”“拍一拍”数字化互动层层递进，有效推进教学过程。第四组代表张李奕老师围绕本节课中的策略支架、语言支架、资源支架、评价支架等四类学生学习支架展开阐述，认为这些支架的构建高效助力学生科学思维的培育。

板块五：专家点评

中心组张晓文老师首先结合课中教学数字化的证据，娓娓叙述了“赛课堂”教学的优势，且保障了整堂课的实施；接着从单元重构、大任务设计、科普剧展演等多个维度表示了老师们对于课堂设计的用心与热忱，认为这种探究精神值得每位老师学习。中心组秦能老师就基于真实情境引发学生认知冲突的设计给予肯定，认为该设计不仅暴露出学生的思维，同时促进学生思维的提升，为教师的诊断性评价给予阶梯。中心组金跃华老师认为本节课的活动设计非常丰富，整节课松弛有度，数字化的教学较传统教学而言，增强学生的学习热情与积极度，鼓励老师们继续在数字化教学指引下有所建树。同时提出问题组织更童趣化、道具设计更优化等中肯建议。

本次活动在老师们的高度评价与积极探索中落下帷幕，今后我们将继续基于新课标展开探索，全方位实现学生科学素养的培育。
