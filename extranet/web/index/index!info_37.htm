落实课标要求 提升学业质量 —— 闵行区五年级英语学业质量调研反馈交流会在闵行区实验小学举行

发布时间 : 2023-09-23   作者（来源）: 殷素婷   阅读 : 695

3/8
←
→

2023年9月19日下午，主题为“落实课标要求，提升学业质量”的闵行区小学五年级英语学业质量调研反馈交流会在闵行区实验小学（畹町校区）举行。本次活动由闵行区小学英语教研员殷素婷老师主持，全区各校英语学科教导、教研组长和五年级英语教师近300人参与了此次活动。

本次教研活动主要包含闵行区五年级英语学业质量调研反馈分析、学校的经验分享和互动交流三个环节。

活动伊始，殷素婷老师从“调研基本情况”、“典型试题分析”、“命题趋势分析”、“学科研究建议”四个方面，对于2023闵行区五年级英语学业质量调研做了全面的数据反馈与分析。

其次，由闵行区实验小学英语教研团队交流分享他们的实践与研究。刘莎老师和马彩红老师对于闵行区实验小学的生源与教师队伍构成做了简要的介绍。从上学年的期末五年级质量监测数据报告可看出，四校区总体发展态势良好。李宝婷老师、马彩红老师和杨涟漪老师分别从“理数据析成因”、“重日常推评优”、“构作业思评价”、“以项目促评价”四个方面，阐述如何以终为始，在新课标引领下，在基于主题育人意义探究的单元整体教学的道路上笃行不待。

紧接着，上海民办圣华紫竹双语学校的谢虹老师从“学校理念与基本情况”、“学校课程特色介绍”、“实践探索赋能学科发展”三个方面介绍了学校如何通过英语课程建设和学科实践探索提升学生的核心素养，夯实学业质量。而后，古美学校的王敏静老师以“聚焦新课标理念，践行教学改革，提升学业质量”为主题介绍了学校近阶段的英语教学改革和研修工作，为老师们提供了理论与实践的经验分享。最后，景东小学的江俊芝老师从“凝心聚力”、“课标领航”、“以研促教”、“多元支持”四个方面交流了作为一所“农村学校”，学校英语教研组如何利用优质资源，积极探索，躬耕课堂，努力提升学生的英语综合能力。

在互动点评环节，鹤北小学的乔文妍老师提到，学业质量的提升不仅局限于课堂上，更包含于课堂外的项目化学习、学科的融合等。明强小学的赵燕华老师肯定了闵行实小的一系列促进英语教学和学生英语核心素养的做法。世外浦江的夏奕舟老师也非常认同各校对学生高阶思维和学习习惯的培养。

活动尾声，殷素婷老师对此次教研活动进行了点评与总结。她高度肯定了四所学校英语教研团队的研修成效。殷老师从“因校制宜，优化教学管理工作”、“因势利导，推动教育教学实践”、“因材施教，深入精准教学研究”这三个方面，为各校在英语校本研修方面指明了未来研究和提升的方向，鼓励各校在新课标的引领下与时俱进地开展精准教学实践，不断提升学业质量。相信在教研员的专业引领下，教师们的努力学习下，定会助力闵行区小学英语教师的专业成长，推动学生的英语核心素养能力发展。

 

撰稿人：闵行区实验小学 李堉珺

审核：闵行教育学院 殷素婷
