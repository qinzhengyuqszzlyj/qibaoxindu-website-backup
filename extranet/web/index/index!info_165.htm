承百年使命 育时代新人——以教育家精神办闵行教育

发布时间 : 2021-10-18   作者（来源）: 徐婧   阅读 : 3128

  • 闵行区第四届“教育学术节”开幕式暨校长论坛顺利举行

  • 闵行区人民政府副区长刘艳致辞

  • 上海外国语大学闵行外国语中学党总支书记、校长吴金瑜作专题报告

  • 上海外国语大学闵行外国语中学党总支书记、校长吴金瑜；闵行区莘松中学党委书记、校长张忆作专题报告

  • 闵行区七宝镇明强小学党委书记、校长姚凤作专题报告

  • 中国教育学会副会长张志敏作专家点评

  • 闵行区教育学院院长朱靖解读方案

  • 闵行区教育党工委书记、局长恽敏霞讲话

2021年10月15日下午，闵行区第四届“教育学术节”开幕式暨校长论坛在闵行区教育学院S301报告厅举行。闵行区人民政府副区长刘艳，闵行区教育党工委书记、局长恽敏霞，原市教委教研室主任王厥轩，中国教育学会副会长张志敏出席活动。教育局科室长及以上干部；全区中小幼、职校、直属单位正职校（园）长、书记；区教育学院全体教师；2021年闵行区教育系统后备干部培训班学员；闵行区第五期“优秀教师培育工程”学员等共计350人左右参与，区域教师1500人线上观看直播。


闵行区教育党工委委员、副局长乔慧芳主持。


于漪先生致辞：赓续红色血脉 培养时代新人

人民教育家于漪先生作视频报告。于老师讲了三方面内容，第一是充分诠释了“双减”“双新”政策的现实背景和价值意义，它是回应社会关切，家庭焦虑，改进教育生态的国家战略布局，是牵一发而动全身的，是贯彻党的教育方针，使素质教育落地的重要举措，学校应拨正航向，明确教育根本任务，发挥孩子成人成长的主体作用，端正理念，提升教育自觉性。其次，教师需要在课堂教学质量上下功夫，避免应试教育和理念跟不上，吃不透带来的不利影响，明确适合学生的课才是最好的，心怀实事求是之意，遵循学科规律，坚持儿童立场，还原儿童成长世界，让孩子快乐动脑筋学习。最后强调“课后服务”是孩子全面而有个性发展的新天地，教育不只是作业与课堂，教育人应该主动作为，面对繁重的“双减”工作，把政策用好，把战略意义变成教育思想，帮助孩子绘好有意义的人生，要赓续红色血脉，培育时代新人，使红色江山代代相传，教育事业灿烂辉煌。


校长论坛：秉承教育初心，探索教改之路，提升育人质量

上海外国语大学闵行外国语中学党总支书记、校长吴金瑜；闵行区莘松中学党委书记、校长张忆；闵行区七宝镇明强小学党委书记、校长姚凤；闵行区莘庄幼儿园（闵行区实验幼儿园）党支部书记、园长郁亚妹作题为《“陪育”具有民族情怀、跨文化素养的时代新人》、《立师德之魂，培育人之本》、《审美.研究.超越》、《拥有与时代发展同步的生长力》的专题报告。四位校（园）长围绕本届学术节主题，聚焦人才培养、教师发展、学校发展、校（园）长专业发展四方面，分别阐述了自己的办学思想及教育实践，做到坚持“立德树人”根本任务，坚守教育初心，与国家时代发展同步，与教师学生成长紧密相连。四场报告各有侧重又紧密相连，以点及面，呈现了闵行校（园）长育生、育师、育校、育己的教育新生态，实现以教育家精神办闵行教育。


张志敏作点评。他对校长论坛的四场报告给予了肯定，闵行校（园）长能根据时代发展要求，有高位思考和扎实的办学实践，注重传承优势和创新发展，提出教育家型校长的三种角色是要循规蹈矩，严格按照党的教育方针和国家政策办学；要顺势而变，在传承中有创新；要学做终身的学习者，严谨的研究者，积极的创新者，最后建议校（园）长要关注未来教育资源的重构，教育空间的再造，教育样态的创新，教育技术的赋能和教育评价的增值这五大变革，积极探索“育时代新人”的时代性应该如何实现。


领导讲话：明使命、正航向，构建闵行教育高质量育人体系

院长朱靖从指导思想、具体安排、组织保障等方面解读第四届“教育学术节”方案。通过回顾前三届“教育学术节”主题，强调了学术节的思想站位和价值意义，并对本届“教育学术节”主题做了阐述，同时希望全区学校和全体教师能积极参与，仰望教育发展的前沿，通过观察与倾听，审视自己的不足，在鉴赏他人学术成果的过程中，提升自己的学术能力。


教育党工委书记、局长恽敏霞指出，要尊重教育的客观规律，将规律与“学、思、悟、践”相融合，充分理解“为教之道在于度，为学之道在于悟”；要注重教育管理和育人过程，提升自身专业度；同时希望闵行教育人能通过本届“教育学术节”，了解前沿思想，启发科研思路和方向，提升自身学术能力；能对“承百年使命，育新时代新人”这一主题有更深远的认识，能始终秉承“立德树人”初心，牢记“为党育人、为国育才”使命，将后阶段的教育工作做的更好，办好人民满意的教育。


副区长刘艳讲话。她充分肯定了“教育学术节”的举办是闵行教育优质发展的助推器和教师学术优教的引擎；称赞了本次学术节主题“承百年使命•育时代新人”立意高远，具有独特的时代意义，彰显了闵行教育人的胸怀和担当，同时希望闵行教育能传承“十三五”发展的经验和成效，在“十四五”期间引导全区教师更好地履行教育使命，落实立德树人，不断提升境界修养，最后预祝闵行区第四届“教育学术节”取得圆满成功。


本届“教育学术节”紧密围绕“教育使命”与“时代新人”这一核心命题，精准契合了国家、上海以及闵行教育发展定位与实践需求，书写着中国教育的“上海篇章”与“闵行经验”。本届“教育学术节”将为闵行17类教育群体定制9大主题，28个专场，涉及16所学校，历时2个月，本届学术节的顺利举办将推进闵行教育教学改革不断走向优质、均衡、高位发展，闵行教育人将明使命、正航向，回归教育初心，遵循教育规律，站在时代之中，学校之巅，探索教改之路，在传承与创新中，建设让党和人民满意的高质量教育体系。
