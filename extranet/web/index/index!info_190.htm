优化学习活动的设计，落实“宏观辨识与微观探析”的核心素养 ------“‘MF-PTTI’式循证教研”化学组研讨活动

发布时间 : 2023-10-27   作者（来源）: 陆艳   阅读 : 718

3/8
←
→

探索循证教研，深化“双新”实施——上师大附中闵行分校市双新项目（“MF-PTTI”式循证教研）如火如荼地开展，2023年10月25日下午，化学组以“如何优化学习活动的设计，落实‘宏观辨识与微观探析’的核心素养”为主题的研讨活动在上师大附中闵行分校321室举行。闵行区高中化学教研员陆艳老师、上师大附中闵行分校校长赵婷老师，上师大附中闵行分校化学组老师，远道而来的龙岩市骨干化学教师代表，以及闵行区各学校的高中化学教师参加活动。本次研讨活动共分为两部分，由五个环节构成，由闵行区高中化学教研员陆艳老师、上师大附中闵行分校化学教研组组长檀文金老师主持。

一、课堂展示

环节一：基于“如何优化学习活动的设计，落实‘宏观辨识与微观探析’的核心素养”课堂展示

首先，李美娜老师和高三（5）班的同学们带来了一节别开生面的课堂，以二氧化碳（CO2）这种我们既熟悉又陌生的物质为研究对象，通过CO2认识历程这一历史线，从制得干冰到现在用CO2制得零碳甲醇，设计了模型搭建、数据推理等学习活动，运用问题串的教学方式，揭秘了干冰分子晶体和CO2共价晶体的结构，探寻了CO2的“前世今生”。此课堂区别于老师讲授、学生听讲的传统课堂，而是通过学生活动，真正使学生成为课堂的主体，充分调动学生的积极性以及对微观不易理解内容的深入思考，有效落实宏微结合等核心素养。

二、循证教研活动

环节二：“‘MF-PTTI’式循证教研——如何优化学习活动的设计，落实‘宏观辨识与微观探析’的核心素养”相关背景介绍

首先，檀文金老师对参加此次教研活动的老师们表达了感谢，接下来就循证教研是什么、为什么以及怎么做，向与会老师进行了介绍。“循证”一词来源于医学领域，本意是“基于证据的实践”。循证教研是一种行之有效、行之高效且能够持续开展的教研模式，基于本学科最核心要解决的问题，化学组确立了“如何优化学习活动的设计，落实‘宏观辨识与微观探析’的核心素养”这一教研问题，基于证据进行教学设计，教学实施产生新的证据，对教学设计进行再优化，有效开展循证教研活动，从而培养学科素养，体现学科育人价值，实现减负增效，落实立德树人。

随后，化学组老师依次就研究背景、证据收集与评估、课堂观察工具向与会老师们进行介绍。关于研究背景，汲丽老师强调“宏观辨识与微观探析”这一素养最能突出化学学科的独特性，也是区别于其他学科的具体体现。在高中化学课程中，“宏微结合”的学习思路，有助于形成“结构决定性质，性质决定应用”的观念，以便于分析和解决实际问题。

关于证据收集与评估，毕炜老师分享了从中国知网、读秀、万方数据三个平台检索到的文献进行评估，初步确定了课堂观察评价量表，为解决如何优化以及如何设计落实“宏观辨识与微观探析”等核心素养提供了课堂观察采集信息的工具。

关于课堂观察工具，基于更好地为循证教研提供证据，评价量表最终优化为“凸显宏微结合的核心素养培育的中学化学课堂教学评价指标体系”，何双老师介绍了评价量表这一证据的构成以及就如何使用进行了介绍。

环节三：化学组就《分子晶体和共价晶体》学习活动设计和宏微结合核心素养落实进行循证教研活动

在檀文金老师的主持下，本次研讨活动进行到最核心的循证教研如何落实到日常的教研活动中这一环节。首先，李美娜老师以今天的课堂展示为例，介绍了如何利用评价量表、文献等证据进行课前的循证，从而进行教学设计。在本节课中设计了数据推理、小组合作进行模型搭建等学生活动，从而更好地感受晶体的宏观性质和微观结构以及它们之间的联系，进而培养学生“宏观辨识与微观探析”的核心素养。

接下来，根据听课过程中使用评价量表收集到的信息，化学组各位老师从不同维度对本堂课进行学生活动设计时使用的证据的效果以及核心素养落实方面进行评价，形成新的证据，优化后续同类型教学内容课堂的教学设计。

郎雪玲老师从教学过程维度进行评价。她表明，李美娜老师基于评价量表以及文献的证据进行学生活动的设计，成功的将三个学习活动有逻辑、层层递进的串联起来，从而呈现的教学过程行之有效地促进学生学习分子晶体和共价晶体的内容，模型搭建、历史线等适切的学习活动把微观层面的知识更加形象生动地呈现给学生，充分调动学生的积极性以及对微观不易理解内容的深入思考，促进学生化学核心素养的落实与发展。

龚奇峰老师从教学设计维度进行评价。他表明，李美娜老师以干冰的微观结构为证，用模型搭建为据，结合化学史，既凸显了宏微结合的观念，也给学生搭建了认知的台阶；以史料循证探索串联课堂主线，以合作动手研究晶胞的构造，用新型燃料的发现揭示学习物质结构性质的应用前景，学生活动的设计合理，证据丰富，并且重视学生为主体的学习活动。

毕炜老师从目标达成维度进行评价。他表明，李美娜老师借助干冰的晶胞帮助学生建立"结构决定性质"的化学观，从知识结构层面，以十分贴切的案例辨析了宏观与微观之间的内在联系。将“宏观辨识与微观探析”和“证据推理与模型认知”的核心素养相结合，通过问题串的活动形式，培养了学生的进阶思维。本节课结构设计合理，情境贴合实际，学习活动设计适切，既符合了现阶段学生的认知水平，又十分聚焦本次教研的主题，始终从宏观和微观两个层面不断的互相论证，目标达成度较高。

何双老师从思维构建维度进行评价。她表明，李美娜老师结合当下时事，借助亚运会开幕式的视频告知学生火炬燃料是首次利用二氧化碳制备的甲醇这样的事实作为证据，引起学生的学习兴趣，启发学生的思维；本节课学习活动是在很多史料证据的基础上进行设计的，也受到学习活动设计的启发，在进行学习活动设计的时候需要基于课本但是也要扩宽自己的视野，可以阅读文献寻找一些适合的证据，基于证据设计情景或者设计课堂整体脉络是她需要不断学习的地方。

汲丽老师从课堂内容、学生表现等维度进行评价。她表明，初中以及高一阶段从性质、一个分子的角度对CO2进行学习，在本节课中关于CO2的学习难度加深，拔高了高三学生对物质结构的认识，以问题串的形式不断锻炼学生的表达能力以及思维能力，以两位同学的课堂表现为例，充分展现出学生思路清晰、顺利流畅、语言精炼的特性，这正是李美娜老师在课堂上提出明确指向性的问题并不断锻炼学生的表达以及思维能力的证据。

随后，檀文金老师对化学组教研活动进行小结。日常开展的常态化循证教研活动是基于评价量表中的不同维度进行课前循证，设计学习活动，教研组根据评价量表收集到的信息对课堂进行研判，作为下一轮优化教学设计的新的证据。

环节四：与会老师代表评价

首先是龙岩市化学骨干教师代表简清荣老师指出，李美娜老师通过设计问题引导学生参与、观察、思考、分析、研讨和总结，让学生充分体验深度学习，进而培养学生个人的关键能力；重视情景任务驱动课堂模式，重视学生深度学习活动体验的培养，让学生体验学科价值和创新意识。并就教研活动进行简短评价，指出教研组通过一定教研活动的循证来最终形成一种适合于大多数学生的教学活动。

其次是闵行区两位教师代表对课例进行评价。上海市金汇高级中学侯月老师对李美娜老师的课例进行了高度的肯定，认为本节课的结构完整，层层递进；利用化学史料为主线，让学生质疑，不断地提出问题，师生互动积极；利用学生活动，给予学生恰当且充分的活动时间。侯月老师也学习到了很好的突破教学重难点的方法。上海外国语大学闵行外国语中学汪青老师肯定了本节课的情景恰当且流畅，从干冰分子晶体“变身”为共价晶体，感受到了CO2的“美”，在未来的教学中也会借鉴问题串的教学设计；但他也建议，在以后的中学化学的课堂教学中是否可以多采用中国科学家的贡献与成就，让学生感受到中国科研的发展与进步。

随后是闵行区两位教师代表对循证教研活动进行评价。上海中医药大学附属浦江高级中学刘莉老师表示，在这样一个新的教研课题下，每个人都有自己的任务，共同向着一个目标前进。上海市闵行中学谭爱华老师以四个关键词对本次循证教研活动进行评价。第一个关键词为“新”，依托优秀的团队和上师闵分优秀的平台，上师闵分化学组贴近“双新”背景，呈现“新”的教研活动设计理念，“新”的教研活动方式以及“新”的教研主题；第二个关键词为“深”，团队成员分工合作，有机合作，深入研究；第三个关键词为“实”，基于学生需求，利用文献和评价量表的使用，对证据进行分析和推广应用，这是“实证”，课堂学生积极参与，教学目标较好达成，这是“实效”；第四个关键词为“研”，区别于传统的教研活动，循证教研类似科学研究的一般过程。

环节五：专家点评

活动的最后，陆艳老师和赵婷校长进行总结发言。

陆艳老师对李美娜老师的课例给予了充分的肯定，也给予了上师闵分学生科学思维能力的夸赞，这都离不开上师闵分搭建的平台。陆艳老师首先从三个角度对李美娜老师的课堂进行评价。第一点：设计理念新颖，顺应课改潮流，凸显学生为主体，学生活动、问题串等的设计巧妙且有效落实学科核心素养；第二点：设计角度新颖独特，一个主角CO2却串联起分子晶体和共价晶体两个重要概念，形成认知冲突，激发学生探究学习欲望；第三点：板书设计富有结构化和艺术性，板书设计永远都未退出历史舞台，其有着媒体无法取代的一些优势，除传递美感外，还蕴含德育教育价值。其次，陆艳老师对循证教研活动进行评价，表达循证教研是一种进步，也是一种必然趋势，可以给其他教研组提供很好的教研思路；循证教研改变了以往教研活动的主题，更持续地聚焦某个主题，更彻底地解决某个问题，解决更具有普遍性和针对性的问题。最后陆艳老师对循证教研进行了总结，并对闵行区青年教师们就教学和教研提出两点建议，一个是防止证据不足，要做好前期证据收集与评估；一个是避免使用证据力度不够，真正做到证据有时效性、有发展性。

赵婷校长对陆艳老师和来参加循证教研的老师表示感谢，对为这次教研活动提供宝贵建议的老师致以敬意与感谢，对化学组团队呈现的循证教研活动的效果表达了肯定，表明化学组循证教研活动贴合循证教研的模式流程，而在后续的磨合过程中可以呈现更加成熟且有效的教研活动。
