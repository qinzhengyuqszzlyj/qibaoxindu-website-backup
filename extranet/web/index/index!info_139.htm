脚踏实地 砥砺前行，成长为新时代好干部 ——2021年闵行区教育系统优秀年轻干部挂职锻炼培训班顺利结业

发布时间 : 2022-09-27   作者（来源）: 曹雅明   阅读 : 2418

3/8
←
→

9月20日，2021年闵行区教育系统优秀年轻干部挂职培训班结业仪式在教育学院举行。区教育局副局长潘蓓蕾，七宝中学党委书记、校长朱越，实验小学党委书记、校长何学锋，教育学院校长发展中心主任俞琳等出席。培训班全体学员及2022年闵行区教育系统后备干部培训班班委等40多人现场参加，2022年闵行区教育系统中小幼副校（园）长、后备干部培训班学员等90人线上参与。

上师三附小李腾蛟作为培训班副班长主持结业仪式。培训班班长、七宝中学范丽作挂职培训总结，她带领学员们对为期一年半的培训进行回顾：10场高质量的专题报告、18场精彩纷呈的专家点评、12轮分组论坛、120人次的主题交流，为期一年的挂职锻炼，线上线下两种不同形态的教育教学方式……感谢教育党工委、教育局、教育学院及挂职学校，为学员提供如此精彩纷呈的培训内容，帮助青年干部汲取时代所赋予的育人智慧，提升在新时代背景下的创新育人能力。

闵行中学华佳齐、闵行四中刘叶、鹤庆路幼儿园王贝贝代表学员发言。他们分别结合自身的挂职学习经历，交流分享所见所思和所想：角色的转变、沉浸式的挂职培训，打开了学员们更大更广的学校管理格局；将理论学习与实践历练相结合，实现工作思路与眼界、能力方法与思维的再次突破；一起在传承中变革，创新中成长，在成人成事中寻找“教育”的最好发生。

朱越、何学锋两位校长代表导师寄语学员们。朱校长希望大家通过此次挂职锻炼，深入体会，用欣赏的眼光看待周边的人，用苛刻的眼神审视这次培训，建议学员们再次对自己一年半的挂职过程进行思考，梳理挂职中的所见所思所想，找到问题的关键，解决工作中之前没有解决的问题。何校长勉励大家，无论岗位是否发生变化，学员们都要将这次高质量的培训，作为资深领导管理者一个崭新的起点，重新树立自己的目标，将这次学习融入将来的工作中，主动影响、带动身边的领导团队。从带教导师、挂职学校中层和班级同学身上感悟作为学校领导干部的境界、素养和智慧，梳理身上的不足和短板，并祝愿大家实现心之所愿的成长和发展。

两位导师代表还为范丽、华佳齐、何敏、李腾蛟、王贝贝、侯惠林6位优秀学员颁发了证书。

潘蓓蕾副局长代表教育局，祝贺各位学员通过一年半的学习和体验，顺利完成了培训任务，同时感谢带教学校和派出学校的校长对此次培训的大力支持。潘局指出，此次优秀年轻干部挂职培训班，是一个承载着闵行教育未来和希望的班级，结业不代表结束，学员们要用好这样的联系继续学习。

接下来的专题培训，在教育学院副院长董学平的主持下，上海市教师教育学院党委书记周增为，为大家带来《教师专业发展能力是落实立德树人的基本保障》的专题报告。周书记在开场互动中就明确指出：教师是学校教育的关键人群，教师要用专业的方法、讲专业的话、做专业的事。随后用“作业设计”这个小切口的例子，引导大家更全面地理解教育的适应性与功能性问题，定位教育的引领作用。周书记指出，教师要有学生立场，教师专业发展要敢于面对和解决学生的真实问题，这也是教师专业发展的价值意义，教师只有能解决这些问题，才能谈得上真正的专业发展。周书记高屋建瓴的观点阐述、现场交流的智慧碰撞，带给学员们更深层次的触动。

本期“优秀年轻干部挂职锻炼培训”给大家的成长也是喜人的。26位后备干部中，5位升任正职校（园）长、书记，7位升任副校（园）长；所有人成为了引领学校发展的核心人物，正带领团队为学校发展作出贡献。学员们将在未来的日子里，带着沉甸甸的成长与收获，回到自己的工作岗位，继续脚踏实地，在实践中努力砥砺前行，成长为一名敢担当、善作为、能成事的好干部，为闵行教育这片沃土尽自己的一份力量。
