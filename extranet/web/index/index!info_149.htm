“闵”而好学传经典 “行”则致远润童心——闵行区经典润乡土计划暨第26届推普周交流展示活动

发布时间 : 2023-10-19   作者（来源）: 倪宏   阅读 : 435

3/8
←
→

2023年10月10日，闵行区2023年经典润乡土暨第26届推普周交流展示活动在华东理工大学附属闵行梅陇实验学校镇西校区举办。上海市教材语管处处长陈爱平、闵行区教育局副局长潘蓓蕾、闵行区教育局普教二科科长马卫锋、闵行区教育学院副院长龚耀昌、闵行区教育局普教二科范亿、闵行区语委办倪宏、华东理工大学附属闵行梅陇实验学校党总支书记方海运、校长汤林出席本次活动。来自十四所经典润乡土项目校的领导和闵行区语言文字示范校的教师代表共同参与。

在第26届全国推广普通话宣传周活动中，闵行区各学校开展了丰富多彩的活动，一段视频“‘闵’而好学传经典，‘行’则致远润童心”，带领各位嘉宾共同重温快乐美好的瞬间。

闵行区教育局副局长潘蓓蕾致辞。她指出闵行区历来高度重视语言文字工作，对标社会主义现代建设的引领要求，以做实基础、提升规范、打造亮点为目标，将语言文字融入到经济建设和精神文明建设工作中，助力创建与全国文明城区相适应的和谐语言环境。闵行区语委将守正创新、深化改革，不断开创与时代发展相适应的语言文字新局面。

华虹小学的吴叶红老师代表经典润乡土项目校发言。吴老师介绍说，学校将经典润乡土活动与校园四季活动相融合，引领师生走近经典，打造诗意校园。华虹小学的学生还声情并茂地演绎了诗歌朗诵《咏梅》。

中国的语言文字是独特的艺术形式，它们不仅是传递信息的工具，更是传承历史、文化和思想的载体。在教育部组织开展的经典诵写讲大赛、中国诗词大会等丰富的诵读、书写、篆刻等交流展示活动中，闵行区的师生获得了诸多奖项。闵行区教育局普教二科科长马卫锋为在语言工作中取得优秀成绩的集体和个人颁发证书。闵行区教育学院副院长龚耀昌为三所闵行区语言文字推广基地学校颁发推广基地示范牌。

立身以立学为先，立学以读书为本。阅读是获取知识的重要途径，也能给人们带来智慧和愉悦的心情。华理梅陇实验学校的学生为与会嘉宾带来优秀原创文学作品朗诵《读书人是幸福人》。

上海市教材语管处处长陈爱平表示，语言文字工作不仅仅是说普通话写规范字，更是推动优秀传统文化的传承工作。上海市语委办将持续和闵行教育携手共进，让语言文字活动走进学生的生活和学习中去，进一步提升普通话的普及率，提升学校文化氛围，持续推进书香校园建设和语言文字示范校的达标建设，助力闵行学生的全面成长。
