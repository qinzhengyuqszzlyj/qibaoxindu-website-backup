立足课堂教学 培育地理实践力——高中地理主题教研活动在七宝中学举行

发布时间 : 2023-10-28   作者（来源）: 周光明   阅读 : 1175

3/8
←
→

当前双新背景下，高中地理教学如何更加科学高效的利用新教材、创新学习方式，将新课程理念下地理实践力的培育贯穿于新课程的教学之中，是一线教师面临的现实难题。2023年10月26日下午，在上海市七宝中学举行的闵行区高中地理教研活动，对这个话题进行的课堂实践探索，给教师带来了诸多有益的启发和思考。

本次活动以“立足课堂教学，培育地理实践力”为主题，邀请了上海财经大学附属北郊高级中学教师发展中心主任、地理特级教师、正高级教师周义钦老师，上海市民立中学副校长、地理特级教师姚伟国老师两位专家莅临指导。闵行区高中地理教师、闵行区地理学科骨干基地学员和部分初中地理教师参加了活动，活动由区地理教研员周光明老师主持。

大家观摩了七宝中学蒋凤丹老师和黎挺老师的两节高三新授课。

蒋老师的《寸土寸金，上海还要不要保留耕地》，以七宝中学 HUB 社的假期社会调研发现为引入，从学生生活中的地理“上海米道”出发，引导学生思考上海耕地的保留问题，进一步衍生出上海乡村建设的重大议题，整节课以师生对话的形式展开，充分体现了以学生为主体的新课程理念。

黎老师的教学内容《做一份东滩鸟类国家级自然保护区的规划》，以崇明东滩鸟类国家级自然保护区为例，在提供遥感影像等详实的图文资料的基础上，以了解崇明东滩保护区的保护对象、影响保护区安全的因子、设计一份保护区的规划为明线，以研究方法的指导为暗线，将鲜活的地理野外考察、地理学生课题研究等素材应用于课堂教学，为大家展示了一节浸润式的、以地理实践力培育为导向的地理课堂。

柳英华老师的主题发言《地理实践力培育·行动和反思》，分享了七宝中学地理教研组长期的探索和经验：做好顶层设计、搭建学习空间、建设课程资源、组织野外考察和开展课题研究等。同时，柳老师也指出了在地理实践力培育中遇到的挑战和瓶颈，如“金字塔”模式的局限性、学生考察意愿的变化、培育群体的不连续性以及外部支持体系的变化等，干货满满，引发了大家的进一步思考。

特邀嘉宾姚伟国老师和周义钦老师对本次活动进行了点评和指导。

姚伟国老师从高三地理课堂、课堂情境和现代课堂的视角，肯定了这两节课在地理实践力培育方面所做的努力和成效，强调了在课堂中落实地理实践力培育的重要性。同时也提出在人工智能高速发展的今天，教师如何拥抱技术，化挑战为机遇，是一个值得大家的思考的问题。

周义钦老师从新课改的教学理念、教学内容和教学方法等方面对两位老师的课堂给予了高度评价，同时就如何进一步提升教学效果提出了宝贵的建议，指出教师在教学中应注重学生的实践操作和问题解决能力。诸多真知灼见，让老师们如醍醐灌顶，受益良多。

周光明老师对上课教师的课堂实践和两位嘉宾的精要点评表示感谢。本次活动为全区地理教师提供了一个学习和交流的平台，展示了闵行区高中地理教研的阶段性成果。大多数教师觉得地理实践力的培育困难很大，难以落实，七宝中学的实践经验给广大教师打开了思路，教师需要在提高认识的前提下丰富自己的教学策略，并加强自身地理实践力的培养。教师要善于延伸课堂教学的时空，教师和学生都需要有良好的问题解决能力，来应对未来社会的发展和不确定性。
