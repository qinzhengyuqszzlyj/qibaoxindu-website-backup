男性视角下“幼儿发展优先”行动——闵行区幼儿园男教师沙龙六月专场

发布时间 : 2023-06-09   作者（来源）: 蔡志刚   阅读 : 571

3/8
←
→

最美人间六月，青年气宇轩昂。闵行区幼儿园男教师再度聚首，共赴一场精神盛宴。2023年6月8日，闵行区幼儿园男教师沙龙活动六月专场在浦江镇第三幼儿园举行。男教师沙龙负责人卢扬老师、吴泾二幼何金奎园长莅临指导，浦江三幼曹培丽园长与闵行区幼儿园男教师沙龙成员共同参与。本次活动由浦江三幼大教研组长曹蕊老师主持。

首先由浦江三幼黄佳伟老师执教大班集体活动《钻圈小能手》。黄老师借助呼啦圈材料，引导幼儿从探索呼啦圈的一物多玩，再到尝试身体移动快速钻圈游戏，给予幼儿充分探索与挑战的机会。同时通过合作钻圈游戏，提升幼儿速度、灵敏性、协调性等身体素质，还培养了团队游戏中智慧运动的品质。

接着秦伟老师执教中班音乐活动《交通工具开来了》，基于幼儿的年龄特点出发，以情景化教学贯穿始终，幼儿在熟悉歌曲基础上尝试改编歌词。活动中秦老师以幼儿兴趣出发，巧妙运用转盘游戏，让幼儿体验在活动中“自己做主”的快乐，让幼儿在艺术领域中感受美的诠释。   

现场观摩结束后，两位执教老师对本次活动主题“男性视角下，幼儿发展优先”进行了反思。观摩老师们结合研讨主题——男性视角下的“幼儿发展优先”行动，展开了积极而又热烈的研讨。来自闵行区各幼儿园的男教师代表针对两节活动进行了梳理。两节活动，无论老师素养、幼儿状态以及基于“幼儿发展优先”活动设计，师幼互动方面都给予肯定。

吴泾二幼何金奎园长对本场活动也给予了高度评价。从两位男老师执教的课看出老师身上良好的教态，眼里有孩子。良好的教态可以看出教师上课的积极状态，能够很好的激发幼儿的学习兴趣。而男性视角下的幼儿发展优先，虽有难度，但我们沙龙的老师们能够主动摸索、积极探寻，对于主体性和主导性的融合，老师们的把握度也一直积极思考和尝试！

相信在卢扬老师的带领下，闵行区男教师沙龙团队将一路“披荆斩棘，行稳致远”，在幼儿发展优先理念的引领下，立足儿童视角，关注幼儿的长程发展，尽情展现出属于男教师的独特风采，为推进闵行幼教高质量发展砥砺前行。

活动最后，浦江三幼曹培丽园长代表幼儿园欢迎各位男教师的到来，很荣幸给了老师们这样一个展示的平台，给予了老师们很多的收获，不仅仅是对于一节集体活动优化，或者说是基于异性视角下对于儿童观的诠释，更多地是老师们在日常这些实践课程研究当中的一种专业的素养性。她表示男教师沙龙活动特别的好，能让老师们找到自己的专业发展的优势和思考点，同时打开对于职业定位的拓宽。
