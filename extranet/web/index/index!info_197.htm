支持幼儿经验连续性生长的集体学习活动 ——闵行区北片幼儿园主题教研活动（七宝中心幼儿园专场）

发布时间 : 2023-10-27   作者（来源）: 张静   阅读 : 322

   闵行区七宝中心幼儿园于10月19日上午组织开展了以“支持幼儿经验连续性生长的集体学习活动”为主题的北片教研活动。本次活动特邀闵行区教育学院学前教育研究室教研员张静老师参与指导，北片公民办幼儿园骨干教师及闵行区第三轮课程基地（七幼基地）研究共同体幼儿园的保教主任共计80余人共同参与。活动由园部负责人陈丽君老师主持。

走进幼儿 看见发展

   莅临现场，老师们拿起幼儿手绘户外活动地图开启了沉浸式观摩。建构活动中，一块块积木的垒高搭建，一个个螺丝螺母的旋转撵紧，在合作商量中，建造属于自己的游戏王国。沙水区里的孩子们，正拿着铲子建沟渠、开河道，摆弄水管感受着水的流动，探索着沙水的奥秘。涂鸦区的孩子们，利用自然物、生活化材料在墙面、地面大胆创作。山坡上、大树下、种植园里，孩子们有模有样的学着消防员灭火，熟悉的拿起放大镜、提起小装备，开始了探索和发现。

聚焦活动 共享智慧

   户外活动结束后，园部负责人陈丽君老师从倾听教师的声音，追随孩子的需求；因园制宜，盘活每一平方米；丰富户外活动场域，满足不同活动需要；提供充足的材料，支持幼儿活动中的探索与发现；增添防晒防护保育设施，为幼儿户外活动保驾护航几方面介绍了园所在优化户外活动二小时中的实践以及关于支持幼儿经验连续性生长集体学习活动组织的思考。

   接着，陈雯老师在基于持续观察幼儿户外建构游戏，链接游戏经验，组织开展了学习活动《意想不“倒”》。鲜活的活动中，幼儿从平面比高搭建到斜坡比稳再到斜坡比高搭建，通过实际操作、直接感知、亲身体验，和同伴合作商量、交流讨论中探索、积累稳固搭建的方法。

   园本教研《共话经验成长 助力幼儿深度学习》在园部负责人陈丽君的带领下，老师们聚焦学习活动，借助活动观察表，通过词条标签记录幼儿表现，思维导图梳理幼儿经验，头脑风暴说教师支持中不断碰撞出思维的火花。

   北片公民办幼儿园骨干教师，带着问题从“支持幼儿经验连续性生长”的角度，谈了自己对于集体学习活动设计与实施的思考和建议。充分肯定了活动设计的来源，对活动过程的探究性和价值性与执教老师和教研团队进行了精彩的互动。

专业点评 助力成长

   最后，教研员张静老师对整场活动做了精彩的点评。从杜威“经验的改造”理论出发，结合集体学习活动，赞许了陈雯老师的专业自信和专业实力，引导教师思考低结构活动与高结构活动之间的关系，同时激发教师在肯定幼儿“不怕多次失败”的过程中更要关注“失败”背后的原因。同时，张老师肯定了教研组的观点表达，鼓励大家都要成为主动且有能力的学习者。

   本次片级研讨，教师们充分认识到“珍视幼儿生活和游戏的独特价值”，“发现和支持幼儿有意义学习的重要性”。支持幼儿经验连续性生长的集体学习活动对促进每一个幼儿发展的意义。
