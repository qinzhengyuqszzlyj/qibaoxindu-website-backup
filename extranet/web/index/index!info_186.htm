柔活活动边界 提升互动质量 ——闵行区北片幼儿园主题教研活动 （古美中心幼儿园专场）

发布时间 : 2023-06-02   作者（来源）: 张静   阅读 : 463

3/8
←
→

2023年5月31日，在古美中心幼儿园开展了以“柔活活动边界 提升互动质量”为主题的北片幼儿园教研活动。闵行区人民政府督导室专职督学徐慧麟老师，教育学院教研中心学前教育研究室张静老师，闵行区教育测评与研究中心项目负责人、责任督学辛丽华老师莅临指导。北片姐妹园、古美街道学联体、民办联合党支部学校和区级重点课题《普惠性政策下区域民办幼儿园办园质量提升的路径研究》课题组成员校的老师们参与活动。

看——发现幼儿主动学习

欧风园大三班的阮冬青老师基于幼儿在户外游戏中玩泡泡活动，设计了集体教学活动《玩泡泡》。幼儿在反复地尝试中探索玩大泡泡的方法，大胆地分享自己的经验。整个活动中幼儿坚持不懈、合作讨论，在愉悦的氛围中体验成功与失败。

听——感受幼儿种植乐趣

欧风园大二班的潘马忆寒老师讲述着幼儿和芋艿之间的故事，在了解、种植、照料芋艿的旅程中，幼儿与植物建立了深厚的情感。幼儿的自主自发的开展“毕业后，芋艿由谁照顾”的讨论会和宣讲团，过程中体现幼儿主动学习的能力。种植活动对于孩子而言，不是单纯地学习知识，是与自然的一次亲密接触，是一种情感的体验、一种生长的喜悦！

研——做有力量的互动支持者

欧风园沈丽洁老师带领小中大各年龄段老师们开展了以“做有力量的互动支持者”为主题微教研活动。教师们通过举例、辩论、经验分享等方式在思维碰撞中逐步清晰互动的力量。最终达成三点共识：1.科探活动中教师不但要与幼儿共情、还需以专业回应。2.在尊重和信任幼儿的同时，不忽视等待的力量，助力幼儿自然生长。3.提升自身互动能力，学会观察、捕捉教育契机，用高质量的互动助力幼儿主动学习、持续探索。

专家引领 智慧共享

张静老师组织开展北片教研。教师们围绕活动现场分成三组，以“在互动中促进幼儿主动学习的启发和建议”展开研讨。每组教师代表分别谈了各自小组的观点。张老师就一些共性的话题进行现场互动。针对现场活动张老师就活动前幼儿经验的准备、教师的分享策略等方面提出了建议，由此推动“幼儿发展优先”的理念更好地践行和落地。

专职督学徐慧麟老师对于活动中体现出的教师专业发展、团队合作等方面给予了充分的肯定，特别对于幼儿园开展的种植活动促进幼儿全面发展的价值给予高度赞赏。同时建议老师能更好的追随幼儿，深入观察，促进幼儿深度学习。

古美中心幼将继续追随儿童脚步，聆听儿童心声，回归儿童本位，磨砺教学能力，在成为有力量的教师的道路不断奔跑。
