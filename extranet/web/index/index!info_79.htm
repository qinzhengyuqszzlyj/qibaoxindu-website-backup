抓机会 攀高峰 促成长 ——热烈祝贺闵行区联队在上海市第八届中学生物理学术竞赛获得二等奖！

发布时间 : 2023-07-14   作者（来源）: 范晓荷   阅读 : 1117

2023年7月5日，上海市第八届中学生物理学术竞赛在上海市南洋中学举行，本届物理学术竞赛由上海市教师教育学院（上海市教育委员会教学研究室）主办，上海市物理学会、上海市教育学会和物理教学专业委员会协办，上海市南洋中学承办。上海市中学生物理学术竞赛（Shanghai Young Physicists’ Tournament，简称SYPT）旨在锻炼学生解决复杂科学问题的能力，并且能够针对这些问题给出令人信服的解决方案。比赛采取团队辩论方式进行，中学生针对开放性物理问题的基本知识、理论分析、实验设计、研究方案、结果讨论等进行辩论，深具科学教育价值。

经过两天的紧张比赛，共计十二个阶段的激烈角逐，闵行区联队“超奇点小队”凭着成员之间的精诚合作和不懈努力，荣获全市第五名，二等奖第一名的成绩，可喜可贺！

“超奇点小队”队是由华东师大二附中紫竹校区新高二年级的伍文诚、乔韵仪、吴文怡3位同学和莘庄中学新高三年级的顾滕缤、谢逸峰2位同学组成。指导教师团队由闵行区物理学科中心组成员邵友娣老师，华东师大二附中紫竹校区刘建老师和莘庄中学蔡香英老师组成。作为领队，邵友娣老师对赛前训练进行了提前预案，精心安排、有序部署。两位指导老师对各参赛成员的研究项目深入指导和实时监督反馈，同学们按计划推进实验进度和改进实验细节，研究实验理论。莘庄中学孙妙琦同学作为观摩队员也一直投入到赛前的准备中。正是各项赛前准备工作有条不紊，才为比赛的成功奠定了良好的基础。五位参赛队员通力合作，课题陈述中自信展示，交流辩论中思维碰撞，评论分析中认真倾听，全面准确，充分展现了闵行学子良好的学科素养、团队精神和奋斗精神。

本届学术竞赛对学生而言，是一次攀登科学高峰的辛苦历程，也是促进学生快速成长的优秀平台，更是一次学习他人、锻炼提升自我的极好机会。成绩的取得离不开团队成员共同的努力付出，也离不开闵行区教育学院范晓荷老师的大力支持和持续关注。



撰稿 :  蔡香英 莘庄中学
审核 :  范晓荷
