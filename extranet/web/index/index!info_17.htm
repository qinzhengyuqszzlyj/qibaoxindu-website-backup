以研为径行致远 砥砺深耕待丰收

——初中道德与法治基于核心素养培育的教—学—评一致性精准教学探究

雨后风凉暑气收，梧桐叶叶报初秋。为了更好地推进闵行区初中道德与法治学科教育的高质量发展和教师教育教学水平的提升，2023年9月7日下午，在明星学校开展了2023学年第一学期闵行区道德与法治学科第一次教研活动。出席本次活动有闵行区全体初中道德与法治教师，本次活动由闵行区初中道德与法治教研员刘芳老师主持。教研活动内容分为三部分，第一部分微讲座：基于核心素养培育的教—学—评一致性精准教学探究的专题研讨、第二部分——榜样的引领——宫伟双老师分享她的单元教学设计实践与思考、第三部分回首过去，展望未来——上学年工作回顾和本学期工作安排。

知识浸润，专题研讨。教研员刘芳老师为我们呈现了一堂精彩的微讲座；基于核心素养培育的教—学—评一致性精准教学探究，讲座从理论到现实，从是什么到怎么做，层次分明的为我们阐述教学平的一致性。刘老师从教学层面的“教学评一致性”和学习层面的“教学评一致性”，深入浅出地分析了“教学评一致性”的内涵与意义。同时提出素养目标的“三问”撰述法——问“双基”、问“问题”、问“成果”，以及达成目标撰述法——能+通过哪些方法（过程）做哪些事（目标分解）+做到什么程度。在怎么做的方面分享了“教学评一致性”视角下单元教学的基本策略——前置单元评价目标、重构单元内容框架、梳理单元基本问题、创设单元核心任务、实施单元主要活动、评价融入学习任务。以丰富的单元教学设计样式，以线串珠生动讲解了如何进行“教学评一致性的教学设计”。落实到学业质量评价中，刘老师结合各类优秀试题分享了如何进行评价设计，强调教师研究规范试题的重要性，要求教师精心设计日常常规性作业，希望老师们与时俱进、加强学习，提高试题设计质量，有效减轻学生的课业负担，切实发挥作业、试题的育人功能。

榜样引领，经验分享。作为2022年上海市青年教师教学评比二等奖的获得者，华东师范大学第二附属中学附属初级中学的宫伟双老师在教研种分享她的参赛经历和参赛单元教学设计实践与思考。宫老师分别从单元主题概述、单元内容交流、单元设计反思和个人感受四个部分展开分享。从分享单元主题的逐步确立的思考过程，谈到“以问题解决为导向”的单元设计核心，在教学设计过程中体现了她对为什么教、教什么的思考。单元整体设计坚持以生为本，通过议题式教学，创设系列教学情境，设计任务驱动议题，来落实学科核心素养。强调“以问题解决”为导向，注重引导学生在问题解决中学会学习，学会迁移，实现知情意行的统一。注重综合运用多种评价方式，促进知行合一，让评价贯穿道德与法治课程学习的全过程和教学的各个环节，发挥以评促教、以评促学、以评育人的功能。

回首过去，展望未来。刘老师对上半年工作进行整体回顾，肯定了在过去一学期里所取得的成绩，也指出了教师在专业发展中面临着教研和培训分离等问题。并围绕扎实推进习近平新时代中国特色社会主义思想和党的二十大精神“三进”工作、加快构建思政（道德与法治）研训一体体系机制、团队建设、以赛促训、课堂教学等多方面对新学期的工作进行了部署与展望。

“九层之塔，起于垒土”。教研，是一场诗意的修行，是激荡思想、碰撞灵魂的过程。路漫漫其修远兮，希望老师们在未来的课堂中继续实践、提升，让道德与法治课堂真正展现它立德树人、育人为本的课程魅力。



撰稿 :  闵行区纪王学校刘翠
