践行数字化转型 探索教学新样态

发布时间 : 2023-05-24   作者（来源）: 徐婧   阅读 : 788

3/8
←
→

5月23日下午，以“践行数字化转型 探索教学新样态”为主题的闵行区小学自然教研活动在实验小学莘松校区举行。闵行区小学自然教研员徐婧和区内50多位老师共同参加了本次活动。活动由实验小学自然学科主任张晓文主持。

教学展示环节。实验小学王暠、蒋雨茜两位老师分别执教了一年级《磁铁能吸铁》、四年级《水的浮力》两节课。王老师通过创设整理实验室材料情境，使一年级学生在“分离大头针和牙签”“探究磁铁能吸引什么”“设计和整理工具盒”活动中体验了提出问题、解决问题和发现一般性质的科学探究过程，引导学生运用所学知识解决生活实际问题，积极培养学生整理收纳的能力。蒋雨茜老师的课堂由欣赏海上航母情境导入，使四年级学生在“体验水的浮力”“测量水的浮力”“探究浮力大小与什么有关”活动中提高获取和处理信息、基于证据得出结论并作出解释的科学探究能力，形成严谨认真和实事求是的科学态度。两堂课都是对“赛·课堂”教学平台的积极实践。王老师利用数字化教学有效降低了低年级学生实验预测和结果记录的难度，活动成果的及时呈现提高了师生互动的有效性。蒋老师利用DIS力传感器设备，在数字化平台支持下实现了实验数据的及时获取、分析和归纳，提高了实验的效率和准确性。

说课与研讨环节。王暠和蒋雨茜两位老师分别对教材内容、教学目标、教学设计思路、教学重点难点和教学特色等方面进行了分析与交流。随后，由实验小学的范博娅、孔倩等几位老师对这两堂课进行了点评。老师们一致认为两位授课老师通过熟练运用“赛·课堂”平台，精心设计活动环节，以过程性评价促进教学，积极呈现出数字化教学在自然课堂中的优势，并在教学中非常关注学生科学语言的表达和探究能力的培养。

项目交流环节。实验小学张晓文老师对市级青年课题《单元视角下小学科学项目化学习设计与实施》进行了交流。主要介绍了课题研究的背景与价值，项目研究内容和单元教学设计要素，并以“蜗牛的家”项目化学习单元为例分享了如何基于核心素养确立项目主题和目标，如何设计驱动问题与课时活动、构建评价体系等，最后分享了项目化学习的实施策略。老师们在认真聆听中对开展项目化学习有了一定的了解和启发。

专家点评环节。教研员徐婧老师对本次活动给予了高度评价。徐老师首先结合张晓文老师的微报告对小学自然学科如何开展项目化学习指明了方向，建议老师们既可以围绕本学科内部的13个核心概念，也可以结合其他学科进行跨学科项目化研究。随后，徐老师结合本次活动主题，对自然教学数字化转型做了深入介绍，指出数字化赋能教学在今天两节课中主要体现为流程数字化和实验数字化。一年级王老师的整个课堂活动在数字化支持下能及时收集到学生的生成资源并给予评价，而四年级蒋老师的课堂运用数字化实验设备大大提高了课堂效率。此外，两位老师在课堂中也充分体现出对学生科学思维的积极培养。

本次丰富的活动内容令老师们收获满满。今后也将有越来越多的闵行自然教师加入到数字化转型的实践中，为实现更为高效的自然课堂而不断努力。
