园本研修，助力教师队伍的快速成长——2018年闵行区保教主任初级研修班学员前往思南路幼儿园学习

发布时间 : 2018-12-24   作者（来源）: 朱凤   阅读 : 1676

12月19日，2018闵行区保教主任初级研修班全体学员前往黄埔区思南路幼儿园学习交流。区教育学院学前教研室蔡志刚主任及相关老师也参加了活动。

学员们参观了思幼中大班的教室，精致、温馨的教室环境，丰富、层次多样的个别化学习内容给大家留下了深刻的印象，思幼的老师还现场解答了学员的疑惑。学员教师无不感叹，思幼的环境和内容创设充分体现了以幼儿发展为本、满足儿童需要的办园理念，反映了管理上的规范及教师们超高的专业水平。

大家观摩了《幼儿驱动问题解决活动设计的“变”与“难”》的现场教研活动。这种“活力型协同研究小组”教研形式，从日常教育教学中的问题出发，不同发展阶段的教师积极参与讨论，发表自己的想法与见解，在思维的碰撞中，挖掘更合适的操作方式和教育行为。让学员们感受到了思幼的教研解决教学中的真问题，更好地满足孩子的需求。

毛尼娜副园长做了题为《教师园本研修与学习管理》的专题讲座，她从教师发展面临的问题、教师团队的建设与管理、园部研修模式、园所文化等四部分介绍了园本研修课程的管理。通过“三色手册”，让教师们从“要我学”转化为“我要学”，不断丰富教师专业发展的资料库，调整幼儿园师资队伍，完善教师培养的相关制度，促进教师的个人发展，从而让教师队伍建设成为幼儿园持续发展的助力。

通过本次活动，让我们看到了思幼教师从关注幼儿到不断思考优质的追求，给学员们带来了思想上的冲击。希望大家好好内化，将学习到的理念和方法运用到今后的工作中，切实提高区域学前的教育保教质量。
