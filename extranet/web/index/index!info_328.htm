把握学科核心素养 开展学历案深度研究 ——区高中通用技术学科市级课题项目系列研讨活动

发布时间 : 2023-11-13   作者（来源）: 刘至宁   阅读 : 529

2023年11月2日和7日，在七宝中学举行了主题为“聚焦项目化学历案研究，推进”双新“教学实施”的闵行区高中通用技术教学系列研讨活动。本次活动以市级课题《劳动育人视角下通用技术学科项目化学历案的行动研究》为抓手，以公开课《延时灯的设计》为载体，聚焦项目化学历案的研究，以研促教推进通用技术学科“双新”教学的实施。

本次活动以高二通用技术选择性必修《电子控制技术》的项目教学展开研讨。高二的学生有一定的电路基础，但电子控制技术里涉及的部分电路对于学生来说还是有些难度。如何激发学生学习的兴趣，引导学生结合所学电学知识，学以致用解决生活中的真实问题？教研组采用项目化学习的方式组织教学，使用电子电路模拟仿真软件+实物电路进行电路设计。引导学生在电路设计的实践中，学习电子技术的基础知识、了解电路设计的基本思路，掌握电路设计及调试的方法、提升运用知识解决实际问题的能力。在项目化学习中提升学生的技术意识、工程思维、创新设计和物化能力。

 11月2日下午，闵行区高中通用技术老师齐聚七宝中学，观摩了崔懿老师开设的研讨课《延时灯的设计》。本节课是模拟电路单元的第3个设计项目，是利用电容器充放电功能实现延时灯的设计的第1课时。教师以设计爬楼梯所需的延时灯为项目主题，组织学生展开研究。

本节课采用电路模拟仿真+项目化学历案的方式组织学生进行电路设计项目的学习。通过tinkercad进行电路模拟仿真实验，利用项目化学历案组织引导学生进行探究并完成记录。希望引导学生在电路探究的同时做好过程性记录，通过学历案辅助学生完成电路设计与调试，理解电子控制系统的工作原理，感受电子电路中信号的变化，掌握电路分析、优化的一般思路和方法，培养解决实际问题所需要的技术意识、工程思维等。

课后，教研员刘至宁老师组织老师们展开了高质量的教学研讨。崔懿老师从本节课的教学形式、教学内容以及教学策略等方面对本节课的设计理念做了深入的阐释。老师们也结合自己的教学探讨如何引导学生从知道到会用，即带领学生跨越从了解电子元件的特性到应用元件设计电路解决问题的认知鸿沟；如何兼顾知识点的落实与学科核心素养的提升；如何设计学历案能引导学生更好地完成项目实践等方面进行了深入探讨。

刘至宁老师在评课中指出，高中通用技术学科教师在新课标引领下，还是要继续转变教学观念，变学生被动学习与主动学习，特别是通过学历案的设计，要给予学生足够的实践活动时间，注重真实的情境设计、选择合适的技术载体、设计合理的技术实践活动、提供恰当的学习支架及资源，放手让学生去学习、实践和交流展示，让学生真正地通过学习或实践过程体验，完成学习任务。同时，可以结合学历案设计配套设计单元教学设计和评价设计来进一步完善学历案的设计与应用。

 

在教研活动中，刘至宁老师还举行了赠书仪式，将她最近出版的力作《区域共同体构建研究》送给老师们进一步研究学习。浓浓的学术氛围激励着在场的每位老师，相信这样教研活动一定会促进闵行区的高中通用技术教学更进一步。

11月7日下午，为了更深入推进项目化学历案的研究，我们邀请了上海市劳动技术学科特级正高级教师吴强老师亲临学校进行深度研讨。本次活动由闵行区劳技教研员刘至宁老师主持，首先项目组崔懿老师对市级课题《劳动育人视角下通用技术学科项目化学历案的行动研究》的研究情况进行汇报，接着吴强老师从劳动课教学如何落实核心素养、如何以学生立场组织教学等角度为课题组答疑解惑，为课题的进一步研究厘清了思路，为如何使用项目化学历案推进教学指明了方向。

本次系列活动有专家的引领、有同伴的研讨；有理论研究、有教学实践。本次系列活动学术氛围浓厚、形式丰富多样，是我区高中通用技术教学研讨的一次有益尝试，是老师们学习研讨共同进步的见证。在“双新”教研的路上，我们继续携手努力前行！

 


七宝中学  崔懿（报道）

2023年11月10日
