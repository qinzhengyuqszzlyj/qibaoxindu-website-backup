【学前 个别化中心组】聚焦问题链接 助推深度学习

发布时间 : 2023-05-23   作者（来源）: 刘时医   阅读 : 919

3/8
←
→

2023年5月22日，闵行区教育学院教研中心学前教育研究室刘时医老师带领区个别化学习中心组的老师们在浦江宝邸幼儿园开展了现场观摩与主题教研。

现场观摩 发现精彩

浦江宝邸幼儿园顾家妮老师结合大班主题《有趣的水》组织实施了主题下的项目式学习活动。基于大班幼儿的前期经验及问题发现，聚焦驱动性问题：“小菜园里我们可以用水做什么?”幼儿在问题驱动下组建自动浇水队、清水营救队、喷泉设计所、水流研究所四个小组进行探究。在分工合作、亲身体验中动手动脑，自由探索，实现与材料的互动、与问题的碰撞，进行智慧共享。

互动研思 悦纳并行

现场观摩后，顾老师从驱动性问题的生成、项目的形成、项目学习活动的设计实施再到评价的三阶段进行了分析。结合案例从幼儿及教师两方面做出深刻的反思。结合现场活动和案例分享，中心组的老师从驱动性问题的设计到主题经验的关联、幼儿个体差异性的关注以及探索的意义、教师的支持等方面进行研讨，肯定了教师对于资源的有效利用及教师的放手，尊重、理解。

专家点评 智慧引领

刘时医老师从课程目标的设计实践与幼儿发展优先的匹配、分享价值的更大化及课程的设计等方面对现场的项目化活动进行了专业的点评，肯定了活动中幼儿的专注投入、家园协作的多方面资源利用。另外，刘老师从运用项目式学习的意义、研究路径的选择、幼儿内驱力的形成—即深度学习的问题、核心素养的问题、自主学习的问题，让项目式学习给予幼儿及教师更大的意义。

始于兴趣，源于问题，陷于探索，忠于坚持，乐于收获。项目式学习的实践研讨为教师的课程领导力厘清思路，为幼儿的学习提供有力支架。以儿童为中心，让实践与幼儿发展优先有效匹配，让项目式学习助力幼儿深度学习。我们将继续砥砺前行，实践出真知。
