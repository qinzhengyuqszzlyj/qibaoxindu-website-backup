闵行区“第二届教育家培育工程”年度总结 答辩互动促成长

发布时间 : 2019-04-08   作者（来源）: 曹雅明   阅读 : 2472

2019年4月2日，闵行区第二届“教育家培育工程”暨闵行教育系统“领航人才培养”年度总结会在教育学院举行。此项目是闵行区教育党工委为实现教育强区而确定的“十三五”期间高层次校（园）长、书记培养项目。上海市教委教学研究室副主任谭轶斌、上海市教育学会秘书长苏忱，闵行区教育党工委书记朱雪平、教育局副局长乔慧芳，“教育家培育工程”项目导师顾志跃、仇忠海、张人利、张志敏、杨荣、郭宗莉、杨静霞等领导、专家出席。闵行区第二届“教育家培育工程”项目全体学员、2018闵行区新任校园长培训班学员、2018闵行区中学后备干部培训班学员等近80人参加。

总结会分中小学、幼儿园两组同时举行，以“学员汇报+专家提问互动交流”的形式展开，分别由闵行区教育学院党总支书记朱靖、副院长龚耀昌主持。

中小学组学员闵行中学校长何美龙、启智学校校长石筱菁、康城学校校长时俊、明星学校校长华志雄、莘松中学校长张忆、颛桥小学校长唐为民、实验小学书记康旻，幼儿园组学员莘庄幼儿园园长郁亚妹、七宝中心幼儿园园长丁春燕、虹鹿幼儿园园长张佩敏、吴泾一幼园长汤军红、闵行四幼园长殷蓓琳，12位校园长分别汇报了一年来在导师引领下的办学实践与探索，以及在教育教学管理和改革发展创新等方面的改变和成长。

学员汇报之后，专家、导师就“如何在学校办学和自身专业成长上找到新的生长点”、“当今教育极具竞争环境下，学校如何在保证质量、满足社会需求与认同等方面取得平衡？”、“如何把校长的进取心转化为教师的工作热情？”、“如何基于学校实际，改变管理方式？”等问题，与每位学员开展互动交流。

专家们肯定了学员的思考和实践，做到个体专业成长与学校可持续发展同步推进，成效明显。也中肯提出学员的思考还缺乏深度与广度，建议今后要多看、多学，注重自身修炼，获得别人帮助的同时也要帮助他人，进而取得更大成长；学会用专业的语言表述专业实践，关注“点、线、面”，即办学有品质，对办学思想、理念、境界有完整的思考和顶层设计，并一以贯之，不断提升对学校、对教育、对管理的认识，逐步形成自己的教育主张。

本次年度总结采用模仿特级校长评审考试答辩的形式进行，这既是一种体验，又是一次高水准的培训。导师、专家还就学员在答辩过程中的陈述及回应方法给予指导。在陈述时要考虑清楚：我从哪里来、我凭什么站在这里、未来去往哪里；并关注三个方面：办学成果、个性化的办学理念、社会知名度。原浦东教育发展研究院院长顾志跃阐述了“教育家”的三个评判依据：一是懂学校，明晰学校的历史沿革、所在社区、教师队伍、学生情况、家长期望等，然后确立学校发展着力点；二是懂教育，了解所处时代对教育的需求和期盼，知道教育的功能。三是懂管理，明白如何把学生带向成功的彼岸，怎样把普通学校办得不普通。校长在答辩时要能用自己丰富的经验、理论、实践，讲清楚以上“三个懂”，能在做好事情的基础上，用独特的理解和主张讲明白怎么做的、为什么这么做。

乔慧芳对专家导师的精心指导表示感谢，充分肯定五位园长学员对区域学前教育的担当，希望在这样的“定向培养”中学会清晰表达，学会梳理与追问，与导师同频共振，在完成“毕业指针”的同时，传承学前教育前辈对教育的情怀与追求，办高品质的园所。

朱雪平总结讲话。他感谢专家导师一直以来对闵行教育的支持与帮助，重申区域开展“教育家培育工程”项目的背景和目的，并对学员提出几点要求：一是做个好学生，主动请教、虚心学习；二是自我领悟，在导师引领下，直面问题，悟出办学的新思维、新想法；三是紧紧抓住契机，认真研读、掌握教育政策导向，在比对中寻找发展方向，从问题入手进行突破，还要多多承担教育改革创新的任务，做好区域辐射与影响，成为闵行教育的标杆。

闵行区第二届“教育家培育工程”项目为期三年，自2018年1月启动至今，已顺利完成第一阶段任务，朱越、唐颖萍、王叶婷三名校长晋升成为特级校长，根据项目初期拟定的“评上即毕业”原则，经局党工委审核通过，莘庄中学校长李国兴、华漕学校校长刘厚萍、华师大附属紫竹小学校长张计蕾进入项目组，我们相信通过不断的实践与探索，会有越来越多的校园长走上更高层次的专业发展领域，并引领与辐射干部队伍建设，助推区域教育高品质发展。
