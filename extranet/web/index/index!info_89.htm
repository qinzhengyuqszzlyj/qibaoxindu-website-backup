沿承于形 贯通于意——闵行区第六期小学校长（田园外小）培训基地在浦江二小开展研修活动

发布时间 : 2023-05-04   作者（来源）: 曹雅明   阅读 : 1477

4月26日，闵行区第六期小学校长（田园外小）基地主持人赵瑛群导师携基地学员——鹤北小学校长蒋红梅、明强二小校长郭芳、昆阳小学书记夏权莅临学员赵美华所在的浦江二小浦涛路校区，开展主题研修活动。活动特邀闵行区新基础教育研究所所长陆燕琴指导。

参观校园 感受“耕读”

研修前，在浦江二小校长王正明和书记赵美华的带领下，导师与学员们参观了学校春耕、夏长、秋收和冬藏四季楼，查看了学校科技、艺术等特色长廊和现代农业生态实验室等育人场域，感受百年老校的新面貌。

汇报交流 品味“耕读”

随后，基地学员赵美华书记向与会嘉宾作了《精耕细读 课程育人》的主题汇报。报告先梳理了学校的历史沿革和发展脉络，介绍了办学规模、、师资队伍等情况，阐述了学校 “耕读树人 健康灵动”的办学理念，再从校本的“红色课堂”和镇域“博乐课堂”两个课程建设出发，详实地介绍了如何围绕学校现有资源，沿承已有的文化传统和新时代育人理念，打造学校特色和镇域课程的思考与实践。

交流环节 互助共生

在交流环节，学员明强二小校长郭芳从课程实践的角度出发，感受到学生心中有信仰，脚下有力量，身边有榜样，心中有目标；教师评价有方法，努力有方向；在学区镇域课程建设中，感受到研究有团队，聚力有成果。学员鹤北小学校长蒋红梅认为赵书记善于教育价值的挖掘和引领，立足本职和核心校联络员的角色，立足学校和学区联盟，借用校内校外资源，开展课程建设，在学生发展和教师发展中取得了一定的成效，同时也期望学校的“耕读”课程能再丰实。学员昆阳小学书记夏权围绕赵书记汇报的课程建设内容，表达了自己的收获和感想，他表示从赵书记和团队在小学理想信念教育校本化的实践中，实现课程育人、文化育人、实践育人和协同育人的同时也实现着管理育人。

专家指导 明晰方向

陆燕琴所长先对赵书记的汇报进行了点评，她表示，赵书记将支部工作与学校中心工作、育人本位和个人专业储备有机整合，在红色文化创建与镇域课程建设中，有前情资源的分析，有策略的过程推进，有创新性的评价方式，有市级课题引领，在专业的表达中体现着管理者的整体性和系统性思维品质；同时陆老师对学校耕读文化的时代内涵提出了建设性的意见。之后，陆老师就当前教师发展中的现状，围绕学校如何在有限的时空里进行再造，向学员们做了专题指导。

导师总结 提出期望

导师赵瑛群校长为本次活动总结。她对学员提出三点希望，一是作为管理者要做学校文化的领路人，校长书记面对新时代学校文化，要有清晰的解读能力，这也是课程领导力的必备能力。二是要有课题研究的意识。管理者要具备整体性和系统性思维，让课程建设更具有科学性、前瞻性和有效性，推动课程高质量发展；三是变革学生的评价方式，立足核心素养，转变育人方式，希望学员们在当前学校成果研究基础上进行再造，生成新研究，形成生长性的评价链，推动学校教育高质量发展。
