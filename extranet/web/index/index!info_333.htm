实践艺术课标 落地核心素养 ——2022学年第二学期浦江镇小学音乐联合教研活动

发布时间 : 2023-05-19   作者（来源）: 刘庆江   阅读 : 581

3/8
←
→

5月18日下午，在上海戏剧学院闵行附属学校，举行了2022学年第二学期第三次浦江镇小学音乐联合教研活动。

 本次活动由上戏附校艺术教研组长黄玉英老师主持，活动主题为“实践艺术课标 落地核心素养（创意实践）”，来自浦江镇相关学校的音乐教师16人，以及音乐研训员刘庆江老师共同参加了本次活动。

本次活动，首先由上戏附校的李庆任老师，执教了一堂以“创意实践”为主题的五年级综合活动课--《吹响我的小芦笙》。李老师从歌曲复习导入教学，通过载歌载舞的形式，和学生一起走进苗家。随后师生共同探究歌曲旋律的奥秘，发现中国五声调式的旋律组成音，了解同头换尾这种中国传统创作手法，并共同进行课堂创编实践。在学会了创作技法后，学生按照个人兴趣特长展开分组编创，他们在活动中充分展现自身能力，综合感受歌曲所展示的苗家节日景象。

随后，李老师对标艺术新课标，围绕“创意实践”这一核心素养，分别从教材解读、学情分析、目标制定、重难点把握以及教学环节的设计等方面进行了说课，并针对本次教学中出现的问题及不足进行了认真反思。在评课环节中，听课老师纷纷谈了自己的观课感悟，一致认为本堂综合活动课，从环节的设置、学生的体验感受，再到最终的自主编创，基本达到了预期目标。大家在交流中碰撞着智慧，在研讨中解答着困惑，教研因沟通而精彩，在互动中树立“以生为本，以学促教，以教导学，以学评教”的教学理念。

最后，音乐研训员刘老师对本次活动进行了精彩点评。他充分肯定了本堂课的成功和带给大家的深刻启示。指出：在综合活动课上，教师要多关注课堂常规的练习储备，为后续的教学提供可能；其次，教师要综合考虑运用多种教学手段丰富课堂教学；再次，音乐教学手段应与核心素养之一的“文化理解”相融合，让学生在掌握音乐技能的同时加深对相关文化的理解。同时，刘老师对于日常课堂中创编课的教学，为年轻教师梳理出了一个较为规范的教学模式，对今后的音乐课堂教学提出了新的要求。

通过此次教研活动，与会教师不仅深入研讨了教育教学方法，还收获了宝贵的教学经验和教育理念。我们相信，这些宝贵的经验和理念必将成为我们未来教学工作的有力支撑，为我们音乐教学工作注入新的活力与动力！
