基于精准教学，落实学科核心素养 ——闵行区高中生物学教学研讨活动在闵行中学举行

发布时间 : 2023-05-30   作者（来源）: 戴赟   阅读 : 431

3/8
←
→

2023年5月24日下午，在主题为“基于精准教学，落实学科核心素养”区教研活动在上海市闵行中学举行。本次活动由闵行中学蒋闰兰执教，闵行中学的陆春花主持，区教研员戴赟主要策划。磨课团队包括区教研员戴赟、中心组张甜、闵行中学生物学学科组全体老师。全区高中生物学教师参加了本次活动。

首先，陆老师分享了“高中生物学图表教学的研究”的微报告。她基于自己的教学经验和研究成果，梳理了高中生物教材中各种图表的数量和类型，并通过案例生动地演示了将图表教学融入生物教学的方法，以及在提升学生比较与分类、分析与综合、模型与建模等科学思维方面的作用。

随后，闵行中学的蒋闰兰老师为展示了“生态系统运行离不开信息传递”的公开课。蒋老师巧妙地以蜜蜂传粉为切入点创设情境串联整堂课的教学，通过有趣生动的视频展示了鸟类的筑巢求偶和毛毡苔捕食蚊子的场景引入，成功地激发起学生们的学习兴趣。然后蒋老师用设计的学习卡组织学生通过分组讨论完成生态系统中不同的信息传递的分类和作用分析。接着，她利用现实案例促使学生思考信息传递在生产实践中的应用，培养比较与分类的科学思维，提升社会责任意识。最后，蒋老师在师生互动中建构生态系统中物质循环、能量流动和信息传递的概念图，不仅建立了知识的内在联系，还进一步建立结构和功能观。

课后，蒋老师与分享了她设计本课的心得体会，包括本章内容的编写特点、教材和学情的分析、单元教学设计以及课教学策略等方面，突出了在设计问题，利用图表信息驱动学生的自主合作学习等方面的的设计。

在接下来的讨论环节中，老师们对蒋老师的开课进行了评价。交附闵分的谢玖清老师指出课中视频形象生动且富有趣味性，能够引起学生的共鸣，推动教学。华理科高的任志红认为本课时的学习卡设计有效地调动了学生的主动学习能力。文来高中的赵琪老师肯定了课堂中使用了农业生产的实际例子有助培养学生的社会责任意识，最后用”山更绿、水更清、天更蓝“总结更能提升本课立意。七宝鑫都的孟辛异老师则关注了课堂中运用的“希沃白板”技术，引发了大家的学习热情。闵行中学的马潇彬老师认为课堂的问题设计是亮点，层层递进、环环相扣，课堂中的图表教学策略也用得很合适。

这次教学活动不仅为我们提供了宝贵的教学经验和启示，也彰显了闵行中学生物教研组在参与区域精准教学、落实学科核心素养方面的积极探索和努力。闵行区生物学科会在越来越优质的区校合作中戮力前行。
