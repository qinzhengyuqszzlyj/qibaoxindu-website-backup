答疑解惑 提升管理智慧——2018年闵行区新任校园长、书记培训活动

发布时间 : 2019-05-24   作者（来源）: 朱凤   阅读 : 2651

5月21日，2018新任校园长、书记办学实践培训活动在莘庄中学举行。特邀闵行区教育学院原院长徐国梁、莘庄中学校长李国兴、莘庄幼儿园园长郁亚妹作为指导专家为学员们答疑解惑。活动分为中小学组和幼儿园组进行。

新任校园长、书记聚焦管理中的问题谈了自己一年的办学实践体会与困惑。在互动交流中，三位特邀专家针对学员的问题提供了可操作性的一校一策，帮助新任校园长、书记掌握学校管理的方法与规律。

幼儿园组11位新任园长从师资队伍的建设、校园环境的改善、机制和制度的建设、后勤队伍的管理等方面分享了自己办学的成功案列，同时也提出了自己面临的困惑：如何推动成熟教师的专业发展，如何打造管理团队，如何形成统一的学校文化等等。郁亚妹指出:园长首先要依法办园、依规办园，坚持底线；从利于园所发展和幼儿发展的大局出发，统一思想、凝聚共识；在处理各种问题时，有谋有略，兼顾各方利益。要学会分析问题，思考表象背后的原因，用赏识的眼光看待每一位教职员工，发挥每个人的特长。

中小学组5位校长、书记围绕“梳理、融入、站稳、借力”等关键词谈了自己的办学实践，同时提出在中层队伍建设、校园文化建设、课程建设等方面的困惑。徐国梁和李国兴指出：校长与中层之间应该保持一定的距离，校长与所有人保持差不多距离有利于工作开展。中层干部的管理能力与其专业能力必须匹配，注重任人唯贤，要率先垂范，严格要求。同时要求每个中层干部在做事的过程中一定要做到眼中有人，面对面的沟通更有温度，才能把事情做实。同时学校要善于借助外力，请专家“把脉”，为学校的发展提出一些思路。但校长要明晰学校发展方向，达成目标，围绕目标采取哪些措施和方法。

大家畅所欲言，即有成功的案列，也有办学的困惑。新任校园长、书记由于岗位的转变，所面临的调整和困难是全方位的，如何做好，需要不断全方位提升自己，不断克服困难。相信新任校园长、书记通过一年的培训，扬强项补短板，站稳岗位，胜任岗位，在传承与发展中，实现学校内涵可持续发展。
