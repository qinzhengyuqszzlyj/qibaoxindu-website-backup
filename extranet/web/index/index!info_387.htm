新学期 新要求 新出发 ——2018闵行区新任校园长带教培训金芸导师组活动通讯（9月）

发布时间 : 2018-09-26   作者（来源）: 曹雅明   阅读 : 2366

2018学年新学期开学之际，马桥中心幼儿园诸伟芳园长在师傅金芸园长的指导下，分析梳理了幼儿园管理和课程发展现状，以及教工工作状态，听取师傅提出的“走进教工，深入一线，思考改变；上善若水，渗透情怀，谋求发展”建议，引领马桥中心幼儿园重“新”出发。

一、新学期：重温制度，重落实

教师上班第一天，徒弟诸园长就组织全体教职工重温了幼儿园的各类制度：《教职工奖惩制度》《教职工工作制度》《教职工安全责任制度》《教职工考核制度》……温故而知新，在重温制度的过程中，进一步规范工作要求，让所有的教工认识到制度是幼儿园发展的保障，是经过教代会通过的大家的共识，所以每位教职工都要执行制度，履行自己的工作职责。平时，诸园长亲自带领各条线老师加强检查和指导，将制度落实情况直接与考核挂钩，促进各项工作有序开展。

二、新要求：加强管理、重跟进

管理团队是幼儿园的核心力量。开学第一周，徒弟诸园长就如何细化管理，组织中层团队进行了讨论，将自己设计的“干部课程监控记录表”，“行政巡查表”等跟条线负责老师进行了分析，提出了：每天一小查，每周一大查，每月全覆盖查的量化要求和检查、反馈、复查的质化检查要求。从刚开始的约着中层去检查，到后来的条线自己去检查；从刚开始的条线汇报笼统无实质，到后来的条线汇报有亮点、有不足、有思考。中层干部的管理责任意识在诸园长的循循指导下也在新学期里有了新变化。

三、新出发：特色梳理、重构建

幼儿园的版画特色课程实施至今，教师中普遍有了希望调整的强烈呼声。版画一直以来是幼儿园的特色，在马桥、在闵行区也是小有知名度。版画到底怎么了？调什么？怎么调？徒弟诸园长利用暑假和核心老师探讨，一起寻找相关资料，一起参观版画展，寻找调整突破口。确立了“玩创”版画的新视角，形成了版画与马桥古文化相结合的新思考。“玩创版画课程重构的实践研究”课题应运而生。

新学期，新要求，新思考，新出发！拥有师徒结对的成长平台，在优秀园长的带教下，相信我们的新园长会和老园所一同发展！

摄影、供稿：马桥中心幼儿园  诸伟芳
